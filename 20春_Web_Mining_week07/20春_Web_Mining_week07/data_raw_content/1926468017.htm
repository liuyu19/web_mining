职位描述：

"岗位职责
1. 参与小米有品电商产品和业务相关技术研究，服务器端需求沟通，架构设计，核心技术研发，线上服务维护等；
2. 负责软件开发过程中的问题分析和总结，提供建议帮助改善研发流程；对业务工程现有技术架构进行优化改进；
3. 和团队其他研发成员一起设计和编码实现产品，并发布上线；
任职要求
1. 统招本科及以上学历，计算机及相关专业，985、211院校优先；
2. 精通 Java 语言和面向对象编程思想，熟练掌握常用的设计模式，掌握io、多线程、多进程等基础知识；
3. 熟悉主流开发框架，对于使用过的框架、代码，透彻了解其原理和机制；能设计，开发高可靠性的基础代码；
4. 具备 3-10 年以Java为开发语言的工作经验，丰富的项目经验，能独立承担服务端产品的开发与维护 ；
5. 掌握软件工程思想，能够对复杂任务分解，熟悉软件迭代开发方法；
6. 熟悉spring, Servlet, JavaBean, Jdbc等；
7. 熟悉Mysql/MongoDB, redis等数据库；熟悉HBase/ElasticSearch/Solr以及相关调优经验；
8. 良好的沟通能力，思维敏捷，思路清晰，工作积极主动，能够承受较大工作压力；
9. 满足下列条件者优先考虑：精通Java Web app开发，参与或研究过主流开源项目，具有大规模高并发访问的Web应用设计和开发经验者，对高并发处理情况下的负载调优有相关经验者优先；"
