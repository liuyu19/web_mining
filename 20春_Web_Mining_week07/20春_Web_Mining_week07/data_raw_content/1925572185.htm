职位描述：

职责描述：
负责全民K歌多终端项目的交互设计；
参与用户研究过程，善于归纳用户需求及体验目标；
能够基于用户需求和产品理念，高效准确地提供相对合理的设计方案；
能够主动发现产品中的体验问题，并给出优化方案；
能够结合项目制订设计规范，并推动在整个团队中的实施；
帮助发掘和思考产品未来的发展和设计方向。
任职要求：
3年以上互联网产品的设计经验，有成功案例优先；
熟练运用用户体验设计方法，并通过高质量的原型清晰地表达设计方案；
对工作充满激情，对互联网趋势和前沿领域有敏锐的洞察力和强烈的兴趣；
具备一定的抗压能力，良好的沟通和理解力；
精通Sketch、Axure等，本科以上学历；
具备较强的AE、动效设计、原型开发能力者优先；
具备英语工作环境或国际化产品项目经历者优先。

企业介绍：

腾讯音乐娱乐集团（纽约证券交易所代码：TME）是中国在线音乐娱乐服务领航者。拥有中国广受喜爱和独具匠心的音乐平台：QQ音乐、酷狗音乐、酷我音乐和全民K歌。腾讯音乐致力于创造音乐无限可能，用科技赋能音乐，人人都可以尽情创作、享受、分享音乐和彼此互动。腾讯音乐提供在线音乐、在线K歌和音乐直播服务，让每一位音乐爱好者能够发现好音乐、获得听、看、唱、表演和社交为一体的全方位音乐娱乐体验。
