职位描述：

岗位职责:
1、负责喜马拉雅资源管理中心 UI 设计(涵盖WebApp，PC客户端)运营活动等设计工作； 
2、和产品交互一起构思与创意，灵活提供视觉解决方案，能很好的掌握设计前沿趋势，懂得设计赋能商业； 
3、进行数据分析和可用性测试，评估产品上线后的效果，并提出用户体验改善计划。
任职要求：
1、教育经历: 本科及以上 ，2 年以上的互联网或移动互联网设计经验，熟悉 Web、iOS、Android 设计规则； 
2、有成熟的代表作与成功项目设计经验,有综合全面的专业能力 ( 如用研方法、具有交互/视觉的专业能力 )； 
3、重视细节，追求像素级精神，热爱交互小动画； 
4、精通 Sketch、PS、AI 、AE 等视觉或动效设计软件； 
5、有 ToB 类商业产品设计经验，有成熟的UI动效作品是很好的加分项。
＊附件请附上作品集或作品链结

【我们是谁】
      我们是喜马拉雅，国家高度重视的新经济龙头企业、互联网行业独角兽。打造国内领先的音频分享平台喜马拉雅，我们拥有海量6亿手机用户，和超5000万海外用户，是中国文化出海的中坚阵地。
【我们的使命】
      用声音分享人类智慧，用声音服务美好生活。我们首创内容生态PUGC，引导行业潮流创新，吸引大咖主播，玩转声音多姿彩，投身音频内容创业。频道覆盖多又广，包含财经、音乐、新闻、商业、小说、汽车等328类过亿有声内容。
【我们的生态】
      秉承万物有声的理念，将有声内容渗透到生活中各个智能终端和场景中。2017年6月，推出国内仅有的全内容智能AI音箱——小雅，布局人工智能领域，打造物联网时代信息获取最便捷的入口。让用户把需求大声说出来，开启了一个语音交互的新传播时代，让声音和知识像水和电一样无处不在，随取随用。
