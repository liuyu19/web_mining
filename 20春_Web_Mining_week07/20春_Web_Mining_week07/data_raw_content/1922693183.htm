职位描述：

岗位职责：
1、参与产品原型讨论，和产品和交互一起构思，灵活提供视觉设计方案，把控设计质量，保证设计的落地；
2、负责中心产品的视觉设计，包括PC端前后台和APP端产品的产品界面、图标和相关设计；
3、有能力制定详细的视觉设计规范，不断优化、维护设计规范，保证产品视觉的统一性；
4、主导前期界面概念探索、视觉风格定义，主动分享设计经验、提高团队的设计能力；
5、负责中心其他部门的辅助性设计需求。


任职要求：
1、本科及以上学历，设计类、美术类专业；
2、5年以上WEB设计和APP设计相关经验；
3、对视觉设计过程有比较深的了解，熟练使用各类设计软件Photoshop\Sketch\Axure等；
4、对用户体验设计和易用性，有敏锐的把握能力；
5、有较强的逻辑思维能力、良好的沟通能力和快速学习能力，具有激情、责任感和执行力；
6、有人工智能相关设计经验更佳；
7、面试时需携带作品集。
