职位描述：

招聘要求（岗位职责）
1.负责公司相关产品的HTML/CSS/JavaScript等前端代码编写，网页标准化及优化研究；
2.解决HTML代码不同浏览器的兼容性问题（IE8以上，及其他主流浏览器）；
3.优化HTML代码及其结构，优化载入速度；
4.维护及优化网站前端性能，提高用户体验；
5.能够配合相关产品完成部分设计工作。
招聘要求（任职资格）
1.大专及以上学历，计算机相关专业；
2.两年以上WEB前端开发工作经验，能够独立完成项目的开发制作；
3.精通HML/CSS等，能够准确细致规划CSS结构，熟悉网站DIV+CSS标准化布局，制作符合W3C规范的页面，并有良好兼容性，有移动端H5开发经验优先
4.精通JavaScript，熟练使用原生JS，以及JQuery；
5.精通流行的前端JS库，如node.js 、zepto.js 、angular.js 、vue.js、 React.js等；
6.有两种或以上的前端框架使用经验，如layUI、mui、easyUI、extjs、bootstrap、dojo、weui等，并能熟练掌握每种框架的优缺点；
7.精通json数据格式的操作；有前后端分离开发经验者优先；
8.熟悉TCP/IP、WebSocket、HTTP等网络通信技术；
9.熟练使用Photoshop、dreamweaver等常用代码切图工具，熟悉图片的制作；
10.具有良好的分析和解决问题能力，对攻关疑难问题具有浓厚兴趣。
