职位描述：

岗位职责：
1、负责创意设计工作，包括视觉识别系统设计、平面设计、自媒体设计，准确把握设计方向，把控设计交付物的质量；
2、负责公司海报、商品目录、宣传品、产品手册、品牌包装等资料的设计与制作；
3、负责公司发布会、展会的现场平面设计工作，并配合做好布展工作；
4、负责公司新产品线上线下的整体视觉推广方案设计。

岗位要求：
1、根据产品定位，时尚变化，能提出构思新颖、有高度吸引力的创意设计及色彩组合搭配技巧；
2、实时把握设计的流行趋势，能够经常提出崭新的创意策略，为产品提供专业的设计意见；
3、有一定的手绘功底，会视频编辑剪辑者优先，熟练使用各类常用软件进行界面设计，如PS、AI、CDR、C4D等；
4、美术、设计或相关专业本科以上学历（优秀人才，专业和学历可适当放宽）; 
5、3-5年以上设计相关工作经验及3年以上主设计师岗位经验；
6、团队意识强，沟通能力佳，有很强的责任心和承压能力。

企业介绍：

华峰集团（www.huafeng.com）创办于1991年5月，是一家以化工新材料为主，以金属、金融、物流、贸易等产业为 辅的大型民营股份制企业。现有员工9000多人，总资产241.19亿元。在全国12个省市建有生产基地，拥有两家上市公司，三家特色金融公司，七家国家高新技术企业，是国内最大的聚氨酯制品产销企业。
　　华峰集团专注于研发生产高品质的化工新材料产品，致力于客户与公司的成功进行创新，致力于可持续发展，建有一个国家级企业技术中心，一个省级重点研究院，两个省级研究院，用创新为客户创造价值，用创新为人类带来美好生活。
      集团目前建有一个“国家级企业技术中心”、一个“国家级行业研发中心”，两个“省级研发中心”，两个“省级研究院”，是超纤合成革、氨纶长丝等五大产品国家行业标准的起草制定单位。集团连续多年名列“中国大企业集团竞争力500强”、“中国制造业企业500强”、“中国民营企业100强”、“中国石油和化学工业百强”。
