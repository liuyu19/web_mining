职位描述：

岗位职责：
1、负责短视频产品的后台服务架构设计和实现；
2、负责多媒体AI平台的建设，负责短视频内容管理平台的建设；
3、负责运营平台的建设，大数据分析与视频PUSH内容推荐。

任职要求：
1、本科及以上学历；
2、3年以上Unix/Linux操作系统下的C/C++/go/java项目的开发经验；
3、熟悉网络编程，精通TCP/IP协议及编程，熟悉互联网应用协议；
4、熟悉面向对象的大型分布式系统设计与开发，了解高并发处理技术；
5、了解微服务架构与设计技术，有性能测试与优化，全链路监控与问题排查经验；
6、具备良好的分析解决问题能力，能独立承担任务和有系统进度把控能力；
责任心强，良好的对外沟通和团队协作能力，主动，好学；

掌握下面一到多项技能者加分：

熟悉Docker/k8s等容器技术， 熟悉TARS GO等开源服务框架；
熟悉微服务框架 server mesh，FAAS， Severless；
了解常用的分布式处理框架与组件如spark/hadoop/ES；
具有数据分析经验或AI平台建设经验；
熟悉GO语言编程，或熟悉大数据处理编程语言python/scala/R。
熟悉使用分布式消息队列如 Kafka， RabbitMQ 等；
熟练使用分布式数据库如TiDB；
具有开源框架Contrihuter经历或者熟读单个TOP项目的开源代码；
