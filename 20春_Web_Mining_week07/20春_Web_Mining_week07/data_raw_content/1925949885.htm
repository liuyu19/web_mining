学历条件：统招本科及以上毕业

后端岗位职责: 
参与华为云服务前台，后台，中间件，数据分析，数据挖掘等设计，开发、测试、运维工作； 
任职要求： 
1.  本科五年，硕士三年及以上前台或者后台开发经验，具有知名企业的工作经历； 
2.  熟悉常用的数据结构和算法，有充分代码编写经验，有良好的编程风格与习惯； 
3.  熟悉Spring、springboot、springcloud框架体系，掌握多个spring功能模块； 
4.  掌握关系型数据库的使用，如：MySQL、Oracle； 
5.  掌握源代码管理工具，如：SVN、Git等；  
6.  熟悉微服务框架以及应用，有分布式系统的设计和开发经验优先。 
7.具有基本的英文文档阅读能力，CET-4以上； 
8. 近四年3跳，两年两跳、创业经历、在职研究生、专升本、三本等都不考虑


前端岗位职责: 
1.  负责Web产品前端UI及交互页面开发及项目定制； 
2.  负责前端html、css、js的设计、编码开发及性能调优，能独立完成系统模块的设计与开发； 
3.  配合后端工程师完成后端数据的接口设计及联调； 
4.  负责前端的维护、优化、重构； 
任职要求： 
1.  本五硕三年以上的WEB前端开发，具有知名企业工作经历。 
2.  熟练掌握使用JavaScript进行Web开发，熟悉html5 + css3的新功能, BootStrap框架等； 
3.  熟悉至少一种后端开发语言, 熟练掌握API开发和调试经验者优先，具有组件化的思想，精通前端应用构建和组件封装； 
4.  精通Vue.js/Reactjs/Angularjs2等前端框架中的一种或多种，并掌握其原理； 
5.具有基本的英文文档阅读能力，CET-4以上； 
6. 近四年3跳，两年两跳、创业经历、在职研究生、专升本、三本等都不考虑
