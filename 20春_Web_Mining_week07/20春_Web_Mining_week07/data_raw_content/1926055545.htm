职位描述：

工作职责：
1、负责物流产品线的前端开发工作；
2、结合项目需求，分析并给出合理高效的前端解决方案；
2、根据产品设计及相应平台（各平台小程序、React-Native、Hybrid APP、H5 web）,利用对应技术或框架实现相应应用；
3、参与功能镶嵌和调试工作，实现产品前端UI和交互开发需求，确保产品在不同平台、设备商兼具优秀的用户体验；
4、参与前端框架及微服务的设计和规范化工作，优化代码实现，建设前端团队、提高产品性能；
岗位要求：
1、全日制本科及以上学历，五年以上前端开发工作经验，有复杂产品开发或框架开发经验优先；
2、精通前端基础，常用布局、LESS/SASS，Webpack、JS基础、ES6、Typescript等；
3、熟悉主流开发框架，如react、VueJS；
4、有Hybrid App、RNAPP，微信小程序开发经验优先；
5、熟悉网页性能优化，并有相关优化实践经验；
6、对移动设备端浏览器特性及兼容性有深入研究，具备移动端WEB开发经验（android、ios）；
7、注重利用合理的设计模式和模块化的组织方式对代码进行架构；
8、有团队协作精神、善于学习，乐于探索，能承受一定的工作抗压力。
