职位描述：

职责描述：
具备航空（机场和航司）行业业务和信息系统分析的能力，为团队提升行业整体经验和解决方案的核心竞争力。
1、深入洞察航空行业趋势，理解行业客户需求并匹配行业解决方案；负责行业客户的high Touch，通过业务以及系统分析，牵引客户ICT投资，挖掘机会点；
2、负责航空行业山头及重大项目的拓展支持及解决方案拓展等工作，参与市场洞察，项目分析，行业研讨，给与一线有效支撑；
3、了解行业价值链，识别主流合作伙伴，负责推动交通行业设计院、咨询公司，行业协会等合作伙伴咨询和建设；
4、负责策划行业精准营销活动、一线赋能、研讨、行业机会引导等工作; 负责日常行业高端客户来访接待，以及行业展会，现场会，以及高端研讨论坛Speaker；
5、围绕目标匹配资源，牵引相关资源，确保航空业务的订收目标达成。


任职要求：
1、具有扎实的技术背景，熟悉ICT技术(IT、IP、统一通信等)，对行业ICT整体架构有全面清晰的了解，有IP、IT背景者优先;
2、具备航空行业信息化的顶层设计能力；了解全球机场/航司信息化发展趋势，了解机场/航司业务和信息系统组成；具备前瞻性、全局性的思维；综合分析能力强，了解某个区域或者国家的航空业发展情况，有一定的国际化视野。
3、能深入理解客户业务，可准确把握客户需求，具备完备的分析、引导客户需求的能力。
4、5年以上航空行业工作经验，曾作为第一责任人负责过大型机场或航空公司项目的建设；
5、有航空公司工作经验和全球派遣者优先。
6、熟悉行业生态圈，在业内有一定影响力；熟悉行业招投标流程和运作模式，具有丰富的大型项目售前运作经验。
