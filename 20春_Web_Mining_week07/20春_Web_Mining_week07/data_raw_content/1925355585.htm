职位描述：
岗位职责：
1、负责京东金融软件平台旗下金融业务系统前端开发工作；
2、优化代码实现，提高产品性能；
3、js前端页面交互设计, 提高用户体验；
4、理解后端架构, 与后端工程师配合, 为项目提供最优化的技术解决方案；
5、配合开发主管，参与HTML5应用架构的设计和规范化工作。

岗位要求：
1、6年以上互联网产品前端工作经验；
2、全日制本科及以上学历，计算机或软件工程等相关专业，扎实的计算机以及网络基础；
3、熟悉ES6,commonjs,amd等模块化规范，有现代化前端组件开发思维；
4、有jQuery、Vue、React、Sass、Stylus 、Webpack、express.js 等前端技术使用经验，精确掌握各种常见布局方式，了解BEM或者其他CSS命名规范等；
5、对SPA应用，MV*模式，前后端分离式开发有深刻理解，解常见的前端本地缓存机制；
6、可独立完成、或协作完成后台前端界面展示，抽象公用组件，并优化代码并保持良好的浏览器兼容性；
7、有过后台前端产品的研发经验，有过大中小型前端项目的重构，迭代经验；
8、具备良好的学习能力、沟通能力、分析及解决问题能力, 优秀的团队协作精神。

加分项：
1、熟悉 TypeScript 者优先；
2、了解后端系统（Java）开发，以及SSH开发框架；
3、了解UI、UE相关知识以及设计能力；
4、跨平台技术开发经验，如Flutter, React Native；
5、 Ios或者Android开发经验优先；
6、喜欢钻研技术，能够快速掌握和应用新技术,有团队管理经验或开源项目贡献经验优先(有GitHub项目, 参与或向开发者提交过 bug 和 patch)。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
