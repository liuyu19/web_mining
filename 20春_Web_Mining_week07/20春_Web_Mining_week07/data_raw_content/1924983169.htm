职位描述：

职责与工作任务：
职责表述一：负责产品（安全气囊，发生器，方向盘，安全带，电子）的验证工作，制定DV,PV及量产后和工程相关的QTP (Qualified Test Plan )
1、收集和分析项目相关的实验技术标准及过往经验. 包括材料标准, 客户标准/要求, 法律法规, JSS内部标准等。
2、从相关部门处收集与本项目相关的过往实验信息. 如项目的实验大纲, 实验报告等.
3、从相关部门处收集其他信息. 例如: 从项目管控部收集生产场地, 供应链情况等信息,从设计部收集DFMEA, DRBFM, 从工艺部收集PFMEA等等.
4、比较客户/JSS内部的实验标准,
比较设计/过程上的变化点,结合上述收集到的输入信息,制定合理的QTP(Qualification test plan).
职责表述二： 验证试验的控制及结果呈现 ，对验证结果作出判断
1、根据被批准的QTP,向相关实验室发行实验依赖
2、确保验证过程中使用正确的实验样品，实验方法和实验治具，定义新的验证方法保证验证过程的质量。
3、及时分析内外部测试的测试结果，在DR会议上汇报验证结果。
4、负责向项目团队汇报更新后QTP, 通过DR2/DR3/DRPV审核并被批准.
5、对测试结果给出判断
职责表述三： 验证报告的制定.
1、收集实验室报告LTR
( Laboratory Test Report ), 实验后样品实物, 实验原始记录(照片, 录像等), 确认实验结果.
2、根据验证工作手册的要求, 完成验证报告。
3、负责向项目团队汇报Validation
Report内容并被批准.
职责表述四： 开发验证方法提高验证质量及为失效问题分析提供支持
1、分析测试方法并开发新的验证方法以提高验证质量.
2、为失效根本问题分析提供支持
3、积极参与支持验证实现过程以确保高质量的验证过程
职责表述五： 测试技术经验积累，形成测试经验手册（UTGL），向客户介绍新的测试技术
1、汇总测试方法、测试矩阵及过往经验
2、向客户介绍新的测试技术/在测试
3、形成测试经验手册（UTGL）
职责表述六：审核并批准PAC ( Product Acceptance Catalog )
审核并批准设计部提供的PAC, 确认抽样频率, 实验方法, 实验条件, 判定标准完整准确, 无错误, 无遗漏等.
任职资格 Qualifications：
1、大学本科以上学历，车辆工程、机械工程、材料工程、电气工程等专业
2、有IATF16949、FMEA相关培训
3、具备3年或以上汽车行业被动安全零部件开发、质量或测试的经验，有验证工程师经验者优先
4、熟悉汽车被动安全产品的各种客户标准/法规要求，了解被动安全产品的开发流程.
5、熟悉AB
or SB or SW or INF 产品知识，熟悉被动安全产品的测试方法.
6、具备流利的英语说写能力
7、熟练使用Excel，PowerPoint，Word等办公软件
8、良好的问题分析与解决以及沟通能力
9、诚实, 责任心强, 团队意识强, 抗压能力,
