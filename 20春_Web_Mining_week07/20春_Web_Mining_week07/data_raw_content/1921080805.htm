职位描述：

1、输出详细产品UI功能的业务需求规范文档及规划设计规范文档。执行跟进软件实施产品UI规划设计，进行UI确认验证体验测试。协助配合，从结合具体产品软硬件情况，进行合理性、可行性，可控性、可阶段性实施的产品线UI的用户体验设计建议和分析；
2、趋势研究：执行行业设计趋势分析，负责Android等平台的UI设计趋势研究分析，制定搭建平台的设计规范，以及持续快速的版本演进
3、竞品分析：协助进行市场调研，竞品分析，制定优秀或差异化竞品的UI功能需求，归纳产品功能特点、提炼易用性、友好性、智能性等特色卖点；
4、设计研究：协助UE人员，参与负责产品的用户行为习惯调研、需求动机分析,制定UI功能的设计目标和可用性目标；
5、交互创新：执行负责交互模式创新、功能卖点创新，并申请专利等知识产权有效防御；
岗位要求：
1.对用研、交互、产品、运营等有一定的认知和理解，有良好的设计讲解能力和沟通能力；
2.把握各产品在视觉设计方向上精准定位，保证各平台视觉规范的一致性；
3.熟知Android、ios平台设计规范，注重细节，也具备全局把控能力；
4.熟练掌握sketch、AI、PS等设计工具，熟悉AE、principle、Framer等动效设计软件更佳；
5.具有活跃的创新思维和优秀的创意能力，能够持续在产品视觉设计方面提供具差异化的设计方案。

企业介绍：

     宇龙计算机通信科技（深圳）有限公司（简称“宇龙通信”）创立于1993年4月，系酷派集团有限公司（香港主板上市公司，股票代码2369）的全资附属子公司。酷派集团是中国专业的智能手机终端、移动数据平台系统、增值业务运营一体化解决方案提供商，专注于以智能手机为核心的无线数据一体化解决方案。
     公司拥有员工6000余人，其中研发人员超过40%，持有专利6000余项，多项技术打破国外公司的垄断。目前，公司已建立起以深圳酷派信息港为总部，以北京、西安、合肥、南京、东莞的研发机构和生产基地为支撑，辐射全国的产业支撑体系，产销规模超过200亿元。
酷派集团是国家级重点软件企业、深圳市政府认定的高新技术企业，先后获得中国优秀民营企业、亚太区500强高科技企业、中国高科技企业50强、中国电子信息100强、广东省知识产权示范企业等荣誉称号，并被国家授予“博士后科研工作站”。经过20多年的发展，酷派集团经历了“创始发展”、“智能终端”、“移动互联网”三个发展阶段，取得了一系列引人瞩目的成绩。其中，全球首创的“CDMA/GSM双网双通终端”技术荣获国家科技进步二等奖，是迄今为止手机终端技术领域所获得的最高奖项。
2014年，酷派完成了“三大品牌、四架马车”的战略布局。“三大品牌”分别为coolpad、dazen、ivvi，每个品牌拥有不同的定位，独立运作与核算；“四架马车”则按业务划分为酷派中国区、奇酷大神、ivvi公司、酷派海外，建立复合多样的商业运营模式。目前，酷派的战略布局已初显成效，随着各子品牌市场份额的不断提升以及产品结构的逐步完善，酷派将实现从中低端市场向中高端市场的跨越。
     随着“互联网+”逐渐成为中国经济发展的新引擎，酷派积极向移动互联网转型，形成“硬件生态+渠道生态+内容生态+用户生态”全新的生态概念，打造强大的互联网生态圈。在PMCC（Personal/Mobile/Cloud /Computer）的指引下，酷派将从B2B向B2B2C转变，聚焦精品策略，转型做消费者品牌。
未来，酷派将通过“青春合伙人计划”，加强平台建设和产业链合作，以互联网共享理念打造合伙人机制。从中国制造到中国创造，酷派将凭借卓越的技术创新能力，致力于“让用户尽享酷派一体化终端和服务的极致体验”，坚定朝着“一流的世界级企业”目标迈进！
