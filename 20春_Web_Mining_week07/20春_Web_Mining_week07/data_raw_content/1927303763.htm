职位描述：

工作职责：
我们需要行业顶尖水准的 GUI 设计专家，负责智能 APP 及带屏类智能产品（带屏智能音箱、车机、电视、一体机等）系统 UI 的视觉设计工作，产出最具行业领先性的设计解决方案，定义新的领域标准。
任职要求：
1. 四年以上互联网视觉设计工作经验，有成功案例/代表作品，曾多次主导设计和制定大型项目的视觉风格与落地，有系列化产品设计经验，能通过建立、优化规范和机制，保障落地品质； 2. 拥有良好的视觉审美和视觉表现能力，对设计流行趋势敏锐且具有前瞻性，并擅长基于品牌基因与产品特色产出富有创造性的设计方案； 3. 拥有较好的用户视角，对视觉设计的商业价值及用户价值有自己的理解，同时对于视觉设计有系统化的抽象和整合能力； 4. 综合能力强，拥有独自驾驭复杂系统 UI 的架构力、表现力、控制力； 5. 有激情、创新欲、责任感，能够自我驱动和快速学习，能够推进自己的想法，具备良好的合作态度及团队精神 6. 良好的思辨总结与沉淀能力，能引领团队的专业建设与发展。 一些加分项 1. 有 智能语音／ IoT／智能软硬件／智能车机 / 跨端跨平台 相关产品视觉设计经验； 2. 有系统级产品设计能力或对设计系统有深入研究。
