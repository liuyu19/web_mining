职位描述：

工作职责：
1. 负责B站商业型产品设计，为产品提供体验、交互、视觉等全方位的设计服务
2. 准确理解产品目标，协同PM及研发团队，以用户角度提出适合于B站的设计创意与解决方案
3. 善于洞察市场环境，深度挖掘用户诉求，分析数据走向，服务于产品体验
4. 沉淀设计方法论并对外输出

职位要求：
1. 5年以上互联网设计经验，拥有成熟落地的互联网产品案例
2. 优秀的全局观、产品理解、逻辑推导、交互及视觉表达能力
3. 高效执行及推动力，责任感及自我驱动强，对前沿设计风向敏感并有强学习能力
4. 强力的团队协作能力，能清晰有效的表达观点
5. 扎实的知识沉淀和分享表达能力
6. 加分项：较了解B站文化属性、用户喜好且能结合应用到设计方案中
