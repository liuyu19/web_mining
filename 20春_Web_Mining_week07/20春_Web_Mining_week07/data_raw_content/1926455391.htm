职位描述：

工作职责：
1、负责数据中台解决方案offering，通过洞察主动发现新机会点，前瞻性地识别行业数字化转型机会点，形成主推方案offering，并通过ISV生态地图的规划，联合伙伴、研发等，构建行业数据中台生态能力，主导数据领域解决方案offering的孵化和复制，并持续提升方案的竞争力和可复制性；
2、面向行业NA客户，为客户业务的数字化转型提供IA（信息架构）顶层设计、咨询规划与布道，引导客户IA（信息架构）和大数据平台TA（技术架构）规划并影响客户决策，参与行业数据中台策略和解决方案设计；
3、指导或者参与重大项目数据中台解决方案的实施，引导建立行业数据中台标准；
4、分析竞争对手信息并制定竞争策略。
任职要求：
业务技能要求：
1、8+年以上数据治理及大数据平台规划和实施经验；
2、对行业的信息架构（IA）和大数据平台（TA）技术架构具备深刻理解，有大型行业客户数据治理规划和实施经验；
3、熟悉各类数据库、数据仓库、数据库中间件及大数据平台相关技术，熟练掌握主数据、元数据、数据标准、数据标签、数据集成等主流工具和方法；
、具备成体系的洞察能力、具备数据专家顾问沟通能力、出色的大型复杂项目运作和资源协调能力；
4、承担过大中型企业数据管理岗位或大型企业数据治理项目项目经理（PM）/技术总架构师（TD）角色。
专业知识要求：
1、熟悉一个或多个技术领域：公有云、私有云、Openstack、IaaS、PaaS、容器/Docker 、HANA、SDN、HPC、IOT、AI、云存储、灾备、等；
2、相关行业公有云解决方案经验者优先；
3、具备重大项目上云解决方案设计经验，具备项目支撑和客户技术方案交流能力者优先，具备IT领域软件技术规划经验和架构设计者优先。
