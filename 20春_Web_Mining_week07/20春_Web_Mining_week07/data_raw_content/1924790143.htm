职位描述：

岗位职责：
1. 能够根据运营需求文档高效高质的完成设计需求；
2. 负责移动商城的资源位视觉设计；
3. 为多个新媒体平台的互动传播提供创意设计；
4. 负责根据创意文案进行视频制作、剪辑和后期加工；
5. 线下印刷及产品包装的设计与制作，了解印刷后期工艺；
6. 品牌周边的创意设计；
7. 展会整体视觉输出。

职业要求：
1. 了解设计流行趋势，对于创意执行，画面构图更重视用户视觉体验；
2. 有高度责任心，有一定的抗压能力；
3. 熟练使用PS,AI等设计相关软件；
4. 有手绘、视频制作经验者加分；
