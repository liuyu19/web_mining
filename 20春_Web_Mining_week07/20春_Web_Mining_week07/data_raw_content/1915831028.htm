职位描述：

职责描述：
3D设计师主要负责项目的视觉风格设计，规划场景中3D元素的色彩、大小、形状、位置，以及模型和材质的风格。
1. 配合产品美术总监制定虚实结合的场景风格，输出场景编辑制作模版和规范；
2. 手绘制作故事板，表现场景及虚拟内容的规划
3. 制作风格帧，指导最终美术效果
4. 及时学习新的软件工具和技术
任职要求：
设计专业背景要求：泛设计专业毕业
 美术学院中造型艺术（美术学）、艺术设计（设计学类）
造型艺术类如：雕塑、摄影
艺术设计类如：动画、环境设计、信息设计、视觉传达、影视/摄影
 2、综合大学中艺术与设计学院中部分设计类专业：
数字媒体技术/新媒体技术、游戏设计、动画设计、导演、编导
技能：手绘技能及软件技能
 较强的手绘功底
 熟练使用Photoshop等设计软件
 如有三维软件、剪辑软件、游戏引擎使用经验更佳
工作经历要求：有设计类工作经验
 游戏公司（3D游戏）：策划、美术（原画、场景美术）
 动画公司：导演、原画、背景、人物设定、机械设定
 虚拟现实/多媒体互动公司：VR/AR节目美术设计
