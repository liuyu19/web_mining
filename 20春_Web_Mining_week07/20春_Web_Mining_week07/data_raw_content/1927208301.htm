职位描述：

工作职责：
1、根据策划文档完成游戏功能模块的前端开发工作
2、与服务端程序员一起设计和实现通信接口和协议
3、对自己所负责的工作有极高积极性，主动跟进与处理所负责模块的bug
4、按时完成上级主管交予的各项任务，并主动学习与研究最新的技术
任职要求：
1、本科及以上学历，计算机、数学等相关专业，有网络游戏开发经验的优先
2、熟悉Lua/Python等至少一种脚本语言，熟悉C/C++/C#/Java等至少一门高级语言，数据结构和算法基础扎实
3、理解3D游戏的基本概念，有UE4、Unity3D使用经验者优先
4、熟悉Windows平台的开发工具，比如：Visual Studio
5、有一个及以上完整游戏上线运营项目经验者优先
6、逻辑思维清晰敏捷，有优秀的团队合作和沟通技巧，有较强的独立解决问题的能力，能承受一定压力和工作强度
7、热爱游戏
