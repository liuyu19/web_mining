职位描述：

职责描述：

1. 结合小学业务发展战略，负责制定小学语文业务发展规划，整体管理语文学科的留存运
营工作；
2. 搭建运营团队，推进语文学科运营项目发展，制定学科运营全年工作节奏，支持完成小
学语文年度预算业绩指标；
3. 负责语文续报业务的整体运营设计和节点管控，参与设计续报流程和节奏，包括：整 体运营规划、物料支持、工具开发，输出培训及运营动作，并结合续报情况，进行
产品、服务的持续优化，包括：对接和推进产品线续报教学节奏落地计划，促进
主讲和辅导老师的配合，设计辅导老师服务框架和关键动作节点的设置
4. 负责小学语文产品外化，与产品研发、教学教研侧、辅导侧，共同管理推进小学语文产
品推广，推动服务升级；参与提升教学服务体验及续报率，并跟踪记录，定期反
馈，积极协助提高辅导老师侧服务品质，并跟踪记录，定期反馈，并提出可
行性解决方案；提升用户满意度，在服务用户的基础上提高续费转化率；
5. 进行小学语文学科日常业务运维，包括业务跟踪，数据分析、运营项目推进，分析运营
效果，提供合理有效的优化建议，并形成方案，推动完成既定的续报目标
6. 语文学科的竞品分析、行业调研、识别、分析用户需求，政策分析。

2. 5 年以上工作经验，具有教育培训行业-语文学科运营相关工作经验优先，了解行业及小

3. 有敏锐的市场洞察力和分析能力，客户感知能力；项目发展规划布局的能力、较强的分
析、解决问题能力，思路清晰，考虑问题细致；
4. 目标导向，有较强的逻辑思维、良好的沟通协调能力和组织管理能力；
5. 善于反思总结，能够通过优化关键节点，促进指标的健康成长。
6. 抗压能力高，执行力和克服困难的能力强，快速学习，不受困于常识和经验。

企业介绍：

    作业帮是一家专注于K12方向的教育科技公司，致力于为全国中小学生提供全学段的学习辅导服务。作业帮以教育为信仰，以科技为驱动，凭借作业帮强大的技术优势，打造了包括拍照搜题、作业帮一课、作业帮问答、作业帮一练、中英作文和作业批改在内的六款明星产品。截至目前，拥有1.8亿+题库数量、1.2亿+月活跃用户 ，是中小学在线教育领军品牌。
   企业秉承“让优质教育触手可及”的使命，致力于借助科技力量提供学生个性化互动教学服务，提升学生学习效率、助力学生全面发展。作业帮团队怀着对教育事业深深的敬畏之心，怀着对创业初心殷殷的坚守态度，始终坚持务实、进取、用户第一的价值观不断优化产品、完善服务，真正做到让广大中小学生和家长方便快捷的享受优质教育。
   作业帮项目由百度孵化，2015年9月正式独立运营，A轮融资由红杉和君联资本领投。于2016年中完成B轮融资，由纪源资本和襄禾资本领投，A轮投资人跟投。2017年8月完成1.5亿美金C轮融资，由H Capital领投，老虎基金、红杉、君联、GGV、襄禾跟投，是K12在线教育领域迄今为止最大规模的C轮单笔融资。2018年7月18日，作业帮正式宣布完成D轮3.5亿美元融资。此次融资由全球知名投资管理机构Coatue领投，高盛、春华资本、红杉中国、GGV纪源资本、襄禾资本、天图投资、NEA、泰合资本等新老股东跟投，泰合资本担任独家财务顾问。
