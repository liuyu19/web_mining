职位描述：

岗位职责：
1. 参与先知产品测试的全流程，包括参与需求分析、设计评审，制定测试计划，设计和执行测试用例，分析总结产品的测试结果，推动问题的解决，提高最终的交付质量；

2. 参与适用于产品需要的自动化测试工具/框架开发，自动化测试用例设计、执行，自动构建测试环境，持续集成等；

3. 保证被测系统的质量，并通过测试流程和方法创新，持续改进提高测试团队的测试覆盖度和测试工作效率。

任职要求：
1. 全日制本科以上学历，计算机、通信、网络等相关专业背景，有计算机视觉产品或云服务产品测试经验优先；具有一定的团队管理经验

2. 熟悉Linux系统，熟悉golang/python/java等面向对象编程，至少熟悉python/shell一门脚本语言；

3. 具备测试领域系统化的知识结构，熟悉完整测试环节，能独立完成功能测试、接口测试、UI自动化测试集成测试；

4. 精通测试流程和测试用例设计方法, 熟悉主流自动化测试框架，如Selenium/ /Jmeter/SoapUI/Testng/Python Requests等，能主动进行技术钻研，有系统性能测试经验；

5. 解决复杂问题和编写自动测试工具和系统的能力；

6. 加分项：能熟练使用jenkins等工具进行持续集成。
