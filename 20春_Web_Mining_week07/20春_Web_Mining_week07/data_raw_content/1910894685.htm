职位描述：

职位基本要求：
* 大学硕士以上学历，计算机信息与技术、电子工程、信息工程、自动化或相关专业；
* 有扎实的研究与开发背景，对相关技术领域有深入理解；
* 有在大型IT、通讯、互联网企业的研究院、专利分析或产品开发部门的5年以上工作经验；
* 有丰富的专利发明经验，对发明的价值评估有一定认识；
* 优秀的人际交往和沟通能力；
* 流利的英语沟通和阅读能力。
 
技术背景要求(经验符合以下领域之一或若干者优先)：
1. 消费电子软件平台
* 在嵌入式软件开发、音视频编解码、用户交互、操作系统、网络协议、BSP、硬件架构中的至少两个领域内具有丰富的开发经验；
* 了解整体软硬件平台的设计和相关各个模块的厂家的特点及产业链的情况。
2. 消费电子硬件平台
* 具有消费电子产品（例如手机/PC/机顶盒/电视等）的硬件设计、开发、测试等经验，以及相关BSP系统开发经验；
* 熟悉相关模块的厂家的特点及产业链的情况，至少深入了解一项硬件模块的原理、集成、测试等。
 
具有以下经验者优先:
* 极客精神，较强的动手能力和深入探究问题本质的习惯；
* 博士学历，计算机信息与技术、电子工程、信息工程、自动化或相关专业；
* 作为主申请人申请过一定数量的专利；
* 具有从技术角度进行专利评估与分析、逆向工程、侵权检测与侵权分析报告构建等经验；
* 技术优秀者可降低专利方面的要求，并在入职后提供专利方面的培训。
 
工作职责：
* 熟悉公司现有专利资产，并开发具有竞争力的专利组合；
* 参与专利评估，从市场应用、技术、法律等角度分析专利的价值；
* 挖掘并开发专利的侵权分析报告，与外部企业开展专利谈判；
* 协调知识产权交易、许可、被许可、诉讼、以及与业务团队的合作以确保在商业策略、技术方向和知识产权战略上的一致。
 
具备以下几种素质：
1. 丰富的技术背景和研发经验，洞悉技术的发展趋势，具备宽广而独到的视野；
2. 具有较强的团队合作意识，并同时具有自我主导完成技术方案价值评估的能力；
3. 具有强烈的以结果和目标为驱动的意识，具备良好的心理素质和强大的抗压能力；
4. 富有自我驱动的能力，具有高度的责任心、积极进取的工作态度。
