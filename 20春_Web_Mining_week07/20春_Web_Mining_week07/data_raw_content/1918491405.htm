企业介绍：

   新潮传媒集团是一家科技媒体创新企业，是中国中产社区第一媒体流量平台，是专注中产家庭消费升级的媒体流量平台。由20年的传媒老兵张继学、二三四五网络创始人庞升东和力美DSP平台创始人舒义联合创办。新潮传媒于2007年成立于成都，目前拥有北京、上海、广州、深圳、成都、重庆、杭州、武汉、天津和南京等20余个子公司，现有员工近4000人。
    新潮传媒融资一期工程已结束，5家上市公司的创始人或董事长联合投资10亿元，分别是二三四五网络创始人庞升东，恺英网络创始人王悦、顾家家居董事长顾江生、欧普照明董事长王耀海、天神娱乐创始人朱晔，公司估值接近50亿。
    目前新潮的三大任务
第一，打破垄断，成为客户和行业垄断者谈判的筹码。
第二，真诚服务，实实在在为客户节约广告费。
第三，合理竞争，让物业公司增加经济收入。
