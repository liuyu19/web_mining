职位描述：

岗位职责：
1、负责产品web端交互设计，对产品用户体验负责;
2、根据需求文档制作交互原型demo;
3、根据产品特性，制定并输出相关设计交互规范和设计说明;
4、对现有产品的可用性测试和评估提出改进方案，持续优化产品的用户交互体验;
任职要求：
1、本科及以上学历、工业设计、计算机、心理学、人机交互、艺术设计等相关专业;
2、2年以上交互工作经验，有1个以上上线商用产品经验;
3、熟悉交互设计流程及设计原则，有一定的产品经验，能协助产品梳理需求和信息架构，独立完产品的交互设计，跟进UI设计；
4、逻辑分析能力强且表达能力佳，能深入挖局问题并提出自己的见解。
5、熟悉UI视觉设计规范，了解前端开发相关知识，熟练使用交互设计常规、主流工具软件；
6、良好的沟通能力和团队协作精神，善于跨部门合作，保证产品顺利上线；
7、有B端经验；

企业介绍：

杭州网银互联科技股份有限公司成立于2004年，是一家致力于为全球用户提供企业级IT管理和云计算技术的服务商。
网银互联通过自建分布式数据中心和全球网络节点，与全球通信运营商、公有云厂商实现资源互联，形成自主创新的云网融合管理平台，为客户提供IDC订制、服务器托管、混合云、容器云、SD-WAN、云上云下数据传输等产品和服务，并以专业的MSP能力为企业提供咨询、评估、方案设计、部署及运维的全闭环服务，确保开发、测试、应用发布等生产环境流程的敏捷、高效、安全，帮助企业数字化转型，实现企业IT向业务的赋能。
