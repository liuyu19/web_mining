职位描述：

工作职责：
1. 助力阿里巴巴生态组织管理数字化转型，通过深入了解组织管理相关领域的业务诉求、对管理者的需求洞察，提供行之有效的设计解决方案； 2. 以用户视角出发，驱动体验升级，洞察用户痛点，提升阿里大脑整体体验及设计品质； 3. 基于各领域数字化场景的数据探索及剖析，提出有创新的、易理解的数据可视化解决方案，最大化数据价值； 4. 基于组织管理各场景的数字化建设，沉淀整体数字化解决方案、设计思考，形成大型组织管理模式、知识体系；
任职要求：
1. 五年以上设计工作经验，对用户体验设计的方法和流程有深入的理解，形成较完整的设计思路； 2. 具备严谨的思辨能力，能够针对复杂业务逻辑下的交互界面提出优秀的设计方案； 3. 对企业级服务和应用领域感兴趣，能够通过深刻理解业务，形成完整的设计方法论和创新的解决方案； 4. 能够独立承担项目的整体设计策略，深刻理解产品和用户痛点，制定体验指标，高效地获得设计成果； 5. 积极主动，善于沟通，有良好的团队合作能力，能很好的阐述设计观点、设计价值等； 加分项： 1. 过往工作中有过企业复杂产品设计经验、项目经验，例如ERP系统、CRM系统等，对 B 类产品有自己独特的理解和认知； 2. 有过数据、智能产品或项目从业经验，对数据敏感度高，对数据生产、研发、应用有有一定的了解，具备可视化设计基础能力； 3. 具备3D、动效、影视能力者； 4. 了解或掌握一门编程语言，例如Python；熟悉例如AntV、Echarts、Highcharts底层数据可视化类库者优先考虑；

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
