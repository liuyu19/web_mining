职位描述：

工作职责:
1、负责国际站移动APP、online产品的界面交互设计，UI设计和风格创意设计工作
2、对产品进行可用性测试和评估，与产品经理沟通，提出改进方案，持续优化产品的用户体验和视觉感受
3、独立完成前端界面的信息架构、流程设计和界面设计，提供高质量的产品原型图或UI稿等设计方案
4、协助完成产品运营策划方案
5、与前端开发协作，保证设计还原度

任职资格:
1、大学本科及以上学历，计算机、美术、工业设计专业优先
2、有2年及以上的全栈设计师经验者优先；有海外设计背景者优先
3、具有较强的团队合作意识和一定的沟通能力，能清晰表达自己的设计意图；
4、熟练使用Sketch、PS、AI、Axure、Mockups或其他设计软件，掌握AE软件者优先；
5、面试必须带上以往设计作品
