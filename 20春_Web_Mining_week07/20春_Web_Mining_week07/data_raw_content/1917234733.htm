职位描述：

工作职责
1. 负责公司现有产品的前端开发工作以及前端相关技术管理工作；
2. 参与构建和开发前端相关的工程项目，组织并参与研发技术难点攻关;
3. 负责构建完善高性能前端架构体系，优化前端开发流程体系，包含和后端的衔接和延伸；
4. 负责 Web前端前沿技术研究与新技术创新，致力于通过技术改善用户体验，负责前端技术在公司的推广和完善。
任职条件
1. 三年或以上的前端开发和架构经验，学历本科或本科以上；
2. 熟悉主流前端框架（bootstrap，jQuery，angularJS，vue，react）中的一种或多种，精通react，vue等， 掌握前端构建工具，如webpack工具的使用；
3. 精通B/S架构开发，精通HTML、CSS、JavaScript、AJAX等，熟悉Jquery、Bootstrap等前后端框架，对html5特性较了解；
4. 熟悉http协议，熟悉与服务端数据交互方式与技术，熟悉websocket/json等数据接口和格式；
5. 在浏览器兼容、性能优化方面有一定的研究；对D3.js、echart.js有实际项目经验，有web地图（百度地图等）二次开发经验者优先；
6. 有较强的沟通能力和逻辑思维能力，工作主动负责，有责任心，有较强的分析和解决问题的能力，技术视野广泛者优先。
