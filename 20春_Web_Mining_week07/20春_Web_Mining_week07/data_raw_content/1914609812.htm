职位描述：

【工作职责】
1、根据游戏功能需求，设计出适合游戏的交互解决方案；
2、把握策划需求，与美术、程序共同配合将功能开发上线；
3、能够深入分析评估一套产品的界面布局、操作设计是否合理，美观及交互反馈表现力效果如何，为产品界面开发提供参考和优化方向。
4、能够与上下游职位良好配合，把握策划需求，向视觉设计师和程序沟通传递设计思。
【职位要求】
1、热爱游戏，愿意投身游戏行业，有游戏交互设计经验优先；
2、能熟练使用photoshop、Ai等任一款设计软件，掌握交互设计方法（如信息架构、原型设计、流程分析等）；
3、具有良好的需求把握和逻辑分析能力，善于沟通表达。

企业介绍：

广州多益网络股份有限公司是中国领先的互联网企业，专注于自主研发和运营，坚持稳健经营、持续创新，在PC端网络游戏、移动游戏、电子邮箱、即时通讯等领域不断推出优秀的产品，为广大用户带去快乐的产品体验。目前，多益网络约有2000名员工，其中80%员工为研发人员，2017年在中国互联网企业百强榜上位列第25位。未来，多益网络将致力打造成为立足本土，走向海外的全球化综合型互联网企业。
十年来，多益网络一直秉承为玩家打造“真正好玩的游戏”的理念，接连推出多款现象级产品。2007年，多益网络推出首款自研网游产品《梦想世界》，大量的创新玩法让其成为当时市场上最具人气的回合制游戏之一。2010年，《神武》登场，以更公平绿色的玩法广受玩家青睐，成为中国回合制发展历程中的重要里程碑。进入移动互联网时代，多益网络先后推出《神武》手游、《神武2》手游、《梦想世界》手游等自研产品，神武系列手游还引领了中国手游行业网游化、社交化的新趋势。
伴随着业务多元化发展，多益网络也在积极构建一个以游戏IP为核心的泛娱乐生态圈，在电影、音乐、综艺等领域进行跨界合作。从邀请知名音乐人许嵩担任《神武2》品牌代言人到携手浙江卫视，多益网络不断创新跨界泛娱乐模式，引领“游戏+音乐”、“游戏+综艺”的泛娱乐潮流。
在游戏代理领域，多益始终与国际知名游戏公司保持密切合作。2017年，多益携手海外知名发行商505Games，获得人气沙盒大作《传送门骑士》国服独家代理权。从自研自运到代理引进，多益网络正逐步走向更广阔的全球市场。
作为一家把自主研发作为核心竞争力之一的互联网企业，多益网络一直坚持技术创新。多益网络已经拥有自主研发的2D、3D游戏引擎，在人工智能、自然语言识别等领域也已经取得了不俗的成果，并推出多益战盟、2980邮箱和多益云等互联网产品。
“虚拟的游戏，真实的幸福”，游戏虽然是虚拟的世界，但玩家在其中感受到的快乐与幸福却是真实的。多益网络将继续秉承这样的理念，志存高远，行始于足，为广大用户提供更多姿多彩的互联网服务。
