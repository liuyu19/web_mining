职位描述：

岗位职责：
1、负责各项业务流程、规范的贯彻执行，确保维修中心及同城或邻近授权钣喷中心向用户提供符合或超越标准的高效、便捷的售后服务，达成最佳的用户体验；
2、负责处理维修中心及同城或邻近授权钣喷中心的用户严重投诉，在授权范围内确保用户投诉得到及时、满意的处理；
3、负责维修中心全面工作：监控维修中心维修标准与日常服务过程、现场管理和服务细节的现场优化，结合团队业绩达成结果进行服务提升整改，包括但不限于制定计划、场景演练、考核测验、过程跟踪等，并对同城或邻近授权钣喷中心进行业务管理协调、支持工作；
4、负责维修中心运营成本控制，确保所有财务信息、文档、凭证等的有效、严格管理；
5、作为维修中心资产负责人，负责所有资产的安全、完整性，做好定期检核、维护工作；
6、负责公司各项规章制度、工作流程的实施、监督，完成公司下达的各项管理、业务指标；
7、负责应对政府职能部门检查，建设及维护政府及其他领域的良好关系；
8、完成总部指派的其它相关工作。
﻿任职资格：
1、8年以上汽车行业经验，担任高端品牌4S店售后管理岗或后市场连锁门店管理岗位2年以上，须具备服务顾问经验2年以上经验，维修经验丰富者优先；
2、持有维修职业资格高级证书，持有低压电工证者优先；
3、大专及以上学历、C1及以上驾照，驾驶熟练；
4、可独立完成对所在城市的售后服务团队进行业务水平提升管理、外部商务谈判；
5、具备极强的用户服务意识、责任心强，并贯彻到日常工作中；
6、具有创新管理思维，组建高效团队的能力，能独立迅速地寻找解决问题的能力。

企业介绍：

    理想汽车成立于2015年7月，致力于用科技改变出行，让更多人受益。
    公司的创始人李想（汽车之家创始人），拥有超过10年的汽车行业从业经验，并把汽车之家打造成为了超过100亿美金市值的上市公司。
    2018年10月，理想汽车的智能电动车品牌“理想”亮相，并发布旗下首款增程式智能电动车理想ONE。理想ONE定位于豪华中大型SUV，满足6-7人的家庭出行需要。它将采用增程电动解决方案，NEDC综合续航超过800公里。在智能系统方面，它将配备四屏交互和全车语音功能。
    理想ONE已于2019年4月正式开启预定，并将在同年第四季度在上述13个城市开始批量交付。
    理想汽车的总部位于北京，生产基地位于常州，直营店面覆盖北京、上海、广州、深圳、天津、武汉、成都、苏州、杭州、郑州、西安、南京、重庆共13个重点城市，为用户提供车辆体验、交付和维修保养等服务。
    理想汽车提供汽车设计、研发、供应链、生产制造、自动驾驶、销售、服务、互联网、软件、汽车电子等岗位，期待你的加入！
