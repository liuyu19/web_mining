职位描述：

岗位职责：
负责部门日常对外宣传创意设计；
负责配合策划与运营需求设计工作；
负责官方网站、微信平台会员部分的视觉设计工作；
负责部门线上、线下主题活动相关设计；
协助部门开发团队APP等系统UI设计。
任职要求：
1.2年以上设计经验和较高的设计水平，对审美色彩有自己独特的认知和见解；
2.熟练使用各种制图软件,如photoshop、ai、coreldraw等相关设计软件；
3.要有良好的改稿精神和抗压能力，时间意识和管理能力强，评估设计的周期，按时完成各项任务；
4.具有良好的沟通能力和表达能力，责任心强，具备较强的审美能力和团队协作力。

企业介绍：

周大生珠宝专注于“周大生”品牌珠宝首饰的设计生产、品牌运营和连锁经营，是国内具有影响力的珠宝品牌运营商之一，也是全国中高端主流市场钻石珠宝领先品牌。2017年4月27日周大生（002867）在深圳证券交易所A股上市。

截至2019年6月30日，周大生共有门店3599家，形成强大连锁渠道网络，带来非凡的集群效应和资源整合优势。
