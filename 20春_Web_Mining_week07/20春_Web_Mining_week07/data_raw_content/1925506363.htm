职位描述：

职责描述：
1、负责广告业务系统的整体设计，以及海量并发和低时延服务架构的设计与开发；
2、负责广告定向、广告匹配、广告播放等关键实时化工程的设计和开发；
3、负责广告生态的技术探索；
工作经验要求：
1、计算机、数学相关专业硕士以上学位，具备硕士、博士学历或ACM获奖经历优先；
2、3年以上推荐算法策略研究、设计与实现经验，熟悉机器学习、深度学习等算法及原理；
专业知识要求：
1、扎实的数据结构和算法基础，精通C/C++、Python、Scala等至少一门开发语言；
熟悉常见的数据挖掘、机器学习、深度学习算法以及TensorFlow等主流深度学习平台；
2、有广告推荐系统相关工作经验者优先，有良好的算法基础及相关竞赛经历者优先；
3、有机器学习顶级会议论文者优先；
