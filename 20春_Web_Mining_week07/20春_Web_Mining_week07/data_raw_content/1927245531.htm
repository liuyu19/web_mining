职位描述：

工作职责：
1.根据场景的流媒体框架如FFmpeg/WebRTC等，设计应用云端一体架构中实时音视频服务的架构设计、开发和性能优化； 2.根据图形系统的渲染结构，设计全新机制接入流媒体框架，支持云游戏的流化框架体系； 3.负责实时音视频流的场景下网络QoS服务的分析及优化。
任职要求：
1.至少3年Linux环境下的C/C++的开发经验，熟悉多进程/多线程编程，具备系统级编程能力； 2.熟悉TCP/IP网络协议，熟悉一种或多种流媒体网络协议（RTSP、RTP、HLS、RTMP、WebRTC)； 3.至少熟悉主流流媒体/实时通信框架的一种，包括FFmpeg、GStreamer和WebRTC； 4.熟悉图形引擎和渲染机制者优先； 5.积极乐观、自动驱动，学习能力强。
