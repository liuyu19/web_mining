职位描述：

职责描述：
1、参与实时音视频产品服务端核心业务的架构、设计、优化及开发
2、参与实时音视频产品解决方案的设计和开发

任职要求：
1、扎实的计算机专业基本功，熟悉 HTTP、TCP/IP 等常见网络协议，2 年及以上工作经验
2、至少熟练使用一门编程语言（C/C++/Java/Python/Golang/PHP 等）
3、对微服务架构组件有实践经验，关注并理解架构的高可用性与可扩展性
4、熟悉至少一种开源数据库或缓存系统，了解常用 RPC 框架如 GRPC、Thrift 等，了解常用消息中间件如 kafka、ActiveMQ
5、具有优秀的技术钻研能力、技术难点攻关能力，分析问题解决问题的能力
6、有异地多活、跨国服务、音视频相关云服务开发经验的优先
