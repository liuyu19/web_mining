职位描述：

1、3年以上前端工作经验，本科
2、熟练使用es6等进行前端代码开发，有良好的架构，精简代码，并在设计的过程中充分考虑到后台的编程。
3、熟练掌握JavaScript语言，包括核心原理、面向对象设计编程；并掌握常用的基于JavaScript的框架或类库。
4、精通vue(必备：vue/vue-router/vuex)技术栈，参与过多款react或vue产品研发或者react(必备：react/react-router/redux/mobx/immutable)，具备独立开 发整套UI组件经验；熟练使用webpack、gulp等构建工具，熟练使用scss、less等。
5、对WEB组件化开发有丰富的经验，要求写过通用组件并应用到项目/产品。
6、优化产品页面的前端性能，保证产品页面的高质高速，解决不同浏览器及其不同版本之间的兼容性问题。
7、有WEB可视化编辑经验者，以及web性能优化，优先录取。
8、有React-Native开发经验，参与或主导多款产品的，优先录取。
