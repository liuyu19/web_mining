职位描述：

算法工程师/研究员：
1. 1熟悉广告/推荐/搜索任一方向基于机器学习的排序，召回方向的1-5经验。
1.2 具备较好的数据分析能力与数据感觉，较好的特征工程能力，熟悉LR/GBDT/FM/深度学习等常见模型。
1.3 推荐方向具备多目标融合优化能力，熟悉信息流业务场景，用户体验趋向的列表优化或用户活跃周期的规律探索能力。
或 广告方向具备很强的精准化模型优化能力，对吸引用户注意力的策略有很好的思路，对特征工程的规模化与价格收益间有很好的方法论。
或 搜索方向具备对关键词级特征，文档相关性特征及权威度特征有深入理解。对复杂多结构文档的结构化处理能力具备一定能力。
