职位描述：
1.掌握至少一种后端语言(Node.js，Java，Python，Golang等)，对新技术充满激情；
2.熟悉 Git 等版本管理工具进行分支管理、工作流程；
3.分析项目需求，能给出良好的解决方案，主要负责业务相关的web后端开发或应用服务架构设计、维护等工作；
4.与其他团队成员合作完成开发及调试工作；
任职要求：
1.良好的CS修养（操作系统、算法和计算机网络）；
2.熟悉Linux，熟悉常用的Shell命令；
3.熟悉 MySQL、Redis
4.理解 TCP/IP，HTTP，HTTPS，WebSocket 通讯协议；
5.良好的沟通、协调能力，具有团队合作精神，有高度责任感和敬业精神，热爱学习，勇于挑战；
满足以下条件的有加分:
1. 有过服务平台开发经验；
2. 对高并发服务端开发有了解的；
3. 对容器化、Kubernetes 有了解的；
4. 对分布式系统有了解的；

  浙江每日互动网络科技股份有限公司（个推）成立于2010年12月，总部位于杭州，在北京、上海、广州等地均有本地团队，核心成员来自于微软、IBM、卓望、百度、新浪、阿里巴巴等知名公司。
      个推专注推送技术7年，至今已服务数十万APP，覆盖数十亿移动终端，典型客户包括新华社、新浪微博、滴滴出行、中国移动咪咕阅读、网易新闻、芒果TV、招商银行掌上生活等众多有影响力的APP。
