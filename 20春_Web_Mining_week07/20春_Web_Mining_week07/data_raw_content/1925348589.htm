职位描述：

职责描述：
1.熟练使用各类设计软件，按要求完成京东互联网医院相关素材设计工作；
2.按要求规划京东互联网医院视觉体系；
3.按要求完成团队内部的各类广告创意及设计工作，包括不限于：主视觉、创意广告素材（创意海报、H5、落地页、效果类素材等）、延展物料（DM单、易拉宝、框架广告等）的创意及设计。
4.按要求完成运营活动相关的平面及视频设计支持;
5.能够充分理解需求，并有独特见解，能够独立完成设计任务，将需求、创意与个人见解转化作品；





任职要求：
1.本科及以上学历，2年以上PC和移动端界面设计经验，计算机、视觉设计、平面设计、广告设计等相关专业;
2. 具备综合类促销活动或多品类活动的设计经验； 具有移动医疗行业从业经验更佳；
3. 精通Photoshop，熟练使用Illustrator，Dreamweaver等设计软件；
4. 对色彩、布局有自己的独到见解，能够灵活控制设计风格，具备出众的创意构思能力；
5. 思路清晰、有较强沟通力、抗压力强，具备良好的团队工作能力。
