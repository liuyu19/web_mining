工作职责：
岗位职责:

任职要求：
岗位要求: 
本科以上学历，计算机或相关专业，2年及以上相关工作经验；
软件知识结构全面，基础扎实，精通常用数据结构与算法；
精通Web后台开发语言至少一种（PHP、Java、Go等）,有一定的架构能力和良好代码规范；
熟悉linux/unix系统与开发环境，熟悉TCP/IP协议，socket编程，熟悉Mysql以及SQL语言，熟悉Kafka等MQ，熟悉Redis等分布式KV存储系统；
高度的责任心、良好的沟通技巧和团队合作精神，正直进取，有上进心，热爱开发事业；
有高可用、高性能、大容量服务系统设计开发经验者、有toB行业经验者优先。
