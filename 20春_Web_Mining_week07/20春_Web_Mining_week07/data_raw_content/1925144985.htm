职位描述：

岗位职责：
1、 支持业务团队进行售前业务拓展，包括但不限于业务和技术梳理、结合商汤产品、技术、生态优势，为客户定制解决方案，推进产品和服务团队进行技术、业务验证，及项目投标技术工作。关键项目交付过程中跨团队的技术设计、管理和协调工作。
2、 为客户提供业务和技术层面的架构咨询服务，理解合作伙伴及客户的业务和功能性需求、非功能型需求、性能及可用性需求，针对客户的业务需求提供基于商汤计算平台技术体系的架构设计方案。
3、 与产品研发团队合作，传递市场需求，共同研发或完善满足特定行业、特定应用场景的产品。

任职要求：
1、 对互联网与云计算行业有清晰的认知和激情，理解客户的业务需求和痛点；
2、精通基于云计算的应用架构设计方法、应用迁移方法、过程及工具；掌握至少一种主流编程语言，熟练使用至少一种数据库产品；
3、 具备结构化思维习惯，至少熟悉一种架构方法论及框架，如TOGAF, ITSA, DODAF等；
4、 应用SOA方法设计服务化应用，熟悉开源 ESB 或 Dubbo开源中间件技术者优先。
5、 学习能力强，有激情，能同期支持多项目；
6、 具备优秀的文档能力，清晰明了地表达架构意图，能够熟练编写各类技术文档；
7、 具有娴熟的沟通技巧，执行力强，具有优秀的团队合作精神、敬业精神；
8、 具有7年以上项目需求分析、方案架构设计工作经验，具有大型企业应用架构经历，较强的客户需求调研和需求分析
9、 有组织跨专业、跨部门协同实施复杂项目的经验，能有效组织团队、积极主动推动业务拿结果。
10、 学习能力强，有激情，结果导向，能同期支持多项目。
11、 具备优秀的文档能力，使用文字、图示清楚地表达架构意图，能够熟练编写各类技术文档；
12、 具有娴熟的沟通技巧，执行力强，具有优秀的团队合作精神、敬业精神
13、 英文能力出色者优先
