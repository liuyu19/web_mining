职位描述：

岗位职责:
1.主要负责公司运营部门相关设计工作。

2. 负责制作电商平台各种大型促销活动专题页面，宝贝描述，商品图片的变化，banner 的设计。

2. 负责新开发的产品进行设计排版，结合商品特性制作图文并茂，有美感，有吸引购买力的详情页页面。

3. 负责商城商品上下架图片的美化设计工作。

4. 负责其他的美术设计和创意工作。

5. 有手绘功底或插画功底的优先考虑。
任职资格:
1. 熟练实用PS、AI、PR、DW等平面设计软件，并有一定的创意能力。

2. 具有较强的审美能力、设计能力，良好的视觉表达能力，能独立完成页面设计制作工作

3. 精通版式设计，对色彩、构图等元素触觉敏感，注重细节设计，设计风格敢去大胆尝试新鲜有视觉冲击里的效果。

4. 具备良好的书面和口头表达能力，对广告创意有较好把握和潮流意识。
