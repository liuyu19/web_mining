职位描述：

职位描述：
1.负责公司平台系统的前端界面设计开发，web端架构设计与实现，小程序前端开发。
2.根据产品需求，完成产品界面设计和逻辑交互功能，做好页面维护及性能优化。
3.按时保质完成开发任务，及时发现并解决开发过程中出现的问题。

岗位要求：
1.能独立完成项目前端设计与开发；
2.精通Web前端技术（HTML/CSS/JavaScript），熟练使用HTML5、CSS3完成web端页面设计与开发，能够很好的解决浏览器的兼容问题及跨平台兼容问题；
3.对原生JavaScript有深入理解，有原生JS封装前端插件经验者优先；
4.熟练掌握各种常用JS框架和Library，如JQuery, Bootstrap, Vue, eChart, 有实际项目开发经验；
5.熟悉 react.js，Node.js, react native优先；
6.熟练使用 webpack等前端自动化构建工具；
7.保持积极的工作态度，具备良好的逻辑分析能力、分析问题和解决问题的能力，良好的沟通能力和团队协作能力；
