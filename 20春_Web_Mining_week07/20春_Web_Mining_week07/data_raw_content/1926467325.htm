职位描述：

工作职责：
一、java 职位要求 1. 一年以上开发及设计经验，Java基础扎实，理解IO、多线程、集合等基础框架，对JVM原理有一定的了解；熟悉面向对象设计开发; 2. 熟悉开源框架(Spring/SpringMVC/iBatis)，并且对其核心思想、实现原理有一定认知； 3. 对主流关系型数据库中的一种有较丰富的使用经验； 4. 熟悉分布式系统的设计和应用，熟悉分布式、缓存、消息等机制；能对分布式常用技术进行合理应用，解决问题； 5. 有高并发、大数据量业务处理的实践经验者优先。 职位描述 1. 负责支付技术研发。基于蚂蚁金服各种领先的产品和技术，开发支付关键链路系统，负责系统的架构设计、编码开发、性能优化、稳定性优化、大促保障等。 2. 深度参与核心功能的设计、开发、测试、上线全过程，以及线上数据分析、功能迭代优化、疑难问题解决过程。 3. 理解和掌握蚂蚁金服的常用架构设计、性能优化、高可用保障理念，并能灵活运用于核心链路系统的优化。 4. 参与双十一、双十二、春节红包的大促准备，包括性能优化、链路压测、可用性优化、大促保障。 5. 跟踪前沿技术和理念，用创新的思路方法解决业务快速增长所带来的技术挑战。通过技术手段驱动业务变化，深挖业务潜力，扩展业务边界。 二、算法 职位描述 1、设计和研发支付、营销、运营等相关业务的算法，满足智能营销、异常检测、波动分析等场景需求； 2、根据业务需求，参与从业务分析、特征分析、模型建立、算法实现等过程，并逐步迭代算法效果； 3、对已有算法与服务进行性能优化，提升算法效能，扩大算法应用场景。 职位要求 1、在机器学习、统计分析、大数据处理等领域有一定的功底，2年以上相关工作经验； 2、优秀的算法应用能力，包括特征提取、模型建立、效果评估、应用部署以及优化迭代等环节； 3、优秀的编程能力，至少熟悉一门常用语言及相关的算法库，如java、python、R等； 4、对时间序列建模和营销场景推荐、效果预估、效果分析等领域有一定的经验优先； 5、熟练使用spark、hadoop、hive等分布式计算优先； 6、计算机、数学或统计学等相关专业优先。热衷于创新技术，适应一定的工作压力； 7、较强的业务敏感度，数据敏感度的优先。 三、测试开发 职位描述 1. 参与支付宝软件产品测试的全流程，包括参与需求分析、设计评审，制定测试计划，设计和执行测试用例，进行缺陷跟踪和软件质量分析等； 2. 执行项目测试，包括制定测试计划，分配测试资源，构建测试环境，执行集成测试，回归测试等； 3. 保证被测系统的质量，并通过测试流程和方法创新，努力提升研发的质量和效率。 职位要求 1. 工科、计算机或其他相关专业本科以上学历； 2. 熟悉C/C++/Java等至少一种编程语言，有Shell或Ruby/PHP/Perl/Python等使用经验者优先； 3. 至少3年以上软件测试开发工作经验； 4. 有性能、安全、白盒测试等专业测试领域经验者优先； 5. 具备Android/IOS无线app产品、互联网广告、搜索、大数据处理、分布式系统、数据库和网络等业务领域测试经验者优先； 6. 熟悉Linux或Unix操作系统; 7. 精通测试流程和测试用例设计方法,能主动进行技术钻研； 8. 解决复杂问题和编写自动测试工具和系统的能力； 9. 很强的逻辑思维能力，谈判的能力和冲突管理的能力； 10. 善于团队合作，理解和适应变化，以结果和行动为准则，努力追求成功。 四、数据分析 职位描述 1. 通过机器学习等技术进行数据分析及建模的方案的系统化设计，负责数据挖掘项目的架构设计工作 2. 分析用户行为，对用户进行建模，设计合理的模型和算法 3. 设计营销算法，ab实验，以数据为驱动的实验调优 4. 对现有系统的架构进行深入分析及优化，进一步提升系统数据和算法能力，构建智能营销平台 职位要求 1. 计算机、数学、统计学、运筹学、机器学习等相关专业，具有扎实的统计和数据挖掘、建模基础； 2. 熟练编程，具有丰富的独立实现算法和调优的经验，熟悉python等脚本语言优先 3. 具备较丰富的基于Hadoop ／ mapreduce ／ Spark等大数据处理项目经验优先 4. 熟悉常用算法，熟悉至少一种常用算法框架（Caffe, sklearn, tensorflow） 5. 对数据敏感，具有良好的逻辑思维能力、理解业务的能力、沟通能力和表达呈现能力，主动性强 6. 在某一算法领域比如推荐、图像、NLP等具备很强的专业技能者优先 五、客户端开发 职位描述 参与支付宝各业务的开发和架构升级，保障数亿笔支付资金的安全流转，服务直面海量用户，开发在亿级设备上运行的代码。根据产品需求开发相关的移动产品，验证和修正测试中发现的问题。 岗位要求 1、1年及以上iOS或Android客户端开发经验；具备良好的编程风格，扎实的编程和数据结构算法基础；深入理解面向对象编程思想，具有较强的设计能力； 2、有独立设计iOS或Android客户端架构经验，熟悉iOS/Android SDK以及相关配套工具，基本功扎实，可以应付各种iOS/Android复杂问题，并迅速给出解决方案； 3、熟练掌握iOS或Android平台下的高性能编程及性能调优，有大规模应用开发经验者优先； 4、具有较强的分析和解决问题的能力；热爱互联网新技术，了解并探索业界技术动向； 5、有强烈的责任心和团队精神，善于沟通和合作。 六、安全 职位职责： 1、负责支付宝业务的安全代码审计、渗透测试和漏洞挖掘等工作，根据安全风险提供安全建议和解决方案； 2、应急响应事件处理。 3、安全攻防技术研究。 职位要求： 1、熟悉WEB安全渗透和漏洞挖掘（黑盒和白盒），熟悉漏洞的内在原理、检测方法、利用手段以及对应解决方案。 2、熟悉JAVA，熟悉Spring框架和使用。 七、前端 职位描述 1. 参与多种平台的应用开发，包括 web 及 native 等产品，负责前端框架和基础组件。 2. 充分理解项目和设计需求，高效完成业务需求。 3. 跟进社区发展，组织团队成员学习研究新技术，分享经验，并转化到潜在项目以及帮助团队成长。 职位要求 1. 前端基础方面：熟悉 ecmascript/css/html 最新规范 2. 应用开发方面：熟练掌握 react/react-native 以及配套社区项目(redux/rxjs/…)，了解社区动向（reactconf 2017/fiber/…），了解 ios 或 android 语言以及对应平台 UI 框架基础。 3. 前端架构方面： 了解 jquery/vue/angular/react 等常用前端类库／框架的设计原理。 4. 工具辅助方面：熟悉基于 node 的工具链体系(gulp/webpack/…).
