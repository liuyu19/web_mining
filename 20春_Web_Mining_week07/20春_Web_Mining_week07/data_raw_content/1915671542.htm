职位描述：

【岗位职责】

1、根据游戏界面进行场景，自然场景，部件，道具等资源的建模及贴图绘制，使界面3D化，增强沉浸感，提升用户体验；

2、熟悉简模3D场景和物件的制作，对模型面数有很好的控制。

3、协助UI设计师创作界面风格尝试和概念设定。

4、负责Unity/NeoX引擎下地图的编辑制作及物件摆放、灯光及气氛营造的工作；

5、根据游戏要求提出相应的界面内的场景优化方案，配合进行优化与渲染效果实验。



【岗位要求】

1、2年以上游戏美术3D项目界面或场景制作经验，对3D界面有兴趣；

2、有一定3D场景设计经验，有较强的建筑结构理解能力，对画面整体构图，地形表现及模型物件搭配有较强的审美控制能力；

3、有完整的3D项目经验，熟悉3D游戏制作流程，熟练掌握Unity3D，3D MAX，BodyPaint，法线贴图等的使用，熟练掌握地形编辑、物件管理、灯光、场景烘培等制作流程；

4、有游戏引擎关卡地图编辑、灯光、场景烘培经验者，优先考虑；
