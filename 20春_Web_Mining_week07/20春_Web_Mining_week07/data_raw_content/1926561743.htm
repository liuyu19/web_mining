职位描述：

1.    负责小米互联网广告（应用商店、信息流等各商业产品）的销售，维护头部客户、拓展新客户，带领团队完成业绩目标；
2.    监控、分析、反馈已售卖广告产品的各维度数据，数据分析能力强，给予优化建议；
3.    协同运营团队、产品团队等，解决客户投放问题并输出结论，合理平衡用户体验与销售策略，有风险意识；
4.    了解互联网金融行业发展动态，了解竞媒广告平台情况，熟悉媒体市场，代理市场，根据市场情况调整营销策略，把握商机；为平台决策提供建设性意见
5.    有销售管理经验、熟悉金融行业客户、有运营管理经验者优先。
