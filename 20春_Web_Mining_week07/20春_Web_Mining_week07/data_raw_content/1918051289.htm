职位描述：

1、 负责产品前端架构体系的设计、建设，包括PC端和移动端，通过技术提升用户体验和可用性；

2、 与产品、设计、后端、测试团队紧密配合，能快速理解、消化各方需求，并落实为具体的开发工作；

3、 负责已有产品线前端迭代开发重构，以及性能优化

4、通过文档、分享等各种形式，帮助团队成员提高技能

任职要求

1、大学本科学历以上，有2年以上web前端工作经验；

2、有较好ES6基础和扎实的JavaScript基础，对前后端分离开发、Web语义化等有深刻理解；

3、 熟练使用Vue、Angular中一种或多种相关技术栈；熟悉W3C标准，熟悉HTML5；熟悉Node开发环境，至少能熟练使用至少Gulp、Webpack、Grunt等构建工具中的一种；

4、至少会一种以上CSS预处理语言，如LESS、SASS、STYLUS等；

5、良好的编码习惯，较强的沟通、协调、理解能力和良好的责任心和团队合作能力，能够带领团队，并能对其他开发人员进行技术指导。
