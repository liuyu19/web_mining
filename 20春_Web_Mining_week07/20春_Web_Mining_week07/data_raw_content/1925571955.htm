职位描述：

职责描述：
1、负责QQ音乐直播相关产品的体验框架搭建和原型设计，输出优秀的交互方案；
2、参与用户场景的分析研究，明确体验设计目标，打磨设计；
3、主动发现体验问题，梳理分析并提出优化，并驱动落地；
4、协同相关角色，参与团队的专业化建设和知识传播。
任职要求：
1、三年以上互联网行业产品的交互设计经验，并有成功案例；
2、对用户需求有敏锐的洞察力，对互联网前沿领域有浓厚的钻研兴趣，保持好奇心；
3、有较好的逻辑思维能力，熟练运用交互工具和方法，对设计有极高的追求，关注细节；
4、善于设计沟通和团队协作，富有激情，有较强的责任感、高执行力；
5、能够承受高强度的工作压力，有良好的组织能力，较强的学习能力；
6、本科及以上学历，工业设计、计算机、心理学、人机交互等专业优先，擅长交互动效者优先。

腾讯音乐娱乐集团（纽约证券交易所代码：TME）是中国在线音乐娱乐服务领航者。拥有中国广受喜爱和独具匠心的音乐平台：QQ音乐、酷狗音乐、酷我音乐和全民K歌。腾讯音乐致力于创造音乐无限可能，用科技赋能音乐，人人都可以尽情创作、享受、分享音乐和彼此互动。腾讯音乐提供在线音乐、在线K歌和音乐直播服务，让每一位音乐爱好者能够发现好音乐、获得听、看、唱、表演和社交为一体的全方位音乐娱乐体验。
