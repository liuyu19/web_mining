职位描述：

工作职责：
1. 深入理解服务管理领域的模式和流程，发现潜在的服务运营风险点，和产品技术、业务运营高效沟通，设计合理的服务安全运营策略架构 2. 充分理解业务特性和场景设计，独立思考某个业务环节或场景可能潜在的风险，设计合理的数据埋点体系，通过数据分析，定量识别潜在的服务运营风险和业务影响 3. 与BI团队紧密配合，通过大数据挖掘，找到行为违规类的特点，快速形成有效的打击策略，持续迭代优化某个业务或场景的服务安全运营效果 4. 针对某个业务或场景建立合理的指标体系，过程中不断完善监控体系，对服务链路中关键控制环节的数据进行分析提炼，与BI配合形成可视化的监控系统快速发现业务风险并形成风险预警指标
任职要求：
1. 本科及以上学历，七年以上工作经验 2. 对业务和数据敏感，能快速发现问题和解决问题，利用经验、统计分析或机器学习等手段，从风险案例或风险数据中理解和挖掘出风险手法和风险特征，并归纳出有效的风险变量 3. 熟悉服务行业相关风险控制指标体系，具有较强的风险分析与指标设计能力以及风险识别、评估、监控、优化等方面的能力 4. 思维开阔，有良好的逆向思维、逻辑思维和结构化思维。有自驱力，能主动思考和学习，勇于接受挑战和承压 5. 良好的团队管理的能力
