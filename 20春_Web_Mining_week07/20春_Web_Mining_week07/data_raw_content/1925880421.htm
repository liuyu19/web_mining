职位描述：

职责描述：
1、根据产品需求文档、UX原型和UI设计稿完成静态页面输出。
2、负责移动端和PC端H5应用开发，编写可复用的前端组件。
3、重构前端界面，实现前端业务逻辑和各类交互效果。
4、优化前端体验和页面响应效果，保持在各浏览器下良好的兼容性。
5、参与前端前沿技术预研，基于产品和技术进行创新，保持产品的技术先进性，持续提升产品用户体验。

岗位要求：
1、本科及以上学历，2年及以上WEB前端开发经验，拥有良好的编码习惯，注重代码质量与代码整洁。
2、熟悉W3C标准以及WEB语义化开发，熟悉ES6规范及模块化开发，熟练掌握html/html5、css/css3、javascript/es6+、vueJS、angularJS、nodeJS 、webpack等开发技术。
3、能够熟练切图，熟练手写DIV/CSS输出静态页面模板，熟练使用ElementUI、Bootstrap、 EasyUI等主流框架。
4、熟悉响应式开发流程，能够解决不同浏览器及版本之间的兼容性问题。
5、有微信小程序、支付宝小程序开发经验者优先。

企业介绍：

微盟集团（Weimob Inc.）， 香港主板上市企业（股票代码：2013.HK），成立于2013年4月，现有员工超3200人，渠道代理商超1600家，注册商户超300万。
微盟是中国领军的中小企业云端商业及营销解决方案提供商，同时也是中国领军的腾讯社交网络服务平台中小企业精准营销服务提供商[1]。微盟围绕商业云、营销云、销售云打造智慧云端生态体系，通过去中心化的智慧商业解決方案赋能中小企业实现数字化转型。目前旗下拥有:微商城、智慧零售、智慧餐厅、客来店、智慧酒店、智慧休娱、智慧美业、销售推、微站、广告助手等解决方案，帮助客户在新零售时代提高运营效率和盈利能力。微盟精准营销业务以大数据、智能算法、营销自动化等技术及优质媒体源，为广告主提供一站式精准营销投放服务。同时微盟旗下微盟云平台通过开放微盟核心产品技术能力，吸引第三方开发者，打造云端生态体系，为商户提供更多应用选择和更好服务。
微盟致力于通过产品和服务，助力企业向数字化转型升级，通过科技驱动商业革新，让商业变得更智慧!
