职位描述：

1. 熟悉从市场调研、需求分析、产品定义、研发实现、生产验证到产品发布运维的整个产品管理流程；

2. 收集市场用户及工程现场反馈的需求及问题，提升产品端到端体验；

3. 牵头撰写详细的产品需求文档及研发设计文档，跟踪产品研发进展；

4. 负责新产品的创新和预研，完成新产品的需求定义、产品功能性能及UI/交互设计；

5. 牵头制定产品研发计划及里程碑，确保项目按时完成；负责把控产品研发推进进度，协调并解决研发过程中的突发问题；

6. 牵头产品的生产验证，协调各方解决生产中出现的各种问题；

7. 负责产品的发布，包括整个生命周期的维护及路标。



职位要求：

1. 本科及以上学历，计算机、电子、通信、自动化、管理或相关专业毕业；

2. 具备两年以上产品经理从业经验；

3. 具备产品研发和产品市场营销的背景和实际经验；

4. 有智能安防或物联网或人工智能行业工作经验者优先；

5. 具备PMP项目管理认证资格优先；

6. 熟悉敏捷agile管理流程、方式和实践者优先；

7. 热爱技术，对相关领域前沿技术感兴趣，具备较强的责任心和沟通能力。

企业介绍：

公司网址：www.tslsmart.com
特斯联科技有限公司是由中国光大旗下基金与IDG资本联合投资的城市级移动物联网生态平台公司，领跑于以物联网为技术基础的“智慧城市”新赛道，为“未来”系列多场景生活、办公和城市管理提供解决方案。目前，特斯联凭借智能硬件、云服务和移动应用的技术核心，重点发展三大业务单元：ABAS 超级楼控，智慧城市、智慧通行解决方案；同时，成立特斯联金融板块，全方位支持纵向、横向的业务板块。
成立至今，公司已形成专业硬、软件技术团队，百余人的技术专家和强大的研发设计能力；拥有国际、国内30多项发明专利。截止目前，特斯联各系列解决方案已成功落地北京、上海、重庆、深圳、武汉、广州、杭州、新疆、云南、贵州、山东等地，覆盖人口达千万量级。
特斯联“智能通行系统”是公司目前着力打造的明星产品：“未来”门锁和门禁系列获得公安部CCC智能硬件认证，市场占有率居前。
公司在北京、重庆、上海、武汉、广州设有办公地点，研发中心分别设于北京和重庆。
