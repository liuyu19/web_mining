职位描述：

职位描述
1.参与公司品牌IP形象（动物）的3D模型开发，一起完成对VI中品牌IP形象的设定
2.与视觉设计师一起落实电商活动设计促成海报设计
3.与设计师一起合作产出日常的KV/Social端传播动图/H5视觉内容
4.配合活动网红/快闪店的设计产出
岗位要求
1.传统艺术类毕业，3D专业或者动画专业，具有良好的视觉美感，设计年限__年以上
2.精通Cinema4d软件及Octane或Arnold/Redshift渲染器，具有动画基础，会AE更佳
3有独立完成建模渲染出图的能力，有电商视觉经验，及表现产品的能力
4.熟练掌握对产品分析能力，知晓运用符合产品的视觉元素
5.熟悉实现方法，高效发现和解决问题的能力
5.在视觉方面有这自己开发挖掘技能的能力
7.有良好的沟通能力及思考能力
8.拥有/了解汽车相关的经验更佳
