职位描述：

工作职责：
1. 负责钉钉事业部各产品的用户体验设计，能够通过自己的洞见与创新，对UX过程方法进行系统化的创新和突破；  2. 具有较强的全局观，了解并参与业务前期产品规划阶段，设计方案或提案体现深度的体验驱动意识，能通过对用户体验的深入挖掘发现新的体验突破点，不仅能够深度满足业务需要，还能为业务带来启发和增值，并更多的影响到企业上下游； 3. 积极主动的向对方提供业务、技术等各类资源信息，针对对方的优劣势提供指导，在传授业务知识、技能的同时切实帮助他人发展； 4. 快速处理，快速落实，多项任务管理、多个项目的有效落实，及时应对突发事件；
任职要求：
1. 拥有大型互联网公司3年以上工作经验，有丰富无线设计经验；   2. 对各种互联网产品有强烈兴趣并有设计触觉，富有设计创造力和激情；   3. 具备良好合作态度及团队精神，并富有工作激情、创新欲望和责任感，能承受高强度的工作压力； 4. 自我学习、自我驱动、自我管理：保持高度竞争性，主动寻求自我提升与发展，超越自我。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
