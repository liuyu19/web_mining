职位描述：

职责描述：
1).独立或主持完成多个客户端模块/组件需求分析、架构设计、核心框架及组件编码等工作；
2).负责客户端在性能优化、稳定性方面的建设,持续从技术上提升性能、稳定性;
3).负责客户端重点、难点的技术攻坚以及技术预研，提升研发效能;

任职要求：
1.本科及以上学历，计算机相关专业，熟悉常见数据结构和算法
2.熟悉Android应用开发框架和特性，有插件化项目经验/跨平台方案优先；
3.熟悉安卓ui框架、线程、进程特性，熟悉安卓性能指标、监控工具及调优方法；
4.具备良好的分析解决问题能力及攻坚能力，能独立承担任务；
