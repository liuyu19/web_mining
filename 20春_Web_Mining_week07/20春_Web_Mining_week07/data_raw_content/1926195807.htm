职位描述：

职位描述：
1. 整体负责公司IT基础架构设计与优化，保障服务的稳定性和可用性
2. 负责和设计自动化运维平台，并协调相关人员推进项目落地
3. 负责完善、优化公司运维体系、流程、制度和规范
4. 研究分布式系统运维技术，负责和参与服务部署设计，提高产品的可靠性和可运维性
5. 处理业务部门以及本部门提出的技术问题，提供技术支持；并根据当前业务现状，制定技术发展方向
6. 主导处理技术疑难问题、重大故障、通过有效的解决方案不断改进提升运维能力和自动化水平
职位要求：
1. 6年以上的系统及网络平台运维经验，具备丰富的大规模服务集群的运维经验，高并发架构经验
2. 精通Linux原理、系统、应用知识，深入了解互联网企业主流和常用的开源系统（如LVS，BIND、zabbix、redis、nginx/tenginx、MFS等）的架构、配置、优化，并有丰富的方案策划、部署、实施和应用经验。
3. 精通Python，Shell，Perl，Ruby等至少一种脚本语言，能熟练使用来完成日常系统运维等工作
4. 熟悉负载均衡，集群架构及其工作原理
5. 有IDC、虚拟化、网络平台、linux等运维经验，具备产品管理能力
6. 深入了解网络及系统的应用技术，熟悉主流业务平台的部署及使用
7. 具有大规模运维平台经验者优先考虑
