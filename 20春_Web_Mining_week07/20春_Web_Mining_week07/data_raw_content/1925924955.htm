职位描述：

职责描述：
1、对app购物体验负责，了解电商用户特性与场景、数据及业务，把控整体购物流程的体验节点，建立健康的用户生命周期，提升用户对京东的信赖与依赖

2、深入了解用研，对体验问题的解决、体验规范的建立有一套自己的方法论

3、善于跨部门合作，快速找到多方收益点，并主导与持续推进方案落地

4、围绕京东平台导购与交易的特性，构建营销玩法与互动，提升存量用户的活跃度

岗位要求：

1、有计算机、心理学、经济学、市场营销等相关知识背景，5年以上移动互联网工作经验，从事过导购交易类、营销类相关工作者优先

2、善于洞察人性本质，懂得抓住用户心理，有C端产品经验。拥有成功和失败的产品案例经验者优先

3、对用户体验管理理念有实操案例和深刻理解，有DAU百万级APP的用户体验管理经历者优先

4、数据驱动，能够根据行业趋势、用户体验与行为细节输出每段增长曲线

5、愿意大胆尝试，擅于团队合作，乐观开朗，有正能量，谦虚好学，强烈的使命感、责任感与自我驱动，抗压能力强

企业介绍：

京东于2004年正式涉足电商领域，2017年，京东集团市场交易额接近1.3万亿元*。2018年7月，京东第三次入榜《财富》全球500强，位列第181位，在全球仅次于亚马逊和Alphabet。

2014年5月，京东集团在美国纳斯达克证券交易所正式挂牌上市。2015年7月，京东凭借高成长性入选纳斯达克100指数和纳斯达克100平均加权指数。

京东商城致力于打造一站式综合购物平台，通过组建大快消、电子文娱、时尚生活三大事业群，服务中国亿万家庭，并以高于行业平均增长速度的发展态势满足消费者日益多元的消费需求。京东商城已完成全品类覆盖，消费品、3C、家电等优势品类年交易额突破千亿大关，未来京东还将培育出生鲜、文旅、时尚、大客户、新通路、拍卖、大汽车、家居家装、大健康等近20个“千亿级品类”。
