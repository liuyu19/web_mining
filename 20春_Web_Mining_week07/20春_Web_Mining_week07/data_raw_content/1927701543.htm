职位描述：

职位描述：
1、负责安卓智能产品的市场调研、竞品分析、产品策划、产品生命周期管理；
2、设计与规划产品功能、业务流程，编写相关产品文档；
3、负责客户端产品交互界面和流程原型设计，熟练使用各种原型制作工具；
4、跟踪产品开发进度，完成产品的开发测试、版本管理、评审发布、上线等相关工作；
5、跟踪产品运营，收集整理客户反馈，持续改善优化产品，提升用户体验。

职位要求：
1、全日制本科及以上学历，有智能设备工作经验；
2、有从事三方支付或支付行业相关5年以上工作经验优先；
3、有收单支付智能类产品设计经验、有相关支付通道对接经验者优先；
4、具备产品分析能力、规划能力、较好的逻辑思维能力；
5、具备较好的沟通、协调能力、较好的抗压能力，能有效组织和推进跨部门项目；
6、能够独立工作、责任心强、工作积极；
7、了解支付行业发展动态及竞争对手的最新动态。
