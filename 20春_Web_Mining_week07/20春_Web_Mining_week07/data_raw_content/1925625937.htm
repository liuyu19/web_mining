职位描述：

职责描述：
1.负责进行产品详情页整体视觉展示设计，能够融合品牌调性，产品详情页转化率；
2. 监控竞品详情展示视觉相关动作，梳理适合的说服逻辑，积极吸收优秀创意和素材，形成共享的素材库；
3. 配合运营节奏，进行产品页面维护及优化；
4.承接产品线上活动页面和海报等设计，产品线下展会物料设计；
5.负责产品的客制化相关设计；
6.为产品企划及营销阶段提供视觉支持。
任职要求：
1.全日制统招本科及以上学历
2.平面设计、电子商务、广告设计等相关专业优先
3.2年以上平面设计工作经验
4.能熟练使用Photoshop等设计软件；
5.对色彩把握敏锐，具有把握不同风格的平面设计的能力；
6.具有良好的沟通协调能力。
