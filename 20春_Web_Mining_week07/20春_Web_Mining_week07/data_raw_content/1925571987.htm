职位描述：

职责描述：
1、负责 QQ音乐全平台终端Web UI / Hybird 设计开发，结合业务目标，保障设计理念还原和效果实现 ;
2、负责制定符合业务需求的前端研发及产品设计方案落地，推动公共UI组件化建设，提升前端页面的良好用户体验创造更高项目价值；
3、关注前端技术和设计的趋势发展和应用，了解泛专业相关互联网资讯，参与创新项目的设计研发。
任职要求：
1、3年以上UI开发/Web前端开发经验，有大型站点前端开发及优化实践经验；
2、熟练掌握 HTML5、CSS3 语言，遵循主流编码规范，熟悉UI组件开发；
3、具备移动端Web / Hybird App 设计研发经验，掌握多平台终端设备适配技巧，掌握移动站点性能体验优化方法；
4、熟练掌握 JavaScript / ES6+，精通一种VueJS, ReactJS 等 MVC/MVVM框架，掌握 Webpack 等自动化构建工具，有实际业务流程优化经验；
5、了解Web动画效果设计研发原理和方案分析，具备方案评估和应用决策能力，有驱动创意动效落地案例者优先；
6、良好的学习、沟通和执行能力，富有工作激情和责任感，能承受高强度的工作压力。
7、对用户体验有深刻的理解，具备良好的审美，关注细节追求体验极致；
8、本科及以上学历，计算机相关专业背景。
