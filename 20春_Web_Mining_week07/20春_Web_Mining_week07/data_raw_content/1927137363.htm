职位描述：

工作职责：
1、能够结合传统调研方法、大数据分析及各种创新方法，洞察淘系核心业务发展趋势及市场机会点，输出可执行的策略建议，支持猫超业务发展； 2、深入挖掘用户痛点及潜在诉求，为猫超业务突破提供消费者端的有效输入和支持； 3、整合资源，推动和实现研究成果的落地应用。
任职要求：
1. 5年以上市场研究、消费者洞察经验，服务和研究领域为：快速消费品、零售； 2. 项目类型：定性专长或定量专长均可，以目标消费者洞察、行业及品类增长为主，重点考虑同时有B端和C端研究经验的； 3. 目标公司： 第一优先级知名的零售（如沃尔玛）、日化、快消、消费电子类企业用研人员；【研究洞察团队】——优选有研究公司背景的，能hands on做项目的。不要只管理agency的，要有丰富的零售研究经验，不要只做产品研发、品牌定位、沟通研究的 第二优先级互联网公司【用户研究团队和平台策略团队】（美团、滴滴、爱奇艺等） 第三优先级市场研究/咨询公司专项研究人员（互联网、FMCG、CPG方向，非纯零研背景）；研究经历-高级研究经理级别；（如ipsos、tns、Nelson非零研团队、其他国内知名调研机构等） 4. 能够独立handle完整项目， 咨询公司必须有配合企业进行项目成功落地的经验。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
