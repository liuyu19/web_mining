职位描述：

职责描述：
1、负责华为电商渠道（国内、国外电商）的视觉风格，保证产品在多渠道的视觉一致性，包含并不限于华为产品详情页、产品创意拍摄、产品视频、活动专题页、Banner、H5等；      
2、主导并输出与视觉设计相匹配并且促进销售的文案；      
3、带领设计团队，并能与周边部门配合，以结果为导向，促进电商销售

任职要求：
1、5年及以上互联网、广告公司、硬件公司视觉设计经验，项目涉及领域广泛有主导型的成功案例，有电商视觉设计者优先
2、具备扎实视觉表达能力，能输出设计方法，并指导团队执行
3、有摄影和视频能力，能使用Cinema4D/After Effects/3DMax/Maya等软件；
4、具备团队管理经验；
5、大学本科设计院校、设计专业毕业，英语可作为工作语言
