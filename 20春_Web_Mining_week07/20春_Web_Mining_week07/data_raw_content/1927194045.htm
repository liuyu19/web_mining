职位描述：

岗位职责:
 
1.      负责手机系统及相关产品(如手机应用)的用户体验研究。
 
2.      结合业务发展现状，通过研究为产品交互设计与策略制定提供迭代建议。解决业务问题，帮助业务不断优化。
 
3.      通过研究，分析用户群体的行为或态度、认知与体验感受。
 
4.      参与建立科学的用户体验指标，通过跟踪、监测和反馈,提升产品用户体验和质量。
 
5.      参与推动用研意识及用研方法的内部普及。
 
职位要求:
 
1.      心理学、社会学、工业设计、人机交互、人因工程等相关专业本科及以上学历，专业基本功扎实。
 
2.      具备独立完成研究项目的能力，熟练掌握定量与定性研究方法，如问卷调查、用户访谈、焦点座谈会、田野调查、可用性测试等研究方法。
 
3.      独立完成数据分析、报告撰写及项目成果汇报并配合业务部门完成解读数据工作。
 
4.      掌握一定的数据分析工具，如spss等，能够进行复杂数据处理和分析。
 
5.      拥有智能硬件相关经验，互联网相关工作背景。
 
6.      海外留学背景者更佳。
