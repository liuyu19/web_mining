职位描述：

任职职责：
1.参与公司项目线下品牌展会/互动体验/品牌商业店、项目策划与创意设想；
2.根据空间设计项目的要求，完成项目的空间布局、模型制作、效果图呈现；
3.对空间设计项目的施工工艺、材质要有足够的了解，并能独立绘制施工图；
4.有项目施工的监理经验，能够把握进度和成本预算；
5.协助部门负责人跟进重要店的设计，完成线下店设计及后期跟进维护工作；
6.负责店铺空间设计标准制定，并形成品牌SI手册；
7.负责SI手册完善，通过对道具、材质的优化，有效控制装修成本；
8.制定产品陈列规范，诠释品牌个性。

岗位要求：
1.本科及以上学历，土木工程、艺术设计、环艺、相关专业毕业；
2.具备五年及以上相关设计经验,资深者优先考虑；
3.有较强的空间理念和造型能力,创意性思维及表达能力强,熟悉各种材料；
4.熟悉使用3DMAX，CAD，illustrator, photoshop等软件；
5.有平面视觉设计能力优先。
请附近期相关作品，或在简历中提供个人作品有效链接！（重要）
