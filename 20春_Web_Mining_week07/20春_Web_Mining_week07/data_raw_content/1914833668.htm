职位描述：

工作职责：
1、负责结合运营的需求，进行首页框架设计、平台大活动设计等；
2，负责在充分考虑用户体验、考虑货品的基础上，进行钻展推广设计、二级专题页面设计、详情页优化等设计工作；
3，负责延续品牌的风格，维护品牌差异性，提升品牌的认知度；
任职要求：
1、大专以上，有2年左右的平面设计经验；
2、艺术院校相关专业毕业优先，手绘能力佳优先；
3，有较强的品牌意识，较强的团队合作意识；
4、思维活跃，善于创新，创造力强；
5、热爱服装，时尚敏感度高；
6，精通Photoshop、AI、Dreamweaver 等软件；

企业介绍：

我们是一家位于中国的领先设计师品牌时尚集团。我们设计、推广及出售时尚服装、鞋类、配饰及家居类产品。我们的品牌组合包括三个阶段的九个品牌，即成熟品牌JNBY，成长品牌（i）CROQUIS（速写）、（ii）jnby by JNBY、（iii）less，初生品牌（i）Pomme de terre（蓬马）、（ii）JNBYHOME、（iii）SAMO、（iv）REVERB 、（v）LASU MIN SOLA及（vi）A PERSONAL NOTE 73，各个品牌均各自面向特定细分客户并拥有基于本集团统一品牌理念「自然而然地做自己」（Just Naturally Be Yourself）的独特设计形象。
我们的产品面向中高层收入客户，该等客户透过时尚服饰彰显个性。我们广泛的品牌组合创建了一种使我们可以满足客户不同生活阶段需求的生活方式生态圈，使得我们打造了庞大、多元化及忠实的客户群。我们于一九九四年透过销售女士服装开展业务。根据灼识咨询进行的调查，我们的成熟品牌JNBY在中国被认为是最独特且最易识别的女士服装设计师品牌，按重复购买的客户数计，在中国十大女士服装设计师品牌中享有最高品牌忠诚度。我们于二零零五年至二零一一年期间扩充我们的品牌组合至CROQUIS（速写）、jnby by JNBY及less，于二零一六年至二零一八年期间，我们进一步推出青少年设计师品牌Pomme de terre（蓬马）、设计师家居品牌JNBYHOME，针对先锋职业男士的设计师服饰品牌SAMO、全新时尚环保品牌REVERB、设计师品牌集合店LASU MIN SOLA及全新的男装设计师服饰品牌A PERSONAL NOTE 73，以令我们的品牌组合更加多元化和细分化，并使我们可为大多数年龄阶层的消费者提供服务。考虑到我们粉丝的购买模式及资讯需求，我们已建立主要由实体零售店、线上平台及以微信为主的社交媒体互动营销服务平台三个部分组成的全渠道互动平台，各部分在我们将潜在粉丝变成忠实粉丝的过程中均起到关键作用，我们旨在打造一套由于其追求我们致力提倡的生活方式而购买的粉丝群体「江南布衣粉丝经济」体系。
