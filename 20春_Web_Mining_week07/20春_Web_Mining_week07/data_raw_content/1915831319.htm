职位描述：

职责描述：
1. 配合美术设计制作场景中所需的虚拟角色、物体、特效、动作。按制作规范管理美术资源。
2. 可以根据原画需求完成模型及贴图（法线与颜色等）的制作，能高度还原原画，拥有完整的手绘贴图能力
3. 熟练使用3DMAX，zb，photoshop，Substance Painter，BD等
4. 根据需求，使用maya或3dmax等软件完成项目中模型需要的绑定及动画k帧工作；
5. 根据需求，使用maya或3dmax等软件制作或优化模型。要求模型面数合理，uv正确；
6. 将模型或者动画导入到虚幻4引擎。确保模型大小合理，无反面等问题。
7. 确保动画能正确播放；在虚幻4引擎中进行基本的场景搭建，材质制作。
8. 及时学习新的软件工具和技术
