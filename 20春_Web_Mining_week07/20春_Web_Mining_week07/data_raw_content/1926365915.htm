职位描述：

【岗位职责】
1. 负责直播/社交/互动玩法产品的设计灵感收集、竞品设计分析、设计趋势分析以及UI风格定义工作。
2. 负责相关产品界面设计、用户体验设计、交互及动效设计以及视觉还原度跟进工作。
3. 推动产品细节体验升级，提高产品易用性、主要页面转化、渗透率、留存率，增强情感化引导和记忆。
4. 提供创意解决方案，发起功能模块的概念设计、前瞻设计，挖掘设计增长点并推动落地。
5. 紧密关注行业发展和流行趋势，关注并分享新颖设计手法、设计风格、设计工具。
6. 推动品牌视觉、IP形象、情感化设计在产品中的落地，提高用户归属感。

【岗位要求】
1. 设计类、艺术类、多媒体、心理学等专业优先，对UI/UX设计有成体系的理解和项目经验，有优秀移动端产品设计作品。
2. 熟练使用sketch（必须），AE、Principle、Protopie（多选一）；PS、AI、C4D（可选）等设计工具。
3. 至少2年以上移动端UI、UE经验，有过娱乐类、游戏类、社交类成熟产品经验优先，有视觉/平面设计经验，品牌设计经验优先。
4. 对设计流行趋势有灵敏的触觉和领悟能力，热衷于设计和解决问题，自我驱动力和学习能力强。
5. 至少拥有一项特有专家能力：交互设计、动效设计、视觉设计、产品思维、品牌设计、3D设计中的一个方向。
