职位描述：

职责描述：
1、负责官网整体的架构与交互设计（移动端，PC端）；
2、参与页面规范的制定和更新，不断探索新技术发展方向，结合实际不断创新；
3、具备对网站、手机、APP架构与交互的理解和分析能力，提高网站及移动端应用的可用性，继而优化设计；
4、协调和推动可用性测试及用户研究，以验证现有和将来的功能设计
任职要求：
1、5年-10年互联网或电商UX架构与交互设计经验，并对设计基础有良好的认知，Interbrand TOP100消费者品牌工作经验者优先。
2、 有大型网站设计经验者优先
3、对用户体验有较深的理解和专业性，对互联网交互设计有深刻理解，具备完整的理论和技术体系。
4、善于沟通，跨团队协作，能表达设计观点和设计价值，能够持续跟踪和关注上线产品
5、工业设计，人机交互，视觉传达，计算机等相关专业，本科及以上学历
6、有成功案例或相关作品（随简历附件或表明有效网址均可）
