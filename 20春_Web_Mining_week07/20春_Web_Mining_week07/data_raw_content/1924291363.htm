职位描述：

职责描述：
1.根据业务部门的要求，独立开发和维护信息系统功能模块。

2.按照客户需求独立完成相应软件模块的设计、开发、编程、测试等工作。

3.依据开发说明书，进行信息系统功能调整，以满足业务部门的个性化信息需求。

4.负责信息设计开发中的系统BUG的查找调整和修正工作。协助信息系统使用部门查找和排除问题。
5.收集、整理系统开发资料和用户需求，保证系统资料的完整性和程序的可开发性。
6.及时完成直接上级交办的其他工作。

任职要求：
1.本科以上学历；
2.开发过ALV、Dialog、RFC、Interface、Enhancement等内容；
3.至少熟悉SAP一个以上模块开发，对业务模块有基本的认识。
4.具有接口开发经验者优先,英语口语流利者优先

企业介绍：

 宁波容百新能源科技股份有限公司，是一家高科技新能源材料行业的跨国型集团公司，专业从事锂电池正极材料的研发与经营，由中韩两支均拥有二十余年锂电池正极材料行业成功创业经验的团队共同打造。

公司拥有优质的国际国内客户，近几年飞速发展，企业规模和营业收入每年倍增，三元材料国内市场占有率和出口市场占有率位居行业第一！公司2017年3月启动上市进程，成为准上市公司，计划2019年完成上市。
