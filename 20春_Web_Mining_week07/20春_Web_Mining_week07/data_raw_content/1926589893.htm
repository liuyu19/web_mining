职位描述：

岗位职责：
1.完善和规范各系列平面形象，对产品、品牌及各项会务及公关活动形象进行升级；
2.全面组织品牌终端形象的定位、主题创意的建设和完善；
3.负责部门管理和员工培养。
任职要求：
1.本科及以上学历，平面设计类相关专业优先；
2.有五年以上平面设计工作经验优先；
3.精通品牌VI管理相关知识体系，具备较强的的平面设计软件操作能力；
4.具有较强的创意、审美、项目管理能力，创新性强。

企业介绍：

周大生珠宝专注于“周大生”品牌珠宝首饰的设计生产、品牌运营和连锁经营，是国内具有影响力的珠宝品牌运营商之一，也是全国中高端主流市场钻石珠宝领先品牌。2017年4月27日周大生（002867）在深圳证券交易所A股上市。

在近二十载发展征程中，周大生一直坚持爱心、诚信、责任、奋进的核心价值观，以领先战略布局，采取轻资产、整合运营型经营策略，专注于品牌运营、渠道管理、产品研发和供应链整合，不断提升品牌形象和影响力，实现公司快速发展。

截至2019年6月30日，周大生共有门店3599家，形成强大连锁渠道网络，带来非凡的集群效应和资源整合优势。
