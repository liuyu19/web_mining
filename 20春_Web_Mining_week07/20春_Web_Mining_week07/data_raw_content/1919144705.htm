职位描述：

岗位职责:
1. 负责/参与相关前端项目的需求分析及研发工作；
2. 负责/参与具体前端项目的架构设计、具体模块、功能的开发工作，包括页面制作以及动画效果等；
3. 负责与团队成员保持良好的沟通协作，确保相关研发工作的顺利实施。
任职资格:
1. 编程基本功扎实，熟悉软件工程和测试方法；
2. 熟练掌握JS/HTML/CSS3/Ajax等前端开发技术，具备一定UI设计能力优先考虑；
3. 熟悉至少一种主流前端框架（React，Vue）并能解释原理；
4. 了解一门后台语言（如：Php/Java等），对前后端联合开发有一定的经验和认识；
4. 对平台化建设和WEB技术有强烈兴趣；
5. 具有良好的沟通能力和团队合作精神、优秀的分析问题和解决问题的能力。
