职位描述：

职责描述：
1、深入理解产品的规划构思，梳理产品需求，给出交互解决方案，完成交互设计工作
2、能够结合业务需求、用户需求及行业趋势，快速输出产品概念设计，并进行验证
3、对竞品的交互设计进行调研、对比分析，整理和积累交互创新想法，不断提升产品易用性
4、协助研发、测试和产品团队，进行新产品设计、原型评估，保证设计效果，并负责产品的持续优化



任职要求：
1、3年以上从事用户体验设计或相关领域工作经验
2、有车联网产品设计经验优先
3、具备较强的逻辑思维能力，熟练掌握业务需求分析、产品需求分解的技巧
4、出色的沟通及语言表达能力，具备较强的学习能力
5、以用户体验为导向，对产品负责，有良好的团队协作能力

企业介绍：

奇瑞控股集团有限公司成立于2010年，以汽车产业为基础，充分利用体制优势与品牌效应，整合多种社会资源，稳步实现多元化发展。
集团旗下投资有汽车、汽车零部件生产与研发、造船、金融、旅游地产、现代服务业等业务板块，200多家子公司，业务范围遍布海外80余个国际和地区。目前集团拥有员工4.8万余人，总资产1200多亿元。
秉持“创新、责任、共赢”的理念，奇瑞控股以产业报国为己任，立志在每一个进入的领域成为行业领先者，致力于成为具有全球影响力、值得信赖的多元化控股集团。
