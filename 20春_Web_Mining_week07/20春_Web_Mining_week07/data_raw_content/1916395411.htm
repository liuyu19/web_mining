职位描述：

职责描述：
1.负责携程商旅市场线上运营活动相关设计工作（包括专题设计、广告BANNER、H5营销以及视频制作等）；
2.负责支持全国运营、行政、人事、销售等相关设计；
3.参与制定商旅品牌VI建立及规范工作，以及一些衍生周边创意设计。
4.能参与公司及团队的创意策划企划方案，能跟进并追踪实时热点，思维活跃，创造力丰富。

任职要求：
1.本科及以上学历；
2.艺术设计，平面设计，媒体设计等美术类相关专业毕业；
3.良好的平面设计能力，优秀的手绘功底，
4.熟练操作PS，AI等设计软件，精通AE等后期以及3Dmax，C4D设计软件优先，具有一定营销设计经验，相关思考及总结；
5.二年以上互联网或大型广告公司经验为佳。

投递时请附上个人作品，谢谢。
