职位描述：

职责描述：
1、针对单个项目编制设计指导性文件和建筑规格更新
2、负责总图、方案、交通流线和绿化布局等，并提供经济性指标供合约部编制预算，提供租赁面积组成供开发部计租
3、协助开发部门编制环评、能评、交评等技术资料
4、内部审核自己和其他设计经理的机电图纸，确保图纸满足内部spec要求以及达到高性价比
5、起草并发起所负责项目的地勘、设计招、评标和相关咨询合同
6、对整个设计规划、设计进度、设计修改和设计成果进行把控
7、编制或审核各类招标文件中给排水、电气技术要求以及项目功能描述、施工节点等内容
8、协调和处理施工或运营阶段出现的专业技术、质量问题，参加施工阶段性质量验收工作和竣工验收工作；
9、参与甲供设备、材料的技术谈判
10、总监安排的其他工作，直接向总监汇报



任职要求：
1、大学本科及以上，机电一体化、电气类专业；
2、7年以电气专业设计背景；兼顾消防、给排水、暖通专业为佳；
3、基本的英语书写及沟通；
4、熟练使用AUTOCAD、office软件；
5、 中高级职称以上优先；

企业介绍：

易商红木集团（ESR）是亚洲领先的物流地产开发商、业主和运营商，为电子商务、零售业以及冷链行业提供仓储设施开发及服务，为企业客户提供物流、生产及办公设施的开发、运营和管理服务，其业务集中在与消费和全球贸易联系最为紧密的重要都市区域。

在联合创办方华平投资集团以及APG、PGGM、加拿大养老金投资公司（CPPIB）、平安和摩根士丹利等全球著名投资机构的支持下，易商红木已发展为中国、日本、韩国和新加坡等亚洲国家市场最大的泛亚地区业务平台之一，已建和在建项目总建筑面积逾800万平方米，并在香港和新加坡设有资产和基金管理办公室。

易商红木还是全球多家顶尖电商公司及冷链物流行业最大的第三方仓储业主之一。电子商务的迅猛发展、整个供应链效率的提高和第三方物流服务提供商的整合，不断推动着易商红木业务的蓬勃开展和快速增长。
公司网址：www.esr.com
