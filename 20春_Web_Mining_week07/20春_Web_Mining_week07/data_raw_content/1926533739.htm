岗位职责：
1、负责金融产品前端UI重构、JS开发工作；

任职要求：
1. 本科或以上学历；

3. 精通Javascript、OOP，理解主流JS框架；
4. 熟悉HTML5，CSS3等技术，有移动web开发经验；
5. 对常见的web性能问题、安全问题有一定了解，并有相关的实践经验。
