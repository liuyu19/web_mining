职位描述：

职责描述：
1. 通过用户调研、数据分析等方法输出用户体验报告（VOC），针对用户体验痛点立项，通过业务流程优化、产品迭代升级、跨部门协作等方式，提升用户体验、留存及复购；
2. 负责搭建核心数据看板，制定项目计划并按进度推进，确保项目按落地执行；
3. 实时向组织同步项目进展，跟踪项目的进度和质量，及时发现项目风险和问题；
4. 积极推进、调整并解决项目落地面临的问题，对项目绩效监管和指导；

任职要求：
1. 统招本科以上学历，5年以上项目经理或业务经营分析工作经验，项目管理专业工作能力突出，具有跨团队项目和交叉带项目的经历；
2. 推动能力强，业务敏感度高，逻辑分析能力强，能够解读数据报告，并将用户声音转化成具体的数据分析，并推动改变发生；
3. 生鲜电商1年以上工作经验或有一线互联网经验者，了解生鲜电商客服、商品、物流、运营的1-2个分支者优先考虑；
4. 抗压能力强，责任心强，做事严谨认真。

企业介绍：

      每日优鲜是一家定位于自营全品类线上超市的移动电商，成立于2014年11月，腾讯投资成员企业，致力让每个人随时随地享受食物的美好。
      目前，每日优鲜已经完成水果、蔬菜、肉蛋、水产、乳品、零食、速食、饮品、轻食、粮油、日百等全品类生鲜布局，并在全国20多个主要城市建立起“城市分选中心+社区前置仓”的极速达冷链物流体系，为用户提供自营全品类精选生鲜1小时达服务。
      2018年9月，每日优鲜完成了高盛和腾讯领投的4.5亿美元新一轮融资。在生鲜电商细分领域，营收增长、用户规模以及盈利能力均保持行业前列，并且凭借创新的商业模式和快速增长，登上科技部发布的2017中国企业独角兽榜单，成为国内数千万用户网购生鲜的重要选择。
