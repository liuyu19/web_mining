职位描述：

工作职责：
1.负责多类终端设备上微信客户端的设计工作，如电脑端、手表端、车载终端、智能硬件等；
2.结合软硬件层面，完成多终端、跨平台的设计工作，探索在触控屏幕以外的交互模式；
3.独立承担项目，为产品功能和特性提供合理的系统化解决方案。
任职要求：
1.2年以上交互设计经验，有多终端、跨平台相关的相关工作经验优先；
2.对各种智能硬件趋势有敏锐的设计触觉，富有设计创造力和激情，有探索研究精神；
3.强逻辑思维能力，掌握业务需求分析、产品需求分解的能力；
4.具备良好合作态度及团队精神，并富有工作激情、创造力和责任感。
