职位描述：

工作职责：
1.负责规划、设计并建立专有云客户安全托管服务体系及安全运营服务体系，服务能力产品化； 2.负责安全服务类项目的项目管理与支持； 3.制定专有云安全技术策略，包括但不限于网络、系统、中间件、云产品、应用的安全检查基线及安全方案； 4.针对网络、系统、中间件、云产品、应用开展安全评估、风险处置等工作； 5.组织安全事件应急处置工作； 6.负责重点客户的项目技术支撑。
任职要求：
1.学历要求：本科及以上；工作经验：5年及以上安全行业从业经历，具备中大型安全服务项目的交付、运营与管理经验，或具备安全托管服务（MSS）设计、运营与管理经验； 2.熟悉主流系统、网络、应用攻击技术原理及防御加固方案、掌握常见的安全漏洞及利用技术； 3.熟悉等级保护、风险评估、渗透测试、安全加固等安全业务，了解相关安全标准和规范及安全体系； 4.熟悉安全事件应急响应流程，能够对各类安全问题和安全事件进行跟踪、分析、处理和解决； 5.熟练使用Java/C/C++/python/perl/shell至少一种语言； 6.熟悉云计算，了解IaaS、PaaS、SaaS等相关技术； 7.具有良好的沟通、团队协作能力、客户需求理解能力和文案编写能力； 8.具备CISP、CISSP、CCIE Security资质者优先。 工作地点：杭州/北京

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
