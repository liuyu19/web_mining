职位描述：

职责描述：
1. 作为第一责任人，根据产品形态及各国服务能力，进行服务模式与流程设计，牵头并组织完成大家电在海外各国的售后服务落地
2. 作为售后服务代表与产品各部门进行有效沟通，评估服务风险，提供服务解决方案，及产品优化建议
3. 以项目形式，对售后内部各业务交付团队进行服务准备监控，对新产品服务交付质量负责，对产品生命周期内服务质量进行监控优化，实现服务闭环
4. 保证服务与销售，产品，市场等各部门的有效沟通，对信息传递及时、准确负责。
任职要求：
1.5年以上大家电产品厂商工作经验
2.从事或参与过产品项目管理工作经历
3.从事过海外项目或服务相关工作
4.流利英语听说读写能力及项目管理能力
5.从事过海外服务工作优先
