职位描述：

工作职责：
阿里云弹性计算(ECS)是阿里云的最重要的云服务产品之一，ECS是一种简单高效，处理能力可弹性伸缩的计算服务。我们始终致力于利用和创造业界最新的前沿技术，让更多的客户轻松享受这些技术红利，在云上快速构建更稳定、安全的应用，提升运维效率，降低IT成本，使客户更专注于自己的核心业务创新。企业级实例产品团队负责ECS实例的定义以及功能、性能、稳定性的持续优化和迭代， 作为阿里云最重要的产品之一把云上的技术红利输出到我们上百万的用户。 岗位职责： 1. 负责ECS产品进入专有云的系统架构设计，包括部署架构，升级架构，运维架构，运维生命周期，容量模型等 2. 支撑ECS专有云重大项目的版本交付，支撑ECS专有云新机型新规格定义与适配 3. 识别ECS专有云定制需求的共性部分，在架构设计上创新，融合到公共云与专有云统一 4. 考虑行业ISV引入的架构设计和合作伙伴联合解决方案
任职要求：
1. 深入理解linux系统，精于容量规划，架构设计，运维设计 2. 精通一门以上脚本语言(shell/perl/python等)，精通一门以上开发语言（Java/C/C++） 3. 熟悉云计算知识，包括虚拟机，VPC网络，分布式存储等 4. 需要具备在企业环境中进行产品交付的经验，交付过私有云的产品的优先 5. 具备很强的ownership, 良好的客户意识，善于团队协作，主动思考，自我驱动力强 6. 有物理网络架构经验优先，有服务器设计经验优先，有大规模管控开发运维经验优先

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
