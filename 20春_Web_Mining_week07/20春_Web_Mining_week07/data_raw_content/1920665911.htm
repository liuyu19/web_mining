29777-腾讯云能源行业解决方案架构师（北京/上海/深圳）（深圳） 腾讯

面议

应聘职位

职位描述：

工作职责：
为能源行业客户和合作伙伴提供腾讯云整体技术平台、业务和技术层面的架构咨询服务，理解客户在研发、运营工作中对网络、计算、存储、物联网等基础设施，数据、技术、业务中台和应用的需求，分析其快速发展过程中，在信息化所遇到的挑战和未来的方向；
根据客户业务的需求和理解合作伙伴的需求，基于腾讯云的技术架构，输出满足客户需求的服务解决方案；
具备独立为行业客户提供有竞争力的互联网+能源解决方案的能力, 联合销售团队整合资源完成技术支撑及系统架构工作，并协助承担售中/后等相关技术管理和推进工作，具备很强的资源协调能力，有一定的项目实施经验。
任职要求：
本科以上学历，5年以上的IT系统架构设计，至少3年以上云计算、大数据、物联网、能源行业信息化架构设计工作经验；
具备优秀的文档能力，清晰明了地表达架构意图，能够熟练编写各类技术文档；
良好的沟通、协调及资源整合能力；
具备优秀的表达能力，思路清晰、逻辑缜密。
具备以下者优先考虑；
1.熟悉openstack等虚拟化架构及主流公有云服务，并能够清晰描述其工作原理和基础的架构逻辑，对不同品牌的云服务区别有一定认知者；
2.熟悉私有云、专有云（面向能源）项目的需求分析、方案设计、传统解决方案云化规划设计、应用迁移和数据迁移等设计工作；
3.具有丰富的企业级应用架构设计或云服务集成实施经验；
4.有一定IT Infrastructure（包含服务器、存储、数据库、网络、中间件）的运维或管理经验
5.熟悉电力、石油、燃气行业信息化系统架构、业务需求和未来升级方向；
6.对能源行业大数据、AI、物联网、数字孪生、综合能源等领域，有一定项目经验或在熟知其中一个领域技术细节者；
7.有能源行业信息化服务交付、设计经验者。 通过腾讯云技术认证或同等资格认证的优先录取。
