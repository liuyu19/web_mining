职位描述：

岗位职责：
1. 据产品需求，编写嵌入式系统硬件总体方案和详细方案，进行硬件选型及系统分析；
2. 针对多用户蓝牙、GPRS、NB-IOT、Lora及门禁产品进行深入研究，提出产品方案；
3. 与手机客户端和服务器软件的开发实现，实现数据传输和实时控制；
4. 制定并编写测试方案，负责硬件驱动的开发调试，与手机APP及后台软件工程师进行联调；
5. 小批量生产、测试、解决生产过程中出现的各种问题，配合施工团队现场处理产品故障。
任职要求：
1. 计算机或相关技术专业的本科及以上学位；
2. 至少三年以上产品开发经验，有独立项目开发经历；
3. 掌握C 开发，熟悉常用数据结构、算法；
4. 熟悉物联网、有低功耗产品开发经验优先；
5. 掌扎实的编程功底和良好的编程习惯，追求良好的用户体验；
6. 具备良好的团队合作精神，快速的学习能力；并且有强烈的事业心和责任心。

企业介绍：

公司网址：www.tslsmart.com
特斯联科技有限公司是由中国光大旗下基金与IDG资本联合投资的城市级移动物联网生态平台公司，领跑于以物联网为技术基础的“智慧城市”新赛道，为“未来”系列多场景生活、办公和城市管理提供解决方案。目前，特斯联凭借智能硬件、云服务和移动应用的技术核心，重点发展三大业务单元：ABAS 超级楼控，智慧城市、智慧通行解决方案；同时，成立特斯联金融板块，全方位支持纵向、横向的业务板块。
成立至今，公司已形成专业硬、软件技术团队，百余人的技术专家和强大的研发设计能力；拥有国际、国内30多项发明专利。截止目前，特斯联各系列解决方案已成功落地北京、上海、重庆、深圳、武汉、广州、杭州、新疆、云南、贵州、山东等地，覆盖人口达千万量级。
特斯联“智能通行系统”是公司目前着力打造的明星产品：“未来”门锁和门禁系列获得公安部CCC智能硬件认证，市场占有率居前。
公司在北京、重庆、上海、武汉、广州设有办公地点，研发中心分别设于北京和重庆。
