职位描述：

职责描述：
1.负责公司核心硬件产品的软件交互设计、原型图设计、人机交互调研、操作交互流程说明、功能设置等；
2.参与用户行为分析（用研/数据分析），根据分析结果对设计方案进行评估和优化；
3.结合业务目标及规划，产品 及 设计团队密切合作，提出创新思路，持续优化产品的用户体验；

任职要求：
1. 3年及以上用户体验设计工作经验；有大型互联网公司的上线落地项目 或 硬件人机交互，软件交互设计经验者优先考虑；
2. 具有扎的交互设计功底，扎实的用户研究和数据分析基础，极强的问题解决能力；
3. 有较强的执行力和沟通表达能力，主动性强，能有效传达和推动自己的想法；
4. 有良好的团队合作精神和协调沟通能力，有与产品、前端等多方密切配合的经验和意识；
5. 全日制统招本科及以上学历，设计相关专业优先考虑；
