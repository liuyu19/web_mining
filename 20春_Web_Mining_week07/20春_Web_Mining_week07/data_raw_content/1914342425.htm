职位描述：

岗位职责：
1. 对接产品经理完成产品的交互和原型设计，保证产品流畅性；
2. 思维逻辑清晰，可以抽象出产品交互框架；
3. 观察产品数据，持续迭代产品功能点；
4. 掌握竞品分析方法，能输出高质量的竞品分析。


任职资格
1、逻辑能力强，能够理清复杂的系统功能流程；
2、具备良好的沟通能力；
3、有较强的学习能力，对时下移动端流行交互方式和设计有研究；
4、具备自驱力，自我驱动成长，善于总结，乐于分享；
5、拥有动效设计能力优先；
6、掌握常用的原型制作工具，如Axure RP、墨刀、Flinto等，并能熟练地制作高保真的原型界面；
7、本科以上学历、3年以上工作经验
8、优秀者以上条件可放宽；
