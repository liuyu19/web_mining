职位描述：

工作职责:
1．负责测试团队技术体系的规划和落实，提供技术培训与指导，提升测试团队的专项测试能力、自动化测试能力，提高整体测试水平
2．负责质量平台的架构设计及实现，通过技术手段提升研发效能、优化测试流程，质量平台包括对功能、自动化、性能、安全、持续集成、测试环境等的统一管理
3．负责解决复杂技术问题，能够发现工作中的效率提升点，开发工作中需要的测试工具
4．调研、引入新技术和新思想，提升整体团队效率和质量，培养后备技术人才
5．对自动化框架、持续集成CI、持续发布CD、敏捷测试、DevOps、测试平台搭建、测试工具开发等有实际开发经验或深入的了解；

任职资格:
1．计算机相关专业，本科以上学历；
2．7年以上复杂系统软件及接口测试经验，至少5年自动化框架设计和实现经验；
3．参与过自动化测试体系构建，有丰富的测试框架、测试平台工具开发应用经验，擅长API接口、UI自动化测试、服务端性能测试以及单元测试；
4．有大型系统的持续集成测试经验；
5．至少精通一门编程语言 （Python、C++、JAVA）；
6．具有较强的业务分析，沟通表达能力和综合协调能力，工作积极主动，责任心强；
7．具备广泛的技术视野和很强的技术前瞻性，了解测试技术的发展,有自己的技术成果；
8．熟悉以下至少其中一项技术业务或者有相关经验者优先：
（1）分布式微服务测试框架
（2）高性能计算
（3）分布式存储
（4）容器、中台、Serverless 等相关云计算技术
