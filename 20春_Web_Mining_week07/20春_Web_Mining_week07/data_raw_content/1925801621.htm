职位描述：

职责描述：
1. 光学镜头模组成像质量评估与缺陷分析，为光学评测提供画质定量分析工具；
2. 光学镜头模组生产设备的相关软件开发，包括UI界面、算法以及硬件控制，为光学镜头模组生产提供技术支持；
3. 光学镜头成像仿真软件开发，为光学设计提供指导性意见；
4. 其他光电类精密仪器、设备的开发。

任职要求：
1. 光电子、光学、物理学、计算机等相关专业，硕士研究生以上；
2. 具备扎实的光学理论基础（几何光学和信息光学）和数学建模能力；
3. 具备编程开发和算法实现的能力，例如UI开发、软硬件通信、图像处理、Opimization算法、光轨迹追踪、光场传输模拟等；
4. 熟悉使用C/C++,Labview等工具 实现算法编程和UI设计；
5. 具备一定的光学设计基础，了解Zemax,TracePro,Code V等光学设计软件；
6. 具备较强的英语文档阅读能力，有日语交流能力优先考虑。

企业介绍：

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，办公室设于北京、上海、西安、香港、东京、洛杉矶、旧金山、鹿特丹、法兰克福等全球18个国家及地区。

大疆是一家致力于持续推动人类进步的科技公司，坚持做空间智能时代的开拓者，不断为世界创造超越想象的科技之美。2006年创立至今，大疆在全球的员工人数已超过14,000人，客户遍布全球二百余个国家和地区。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十三年间，大疆不断革新技术和产品，开启了全球“天地一体”影像新时代；大疆的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。
