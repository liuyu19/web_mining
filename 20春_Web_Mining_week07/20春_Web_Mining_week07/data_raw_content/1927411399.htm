职位描述：

工作职责：
1、负责区域现有服务市场的顶层设计和服务商日常管理，推进服务商完成1688平台的各项运营要求和指标； 2、根据1688平台战略方向和产地规划，对现有服务商进行分析诊断，并赋能服务商，提升其深度服务运营和控商控品能力； 3、协同当地渠道经理共同推进当地市场服务商招商，完善服务市场生态布局，繁荣当地服务市场； 4、分析所负责市场产地属性和市场特点，发现相应市场机会，找到业务突破点，制定对应业务策略，推进平台战略在当地落地；
任职要求：
1、5及以上相关工作经验，包含B类或C类行业电商运营管理，商家运营等方向； 2、具有优秀的沟通协调能力，有项目管理及店铺全链路运营经验，曾经为行业top商家提供服务咨询，帮助商家提升店效水平及运营效率；具备良好的商务合作及客户拓展能力； 3、有独立的行业思考的全局性和前瞻性，较强的逻辑思维，敏锐的市场洞察力及分析能力，掌握行业动态及消费需求； 4、有较强的自驱力及快速学习能力，能够适应一定的出差工作强度； 5、全日制本科及以上学历，有渠道商或代理商管理工作经验优先。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
