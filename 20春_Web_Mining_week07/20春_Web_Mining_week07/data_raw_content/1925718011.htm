职位描述：

1. 负责新社交类项目相关设计工作，进行高质量的产品UI设计、动效设计、参与设计规范的制定；
2. 参与产品构思及移动端界面优化及迭代，提升产品易用性，挖掘新功能，把控整体视觉产出；
3. 帮助产品挖掘更多商业性互动玩法的可能性，并产出demo；
4. 营造学习氛围，分享设计经验及方法论，帮助团队成长；

职位要求：
1. 设计类专业背景，3年以上经验，有优秀移动端app设计作品，声音社交类产品优先；
2. 对设计流行趋势保持灵敏触觉，富有激情、创新欲望以及持续学习能力、自我驱动力、洞察力；
3. 具备业务推动力及良好的抗压能力、合作态度、团队精神、责任感；
4. 逻辑思维能力强，兼顾掌握交互设计知识，优秀的动效设计技能；
5. 必须项：熟练使用Sketch、Photoshop等基本设计工具，加分项：AE、Principle等同类动效软件、手绘能力强；
