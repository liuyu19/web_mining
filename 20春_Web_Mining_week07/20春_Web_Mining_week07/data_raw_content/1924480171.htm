岗位职责：1、负责集团金融科技输出探索和落地，包括对外拓展合作方（包括但不限于金融机构、同业、场景方等）、行业最新动态收集、合作方案撰写、业务合作落地等；2、根据合作方需要创新合作模式，为不同场景提供定制化合作方案以及行业解决方案；3、配合部门产品侧的创新需求，做好政策梳理、外部信息收集、交易结构设计、可行性分析等工作；4、跟踪和维护对外合作关系，不断深挖合作；5、沟通和协调内部相关部门，推进业务快速落地
