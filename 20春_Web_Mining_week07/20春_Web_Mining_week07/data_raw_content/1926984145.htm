职位描述：

职责描述：
1. 规划入境市场和香港钱包整体的火车票预订流程，负责整体的预订流程优化迭代；
2. 积极与火车票BG配合，提升预订流程的用户体验，对火车票整体漏斗转化负责；
3. 负责火车票入境市场和香港用户的用户调研和竞品调研，通过有效调研找出与大陆本地市场的差异点，提升整体的购票体验，定期输出调研报告；
4. 负责火车票辅营增值类产品：规划入境市场整体的火车票辅营增值产品，优化迭代已有辅营产品（VIP FEE，BOOKING FEE，预订FEE，VIP贵宾厅），挖掘新的辅营产品（保险等）及增长点，制订辅营产品的策略；
5. 积极与火车票BG以及相关服务合作，在不影响用户体验的情况下，提升辅营产品的营收；
6. 对火车票整体的营收负责，完成辅营的前端产品落地方案与需求迭代跟进；
7. 火车票前端客服问题对接反馈，定期梳理跟进客服反馈的问题，将合理客诉吸收整合上线；
任职要求：
1. 至少3-5年互联网交易类产品经验，有交通或OTA行业优先；
2. 对市场和商业有很好的敏感度，熟悉交通类行业的营收模式，能够建立业务线营收模型；
3. 有一定数据分析能力，能够通过用户调研分析和活动数据分析进行产品优化迭代升级；
4. 有较强的沟通协调能力和抗压能力，能够在高压环境下进行资源的沟通协调；
5. 具体良好的英文书写能力。

企业介绍：

同程艺龙由同程旅游集团旗下的同程网络板块和艺龙旅行网在2017年底合并而成。
同程旅游集团是一个多元化旅游集团，中国领先的在线旅游服务商和一站式旅游预订平台，是“中国民营企业500强”和“中国旅游集团20强”企业。同程旅游集团下辖同程网络和同程控股两大板块，其中同程网络业务涵盖同程机票、酒店、火车票、汽车票等多项业务。
艺龙旅行网是中国领先的在线旅行服务提供商之一，深耕OTA行业多年，秉承专业团队的管理运作，在酒店等领域享有盛誉。
合并后，同程艺龙定位于领先的在线旅行服务平台，将同程网络和艺龙各自核心业务优势互补，并占有微信钱包中机票火车票的入口和酒店入口。目前同程艺龙已有近两亿的消费会员，2017年服务超过五亿人次，合并交易额超过千亿人民币，并已实现年超过五亿人民币的规模化盈利。
