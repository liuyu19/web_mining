职位描述：

职位描述
岗位职责：
1. 负责IOS客户端软件的研发，组织代码review，负责基础库和模块架构设计和开发；
2. 对软件产品的持续改进，优化性能，提升开发效率和优化项目流程和存在的问题等。
3. 负责编写技术规范文档并定期给技术团队进行技术培训。
4. 对产品新feature进行调研，技术方案选项及评审

任职要求：
1、本科及以上学历，五年及以上工作经历，三年以上iOS应用开发经验，对技术具有一定热衷；
2、精通Objective-C、Mac OS X、X code，精通IOS SDK中的UI、网络、XML/JSON解析等开发技巧；
3、对软件产品有强烈的责任心，具备良好的沟通能力和优秀的团队协作能力；
4、具有阅读英文文档的能力，能简单进行英文书面沟通；
5、对终端产品的UI/UE有独到的见解与认识，追求良好的用户体验，能不断自我学习和改善。
6、精通常用软件架构模式，熟悉各种算法与数据结构，数据库，多线程，网络编程（Socket、http/web service）等。
7、对音视频有开发经验优先。

【我们是谁】
      我们是喜马拉雅，国家高度重视的新经济龙头企业、互联网行业独角兽。打造国内领先的音频分享平台喜马拉雅，我们拥有海量6亿手机用户，和超5000万海外用户，是中国文化出海的中坚阵地。
【我们的使命】

【我们的生态】
      秉承万物有声的理念，将有声内容渗透到生活中各个智能终端和场景中。2017年6月，推出国内仅有的全内容智能AI音箱——小雅，布局人工智能领域，打造物联网时代信息获取最便捷的入口。让用户把需求大声说出来，开启了一个语音交互的新传播时代，让声音和知识像水和电一样无处不在，随取随用。
