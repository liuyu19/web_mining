职位描述：

岗位职责：
1. 根据产品需求，制定开发计划，进行版本迭代;
2. 对现有产品提出优化方案并实施;
3. 对终端UI / UE有自己的感知，理解安卓设计，追求用户体验;
岗位要求：
1. 本科及以上学历，计算机相关专业;
2. 4-5年Android实际开发经验;
3. 精通Java编程语言，良好的数据结构、算法基础;
4. 熟练使用Android网络框架及类库，并了解工作原理;
5. 熟练掌握Andorid图形绘制和事件分发机制;
6. 了解组件化，插件化等开发模式和技术，
7. 热爱移动产品研发，乐于学习钻研与分享
8. 有MVP使用经验和Kotlin语言开发经验优先
