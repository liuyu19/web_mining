职位描述：

职责描述：

1、负责云计算IAAS运维管理平台的软件架构设计，提供健壮的软件架构及灵活的页面布局；
2. 构建业界领先的自动化IAAS运维平台，高效管理IAAS基础设施，包括设备管理、配置管理、性能监控等， 打造以智能化、自动化为核心的运维平台，提高平台易用性、可靠性；
3、负责运维平台的业务规划、技术路标管理，完成技术竞争力构建与落地计划；
4、负责运维平台的可服务性设计，确保实现高可用、高性能、高扩展性等架构竞争力要求；
5、负责运维平台的关键核心软件代码的编码及技术指导；

任职要求：

1. 精通Java/Python/JS中任意一门语言，具有分布式软件系统架构设计及重构能力；
2. 熟悉业界典型的云计算/大数据运维系统架构，熟悉自动化设备管理、配置管理、性能监控平台设计，有大规模分布式集群自动化运维经验；
3. 对学术界在云计算、虚拟化等领域新技术有较好理解，了解AWS、VMware等云产品相应运维特性能力；
4. 责任感强，逻辑思考能力强，沟通能力良好，执行力强，能够接受较大的工作压力；
5. 具有较好团队合作精神和较开放的技术思路，具有基于全栈能力的业务视角；
