职位描述：

职位描述
1、负责产品功能设计和编码工作；
2、参与软件产品项目规划工作，制定具体项目实施方案；
3、根据开发进度和任务分配，完成相应模块软件的设计、开发、编程任务；
4、负责项目相关文档的编写；
任职要求
1、本科以上学历，具备计算机类专业背景；
2、熟练掌握HTML. CSS、Javascript、Jquery、XML、Ajax等技术，熟悉Weblogic、WebSphere、 Tomcat中的一种应用服务器
3、了解面向对象编程，具有一定的设计能力,熟悉设计模式（Design Pattern）；
4、熟练掌握 Eclipse 开发工具，对界面的易用性、合理性、美观等元素敏感；
5、严谨的编码风格，良好的界面设计能力，较强自学能力;具有较好的质量意识，文档编写能力；
6、有多个大型J2EE平台下企业应用开发经验者优先考虑；
7、能够适应出差
