职位描述：

岗位职责简述：
1、负责OSB系统集成需求分析，规划应用系统集成总体设计方案；
2、负责OSB应用系统集成详细设计，负责服务接口开发、规范设计；
3、负责OSB平台的安装部署、客户化配置、开发、测试等工作；
4、负责监控系统运行健康状态，及时优化系统性能。
专业素质要求：
1、本科及以上学历，计算机相关专业，至少3年以上相关工作经验
2、有使用Oracle SOA套件、Oracle服务总线(OSB)和Oracle WebLogic server的开发经验
3、具有使用JAVA对OSB进行功能扩展的经验
4、充分理解传统开发平台、工具和概念包括Java / JEE,Oracle / RDBMS,JDBC / ORM、事务管理、面向方面开发,Struts,Web服务开发,实时应用程序/并发性
5、熟悉常用的开发工具如Eclipse IDE,ant,Weblogic / JBoss / Tomcat,SVN / VSS和关系数据库(如Oracle数据库）
6、具有一定的架构设计和研发能力，熟悉应用单元测试设计者优先。

广州赛意信息科技股份有限公司（GuangZhou SIE Consulting Co., Ltd.）

面对新的发展机遇和目标，赛意信息将在不断的实践中累积适合中国市场的信息化服务解决方案，务实创新，成为中国企业信息化服务领域的先驱。
