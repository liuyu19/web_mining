职位描述：

工作职责：
-负责面向支付宝的商业广告数据产品整体规划、设计及落地，提升业务数据决策效率和质量 -负责调研、收集、整理数据相关的业务需求，形成系统化产品方案，包括但不限于数据埋点设计、全链路数据治理、指标体系建设、数据可视化等 -负责数据方向的业务协同，包括但不限于BI、运营、技术等团队，对关键业务链路数据完成相关分析工作 -持续跟进评估数据产品的业务满足情况并进行完善；
任职要求：
-本科及以上学历，熟悉互联网广告业务优先 -具有3年以上数据产品或广告产品相关经验 -对数据驱动业务具有深刻理解，独立负责过数据产品从0到1搭建优先，熟悉mysql / hadoop / python等数据处理与分析工具优先 -具备出色的沟通协作能力和自我驱动意识，具有强烈的责任心和上进心，具备良好的团队合作精神，积极主动，善于调动合作业务团队积极性

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
