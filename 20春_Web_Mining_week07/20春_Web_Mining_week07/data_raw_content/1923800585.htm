职位描述：

【职位描述】
1. 参与和制定产品战略、市场调研及竞对分析；
2. 产品核心能力提炼，设计产品原型，撰写产品文档并组织产品评审，协调设计部门资源完成UI设计工作；
3. 联动市场部、商务、售前对外宣传产品，负责面向政府部门（公安、综治等）推广产品和封装解决方案；
4. 与研发团队、测试团队进行有效沟通、协作，产品研发进度、质量、成本进行管理；
5. 与客户、销售、项目团队协同工作，对于重点市场，协同各部门联合研发，对产品落地负责；
【任职资格】
1. 大学本科及以上学历，有4年以上产品经理工作经历；
2. 有丰富的企业级产品设计经验或成功的产品实施经验；
3. 具有较强的逻辑分析能力、口头表达能力、文档书写能力和推动策划能力；
4. 对数据和数据挖掘，数据分析技术具有强烈的敏锐度，能够从客户和业务的角度来进行技术实现；
5. 责任心强、跨团队合作能力强，具备良好的沟通协调和自我驱动能力，能承担一定压力；
6. 具有公安、综治、智慧社区相关产品或项目经验丰富者优先；
7. 国内一线互联网公司或企业级服务公司从业者优先；
