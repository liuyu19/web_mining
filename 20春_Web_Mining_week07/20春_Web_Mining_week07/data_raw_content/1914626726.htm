职位描述：

工作内容：
1、负责公司各产品的页面视觉设计及日常维护；
2、线下及线上各类宣传物料设计；
岗位要求：
1、设计类相关专业本科以上学历，3年以上平面设计工作经验及丰富的电商设计经验；
2、有较强的美术功底及审美能力，较强的平面设计和网页设计能力；能够准确把握整体风格及视觉表现；
3、良好的创意能力和领悟力，能够独立完成各类宣传物料及页面设计；
4、熟练使用Photoshop / Illustrator / Dreamweaver等设计类软件，对图片处理和视觉呈现有较好的控制力；
5、良好的沟通能力和领悟力，团队合作意识强，敬业，有责任感，能够承受工作压力。
