职位描述：

职位描述：
1. 负责蛋白纯化的工艺开发和放大，能合理分析、解决纯化过程中出现的问题；
2. 掌握各种色谱层析、膜分离等技术的原理和实际操作；
3. 保证生产正常进行的情况下参与纯化工艺的优化和改进；
4. 参与生产异常的工艺分析，并负责相应的工艺技术调查研究。

任职要求：
1. 本科以上学历，生物工程、生物技术、生物制药、生物化学等生物相关专业；
2. 具有蛋白分离、纯化技术工作经验者优先；
3. 熟练掌握各种蛋白纯化技术和放大工艺；能够熟练查阅英文文献，独立开发蛋白纯化工艺；
4. 熟练掌握蛋白质分析的相关实验技能，具备扎实的理论基础和实验操作能力。可独立进行文献检索、阅读与信息提取及实验设计和分析；
5. 具有较强的协调能力、学习能力和良好的沟通能力，做事严谨负责、注重细节，具有较强的总结、归纳和创新能力。

企业介绍：

      宜昌东阳光长江药业股份有限公司成立于2001年8月，座落于山水秀美的滨江小城—湖北宜都，注册资本4亿元，现有员工1000余名。公司是国家高新技术企业，香港东阳光药（1558.HK）主体公司，先后荣获湖北省十佳外商投资企业、医药十强企业和湖北名牌等多项荣誉，被列入湖北省生物产业发展规划（2008-2015）纲要。公司目前是我国防治甲型H1N1流感药物—磷酸奥司他韦（可威）的国家战略储备及军需特供药品生产基地，也即将成为中国领先甚至对世界都有重大影响的全系列胰岛素生产基地。
      公司现有十多个具有自主知识产权的一类新药正在研发中，涉及抗病毒、心血管、抗肿瘤等领域。公司一号厂区现有片剂、胶囊剂、颗粒剂、干混悬剂等一系列制剂产品生产线及多条合成原料药生产线，以普、新、特药为主，并全部通过国内GMP认证。同时有多个产品成功拿到欧洲药政部门颁发的COS/CEP证书及美国的FDA认证。
     三号厂区于2005开始涉足治疗糖尿病领域产品的前期研发，并于2008年取得胰岛素产品生产工艺的突破，2009年进行工厂化建设。现有重组人、甘精、门冬3种原料药及6种制剂产品，拥有自主的知识产权，生产技术、产品质量世界一流。
