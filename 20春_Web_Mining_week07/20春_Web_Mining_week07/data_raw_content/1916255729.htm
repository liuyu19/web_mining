职位描述：

蚂蚁金服-支付宝开放平台&小程序 技术文档工程师 （Technical Writer）

- 产出支付宝开放平台和小程序的业务文档、API文档等形式的完整内容体系，为用户快速上手使用开放平台的功能和小程序提供完整的支撑。
- 和产品团队紧密联系，根据产品定位设计产品信息模型，组织内容开发，保持内容、产品的强一致性和准确性。
- 负责整合、分析各渠道的用户反馈，制定内容改进方案建议、跟踪改进进度及改进效果，负责用户体验持续改进计划。
- 负责对产品团队进行文档写作培训、文档质量标准宣导，提高技术团队整体写作能力。

岗位要求:
- 本科或研究生，信息类毕业（计算机，信息工程，软件，自动化，电子工程，通信，数学等）或者有IT从业经验2年以上；
- 有以下工作经验：系统开发 / 测试 / 运维 / 质量控制及改进 / 项目管理 / 售前咨询 / 售后支持 / 技术文档写作 / 信息开发。
- 有快速学习能力，能尽快学习掌握云计算各种产品，做到理解原理，有能力验证产品，开发运维等。
- 具备良好的项目管理、表达、沟通及协调能力。
- 具备优秀的语言理解能力、信息架构能力及文挡写作能力。
- 有良好的客户服务观念，工作责任心强，注重团队合作。
- 优先录用条件：
- 了解 云计算/ BI / 移动开发 等相关技术、有开发经验。
- 英语水平佳，大学英语六级以上水平，具备良好的读写能力。
