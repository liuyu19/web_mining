职位描述：

【岗位职责】
1.参与影像研发技术路线及器件选型，制定影像测试标准，和研发深度合作，保障影像效果；
2.主导camera HDR/美颜/AI拍照/专业相机/图库功能、性能、稳定性测试，全面把握功能及非功能需求，设计针对不同产品的测试策略；
3.负责制定相机画质评测标准，设计评测体验方案，跟踪行业标准；
4.负责对产品的图像质量进行主客观的测试评价，与算法、软件、硬件紧密合作优化产品；
5.主导camera测试工具、测试技术、测试平台的设计和引入，并推动方案动落地，提升测试效率；
6.编写测试计划、规划详细的测试方案、编写测试用例。
【任职要求】
1.光学，图像处理，光电信息，计算机等相关专业，本科5年以上工作经验，硕士3年以上工作经验，有知名手机厂商或模组厂工作经验者优先；
2.熟悉RGB摄像头模组、Sensor、VCM、Lens、蓝玻璃等相关器件制程和品控；
3.熟悉镜头模组的鬼影、Flare、偏心等异常问题处理；
4.熟悉CMOS/CCD传感器的特性Noise/SNR/暗电流/ERS/GES/灵敏度/QE等评估与测试方法；
5.掌握imatest, IE , DXO等的影像评测体系和使用，熟悉主要图像质量测试方法，熟悉相机测试标准；
6.熟悉ISP原理及架构，有相关调试经验优先考虑；
7.了解各种传统图像处理算法，包含但不限于降噪，HDR，超分辨率，锐化，颜色管理等，图像或者视频相关处理经验都可。

企业介绍：

奥比中光成立于2013年1月，是我国人工智能3D传感领域独角兽。公司总部位于深圳，在美国西雅图、上海、广州、西安设有分部。奥比中光是继苹果、微软、英特尔之后，全球第四家，也是全亚洲第一家能量产消费级3D传感器的厂商。目前，奥比中光已推出多款3D传感摄像头，产品已应用于智能手机、智能机器人、智慧客厅、新零售、VR/AR、工业测量等多领域。2018年5月，奥比中光完成由蚂蚁金服领投的超两亿美金D轮融资，估值已超10亿美元。
