职位描述：

职位描述：
1. 负责APP服务端设计和研发工作；
2. 负责APP推送相关服务的开发；
3. 负责APP相关订单、用户数据等开发和维护；
4. 负责设计服务端高可用的通用框架；
5. 参与APP客户端接口和功能设计；
6. 能够处理日常复杂的业务需求，协调夸部门的沟通工作。


任职要求：
1. 本科及以上学历，计算机、信息管理等相关专业优先；
2. JAVA基础知识扎实，理解多线程、集合等基础框架，对JVM原理有一定的了解；
3. 熟悉SSM， Spring Boot等开源框架；
4. 熟悉分布式、缓存、消息等机制；
5. 掌握多线程编码及性能调优，有高并发、高性能系统设计和开发经验优先；
6. 熟悉Linux 操作系统和大型数据库（Oracle、MySql），对sql优化有丰富的经验；
7. 学习能力强，适应能力好，具备耐心/细心的品质；
8. 追求编写优雅的代码，具有良好的技术敏锐度，能从技术趋势和思路上能影响技术团队。
