职位描述：

岗位职责:
1. 独立完成项目分解、负责设计团队工作安排、业绩考核，带领设计团队高效完成工作任务；
2. 负责线下美陈展与门店相关视觉设计，参与品牌VI设计、根据市场需求与时下流行风格不断完善品牌VI；
3.负责把控品牌整体风格及视觉形象，确保设计风格符合品牌定位，提升品牌在线上线下的整体视觉效果及用户体验；
4. 负责品牌宣传、文化推广等设计需求等实现，包括视觉包装、产品包装、各类宣传手册、平面海报、宣传物料等。
任职资格:
1. 本科以上学历，设计类相关专业；
2. 三年以上平面设计工作经验，能熟练使用PhotoShop/AI/等图像处理和设计工具；
3. 具有扎实的设计功底和手绘功底，良好的创意思维和理解能力；
4. 熟悉印刷工艺、广告材质，具备项目管理能力及团队管理能力；
5. 具有较强的自我驱动力，责任心强，能承受较大的工作压力；
6. 具有连锁门店总部视觉管理经验或儿童游乐、主题乐园、玩具、动漫、家庭娱乐中心、IP产业等行业从业经验者可优先考虑。
