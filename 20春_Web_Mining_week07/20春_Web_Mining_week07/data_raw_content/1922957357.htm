职位描述：

职责描述：
1、负责转转腾讯生态、优品业务、前端架构等方向研发（pc/m/app[rn/hybrid]/小程序）；
2、掌握转转前端一个技术方向，如App web开发调试，前端工程化、前端性能、webview技术、离线存储技术、UI组件化、前端自动化测试、web安全等；
3、解决业务前后端开发模式、运营项目效率、UI标准化方面的难题，并产出通用解决方案。


任职要求：
1、本科以上学历，计算机及相关专业；
2、2年以上前端开发经验，负责过复杂应用的前端设计和开发
3、精通web前端技术（js/css/html），熟悉主流框架类库的设计实现、w3c标准；
4、熟悉移动端开发、自适应布局和开发调试工具；
5、掌握一门后端语言（java/php/python/ruby/golang/nodejs），对前后端合作模式有深入理解；
6、有良好的产品意识和团队合作意识，能够和产品、UI交互部门协作完成产品面向用户端的呈现；
7、有技术理想，致力于用技术去推动和改变前端研发。
