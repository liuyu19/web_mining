职位描述：

职位描述:
• 基于移动互联网的支付、金融应用的视觉设计工作；
• 配合产品/业务小伙伴们，理解业务发展、产品功能和用户需求分析，能够独立完成产品交互流程，界面交互结构并最终输出高保真视觉设计；
• 具有企业VIS意识，结合产品的用户定位，探索视觉样式的多样性，通过形成的产品视觉体系能够满足企业形象、产品特征和用户定位三者之间的平衡；
• 通过内部、外部等多方渠道获取用户反馈，对现有产品能够开展用户研究和可用性测试，用设计方式解决产品乃至业务中无法体现的问题；
• 规范产品在用户体验上的统一性和一致性，编写、维护和更新产品的相关设计文档和规范。
职位要求：
• 专业不限，但最好与人机交互、心理学、软件工程、艺术设计、工业设计或相关设计相关的本科以上学历；
• 英语4级以上，具有一定的听、说能力；
• 具有移动互联网设计工作经验4年以上相关工作经验；
• 具备前瞻的设计理念和方法论，关注设计、金融行业动态及移动互联网产品趋势； 
• 熟悉，精通Sketch/Photoshop/Illustrator等主流设计软件即可；
• 熟悉iOS、Android和H5/Web平台应用界面控件视觉样式、动画、尺寸等细节特性，
• 喜欢金融理财，有较好的沟通能力，有更多的设计之外的爱好，具备团队和跨团队合作精神、设计表达和展示能力较强者优先；
• 应聘时请携带本人近期作品集。
