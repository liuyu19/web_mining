职位描述：

职责描述：
1. 负责视觉人工智能在线服务产品的设计、开发、部署和优化工作
2. 根据产品和项目需求，参与需求对接，负责分布式系统架构和框架组件的设计选型、开发调优，推动系统架构和框架在各业务系统的落地实施
3. 持续改进系统设计和实现，优化系统性能，保障系统稳定、高效运行

任职要求：
1. 本科或以上学历，计算机或相关专业毕业，5年以上工作经验
2. 计算机基础扎实，有良好的数据结构与算法功底，熟悉Linux系统和网络编程
3. 熟练掌握C++/Golang/Java等任一语言，有相应的服务端开发经验和完整的产品级项目经历
4. 具备分布式/高并发在线系统设计和开发经验
5．有微服务开发经验，熟悉Docker、Kubernetes容器平台相关技术者优先
6. 注重代码质量，有良好的自我驱动能力，工作严谨，责任感强，有协作精神
7. 学习能力强，具备优秀的分析和解决问题能力，具备良好的沟通能力和团队合作意识
