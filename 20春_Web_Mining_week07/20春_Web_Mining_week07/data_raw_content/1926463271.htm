职位描述：

高级硬件产品经理  （安防摄像头方向）

1. 负责开发生态链硬件产品（摄像头产品），包括产品的规划，设计和项目执行以支持生态链物联网产品的发展和布局； 

2. 进行产品的深度市场调研以了解客户的需求和痛点，对顾客画像和行为及需求分析，进行产品规划，概念定义，功能设计等； 

3. 负责与研发团队（包括结构、质量、ID等）或公司其他部门（策划、销售等）沟通明确需求，确保产品按照规划设计与实现 

4. 追踪产品的整个生命周期，关注用户体验与反馈，及时发现和解决问题 

5. 持续优化、改进产品的体验与性能，实现产品有序更新与迭代 


任职要求： 

1.工业设计、通信、计算机等相关专业，本科及以上学历 

2.摄像头或安防监控行业5年以上产品或研发经验，精通SMB级安防行业解决方案构建 

3.熟悉硬件产品设计、研发、开模、生产，从无到有的完整过程 

4.熟练使用Excel、PPT、Visio等办公软件，具备一定的数据分析能力

5.具备良好的协调沟通、创新能力及合作精神，团队意识强。
