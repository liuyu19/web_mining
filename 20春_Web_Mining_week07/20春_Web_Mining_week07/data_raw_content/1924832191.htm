职位描述：

职责描述：
1. 主站PC&APP端平台产品&营销导购内容的视觉体验设计，独立把控风格与推动进度；
2. 尝试探索前瞻性的视觉风格，创建及维护设计系统和平台链路规范体系；
3. 对于动效互动设计趋势敏感，有手绘能力及经验加分；
4. 参与团队的专业化建设和设计思维传播。

任职要求：
1. 三年以上相关工作经验；对设计定位、风格构建、平台产品规范沉淀有一定的理解；
2. 善于交流、具备标准化团队协作流程思维及合作精神，能承受一定的工作压力；
3. 关注细节，富有激情，对设计趋势有敏锐的嗅觉；
4. 追求极致的设计，有卓越的界面结合图形表达能力和视觉创造力。
