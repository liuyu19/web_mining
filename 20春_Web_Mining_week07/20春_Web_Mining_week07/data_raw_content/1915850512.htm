职位描述：

工作职责:
1. 负责公司产品或项目Web前端研发，并配合后端的开发。
2. 根据需求实现页面交互效果。

任职资格:
1. 熟练掌握前端开发技术(HTML5、JS、JSON、XHTML、CSS3)，了解各项技术的相关标准
2. 3年以上的web前端开发经验，前端基础扎实,熟悉响应式布局，熟练使用CSS3，ES6语法；
3. 熟悉前端模块化开发、有性能优化和跨浏览器兼容实践经验，具备实际部署优化的经验；
4. 对前端工程化开发有深入了解，熟悉grunt/gulp/webpack等构建工具配置使用；
5. 熟练使用React, Vue, Angular中的一种框架以及其生态的UI框架，2个以上项目的实际经验；
6.熟悉前端性能优化，并对用户体验、可用性、易用性有深入理解；
7.熟悉使用至少一门后端语言的优先考虑。
