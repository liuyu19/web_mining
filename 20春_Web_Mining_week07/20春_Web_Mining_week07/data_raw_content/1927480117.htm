职位描述：

岗位职责：
1. 主导产品系列视觉风格定义，形成设计语言，输出产品产品上市及持续营销阶段的关键节点物料，包含但不限制于：产品主视觉、发布会KV、keynote、产品图、产品卖点图，节日KV视觉等；
2. 根据项目目标及产品卖点，组织概念讨论，定制视觉创意方案，清晰表达设计意图；
3. 根据项目目标及内容，合理统筹内外资源，输出执行计划，并推动视觉内容的落地，保障落地效果；
4. 优化设计流程，打造高效的业务协作模式。

岗位要求：
1. 本科及以上学历，设计等艺术类高校相关专业；
2. 3年以上平面/视觉/品牌等设计领域工作经验，熟练应用Adobe CC等通用绘图软件；
3. 对品牌及流行趋势有敏锐的洞察力，富于创新、思维灵活，具备良好的审美品位；
4. 优秀的组织协调能力及沟通理解能力，有良好的团队合作精神。
