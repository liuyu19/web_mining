职位描述：

职位描述：
1. 负责华云私有云产品从0到1的UX设计工作(包括B端WEB后台产品、Windows、Linux、MacOS PC端、移动IOS、Android手机、平板端)从设计策略到设计执行的全流程工作；
2. 负责制定迭代界面设计标准和规范，编写界面交互设计说明书，梳理交互逻辑和交互原则；
3. 配合产品经理完成界面的信息架构、流程设计和原型设计。完成界面交互行为和功能的改良，提高产品的易用性；
4. 对产品功能点的数据变化与用户满意度有极高的敏感度，灵活调整设计视角与工作方式；
5. 关注并分析流行产品设计趋势，研究目标用户需求，体验诉求与审美倾向，并迅速优化现有产品。
任职要求：
1. 本科及以上学历，设计类相关专业；注重能力与天赋，不限经验年限；
2. 具备成功商业化产品的项目经验；有中大型B端工具类产品设计经验者优先（简历请附作品）；
3. 精通各类交互与视觉设计软件；熟悉Web、Windows、Android端的官方设计规范，同时是深度用户；
4. 精通设计软件如Figma、Sketch、Adobe XD、PS等，熟悉其配套工具链（版本控制、原型、向开发交付），如Sketch (Design) + Abstract (Version) + Zeplin (Handoff) + InVision (Present / Prototype)；
5. 保持开放的学习心态，跨平台设计能力强，理解华云的产品与商业属性，主导设计的产品目标用户习惯Windows平台；
6. 具备一定的产品思维和较好的用户同理心，能够参与用户研究、交互设计、设计原型开发等工作；
7. 具有优秀的视觉设计能力，扎实的美术基础，对视觉设计有激情和创新精神；极度重视细节，追求像素级完美，热爱人机交互与用户洞察；
8. 具有良好的逻辑思维能力，能够快速理解产品和需求，对前沿科技，设计探索有浓厚兴趣，自我驱动力强；
9. 良好的团队合作精神，富有工作激情、创造力和高度责任感；
10. 有良好的业务理解力，对云计算行业有一定了解的优先。

云服务提供商，是国家课题承接单位、中国十大云计算解决方案提供商。
