职位描述：

工作职责：
加入全球最大的企业服务平台，打造创新性的、最优秀的、高效的下一代企业级实时音视频通信网络服务。 1 、负责公司视频引擎和云服务的架构设计，持续优化提升用户体验； 2 、负责视频编解码算法在不同终端（iOS/Android/Windows/Mac）的适配开发和优化； 3 、负责解决音视频产品开发和实施中的各类关键视频技术问题，协调业务团队推动实时通讯产品落地； 4、 组建掌握关键技术的、世界一流的音视频研发团队。
任职要求：
1. 熟悉会话控制协议，至少熟悉一种以上的会话控制协议； 2. 熟悉MCU功能，熟悉Realtime Network的架构； 3. 了解掌握视频压缩技术的基本原理，包括H.264, H.265等Codec； 4. 了解各个平台的多媒体技术，包括Windows、iOS、Android等平台Capture、Render的实现机制； 5. 熟悉Realtime Media的拥塞控制和错误恢复技术、了解实时通讯相关的质量主客观评测； 6. 至少具备声学数字信号处理，图像处理，计算机视觉其中一类算法工作经验； 7. 有优秀的沟通能力，团队协作能力和组织建设能力。
