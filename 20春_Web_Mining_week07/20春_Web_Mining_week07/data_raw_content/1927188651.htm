职位描述：

职责描述：
1. 配合产品经理和UI设计师，提升产品用户体验，改善产品易用性。
2. 负责产品特性的持续迭代工作，根据产品迭代节奏输出前端设计方案。
3. 负责前端合下框架设计与功能开发。

任职要求：
1. 精通HTML/CSS/JavaScript，熟悉W3C标准。
2. 熟悉React/Vue.js/jQuery/Angular JS/Node.js/Sea.js和Nginx等相关技术。
3. 掌握UI设计原则，对用户体验设计又一定的认知。
