职位描述：

【岗位职责】
1. 参与KLOOK各业务线的iOS APP的设计和研发工作
2. 负责管理自身项目和需求的优先级、按时高质量交付
3. 与产品经理沟通需求，进行相关的技术文档编写，方案设计
4. 设计良好的代码结构，不断迭代重构
5. 指导并带领初级工程师共同完成研发任务
6. 和产品同学一起提升国际化APP的用户体验

【任职资格】
1. 5年或以上iOS开发经验，有成功开发并上线到AppStore的经验
2. 良好的计算机基础，数据结构和算法。对操作系统相关的线程、内存、文件有深入了解的优先,熟练使用Objective C，熟悉iOS的Runtime, 内存管理机制和多线程开发，或具有良好的C/C++语言基础，了解内存和指针概念
3. 对于网络编程有一定的经验，对于HTTP的协议栈能够有深入的认知。
4. 对于优化程序的性能有一定经验，了解并使用面向接口开发模式，对http协议有一定了解，并可以熟练使用，熟悉XML和JSON数据结构
5. 对于MVC框架或常见设计模式有自己的理解，并熟练使用，有良好的组织代码习惯
6. 逻辑思维能力强，思路清晰，有创造性，有良好的英文资料阅读能力

【KLOOK福利】
1、舒适、健康、人性化的办公环境；
2、扁平化的团队管理，业务飞速发展，你的能力和价值很快会被大家看到；
3、开放和充满激情的国际化团队，男女高颜值并且比例协调，每个人都有那么一点传奇；
4、独特的创业团队氛围会让你在完成工作之余，还有自己的时间学习、成长；
5、还要跟Homie同富游：团队足迹踏遍国家50+，等你来一起占领地球；
6、团队天天下午茶，周周小聚餐，月胖三斤不是梦。

企业介绍：

【KLOOK是谁】
KLOOK客路旅行提供世界400个目的地的旅游体验服务，包括景点门票、一日游、独特体验、必吃美食、WiFi及当地交通等。KLOOK与超过10,000位供应商伙伴合作，提供超过100,000种活动及旅行服务。目前，KLOOK总融资额超35亿元人民币，获得创投机构的支持，包括红杉资本中国基金、软银愿景基金、经纬中国、高盛、博裕资本、TCV 、微光创投及华创资本等。


【KLOOK客路团队】
- 于2014年成立于香港，在香港（总部）、深圳（全球技术研发中心）、台北、新加坡、首尔、曼谷、吉隆坡、东京、马尼拉、胡志明市、雅加达、迪拜、孟买、宿务及阿姆斯特丹等地设有29个当地办公室；
- 国际化团队，超过2000名团队成员，来自超过30个国家及地区；
- 平台已汇聚500多万条真实的参与评论，帮助旅行者更轻松地选择的自由行活动；
- 支持41种货币支付及12种语言：英语、简中、繁中（港／台）、韩语、泰语、越南语、印尼语、意大利语等。

【KLOOK荣誉】
-  KLOOK App多次获得App Store和Google Play“年度精选App”荣誉；  
-  KLOOK提倡体验式旅行，上千位KLOOK旅行体验师深入全球，为用户甄选出最精华当地玩乐；
-  KlOOK为用户提供即时、简便的服务，包括会员专享价格，免排队通道以及其他特权。
