平面设计师

职责：
1.   负责公司的新品上市、营销活动、新媒体宣传等方面的视觉设计工作
2.   独立负责项目相关的平面创意和设计工作
3.   负责完成领导交代的其他工作


要求：
1.   本科以上学历，平面设计、艺术设计等相关专业，有2年以上相关工作经验
2.   精通Photoshop、Illustrator等设计软件
3.   有良好的视觉审美和商业化思维
4.   有良好的沟通和团队合作能力
5.   执行力强
6.   善于手绘、摄影或精修人物能力者优先
