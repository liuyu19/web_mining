职位描述：

岗位职责:
1.    负责小米电视OTT应用商店的应用引入和开发者服务，包括应用评估、合作洽谈、协议签署、上架跟进、结算等。
2.    深入挖掘优质应用潜力，通过应用联运、内容引入等方式提升用户量和收入，提升用户体验和满意度。
3.    负责小米影视会员或儿童会员等增值服务，如联合会员、异业合作等，提升会员转化和收入。
4.    参与或负责其他新业务方向的调研、开拓及落地执行。
任职要求：
1.    本科以上学历，2年以上互联网商务或媒介异业合作等工作经验，有BAT工作经历、大型项目合作经验者优先。
2.    具备良好的沟通能力和商务拓展能力，形象气质佳。
3.    擅于思考分析提出解决方案，逻辑清晰，执行力和推动力强，具备一定的调研能力。
4.    海外留学经历更佳
