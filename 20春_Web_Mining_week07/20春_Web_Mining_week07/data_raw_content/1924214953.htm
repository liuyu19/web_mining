职位描述：

职责描述：
1. 负责幸福APP的H5页面开发；
2. 负责前端产品的页面制作，根据UI设计图完成html5应用的编码；
3. 测试维护并持续优化现有产品，修复bug，提高前端的用户体验及性能；
4. 优化前端体验和页面响应速度，并保证兼容性和执行效率；
5. 配合开发工程师一起研讨技术实现方案，进行应用及系统整合。

任职要求：
1、本科及以上学历，3-5年APP类H5应用开发经验；
2、精通HTML、CSS、JavaScript、AJAX、Jquery、Angular等，能快速高效实现各种交互效果；
3、能独立完成相关页面切图工作；
4、能够很好地与产品、设计师和后台开发人员沟通协作，具有良好的团队合作、协作精神；
5、具有与移动端混合开发经验优先考虑；
6、合理优化js / css等代码，使得网站体验更加优越

企业介绍：

方太集团（以下简称“方太”）创建于1996年。二十一年来忠于初心，始终专注于高端嵌入式厨房电器的研发和制造，致力于为追求高品质生活的人们提供优质的产品和服务，打造健康环保有品位有文化的生活方式，让千万家庭享受更加幸福安心的生活。
方太业务涉及厨房电器、集成厨房以及海外事业三大领域，其中FOTILE方太品牌专注于嵌入式厨房电器业务，现拥有吸油烟机、嵌入式灶具、嵌入式消毒柜、嵌入式微波炉、嵌入式烤箱、嵌入式蒸箱、热水器、水槽洗碗机、蒸微一体机、净水机十大产品线。
方太目前在全国已有员工18000余人，除雄厚的本土设计实力，方太还拥有来自韩、日等地的设计力量以及高端厨房生产设备及国际工业制造先进技术。方太坚持每年将不少于销售收入的5%投入研发，拥有包含厨房电器领域专家在内的400余人的研发人才团队，国家认定的企业技术中心和行业前沿的厨电实验室，包括两个“中国合格评定国家认可委员会认可实验室”，分别在德国、日本等地设立研究院，并于2010年被国家知识产权局评为“全国企事业知识产权试点单位”。截止2018年1月，截止2018年8月，方太已拥有了2066项专利，其中实用专利数1405项，发明专利数量316项。雄厚的科研力量，确保了方太的创新实力。
方太，一家以 “人品、企品、产品三品合一”为核心价值观；以“为了亿万家庭的幸福”为企业使命；以使命、愿景、价值观驱动；以22年来对高品质厨电的专注与坚持；向着“成为一家伟大的企业”宏伟愿景大步迈进。
