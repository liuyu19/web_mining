职位描述：

岗位职责:
1.参与流媒体服务器架构的设计与开发；
2.实现直播、短视频系统/子系统架构和开发；
3.对直播、短视频系统不断的改造与优化；
4.负责技术难点的攻关和预研工作；
5.线上直播、短视频系统的维护。
任职要求:
1.计算机相关专业，3年以上多媒体相关开发经验；
2.具有良好的数据结构和算法基础、工程素养以及关于Paper, IETF RFC与Open Source 的基本阅读与理解能力；
3.精通C/C++、Socket编程、网络编程、多进程多线程编程技术；
4. 熟悉音视频的编解码及网络传输优化；
5.熟悉H.264、H.265、AAC、FLV、TS、MP4等标准规范，熟悉RTP/RTCP/RTMP/HLS/DASH/HTTP等相关协议规范, 能基于开源音视频编解码软件（如ffmpeg,x264等）定制和优化；
6.熟悉TCP/IP协议栈，熟悉网络通信协议以及Webrtc系统架构和通讯标准，
7.熟悉分布式系统原理，理解集群开发和高可用相关实践，擅长Linux平台下高性能软件架构与设计；
8.熟悉常用开源软件如MySQL、Redis的基本操作，了解相关原理及其实现；
9.了解Go, Lua, Python语言者优先。
