职位描述：

岗位描述:
1. 把控贝仓、贝贝等App整体的营销设计工作，进行高质量的营销视觉界面设计，涵盖店主运营、粉丝运营、平台运营等版块设计工作，互联网大促经验丰富者优先。
2. 推动各营销主题整合推广，能够负责创意传播、营销传播、事件传播、活动大促等创意视觉设计工作。
3. 负责团队规划 、管理，把控创意方向和产出质量；带领团队制定设计标准，用数据评估设计结果，并推进落实，从设计侧推动产品优化。
4. 具备团队合作精神，提供专业的设计支持，分享设计经验、推动并提高团队的整体设计能力，给团队带来优秀的设计经验。

岗位要求:
1. 本科以上学历，美术、视觉传达、多媒体设计等相关专业，4年以上工作经验，有团队管理经验者优先。
2. 有扎实的美术功底、热爱设计、拥有丰富的营销视觉设计经验，能对造型、颜色、排版、字体设计等均有良好的把控能力。
3. 思维开阔，有强的动手能力，具有敏锐的设计触觉，能将时下的趋势和设计结合。
4. 较强的思维能力，具有良好的沟通和设计表达能力，良好的团队协作态度，主动积极、踏实细心，协调能力强，能独立担当高强度的设计工作。
5. 有Dribbble Player和Behance高质量作品加分；有大型互联网工作经验，或电商独角兽企业工作经验优先。
6. 熟练应用PS（必备），AI（必备），手绘（加分），C4D（加分），AE（加分）等设计工具&技能。

企业介绍：

集团介绍
贝贝集团创建于2011年，旗下拥有贝贝网、贝店、贝仓、贝贷等平台，致力于成为全球领先的科技零售平台。

贝贝创始团队来自阿里巴巴，先后获得IDG资本、高榕资本、今日资本、新天域资本、襄禾资本、高瓴资本、红杉资本、创新工场、北极光创投等数亿美金的风险投资。

愿景：成为全球领先的科技零售平台

使命：让更多人过上更好生活

核心价值观：专业可信赖

旗下业务：
1、贝贝：
贝贝是国内领先的母婴购物平台。贝贝目前拥有1亿家庭女性用户，通过专业买手的层层精选，消费者可以在贝贝买到各种价廉物美的母婴、居家百货、美食、服饰等商品
2、贝店：
贝店成立于2017年8月，是国内领先的社交电商平台，秉承“让更多人过上更好生活”的使命，为消费者提供居家、服饰、美食、美妆、母婴等全球好货。与传统平台不同，贝店通过人与人的社交化分享传播，实现消费者、店主以及供应链的三方连接，将精选好货送达消费者手中
3、贝贷：
贝贷是贝贝集团旗下的消费金融平台，联合银行等金融机构，为亿万家庭提供分期购物、消费借贷等金融服务。
4、贝仓：
贝仓是贝贝集团旗下的品牌特卖平台。贝仓通过S2B2C的创新模式，一端连接源头品牌商，通过专业买手挑选有品质保证的高性价比货源；一端连接微商、实体店主、代购等分销商，提供货源、供应链、开店工具等一站式解决方案，以社交化方式销售商品。

贝贝是一群志同道合的小伙伴快乐工作的地方，我们希望成为一个专业可信赖的团队，为用户和社会持续创造更多价值，成为全球领先的科技零售平台。“优秀是一种习惯”是我们的座右铭，我们相信平凡人可做非凡事，我们渴望用自己的青春让更多人过上更好生活。
