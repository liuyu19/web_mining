职位描述：

职责描述：
1、 参与公司运维平台的开发，改进用户体验；
2、参与数据中心海量资产运维平台的开发；
3、提高系统运维工作的自动化水平，开发建设云计算的相关组件；
4、开发维护企业级别监控、日志、报警系统

任职要求：
1 、熟悉JavaScript, HTML5, CSS3 的基本技术，至少掌握一种主流框架（Angular，React，Vue）；
2、熟悉Linux操作系统，掌握Python、Java、Golang中的至少一门语言；
3、熟悉Mysql、Redis、RabbitMQ、Celery、MongDB的基本技术；
4、熟悉配置管理工具（Ansible/Salt/Puppet）中的一种或几种 ；
5、熟悉Django、Tornada、Flask 等主流python框架者加分；
6、有Jinkens CI/CD、Harbor、Docker、K8S、ELK、Promethues、Grafana、Zabbix相关经验者加分；
7、对技术有强烈的进取心，具有良好的沟通能力和团队合作精神、优秀的分析问题和解决问题的能力。

企业介绍：

依图智能安防技术主要由人工智能算法、高性能计算、分布式计算和存储、大规模运维等构成。其中视觉智能算法主要涉及人脸识别、车辆车型识别、文字识别、目标跟踪、图像特征搜索等技术。

依图技术团队来自MIT 、Google、阿里巴巴等顶级学术和工业机构，将最前沿的美国学术理念和扎实的工业界实战经验相结合，应用先进的人工智能算法，包括深度学习、统计建模技术，创新性的解决技术和应用结合的难题，能有效处理大规模样本（亿级训练样本）和少量训练样本的机器学习问题，具备丰富的世界领先的技术落地经验，包括超大规模识别（十亿级比对）、超低误报率（亿分之一误报）、多类型终端感知（静态图片、动态视频、低清画质、网纹噪音）、多人种跨年龄段识别等复杂应用场景 。
