职位描述：

工作职责:
1. 负责产品设计的前端开发工作，能够熟练手写HTML/CSS/JavaScript；
2. 基于JavaScript框架的应用和项目开发；
3. 负责与Web UI设计师、软件开发工程师协作完成web前端的开发工作；
4. 能够快速学习并使用部门内部开发的JS框架和前端规范
5. 对前端技术的发展趋势保持关注，能不断学习业界最新技术
任职资格:
1. 5年以上前端开发经验, 具有良好的交流沟通能力和团队协作能力；
2. 精通html/xhtml、css，对web前沿产品和技术感兴趣并愿意做研究
3. 熟悉Web2.0，了解浏览器端原理，掌握网站性能优化基本知识；
4. 熟悉js的运行机制，熟悉jQuery/YUI/MooTools/Dojo中至少一种框架；了解框架设计原理、适用场景，若能灵活一些思想进实际工作更好；
5. 有asp 、asp.net、php 或 jsp 相关经验的优先

企业介绍：

同程艺龙由同程旅游集团旗下的同程网络板块和艺龙旅行网在2017年底合并而成。
同程旅游集团是一个多元化旅游集团，中国领先的在线旅游服务商和一站式旅游预订平台，是“中国民营企业500强”和“中国旅游集团20强”企业。同程旅游集团下辖同程网络和同程控股两大板块，其中同程网络业务涵盖同程机票、酒店、火车票、汽车票等多项业务。
艺龙旅行网是中国领先的在线旅行服务提供商之一，深耕OTA行业多年，秉承专业团队的管理运作，在酒店等领域享有盛誉。
合并后，同程艺龙定位于领先的在线旅行服务平台，将同程网络和艺龙各自核心业务优势互补，并占有微信钱包中机票火车票的入口和酒店入口。目前同程艺龙已有近两亿的消费会员，2017年服务超过五亿人次，合并交易额超过千亿人民币，并已实现年超过五亿人民币的规模化盈利。
