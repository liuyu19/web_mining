职位描述：

岗位职责：
1. 对公司现有产品（豆芽及B端相关产品）的可用性进行测试和评估，并进行持续性的优化迭代。
2. 配合公司相关部门需求规划产品及创新，做好竟品分析，梳理功能流程和页面流程，输出高质量的交互原型；
3. 提出三端交互方案，并输出相关规范标准文档；
4. 负责维护和更新界面设计标准和规范，并定期做质检报告汇报。
任职要求：
1.工业设计、交互设计、心理学、人因工程相关专业背景专科及以上学历，5年以上交互设计工作经验；
2.对交互设计以及用户体验有较深的理解，可独立完成；
3.有大局观，能够分析业务需求并加以归纳，分解交互需求；
4.个性乐观、开朗、优秀的沟通与协调能力；
5.有创新意识，关注细节，敢于决策。
技能要求：
1. 熟练使用Axure，Sketch，Visio等交互设计软件；
2. 深刻理解桌面端/移动端样式规范；
3. 有AntDesign、WeUI、Bootstrap、微信小程序等前端样式库项经验；
4. 熟练使用语雀、markdown等文档类工具；
