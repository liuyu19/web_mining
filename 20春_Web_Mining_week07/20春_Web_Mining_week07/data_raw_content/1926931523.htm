职位描述：

工作职责:
1. 参加产品需求和设计文档评审，从测试角度分析可能的问题并提出建议；
2. 参加营销云2C业务线核心模块的WEB端、APP端各项功能测试；
3. 编写测试用例、设计测试数据、执行功能测试、跟踪测试进度、验证线上问题；
4. 在现有接口自动化测试平台上，完善营销云核心功能的接口测试用例；
5. 对现有系统测试用例，参与UI自动化测试脚本编写和维护；
6. 在测试各环节与开发、产品等部门沟通保证测试输入和输出的正确性和完备性；
7. 参与产品质量评估，持续改进研发测试流程；
8. 帮助研发定位问题，并对产品提出改进建议。


任职资格:
1. 全日制计算机相关专业本科以上学历;
2. 熟悉软件测试流程与测试用例设计方法;
3. 能够独立编写测试用例，执行测试过程，完成问题报告；
4. 熟悉至少一种开发语言或脚本语言；如：Java，C++, Shell，Python，Perl，Ruby等；
5. 熟悉常见的数据库（Oracle、MySQL等），熟练使用常用的SQL语句进行数据库操作；
6. 熟练掌握至少一种测试管理工具和缺陷管理工具；
7. 熟悉常用测试框架TestNG, Robot Framework, Selenium, Appium等;
8. 对软件测试有浓厚的兴趣和丰富的经验，有很强的分析能力和定位问题的能力；
9. 具备良好的沟通能力、团队合作和质量意识，能承受工作压力；
