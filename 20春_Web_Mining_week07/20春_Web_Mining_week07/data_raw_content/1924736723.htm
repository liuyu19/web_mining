职位描述：

工作职责：
负责手游产品海外版本系统本地化设计，对主流游戏的游戏内界面设计、经济系统、社交系统等系统模块了解；
与美术和开发进行良好的沟通合作，保障游戏的功能实现及效果分析；
提炼海外用户需求或通过版本效果分析来梳理需求，推动版本持续调优完善。
任职要求：
对各种游戏平台（如战网，EPIC平台）的流程机制熟悉的优先，对界面UI理解深入的优先，对海外合规方面熟悉的优先；
参加过大型手游的研发，有国际化项目或留学经验优先。英语能力好优先，沟通能力好优先；
深入体验大量成功游戏的系统，了解系统设计的逻辑，对界面UI，账号，社交，支付，商业化等任一模块有过深入了解；
3年以上游戏策划经验。
