职位描述：

岗位职责：

1、根据产品需求，负责后端应用服务的模块设计和开发，并参与部分架构设计；

2、负责产品的框架研发以及相关产品线的分析与设计以及开发；

3、负责与初级和中级程序员以及外部供应商的研发人员进行技术沟通与协同。

任职要求：

1. 全日制大学本科以上学历， 计算机或相关专业；

2. 5年及以上Java开发经验；

3. 在网络、操作系统、程序语言、算法、数据结构、数据库等方面具备一定理论基础；

4. 深入理解JVM、Java设计模式、OO思想， 在Java内存管理、性能调优、并行开发等方面具相当经验； 精通Java Web 编程； 能够编写多线程、socket，IO等相关的高质量代码，熟悉Java单元测试；

5. 熟悉掌握spring、springmvc/struts、mybatis/hibernate等框架基础原理，精通其中至少一套组合的应用开发；

6. 熟悉tomcat、webLogic、websphere等主流应用服务器之一；

7. 熟悉MySQL，SQL SERVER，ORACLE 之一，具有良好的sql编写和优化能力；

8、精通B/S结构，掌握JS,JQuery,json,ajax等技术。

9. 具分布式系统开发经验， 熟悉负载均衡、缓存、消息等相关机制。 精通 redis， nginx, kafka，zookeeper、分布式mysql其中一项或数项者优先；

10. 具微服务经验， 熟悉spring cloud 或者dubbo者优先；

11. 同时掌握其他主流开发语言者优先，包括并不限于： C++、C#、Python、Node.JS;

12. 有大型、超大型互联网应用平台设计、开发经验者优先。对手机端APP开发亦有一定程度了解、甚至熟悉者更佳；

13.有金融类、钱包类、支付类、电子商务类项目核心功能开发的优先。

14.有数据开放平台或者其他大型项目核心功能开发经验者优先.

企业介绍：

宗申集团始创于1982，现已成长为一家总资产和销售收入均已突破200亿元人民币的大型集团企业。宗申制造工厂（或子公司）遍布在中国重庆、广东、江苏、河南、河北、天津和越南、泰国、巴西等地，正逐步成为“世界一流中小型动力系统集成服务商”。

宗申集团连续16年入选重庆工业50强，连续入选重庆市“纳税50强”，是中国民营企业500强、中国机械500强、中国制造500强，并入选中国企业500强。宗申拥有国家级“企业技术中心”、“机械工程专业博士后科研工作站”，宗申检测中心通过CNAS国家实验室认可。

宗申与意大利比亚乔（PIGGIO）、塞尔瓦（SELVA）、巴贝锐（BARBIERI）、白俄罗斯戈梅利等国际知名企业在中国设有合资工厂；宗申还与合作伙伴在上海、深圳、香港、加拿大、美国等地设有包括水上飞机运营、地产业务开发在内的合资运营公司。
