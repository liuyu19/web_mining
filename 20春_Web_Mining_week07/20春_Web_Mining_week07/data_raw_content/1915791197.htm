职位描述：

工作职责：
1. 负责整条B或者C端平台业务设计落地，能够根据产品目标设计并提炼最优解决方案，输出完整的对接页面及说明文档； 2. 把控产品的整体UI风格，保证产品的品质与审美；规范产品在用户体验上的统一性和一致性；； 3. 通过与产品讨论，在现有的页面结构及用户操作流程上完成细化体验； 4. 完成 design system 的规范搭建：包含 文字、色彩、icon、组件等等，规范化设计文件输出； 5. 对接相应的插画设计； 6. 完成页面的动效化（包含页面动效及icon动效）并输出文件和规范对接文档； 7. 与其他设计师、产品经理、工程师紧密沟通协作，共同达成目标。
任职要求：
1. 3年以上设计经历，在B端及C端设计有丰富经验，有成功案例者优先； 2. 有体育相关项目经历，有海外相关项目经历； 3. 扎实的且完整的设计理论基础，且创新意识强，能够洞察到当前设计的发展趋势，关注最新的技术和设计动态； 4. 具备娴熟的设计技巧和逻辑思维能力，能够利用设计对产品服务； 5. 具有良好的沟通及呈现技巧，具备工作责任心与团队精神； 6. 结果导向，有良好的自我驱动力、沟通能力和抗压能力； 7. 具有优秀的中英文文档写作能力，良好的英文口头表达能力。

阿里巴巴集团的使命是让天下没有难做的生意。
