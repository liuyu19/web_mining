职位描述：

岗位职责：
1、负责根据策划需求文档，设计制作符合项目美术风格的游戏界面、交互动画等；
2、提供高质量的游戏产品界面设计方案，对游戏产品进行视觉改进和体验研究；
3、能独立工作或与团队成员协作完成整体界面设计制作工作，正确理解策划需求案和项目需求；
4. 保持与相关部门（策划组、程序组）的沟通顺畅，理解项目，研究同类产品优势，并乐于分享经验。
任职要求：
1、3年以上端游或手机游戏产品界面设计经验；
2、有良好的美术功底和审美，熟悉写实风格，能够适应不同美术风格和配色方案，有二次元项目设计经验；
3、熟练使用Dreamweaver、Photoshop、Illustrator等相关软件;
4、具备较好的沟通能力和逻辑思维能力，有良好的团队合作精神，责任心强、能承受一定的工作压力。

企业介绍：

全球领先的互动娱乐品牌
基于全球化的视野，立足于移动互联网时代，英雄互娱致力于为海内外用户提供优质的移动游戏。
以泛娱乐为战略目标，推广“移动电竞”概念，打造全方位的移动电竞娱乐方式，开创全民互动娱乐新时代。英雄互娱的创始团队由游戏领域经验丰富的行业精英组成。 团队成员都有着丰富的移动游戏自研和发行经验。
一流的移动游戏发行团队，庞大而稳固的告联盟网络渠道、终端预装渠道、第三方应用商店、开放平台渠道和各大移动运营商渠道。
除发行自主研发的游戏产品外，还积极寻求优质精品网游产品进行独代、首发和联运合作，与国内外优秀的手机游戏开发商共赢手机游戏市场。
