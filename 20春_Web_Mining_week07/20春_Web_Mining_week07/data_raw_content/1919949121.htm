职位描述：

工作职责：
游戏产品界面设计，优化产品界面用户体验；
界面动态效果创意制作；
进行系统功能图标，道具图标的概念设计；
负责游戏宣传所需平面设计；
负责图标的外包质量审核把控以及进度管理，保持风格统一性和资源规范性。
任职要求：
大专以上学历，设计相关专业；
熟练掌握photoshop、illustartor等图像设计软件；
深厚的美术功底，色彩感觉敏锐，想象力丰富，具有独特的审美能力，具备高品质的视觉及创意设计指导能力；
3-5年的UI相关的工作经验，有过至少2款以上完整的手机游戏的UI设计经验，具有较强的学习能力和钻研精神；
能驾驭Q版UI美术风格，并且风格能适应全球化，如果能驾驭其他多种美术风格最佳；
能够统一UI元素，统一UI风格，并能与原画很好的融合，同时为UI特效预留空间，方便特效表达；
有强烈责任心，逻辑思维及分析能力强，有较强的分析判断能力，有较强的项目推动能力。
