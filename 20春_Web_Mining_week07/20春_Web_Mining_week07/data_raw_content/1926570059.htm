职位描述：

职责描述：
1、负责iOS客户端应用架构和开发工作；
2、独立架构、完成 iphone客户端程序的开发；
3、验证和修正测试中发现的问题，保障体验和质量；
4、支持公司内部 ios 开发技术培训；

任职要求：
1、本科及以上学历，计算机、通讯相关专业毕业，5年以上iOS客户端开发经验，精通 Objective-C 程序设计，精通 iOS SDK 及相关开发工具；
2、熟悉 iOS 框架及各种特性，熟练使用iOS的各种UI组件并了解其内部原理，了解基本的交互知识；
3、熟悉cocoa touch，core data，iOS runtime，精通OS X/iOS下的并行开发、网络、内存管理、GUI开发；熟悉多线程开发，熟悉 http 工作原理，理解常用数据交换格式（JSON、XML 等）；
4、熟悉iOS平台常用的开发模式，理解MVC，MVVM并在项目中有所实践；
5、有iOS的UI控件有优化经验、网络优化、内存优化、性能调优经验；
6、 有大型app项目经验者优先；有跨端技术开发经验优先。
