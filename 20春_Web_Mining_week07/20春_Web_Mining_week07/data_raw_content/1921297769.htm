职位描述：

工作职责：
1. 负责品牌 VI 体系设计执行与完善、品牌衍生体系完善：如品牌字体、品牌规范、周边延伸应用、礼品包装、互动传播、审查机制等；
2. 对品牌包装、形象推广、Motion Brand Identity 和互联网设计等领域，有着浓厚兴趣且掌握相关的优秀专业能力，具备从创意构思到画面执行的能力，有一定的策划和文案撰写能力；
3. 负责品牌识别多端表达规则制定与执行管理，包括：线上产品全链路应用、线下应用等方案和设计规范制定、维护；
4. 负责企业品牌衍生商品/礼品概念规划、商品研发设计、创意包装与展示；
5. 负责品牌创意设计团队的管理，负责工作规划与协调，设计评审，优化工作流程，设计师工作评估与指导。

任职要求：
1. 本科及以上学历，五年以上品牌设计经验，具备集团企业、互联网平台层级品牌VI 设计管理经验，拥有品牌策略、创意策划文案能力者加分；
2. 具备抽象概念转换成创意视觉的设计执行力（3D /影视剪辑 /动效能力加分）；
3. 具有敏锐洞察力，并能让项目创意真实落地，有清晰的沟通表达能力和逻辑思考能力，能主导项目且有较强团队协作能力，主动积极，踏实细心，同时具备可独立把控和完成项目的能力；
4. 有专业知识深度与视野，具有丰富的想象力，热衷于品牌文化设计和视觉设计，善于把握潮流动向，能够结合自身设计经验，根据业务需求推动方法论建设以及相关研究工作；
5. 有企业品牌形象延伸商品化经验，从概念策划、创意方案、商品研发生产全流程经验者优先；
6. 投递简历请附带作品集或作品链接。

企业介绍：

流利说®（NYSE：LAIX）是领先的人工智能驱动的教育科技公司，由王翌博士和胡哲人、林晖博士于2012年9月共同创立。作为智能教育的倡行者，流利说®拥有一支业内领先的人工智能团队，其自主研发的人工智能英语老师，基于深度学习技术，能够为每一位用户提供个性化、自适应的学习课程。

经过多年积累，流利说®已拥有巨型的“中国人英语语音数据库”。在此基础上，公司自主研发了领先的英语口语评测、写作打分引擎和深度自适应学习系统，致力于为用户提供一整套系统性的英语学习解决方案，从听、说、读、写多个维度提升用户的英语水平。

2018年9月，由于品牌升级，公司正式启用“LAIX”作为集团化运营的新名称。LAIX 前三个字母意为：Learning + AI，以人工智能驱动教育，引导新的学习方式；而“X”则代表着未知，寓意解锁人生的无限可能。

流利说®旗下目前拥有包括“流利说®英语”、“流利说®雅思”在内的英语教育类 APP，以及“流利说®懂你英语”、“流利说®阅读”、“地道发音课”等英语学习产品。截止2019年3月31日，旗舰App“流利说®英语”的注册用户数已超过 1.2 亿人，付费学员超过 250 万人，覆盖全球 175 个国家，全国 384 个城市，亦拥有国内领先的英语学习 App 和活跃的语言学习社区，曾被苹果公司评为“App Store精华”和“2013年度精选 App”。

2018年9月27日，流利说®以股票交易代码“LAIX”正式挂牌纽交所，以其开发的教育 3.0 模式，被誉为“AI+教育”第一股。上市前，流利说®曾获得IDG、挚信资本、GGV、华人文化产业投资基金 (CMC)、心元资本、双湖资本、赫斯特资本等知名VC的多轮投资。
