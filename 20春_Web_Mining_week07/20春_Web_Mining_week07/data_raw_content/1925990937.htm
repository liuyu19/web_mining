职位描述：

职责描述：
-负责商业产品的视觉设计工作，参与平台的相关设计以及产品需求设计，能够输出符合业界趋势的有竞争力的视觉设计方案
-快速理解用户及产品特征及市场定位，分析设计趋势，负责To B相关产品的界面视觉设计、运营设计、品牌设计、创意输出
-参与团队建设，组织设计经验的分享和传承，提高团队的设计能力

任职要求：
-3-5年以上用户体验设计相关工作经验，视觉传达、数字媒体设计、艺术设计等相关视觉设计专业背景，能够带领、组织设计师进行设计工作
-具有深厚的设计理论与娴熟的设计技巧，对业界和行业有深入的了解，善于捕捉流行趋势，并善于分析和总结，有提炼设计语言整合设计规范的能力
-热爱设计，富有激情，拥有宽广设计视野与审美标准
-独立支持多项目多任务，并有一定项目管理能力以及良好的带队设计与跨部门沟通协调能力
-擅长手绘、C4D或4A广告公司、影视宣传公司工作背景者优先考虑，投递时务必附带作品集
