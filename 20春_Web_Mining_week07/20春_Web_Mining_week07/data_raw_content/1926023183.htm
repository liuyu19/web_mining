广州赛意信息科技股份有限公司（GuangZhou SIE Consulting Co., Ltd.）  

股票简称：赛意信息，股票代码：300687。成立于2005年，是国内成立较早、目前规模较大、经验丰富的IT咨询服务公司之一，有着企业信息化ERP咨询顾问的“黄埔军校”的美誉。经过十多年的经营和发展，赛意信息以广州为总部，在上海、深圳、顺德、武汉和济南设有分、子公司，在北京、成都设有办事处，拥有2500余名具备丰富IT咨询和实施经验的复合性人才。

赛意信息以ERP系统实施为核心，拥有360°的企业管理信息化解决方案的能力，为数百家中国企业及跨国公司提供了优质的信息化服务。经过多年累积，赛意信息已在家电、电子、钢铁、机械、航空、通讯、金属、汽车和贸易等行业建立了深厚的解决方案优势，服务客户包含：华为、美的、海航、松下、索尼等近千家国内外品牌企业。在长期的实践中，赛意信息申报并获得多项国家技术发明专利和实用新型专利，数十项软件著作权，并荣获“国家高新技术企业”、“ORACLE黄金合作伙伴”、“广州市首批百强科技小巨人”、“广东省诚信示范企业”、“ISO9000：2000质量认证企业”和“CMMI认证企业”等近百项资质荣誉 。

面对新的发展机遇和目标，赛意信息将在不断的实践中累积适合中国市场的信息化服务解决方案，务实创新，成为中国企业信息化服务领域的先驱。
