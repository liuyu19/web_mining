职位描述：

职责描述：

1、负责互联网+AI的医疗运营规划，运营体系的打造，基于平台的内容运营、活动运营、用户运营、对外的培训体系。
2、负责移动医疗APP/微信平台的运营规划，包括流量体系、会员体系、微信个人号、现实与线下的活动、电商导流等的产品运营详细方案策划。
3、规划实体医院的互联网医院申请、对接卫健委的政策支持和在线电子处方流转的运营。
4、对区域的医--患复诊、医--医互动场景化思考，不断发现用户需求及痛点，提供有效的运营解决方案。
5、深入理解医疗和健康产业延展服务的用户体验，在项目推进过程中的跨部门协调沟通工作，协作共赢的同时，确保产品按时完成并高质量上线；



任职要求：
1、本科及以上学历，3年以上移动互联网经验，有电商类、移动医疗运营经验者优先考虑；
2、对移动医疗互联网产品运营发展趋势有良好的判断能力，具备数据分析能力；
3、逻辑思维能力强，具有良好的沟通与语言表达能力，良好的团队协作、执行力与学习能力，以及产品相关文案的撰写能力；
4、工作积极主动，责任心强，对待工作富有热情，能承受工作压力。
5、有平安好医生、阿里健康、微医、丁香园等医疗健康平台或类似平台运营经验优先。
