职位描述：

工作职责：
1.负责医疗影像相关产品的产品规划和落地，可以独立负责2-3条产品线的研发；
2.主导产品从设计到上线的全流程。产出产品需求文档，包括：功能、特性、使用流程、UI/UE、性能要求等，协同相关团队进行产品落地。在过程中不断分析数据和客户调研，进行敏捷迭代优化，持续提升产品体验；
3.制定产品和项目的目标计划，负责把控产品和项目的进度，协调突发问题与质量管理工作；
4.组织团队，带领团队；

岗位要求：
1.医学相关专业背景优先，或深耕于医疗影像产品领域3年及以上优先；
2.具备至少2年以上影像产品研发经历，或具备3年以上互联网医疗产品研发经历；
3.具备良好的客户沟通能力，对于医疗影像市场有自己独到的见解；
4.具备带团队的能力，有不错的领导力和组织能力；
5.工作地点优先杭州，或能够接受长期出差杭州；

企业介绍：

依图智能安防技术主要由人工智能算法、高性能计算、分布式计算和存储、大规模运维等构成。其中视觉智能算法主要涉及人脸识别、车辆车型识别、文字识别、目标跟踪、图像特征搜索等技术。

依图技术团队来自MIT 、Google、阿里巴巴等顶级学术和工业机构，将最前沿的美国学术理念和扎实的工业界实战经验相结合，应用先进的人工智能算法，包括深度学习、统计建模技术，创新性的解决技术和应用结合的难题，能有效处理大规模样本（亿级训练样本）和少量训练样本的机器学习问题，具备丰富的世界领先的技术落地经验，包括超大规模识别（十亿级比对）、超低误报率（亿分之一误报）、多类型终端感知（静态图片、动态视频、低清画质、网纹噪音）、多人种跨年龄段识别等复杂应用场景 。
