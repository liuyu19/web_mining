职位描述：

1.负责前端页面代码的编写，优化及特效的实现；

2.完成前端需求理解、功能设计、迭代开发；

3.快速响应业务需求，及时解决前端问题。

岗位要求：

1. 统招本科及以上学历，计算机相关专业，至少2年以上Web开发经验；

2. HTML、CSS、JavaScript基础扎实，了解HTTP协议以及熟悉浏览器兼容性、终端适配方案；

3.熟练应用Vue及相关组件，熟悉ElementUI或AntUI等常见UI组件库；

4. 良好的UI感知与审美能力，细节控，有数据可视化经验加分；

5.责任心强，具备良好的团队合作精神和承受压力的能力。
