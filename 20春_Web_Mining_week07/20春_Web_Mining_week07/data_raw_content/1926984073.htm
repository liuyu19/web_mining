职位描述：

职责描述：
1、负责车联网相关产品的规划和原型设计；
2、把握用户需求，主导市场分析，竞品分析，用户研究等工作；
3、完成产品需求文档输出，包括功能，业务流程，界面原型等；
4、对用户体验负责，协调资源，跟踪产品落地和制定迭代计划；
5、对产品数据进行分析，和持续性优化和改进


任职要求：
1、热爱生活，乐观积极，善于总结和归纳，长于沟通和表达；
2、至少4年以上互联网以及移动端产品规划经验，并主导过产品全生命周期管理；
3、熟悉Visio、Axure、MindManager等常用设计软件，能够制作产品原型，能够输出工作类PPT；
4、较强的市场分析和需求管理能力，注重细节，追求完美，富有创新意识；
5、对产品数据敏感并有产品数据运营的经验
6、严谨，能够输出高质量的产品产出物，可展示作品。
7、面试时需要带上自己设计过的的产品范例

企业介绍：

奇瑞控股集团有限公司成立于2010年，以汽车产业为基础，充分利用体制优势与品牌效应，整合多种社会资源，稳步实现多元化发展。
集团旗下投资有汽车、汽车零部件生产与研发、造船、金融、旅游地产、现代服务业等业务板块，200多家子公司，业务范围遍布海外80余个国际和地区。目前集团拥有员工4.8万余人，总资产1200多亿元。
秉持“创新、责任、共赢”的理念，奇瑞控股以产业报国为己任，立志在每一个进入的领域成为行业领先者，致力于成为具有全球影响力、值得信赖的多元化控股集团。
