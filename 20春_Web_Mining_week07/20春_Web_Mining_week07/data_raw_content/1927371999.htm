职位描述：

业务介绍：
我们是阿里巴巴新零售事业群-1688技术部-无线研发团队，负责支撑全球最大的B类电商1688APP。1688依托于强大的产地供应链与线上分销体系，覆盖40%以上的淘宝商家及社交电商的货源供给，是国内最领先的线上采购批发平台。

技术介绍：
历经多年电商业务的规模化增长，1688承载了复杂多样的业务形态，也沉淀了极其雄厚的技术能力。在这里你不仅能够广泛吸收用户增长、泛内容营销、智能企业工作台等业界最领先的业务解决方案，更能够在智能UI、边缘计算和端智能、servless等全栈技术领域中突破创新。在这里你不是一颗螺丝钉，而是有机会一起成为电商发展的造风者。

团队文化：
我们是一个追求极致、创新、敏捷的研发团队。敢跨界、有担当，极致的客户价值和体验是我们的做事准则；知商业、懂业务，通过技术创新赋能业务是我们对自己持之以恒的要求。

工作内容：
1. 负责APP用户增长、直播短视频、内容平台、社交电商平台的业务承接，技术沉淀工作。
2. 负责APP整体的开发及架构设计，能带领团队完成设计、实现、关键技术验证、选型等工作，提供一流的体验和稳定性;
3. 主导移动规范制订、技术文档编写、研发流程改进。
4. 调研业界前沿技术并结合产品运营落地（端智能，UI自动化生产，跨平台框架等）

职位要求：
1. 本科及以上学历，计算机或相关专业， 熟悉操作系统、算法与数据结构、多线程、网络编程等计算机基础知识；
2. 三至五年iOS / Android开发经验，精通移动开发框架，熟悉组件化开发、性能优化、软件架构、动态化方案等，有算法工程、前后端经验者优先考虑；
3. 有完整的iOS / Android 项目经验，至少参加过一个完整的商业级手机应用或游戏开发项目；
4. 有强烈的责任心、团队精神和技术热情，善于沟通和合作。
