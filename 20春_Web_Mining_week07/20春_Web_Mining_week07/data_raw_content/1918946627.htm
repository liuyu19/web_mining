职位描述：

岗位职责：
1. 与内外部客户进行沟通，了解需求，并发掘潜在需求；
2. 对内外部平台产品进行原型设计，并调动研发、UI/UE、测试等资源，推动产品按时上线；
3. 负责产品需求分析，产品规划设计和定位，对产品进行整体把握，确保达成相关目标；
4. 负责平台化相关产品、项目的需求管理，包括需求调研、分析，撰写需求文档。

任职要求：
1、具备3-5年平台化产品经验；
2、具有较强的逻辑性和分析能力，有SAAS、PAAS平台工作经验；
3、优秀的沟通能力，主动而有效协调跨部门资源，责任心和团队精神；
4、思路清晰，学习能力强，善于归纳总结，注重产品闭环。

企业介绍：

深圳市柔宇科技有限公司（以下简称“柔宇科技”）是全球柔性显示、柔性传感、虚拟现实显示及相关智能设备的领航者，专注于下一代人机交互及万物互联技术和产品的研究、开发、生产及销售。
柔宇科技于2012年在美国硅谷、中国深圳及香港同步创立，五年内先后获得国内外知名风险投资机构的六轮共同投资。2014年柔宇在全球第一个发布了国际业界最薄、厚度仅0.01毫米的柔性显示屏，引领了柔性显示和柔性电子产业的新潮流。公司创立四年估值突破35亿美元，成为全球成长最快的独角兽科技创业公司之一。
柔宇科技的自主核心技术包括0.01毫米全球最薄彩色柔性显示屏和新型柔性传感器，核心业务包括基于“柔性+”平台的一系列新型智能终端产品及行业解决方案。柔宇已在国内外储备1700余项核心技术知识产权，并将产品销售到20多个国家和地区。
柔宇投资超过100亿的国际柔性显示基地及全球首条具有自主知识产权的类六代全柔性显示屏大规模量产线于2015年10月在深圳筹建，满产规模超过每年5000万片柔性显示屏，可支持大量消费电子、智能交通、智能家居、运动时尚、教育办公、机器人等行业应用，并已与国内外行业伙伴展开广泛合作。
2017年，柔宇在业界首次提出了“柔性+”理念，即柔性电子作为平台型技术，能广泛应用到各行各业，与人工智能和万物互联将深度融合发展。目前，柔宇科技的柔性电子技术已经陆续与各行各业的企业建立了合作伙伴关系，并通过B2B与B2C模式不断推出新的量产产品走入市场。“柔性+”正在逐步走入人们生活的方方面面，应用在消费电子、智能交通、智能家居、运动时尚、教育办公、机器人等多个行业，为用户带来全新的产品设计方式和体验。
