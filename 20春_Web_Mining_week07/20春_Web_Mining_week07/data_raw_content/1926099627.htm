职位描述：

岗位职责：
1. 负责2B端销售体系及供应链相关系统架构规划及设计，深入理解业务和需求，能设计可扩展、高可用、高性能、稳定安全的系统；
2. 负责系统整体技术规划及跨部门合作项目边界划分，主导团队技术方案评审等工作；
3. 负责完成重要业务模块、核心框架的搭建及编码实现，能够把控项目质量和进度；
4. 对新技术快速学习推广并能指导团队成员，提高团队技术能力及战斗力。

任职要求：
1. 全日制本科及以上学历，扎实的计算机专业基本功，具备较强的编程能力和质量意识；
2. 6年以上开发经验，精通Java及面向对象设计开发，熟悉JDK源码，研究过优秀开源软件的源码并有心得；
3. 精通MySQL应用开发，熟悉数据库常用性能优化技术，及NoSQL、MQ使用场景以及限制；
4. 能够指导和解决开发过程中遇到的各类技术难题，保证软件开发正常进行；
5. 有大型互联网后台服务设计经验，精通高并发及分布式系统设计，能依据业务场景做好技术选型，例如负载均衡、弹性扩容、微服务、缓存、分布式存储、分布式事务、消息中间件、RPC框架等；
6. 具备很强的沟通能力、组织能力、分析和解决问题的能力及团队协作精神，有强烈的自驱力、责任心和使命感。
7. 熟悉2B端供应链、汽车相关业务，有相关系统建设经验优先；
8. 有大型复杂分布式系统设计开发经验或系统重构经验优先；
9. 熟悉领域驱动设计方法，并具有实际落地经验优先。
