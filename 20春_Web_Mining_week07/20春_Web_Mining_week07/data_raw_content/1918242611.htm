职位描述：

工作职责:
岗位职责：
1. 公司及所属品牌视觉系统的建设及维护，确保品牌在传播上的一致性；
2. 梳理公司及品牌视觉需求，提供品牌分析视觉报告，并参与完善品牌VI系统设计；
3. 输出品牌设计战略、方向，以及品牌在线上线下的应用设计；
4. 具有品牌意识，逐渐建立适合公司品牌推广的创意设计语言；
5. 配合相关部门一起进行大型品牌市场活动创意与设计。

任职资格:
任职资格：
1. 一定要有作品集；
2. 具备三维效果图技术，可熟练操作C4D等3D软件。
3. 具备稳固的基础视觉的理解和实现能力（栅格系统grid system，字体排版typography，布局排版layout）；
4. 出色的观察力以及视觉洞察；
5. 对于设计趋势的变化敏感，并能有效应用在业务场景里；
6. 追求完美，追求有趣，追求挑战；
7. 3年以上知名设计公司、企业的业界经验；
