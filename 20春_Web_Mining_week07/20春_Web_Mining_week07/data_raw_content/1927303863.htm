职位描述：

工作职责：
1、负责天猫国际、天猫海外产品端的用户体验设计工作，包括导购产品、基础链路、中后台产品等，确保产品易用性、扩展性及设计品质； 2、负责从前期业务对接、策略制定、设计输出、开发落地以及迭代跟进的整体工作，能够独立完成信息架构、界面布局、操作流程的产品视觉设计，通过机制和流程的建立持续保障基础体验可用； 3、对业务有全局的理解和判断，能够联合业务、用户研究及开发等上下游团队，寻找设计机会点并驱动业务产生增量价值； 4、工作地点主要在杭州，未来有机会去海外出差或工作。
任职要求：
1.具备基础设计能力：具备专业的体验设计理论基础，对用户体验设计原则、方法、流程有深入理解，有较好的用户视角、较高的审美水准、广阔的业务视野； 2.具备商业洞察能力：能够理解商业背景及目标制定设计策略并综合利用多种设计手段产出创新性、前瞻性的体验设计解决方案； 3.具备专业沉淀能力：能够将项目中的成功或失败因素进行提炼，通过产品化、方法化、机制化等方式，形成可持续的成功运作模式； 4.具备组织协作能力：具备优秀的沟通协作能力和表达能力，能够同跨专业跨岗位进行项目协同，保质保时保量支持业务运转； 5.具备同类业务经验：海淘业务背景、跨文化跨地域工作经历、英语表达能力强的面试者优先； 6.简历请附作品（必须有软件界面或网站设计作品）。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
