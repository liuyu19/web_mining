职位描述：

岗位职责
1. 负责Web端和H5及微信小程序的研发及优化工作；
2. 与UI设计师沟通交互、视觉设计的相关问题；
3. 参与产品的调试、测试及维护工作；
4. 在理解产品业务的基础上，提升产品的用户体验，技术驱动业务的发展。

任职要求
1. 计算机相关专业优先，本科及以上学历；
2. 2年及以上互联网前端工作经验；
3. 熟悉H5及微信小程序开发；
4. 熟练掌握 JavaScript、CSS、HTML5、DOM、性能优化等前端技术，对主流前端框架Vue有深入应用并深入理解其设计原理；
5. 熟练掌握页面架构和布局，对CSS/Javascript性能优化、解决多浏览器兼容性问题有一定的经验；
6. 熟悉模块化、前端编译和构建工具，如Gulp、Webpack等；
7. 有较强的逻辑思维能力，善于分析、归纳、解决问题；有很强的学习能力，责任心强，具有良好的团队协作与沟通能力；
8. 热爱前端技术，有较强的学习能力，有强烈的求知欲、好奇心和进取心 ，能及时关注和学习业界最新的前端技术。
