职位描述：

职责描述：
1. 构建全球服务体验管理体系，包括组织、流程、系统等；
2. 建立用户服务体验闭环管理体系，绘制用户体验地图；
3. 评估客户服务体验，分析服务体验指标波动情况，定期发布用户体验报告，对异常情况做深入洞察及剖析，精准定位问题根因并提供解决方案；
4. 设计服务体验方案，制定服务体验标准，提炼体验亮点，推动并监控体验指标达成；
5. 倾听内外部用户声音，协同全球服务组织快速推进用户体验项目落地，提升用户满意度。

任职要求：
1. 本科及以上学历，书面英语可作为工作语言；
2. 在客户体验领域工作5年以上，有服务体验/客户洞察/服务设计/数据分析/产品运营工作经验尤佳；
3. 服务意识强，有具备优秀的逻辑思维和敏锐的洞察能力；
4. 具备优秀的项目管理能力与跨部门沟通协调能力，能够协调部门内外的资源快速解决问题，保证项目落地。

企业介绍：

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，办公室设于北京、上海、西安、香港、东京、洛杉矶、旧金山、鹿特丹、法兰克福等全球18个国家及地区。

大疆是一家致力于持续推动人类进步的科技公司，坚持做空间智能时代的开拓者，不断为世界创造超越想象的科技之美。2006年创立至今，大疆在全球的员工人数已超过14,000人，客户遍布全球二百余个国家和地区。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十三年间，大疆不断革新技术和产品，开启了全球“天地一体”影像新时代；大疆的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。
