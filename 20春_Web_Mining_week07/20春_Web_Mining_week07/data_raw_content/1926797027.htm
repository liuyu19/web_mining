职位描述：

职责描述：
1、 区域目标管理，根据总部下发的KPI，进行城市内目标拆解和资源协调，并进行目标追踪跟进；
2、 配合总部建立大区规范、高效的交付中心运营管理体系，标准化业务流程落地，赋能城市运营管理；
3、 对接总部和城市车辆交付、运维业务，统筹规划各城市开城相关业务，确保各环节的合规性；
4、 建立数据运营体系，定期输出大区运营数据报告，定位城市问题及风险，提供优化方案并进行效果追踪，对交付中心运营指标负责；
5、 配合总部建立标准化服务管理体系，设计体验标准流程并推动落地，推进城市落地；
6、 负责体验专项治理的问题拆解，整合资源输出服务设计方案，并推动落地；
7、 对交付中心运营中的卡点，有效复盘，流程迭代和优化，提升人效和时效；
8、 大区交付中心安全管理隐患的检查和处置；
任职要求：
1、本科及以上学历，互联网运营管理、汽车行业背景，3年以上者优先；
2、具备标准规则流程输出能力，具备一定的互联网产品运营逻辑思维；
3、 数据导向，通过数据分析制定运营策略能力，能思考提出解决方案，了解并会使用SQL优先；
4、对用户价值和体验管控有清晰的认知及独特见解；
5、有较强的沟通能力、自我驱动能力，适应能力强
6、忠于目标，认同企业核心价值观，抗压能力强，能迅速推动问题的解决。

关于我们

滴滴出行是一站式移动出行平台；为5.5亿用户提供出租车、快车、专车、豪华车、公交、代驾、企业级、共享单车、共享电单车、共享汽车、外卖等多元化的出行和运输服务。在滴滴平台，超过3100万车主及司机获得灵活的工作和收入机会。

滴滴出行与监管部门、出租车行业及社群积极协作，致力于以智慧交通创新解决全球交通、环保和就业挑战。滴滴也和汽车产业链企业建立广泛的合作，携手打造面向未来的汽车运营服务平台。

在全球范围内，滴滴与Grab、Lyft、Ola、99、Taxify、Careem构建的移动出行网络触达全球超过80%的人口、覆盖1000多座城市。目前，滴滴通过旗下99平台服务巴西，在墨西哥和澳洲提供滴滴品牌的出行业务；并在日本通过合资公司提供网约出租车服务。滴滴始终致力于提升用户体验，创造社会价值，建设安全、开放、可持续的移动出行新生态。
