职位描述：

职责描述：
1、定义规划华为手机各系列调性，牵引产品定义及开发。
2、负责华为手机各个系列的视觉形象（ID、UI、视觉传达等），管理各系列品牌的继承和一致性。
3、负责分析研究具有代表性的品牌，潮流趋势，牵引产品规划及设计方向。
4、研究用户的消费习惯、价值观、生活方式等，形成用户画像，提炼规划对应的视觉风格。

任职要求：
1、美术、视觉传达、认知心理学、产品设计等相关专业本科以上学历。
2、具备较高的艺术修养，富有创造力、优秀的视觉；有良好的整体把控力、优秀的策划能力。
3、对年轻群体以及当下流行趋势有敏锐的洞察力。
4、富有工作激情、创造力和责任感，具有良好的沟通理解能力及较强的团队合作精神。
5、有品牌策划、广告策划、视觉传达、ID设计、UI设计等相关工作经历优先。
