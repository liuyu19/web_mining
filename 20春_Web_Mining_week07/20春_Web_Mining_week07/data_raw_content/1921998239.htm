职位描述：

岗位职责:
1.负责APP产品（包含PC端和移动端）的视觉风格定位以及UI界面设计；
2. 优化用户体验，与产品组合作分析用户的操作习惯和偏好，从用户体验的角度分析进行界面的易用性和功能分析；
3. 推进界面以及视觉交互设计的最终实现；
4. 参与设计体验流程和规范制定；
5、跟进设计方案的执行和产品表现，提出产品的改进方案；
6、参与产品的需求讨论，提供视觉和交互建议；
7、整理和维护产品的交互规范和文档。
任职资格:
1、统招全日制本科以上学历;
2、三年以上互联网/APP产品经验,一年以上产品经理工作经验优先；
3、熟悉使用各种产品工具：如Visio、AXURE、mindmanager等，智能机深度使用者，同时熟悉Android和iOS系统；
4、具有逻辑性强、擅长分析和需求调研，竞争对手分析，数据分析及解读能力；
5、具有敏锐的市场洞察能力和自我驱动力；能够适应高压力、迅速的工作风格。

企业介绍：

     合生创展集团有限公司（合生创展）1992年进军房地产业，1998年在香港联交所主板上市（代码：0754），是致力发展住宅地产、商业地产、酒店地产、旅游度假产业和物业管理产业等泛地产事业的大型综合性企业集团。
   一直以来，合生创展秉承“优质生活、完美体现”的企业理念，开香港地产集团大规模投资中国大陆之先河。香港上市期间，合生在国际最严格的金融市场上一直表现卓越，在国际范围内获得了市场的认可；2005年8月，合生创展成功引入了两大策略股东“老虎基金”及新加坡政府投资公司“淡马锡”，在国际范围内获得了金融市场的高度认可；成立至今，合生创展始终注重审慎的财务管理，保持着稳健的经营表现与领先于行业的盈利水平，复合增长率达20%。
   合生创展立足经济发展快速的国际化大都市，形成了“对接世界，影响中国”的发展态势。依靠丰厚的土地资源储备，以北京、上海、广州三大城市为轴心，积极向中国二三线城市发展，构筑起“环渤海、长三角、珠三角”、三大经济圈为一体的事业版图，并从单一的优质住宅提供商逐步转型为全系地产的领导者。
     在住宅地产板块，合生创展已经初步建立了“高端物业、高档住宅、精品大盘、都市湾区”四大住宅产品线。短短几年内，连续开发了合生•珠江帝景、合生•珠江绿洲、合生•珠江罗马嘉园、合生•霄云路8号、合生•世界村、合生•世界花园、合生•滨江帝景、合生•京津新城、合生•天戴河、合生•佘山东紫园、合生•东郊别墅、合生•财富海景公馆、合生国际城、合生•上观国际等优质楼盘。
     在商业地产板块，合生创展涵盖了“城市综合体、精品写字楼、高端商业、酒店&高尔夫度假、科技产业园、文教地产、养老地产“七大核心业务板块，整合各城市的优质商业资源，牢牢把握中高端商务办公、生活消费和娱乐休闲等城市未来经济发展的增长点，实现资产价值升值的快速化、最大化和可持续化。先后开发了合生•财富广场、合生•国贸中心、合生•国际广场、合生•国际大厦、合生•德胜大厦、合生•时代帝景、合生•麒麟社等知名商业中心。
   自成立至今，20余年的时间里，合生创展累计开发面积3000多万平方米，成功打造近百个高品质项目，深得100万业主信赖，业已成为中国大陆表现最佳、开发规模最大、业主数量最庞大的房地产发展商之一。
