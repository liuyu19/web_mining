职位描述：

1. 负责小米金融iOS、Android界面设计；
2. 把控视觉风格，传递符合产品定位的设计理念；
3. 从设计角度系统的解决体验问题。

任职要求
1. 专业基本功扎实，三年以上移动端设计经验；
2.对设计趋势、移动设计、规范有深刻的理解。从设计发起研究、有成熟的设计方法论。
3.对交互设计有所了解，能够参与交互设计的一些工作，熟悉常用的方法论。
4.懂得用研和数据分析的一些基础方法，辅助设计决策。
5.需要有超强的：表达能力、沟通能力、逻辑思维能力，能够独立带项目。
