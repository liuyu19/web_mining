职位描述：

岗位职责：
1. 负责C端产品设计，涉及APP/M/PC三端；
2. 系统化的进行用户需求的调研和分析，参与相关设计流程、设计标准以及视觉元素规范的制定，持续跟进产品的交互、视觉体验优化；
3. 理解业务背景，结合业务目标提出产品设计方案和完成UI设计稿，协同上下游推动项目落地，并根据数据及反馈结果，持续优化；
4. 对产品功能点的数据变化与用户满意度有极高的敏感度，灵活调整设计视角与工作方式；
5. 跟踪把握设计与创意的流行趋势，对设计和用户体验提供新的想法和观点。
6. 与业务运营、产品经理、前端工程师密切合作，完成日常的需求与项目工作，根据产品需求，提供优秀合理的设计方案。
岗位要求：
1. 2年以上设计相关工作经验，有线上完整的设计作品，较好的审美水平，极度重视细节，能够驾驭多种设计风格；
2. 掌握专业的用户体验设计的方法并熟练应用，对产品设计有实践出真知的深入理解；
3. 具备视觉风格原创力、建立规范与组件库经验；
4. 具备全链路设计思维，具备交互体验与产品创新思维；
5. 优秀的沟通/协调能力、较强的主动性和自驱学习态度；
6. 具备一定的英语沟通能力、擅长3d等优先；
