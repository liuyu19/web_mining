职位描述：

职责描述：
1.   负责平台健康频道和医疗频道的内容模块的运营，包括内容编辑，用户健康头条，明星医生包装，粉丝互动，信息监测等；
2.   负责平台内容的视频、音频、图文的信息编辑与内容梳理与设计，对标题、内容、排版、图片等内容结果负责；
3.   负责不断优化用户的黏性与依从性，培养用户习惯；对用户的阅读数、评论数、转发数、收藏数等数据结果负责；
4.   撰写各类推广宣传稿件（新闻稿、邀请函、合作函等）、策划方案、报告等，建立公司的品牌形象；

任职要求：
1、本科及以上学历，3年以上移动互联网经验，有电商类、移动医疗运营经验者优先考虑；
2、对移动医疗互联网产品运营发展趋势有良好的判断能力，具备数据分析能力；
3、逻辑思维能力强，具有良好的沟通与语言表达能力，良好的团队协作、执行力与学习能力，以及产品相关文案的撰写能力；
4、工作积极主动，责任心强，对待工作富有热情，能承受工作压力。
5、有平安好医生、阿里健康、微医、丁香园等医疗健康平台或类似医疗健康管理平台内容运营经验优先。
