职位描述：

岗位描述：
1. 负责办公平台整合、办公信息化自动化智能化相关系统的规划与设计，拥有AI技术背景更佳；
2. 独立完成产品的原型、流程和交互界面设计，编写完整的产品需求文档（PRD），协助开发团队理解和掌握需求；
3. 独立完成从产品开发过程中的进度管理与跟踪，组织产品测试，后期产品培训等全过程；
4. 负责产品上线以后的功能完善与优化、BUG跟踪、收集改进意见、提供改进方案，引导用户熟悉使用产品；
5. 负责统计产品的数据及用户行为分析；
6. 定期对行业相关产品进行跟踪与分析；

任职要求：
1. 具备3年以上互联网产品经验，本科学历，英语六级及以上，能轻松阅读外文；
2. 思路清晰、善于学习与思考，对国内外互联网产品与技术发展有敏锐的观察力和求知欲；
3. 熟悉人工智能相关知识与算法，特别是智能机器人问答领域；
4. 具备良好的项目管理思维、逻辑分析能力、沟通表达及协调能力，带领过项目型团队；
5. 做事有责任感，具备良好的沟通能力、团队合作能力、执行力及心理承压力；
6. 注重用户体验及交互设计。

企业介绍：

同程艺龙由同程旅游集团旗下的同程网络板块和艺龙旅行网在2017年底合并而成。
同程旅游集团是一个多元化旅游集团，中国领先的在线旅游服务商和一站式旅游预订平台，是“中国民营企业500强”和“中国旅游集团20强”企业。同程旅游集团下辖同程网络和同程控股两大板块，其中同程网络业务涵盖同程机票、酒店、火车票、汽车票等多项业务。
艺龙旅行网是中国领先的在线旅行服务提供商之一，深耕OTA行业多年，秉承专业团队的管理运作，在酒店等领域享有盛誉。
合并后，同程艺龙定位于领先的在线旅行服务平台，将同程网络和艺龙各自核心业务优势互补，并占有微信钱包中机票火车票的入口和酒店入口。目前同程艺龙已有近两亿的消费会员，2017年服务超过五亿人次，合并交易额超过千亿人民币，并已实现年超过五亿人民币的规模化盈利。
