职位描述：

工作职责：
1. 负责沟通协作解决美术方面的技术难题;
2. 对接程序技术侧，优化文件修改BUG等问题;
3. 熟悉并研究UE4引擎渲染架构，给予提升美术效果的技术支持，可帮助团队编写优化辅助工具的使用，如毛发插件，布料插件，材质渲染等;
4. 熟练掌握UE4美术流程，能够快速搭建符合项目的场景;有C4D软件使用经验，会AE,PR后期软件
5. 对材质，灯光，3D动画，渲染有足够的理解，可独立通过蓝图制作相关内容。
任职要求：
1. 有一定的美术认知，熟悉美术开发和PBR流程;
2. 熟悉c/c++、unity3d、ue4等相关技术;
3. 熟悉美术文件的优化整理和对接程序技术的优化;
4. 良好的沟通能力和团队协作能力，需不断钻研软件功能，紧跟市场发展步伐。
5. 有TA的相关工作经验。
