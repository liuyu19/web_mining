职位描述：

职责描述：
1、负责各产品线的UI设计、视觉定位，把控产品的设计方向；
2、支持运营/品牌相关活动的视觉设计工作，包括主视觉、H5、海报、专题等内容的创意设计和执行；
3、制定视觉设计规范，定期收集和整理通用设计素材，保证设计实现的可复用性和用户体验的一致性；
4、紧跟设计流行趋势，把握竞品设计动态，定期分享，驱动设计创新；
5、把控实现质量，跟踪设计效果，根据用户研究及数据反馈不断提升产品设计体验；
6、与交互和用研紧密结合，帮助团队共同构建UED思维体系；
任职要求：
1、美术、视觉传达、艺术、工业设计、计算机等用户体验相关专业；
2、4年以上视觉设计经验，主导过完整的互联网/移动端产品视觉设计工作，有相对完整的设计规范编写经验，并有成熟的设计产出和有效的设计落地；
3、较高的审美水平，专业的设计基础，精通Photoshop、Stetch、Illustrator等常用设计工具，了解Html、JS等基础前端知识，能设计出符合技术开发的视觉界面；
4、熟悉各平台的行业设计规范，并善于根据具体产品对现有规范进行优化，形成更具指导性和可持续的设计准则；
5、具备交互设计的理解和分析能力，能将交互原型中的信息内容用专业的视觉语言合理的表达，并结合设计创新进行有效的整合和体验提升；
6、独立思考，对设计过程和结果有自己的观点，能系统清晰的结合目标阐述完整的设计思维；
7、对新事物有好奇心，对于设计流行趋势充满学习和研究的热情。
8、善于沟通与合作，能积极有效地与开发沟通，跟进设计效果的最终实现

企业介绍：

京东于2004年正式涉足电商领域，2017年，京东集团市场交易额接近1.3万亿元*。2018年7月，京东第三次入榜《财富》全球500强，位列第181位，在全球仅次于亚马逊和Alphabet。

2014年5月，京东集团在美国纳斯达克证券交易所正式挂牌上市。2015年7月，京东凭借高成长性入选纳斯达克100指数和纳斯达克100平均加权指数。

京东商城致力于打造一站式综合购物平台，通过组建大快消、电子文娱、时尚生活三大事业群，服务中国亿万家庭，并以高于行业平均增长速度的发展态势满足消费者日益多元的消费需求。京东商城已完成全品类覆盖，消费品、3C、家电等优势品类年交易额突破千亿大关，未来京东还将培育出生鲜、文旅、时尚、大客户、新通路、拍卖、大汽车、家居家装、大健康等近20个“千亿级品类”。
