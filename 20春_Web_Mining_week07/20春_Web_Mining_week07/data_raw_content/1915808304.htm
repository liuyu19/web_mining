职位描述：

岗位职责:
1.根据业务需求，协助业务部门编制相应IT项目需求书，同时协助开发人员开发相应业务系统；
2.参与各业平台管控模式设计、业务流程梳理及优化信息化解决方案设计与管理软件系统的咨询服务；
3. 调研各职能平台业务信息化支撑问题点、业务需求点提供相应咨询服务并书写相应需求报告与解决方案；
4. 业务需求的 软件原型或功能原型的设计、开发，软件技术设计规格书编制；
5.能承担项目经理的职责，推动企业中制造信息化建设；
6.对工厂服务器及网络运维管理有一定的了解，可以解决常规日常业务
任职资格:
1.具备一定的项目管理经验，能够管理控制与推动项目进度；确保项目有效落地；

2.思维清晰，具备良好的沟通表达能力和书面文档撰写能力

3.良好的团队协作能力、责任心,思路清晰，沟通有效。&1. 熟悉制造业生产制造业务，有至少3年制造行业MES、WMS,QM等制造信息化的从业经验，对现场生产制造管理、生产跟踪、质量管理、生产物流等有深刻理解；对自动化设备整合，产线设备信息采集等均有相应的了解;

2.有一定的开发能力（JAVA,.NET）等，熟悉数据库等;

3.能根据业务需求的编制软件原型或功能原型的设计、开发，软件技术设计规格书

4.服务器及网络运维管理有一定的了解，可以解决常规日常业务

5. 熟练使用各类办公软件PPT、Word、Visio、Excel等,&1.本科及以上学历，5年以上工作经验，有软件开发背景与经验（条件优势者可大专）

2.工厂自动化系统经验，熟悉一般工厂底层控制设备者,熟悉产线数据自动采集；

3.具有家电、电子制造等离散制造行业背景；

4.制造行业信息系统建设与整合项目经历&良好的团队精神和严谨的工作态度，能承受工作压力，良好的学习能力和自我提升愿望

企业介绍：

欧普照明股份有限公司是一家创新型快速发展的公司，成立于1996年8月，是一家集研发、生产和销售于一体的全球化照明企业，现有员工6000多人。产品涵盖家居、商照、电工、光源等领域，主要产品包括节能灯、吸顶灯、支架、筒射灯、LED照明等，在国内市场占有率已处于领先地位，同时，欧普自主品牌产品的海外业绩也相当喜人。
   经过多年的努力建设，欧普拥有了一支强大的营销队伍和完善的国内外营销网络。目前国内有32个办事处和近3000家品牌专卖店，5000多个家居照明产品销售网点，3000多个超市网点，500多个商业照明产品销售网点，覆盖95%县级以上城市，并逐步深入乡镇。同时，欧普照明的产品也远销世界五十多个国家和地区，其中自有品牌占70%。在欧洲、中东、南美、南亚、东南亚、印度、南非等地区设有分公司。
   2008年，欧普照明股份有限公司搬迁至上海，开始了更加全面的国内市场覆盖及更加前沿的海外销售布局；2011年，欧普苏州工业园已投入使用，占地750亩，标志着欧普照明已经迈上了一个新的发展台阶，为打造照明行业的民族领袖企业打下坚实的基础。2012年，迪拜分公司正式成立运营，标志着欧普的国际化进程有着实质性进步。
   欧普照明以“用光创造价值”为光明使命，以“打造全球化照明企业，成为中国照明领袖品牌”为企业目标，在努力塑造产品品牌和雇主品牌的同时，致力于绿色、和谐、低碳的新生活主张，以领先的技术、节能环保的理念，为人们的工作和生活营造优质的光环境而不懈努力。
   欧普照明的用人理念：不求无过，不得无功，德才兼备，以德优先。我们鼓励员工勇于面对困难，承担对自己和对社会的责任，责任，是欧普人发展的动力；我们欢迎不同文化背景的员工，以接纳和鼓励的态度对待新思想和新事物，创新，是欧普人的标签；我们大力投资员工培训和发展，有针对性的准备了大量的管理培训项目，志在帮助每一位员工提升自己，因为我们希望每位员工和欧普共同成长，速度，是欧普人前进的姿态！
   专注光的价值，感受光的魅力！热情、敬业的欧普人，正在为打造中国照明领袖品牌而不断进取，不断创新，将光与爱播撒人间！
