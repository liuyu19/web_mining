职位描述：

职责描述：
1.根据公司的发展战略与规划，负责携程商旅APP产品的需求梳理、产品策划、流程设计、PRD撰写等工作
2.根据业务需求、用户行为分析，进行界面的信息架构设计和原型设计
3.负责跨部门协调和沟通，推动UI、开发、测试、运营等人员紧密合作达成产品目标，保证产品实施阶段的进度和质量
4.通过市场调研、竞品分析、运营数据报表等方式收集并分析市场和客户需求，研究分析运营成本与效果，制定并实施产品优化方案
5.负责产品团队管理：工作的分配，人员的培养、激励与考核

任职要求：
1.五年以上互联网产品工作经验，两年以上APP端产品工作经验；具备团队管理能力
2.本科学历及以上，专业不限，如有出色的UI设计能力优先考虑
3.能独立推动产品迭代，承担包括需求收集、输出、评审、推动、测试、运营等相关工作
4.具有产品规划能力，包括需求发掘/收集和管理、市场和竞品分析、设计产品功能、跨部门协调沟通、线上产品监控等
5.拥有良好的逻辑分析思维和组织协调能力、时间管理能力，能够对事务并发处理，有激情和拼劲，敢于承担高强度工作
