职位描述：

工作职责：
1. 负责游戏客户端引擎功能研发及美术工具开发。
2. 负责游戏客户端的性能优化及资源优化。
3. 协助美术持续改善制作流程，优化开发效率。
4. 协助策划将相关游戏玩法在引擎中落地。

任职要求：
1. 有3年以上的引擎客户端开发经验者优先考虑；
2. 掌握C++，熟悉至少一门脚本语言如Python；
3. 在渲染、动画、材质、UI、AI、美术工具构建等任一方向有具体实践或专精；
4. 乐于沟通合作，有团队合作精神，可以主动的推进业务；
5. 热爱游戏，对效果和体验有追求，有游戏经验者优先；
6. 可承受一定强度的工作压力。
