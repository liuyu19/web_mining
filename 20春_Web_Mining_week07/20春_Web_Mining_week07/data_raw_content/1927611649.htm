职位描述：

工作职责：
1.负责web产品的UI设计及创意视觉设计工作；
2.负责核心产品整体视觉风格定义，优化已有的设计规范形成可持续的设计准则；
3.带领团队设计品质和创意方向规划，推动业务视觉品质提升，提高团队综合设计能力；
4.分享设计经验，共同推动团队设计能力的提升，和团队成员一起成长。
任职要求：
1.本科及以上学历，四年以上UI设计经验，有tob或中后台设计经验优先；
2.丰富的设计理论知识和对流行趋势敏锐的洞察，对UI设计趋势有灵敏触觉和领悟能力；
3.能灵活应对不同类型的业务，具备较强个人的设计特点和观点，能系统清晰的结合目标阐述完整的设计思维；
4.关注设计前沿趋势，对设计探索感兴趣 ，有系统化的设计思维方式；
5.良好的团队意识，富有工作激情，创造力和责任感。
