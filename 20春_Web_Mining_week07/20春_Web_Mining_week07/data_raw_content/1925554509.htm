职位描述：

职责描述：
1. 主要负责京东钼媒的品牌形象设计与品牌推广，企划工作的平面设计、协助完成公司营销推广项目的整体策划创意设计；
2. 负责日常京东钼媒自媒推广的运作，负责公司对外形象的建立与宣传，建立公司与行业媒体的交流，配合完成日常推广宣传工作；
3. 负责各类线下品牌活动的视觉画面营造，主导活动视觉效果及物料设计。
任职要求：
1. 美术、设计或相关专业本科以上学历，5年以上品牌设计工作经验；
2. 有公司产品品牌形象设计、品牌推广、展览展示和公关活动等相关设计工作经验与成功案例；
3. 熟悉印前流程，能独立完成公司的VI、海报、DM画册、包装的设计和排版；
4. 了解活动设计，根据需求完成活动视觉效果及物料设计排版；
5. 对设计有自己的见解，有较强的手绘功底，可借助鼠标、手绘板进行手绘元素与插画创作；
6. 具备良好合作态度及团队精神，富有工作激情、创造力和责任感，较强的抗压能力。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
