职位描述：

职位描述：

1. 负责小米有品自研产品包装设计及平面VI的设计维护工作；

2. 负责品牌运营设计，如快递箱、宣传海报、DM单及公司运营类的产品包装；

3. 负责设计的前期调研、设计资料收集、分析工作，根据品牌定位设计产品内外包装等平面设计工作；

4. 包装物料印前准备，排版、监督印刷等相关工作；

5. 总结包装平面设计规范，整理设计规范文档，并定期更新规范文档。



职位要求：

1. 有良好的的审美能力，对设计或时尚有敏锐的洞察力；

2. 具有电商平台相关包装设计3年及以上工作经验；

3. 包装设计、艺术设计、工业设计或纯艺术专业，本科以上学历；

4. 精通使用AI、PS、CDR、Key Shot、C4D等相关设计软件；

5. 熟练运用包装工艺，熟知特种工艺相关标准；

6. 具有良好的语言沟通能力及团队合作精神；

7. 广告公司负责国内／国际一线品牌电子产品品牌的包装设计师优先考虑；

8. 应聘时请携带本人近期作品。
