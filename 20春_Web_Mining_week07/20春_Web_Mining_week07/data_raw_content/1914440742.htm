职位描述：

岗位职责：

1、负责互联网金融相关产品后台服务的设计和开发；

2、解决开发过程中的技术难题；

3、保证系统的稳定性及可用性，安全性；

4、对团队成员进行有效的指导，增强团队技术能力；

任职资格：

本科或本科以上学历，扎实的Java基础；

-熟练使用常用MVC、ORM框架，能了解其原理；

-精通Java，具有4年以上系统开发、设计经验，熟悉分布式系统开发；

-优秀的数据库设计和优化能力，至少精通一种数据库应用;

-熟练掌握面向对象设计的基本原则和常用的设计模式并有丰富的使用经验；

-熟练掌握 Java 及面向对象设计开发，对部分 Java 技术有深入研究，研究过优秀开源软件的源码并有心得者优先；

-丰富的互联网系统设计经验，有大型互联网金融系统架构经验者优先；

-研究过 http 协议、缓存、jvm 调优、序列化、nio、多线程、RPC 调用框架等，有相应实践经验者优先

-熟悉sping、spring cloud 框架，能够在此框架下进行业务开发；

-对互联网金融业务、账户系统有丰富经验者优先；

-有金融信贷系统开发经验者优先；

-优秀的学习能力，分析和解决问题的能力和强烈的进取心；

-诚恳、踏实，对技术和工作充满热情。

企业介绍：

众安在线财产保险股份有限公司（以下简称“众安”）是中国第一家互联网保险公司，于2013年11月6日揭牌开业，2017年9月28日在香港联交所主板上市，股票代码为6060。

众安总部位于上海，不设任何分支机构，完全通过互联网展业。由“保险+科技”双引擎驱动，众安专注于应用新技术重塑保险价值链，围绕健康、消费金融、汽车、生活消费、航旅五大生态，以科技服务新生代，为其提供个性化、定制化、智能化的新保险。于2019年上半年，众安服务逾3.5亿用户，其中35岁以下的人群占比58%。

在科技赋能保险的同时，众安将经过业务验证的科技对外输出。2016年11月，众安成立全资子公司众安信息技术服务有限公司（以下简称“众安科技”），将技术方案产品化，向海内外市场输出科技产品和行业解决方案。2019年上半年，众安科技共与170多家客户签约，截至目前的海外合作伙伴包括日本历史最悠久的财产保险公司SOMPO、东南亚领先的O2O平台Grab、新加坡的综合保险机构Income等知名企业。

以“科技驱动金融 做有温度的保险”为使命，秉承“简单、快速、突破、共赢”的价值观，众安将继续乘风破浪，砥砺前行，开启真正的新保险时代。
