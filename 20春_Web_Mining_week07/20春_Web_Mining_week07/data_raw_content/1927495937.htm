职位描述：

工作职责：
1. 负责AR虚拟场景3D建模和修改，基于PBR流程制作贴图，熟悉材质系统和制作出符合视觉的材质，以及美术场景库整理和优化；
2. 运用引擎资源做整合场景，布置灯光以及烘焙灯光贴图，准确表达场景氛围；
3. 根据需求设计出粒子特效，角色特效，场景氛围特效，UI特效等；
4. 根据创意设计视频需求，把控动画影片的镜头和节奏，引擎渲染输出成片。
任职要求：
1. 优秀的场景物件塑造能力、气氛把控能力、布置灯光，材质制作，地形编辑制作，场景特效气氛；
2. 熟悉引擎动画系统、特效设计，包括镜头动画、定序器动画和蓝图驱动动画；
3. 熟知PBR流程，熟练运用Substance Painter绘制贴图；
4. 有良好的美感，了解分镜头、光线和色彩的运用；
5. 熟悉美术文件的优化整理和对接程序技术的优化。
6. 会C4D，了解AE，pr等后期制作软件。
