职位描述：
1. 负责OTT研发中心服务端设计与开发工作；
2. 负责高可用架构的设计、实现和维护；

职位要求：
1. 统招本科及本科以上学历，计算机相关专业，3年以上服务端研发经验；
2. 熟练使用Python，Django；
3. 熟练使用MySQL、HBase、Redis等关系/非关系数据库的使用，了解基本的数据库设计、优化原则；
4. 熟悉Linux开发环境和代码管理工具，熟悉网络编程，了解操作系统相关知识；
5. 部分岗位涉及国际化业务，有良好英文沟通能力、文档阅读能力者优先;
6. 热衷于高可用架构、性能分析与调优，勇于面对技术挑战，追求技术卓越。
优先条件：
1. 开源社区贡献者，有修改开源框架经验者；
2. 具有高并发大流量在线服务开发经验者。
