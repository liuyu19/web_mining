职位描述：

职责描述：
1.根据公司规定与项目需求，负责PC端和H5项目高保真页面原型的重构和维护；
2.负责将VI视觉稿切分、设计、编码，转化为HTML高质量静态页面；
3.根据项目要求，结合Css3、Javascript、Php等工具，实现web页面动效、各种逻辑状态展现及交互效果；
4.根据团队要求，做好代码版本管理与维护，并完成各种开发文档的编写和维护；
5.负责担当UED团队与开发团队的沟通纽带，能够准确有效沟通流程和细节，确保产品用户体验的贯彻；

任职要求：
1.本科及以上学历；
2.熟练掌握HTML5和CSS3，熟悉PHOTOSHOP，AE等设计工具，熟悉使用DIV+CSS完成HTML页面布局，可以制作简单静态页面及动效，可以制作视频
3.对微信小程序及H5开发有一定了解和项目经验
4.能通过设计稿的交互批注和功能文档来实现JS交互
5.擅于开发UI（包括JS微交互和动画效果），需要在还原设计稿UI方面有过硬能力
6.需要熟练VUE，熟悉前端框架VUE，React
7.了解NODEJS，PHP优先
