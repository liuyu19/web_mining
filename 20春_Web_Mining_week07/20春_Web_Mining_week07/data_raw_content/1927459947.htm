职位描述：

工作职责：
1. 负责创新业务iOS平台的研发工作，包括信息流，短视频，直播，订阅等子业务系统的研发； 2. 独立完成项目的系统分析、设计，并主导完成详细设计和编码的任务，确保项目的进度和质量； 3. 参与移动平台基础框架的研究，设计和实现、关键技术验证和选型等工作； 4. 解决疑难杂症，提高客户端的性能和稳定性
任职要求：
1. 大学本科及以上学历，计算机相关专业，三年以上iOS开发经验，基础扎实，数据结构，网络等基础知识掌握牢固； 2. 熟悉移动平台知识；了解基础的系统原理和系统组件，了解不同终端和系统版本的特性差异；精通UI原理和开发；了解流行的开源组件原理；了解编译构建的基础知识 3. 熟悉iOS操作系统特性和内部运行机制，精通UI开发，熟练各种UI组件并了解其原理； 4. 熟悉Object-C，熟悉C/C++，了解基础的网络通信机制、前端基础知识、多线程基础知识。 5. 熟悉常见设计模式，掌握 MVP、MVVM。能够独立完成模块设计，编写开发相关设计文档 6. 熟悉iOS调试工具和方法，可以应付各种iOS复杂问题，具备良好的分析解决问题能力，有技术难题攻关经验； 7. 有丰富的客户端应用架构经验，可以独立承担业务功能的设计和研发； 8. 至少做过2个大中型项目的开发实施优先，有视频类、直播类、资讯类项目经验者优先 9. 能协调产品、交互、测试推进项目落地优先，有安全性、性能优化经验的优先
