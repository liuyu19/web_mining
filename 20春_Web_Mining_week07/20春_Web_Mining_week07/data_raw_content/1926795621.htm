职位描述：

工作职责：
1、负责腾讯游戏内容中台前端相关业务需求的研发工作
2、参与前端工程化体系建设，提升开发效率
3、与各个角色团队配合完成项目规划、设计与开发
任职要求：
1、本科以上学历，计算机软件或相关专业3年及以上互联网业务开发经验；
2、前端基础扎实，熟练掌握 HTML、CSS、JavaScript、Node.js；
3、熟悉 React 或 Vue.js 等主流开发框架，以及前端工程化相关工具的使用；
4、熟悉 Node.js 及主流开发框架 (Koa.js、Egg.js)，或有其它后端语言经验；
5、有良好的编码规范及沟通能力、总结能力和独立解决问题的能力；
6、对用户体验、交互设计有一定的研究；
7、有良好的沟通能力和团队协作精神,能承受较大的工作压力。
有开源作品或活跃的社区贡献者优先
