工作职责：

熟悉java/kotlin语言，具备良好的编码风格，熟练代码管理工具git；
具备良好的定位根本原因并解决问题的能力，良好的学习和分享习惯，工作积极主动，责任心强；
具有音视频编解码、视频图像处理、native开发经验者优先。
