职位描述：

岗位职责：
1、根据产品需求和UI设计图，使用前端技术开发极具用户体验的PC、移动网站；
2、持续的优化前端体验和页面响应速度，并保证兼容性和执行效率；
3、参与设计并编写WEB前端组件应用；
4、配合后台工程师一起研讨技术实现方案，进行应用及系统整合。

任职要求：
1. 计算机相关专业本科或以上学历，5年以上Web前端开发经验及对前端技术有浓厚兴趣，关注前沿的技术
2. 熟悉HTML5，CSS3，Javascript，Jquery等前端技术
3. 熟悉MVC/MVVM架构，使用过Vue.js进行过项目开发；
4. 了解服务端开发，熟练使用Node.js优先
5. 具有跨平台应用的开发经验，了解各主流浏览器下的JavaScript差异
6. 熟悉W3C标准，对表现与数据分离、Web语义化、网站性能优化等有深入了解
7. 沟通能力强，责任心强，思维逻辑性强，有团队合作精神。

企业介绍：

思必驰科技（AI Speech—Artificial Intelligent Speech）是剑桥大学团队创立、中国领先的智能语音公司，是国内仅有的拥有人机对话技术，国际上极少数拥有自主产权、中英文综合语音技术（语音识别、语音合成、自然语言理解、声纹识别、性别及年龄识别、情绪识别等）的公司之一，语音识别（通用/车载/家居）处于国际领先水平。

思必驰团队的使命是：智能硬件和移动互联时代，让人机交互更自然、方便，成为人工智能时代的领先者。思必驰“对话工场”语音云已经被诸多智能家居、车载、穿戴公司采用，如阿里，小米，联想、索尼、inwatch、长虹等，帮助实现对话式的智能设备控制及服务；并与多家互联网公司合作，如京东、同程、搜狗等，实现自然语言交互的移动搜索。
思必驰成立至今，先后获得联想、启迪、DCM等知名VC投资，并先后获得剑桥大学商业计划奖、剑桥大学Downing基金、英国政府研发基金、国家电子发展基金、重点科技支撑计划以及各类领军企业基金。

思必驰团队是一个满怀激情、创新、多元化的年轻大家庭，来自剑桥、新加坡南洋理工、以色列理工、清华、北大、中科大、哈工大等知名高校，及百度、小米、索尼、华为、联想等知名互联网和智能硬件公司。思必驰在北京、苏州、深圳设立公司，与上海交大建立了智能人机交互实验室。

发展历程：
2014年，获得国际顶级投资机构DCM投资，加快推动智能语音交互技术在智能硬件和移动互联网的应用。
2014年，将教育事业部剥离，成立驰声科技有限公司，思必驰专注智能硬件和移动互联网。
2012年，获得来自联想、启迪投资，加速人机对话等系列技术研发、拓展移动互联网。
2008年，回国落户苏州工业园区，正式开启语音技术国内产业化。获得了江苏省领军、姑苏领军。
2007年，公司正式成立于英国剑桥大学。并先后获得剑桥大学商业计划奖、剑桥大学Downing基金、英国政府研发基金。
