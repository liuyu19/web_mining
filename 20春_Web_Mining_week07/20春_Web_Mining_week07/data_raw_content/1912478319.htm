职位描述：

岗位职责：
1. 负责产品需求采集，需求分析、产品规划，用户体验研究、市场分析、产品功能的设计等；
2. 对公司产品进行全面分析、设计，如功能，特性，使用流程，UI/UE，性能要求等；
3. 负责产品上线后的跟踪优化和运营数据工作，持续推进产品优化；
4. 组织产品与运营模式的内外部调研分析，进行用户数据分析并找到对应的提升方案，为决策提供核心依据，根据公司和产品发展策略，制定所负责产品线的发展和实施计划；
5. 协助研发中心撰写产品需求文档，供设计、开发、测试人员明确产品需求和进行开发、测试；
6. 关注业界动态和用户对于产品各项功能的反馈，提出产品升级建议，不断增强用户体验。
任职要求：
1．本科及以上学历，5-8年互联网产品设计经验；
2．有良好的产品分析、产品设计、总结归纳能力，熟悉产品与运营的协作流程；
3．熟悉磨刀、axure、mindmanager、visio等产品设计工具，对技术实现和可行性、技术成本有良好评估能力；
4．具备优秀的沟通表达能力，团队管理、沟通和协调能力，抗压能力强；
5．熟悉互联网平台开发基本流程，具有资深技术背景者优先考虑；
6．有大型互联网产品项目从业验者优先考虑。

企业介绍：

宗申集团始创于1982，现已成长为一家总资产和销售收入均已突破200亿元人民币的大型集团企业。宗申制造工厂（或子公司）遍布在中国重庆、广东、江苏、河南、河北、天津和越南、泰国、巴西等地，正逐步成为“世界一流中小型动力系统集成服务商”。

宗申集团连续16年入选重庆工业50强，连续入选重庆市“纳税50强”，是中国民营企业500强、中国机械500强、中国制造500强，并入选中国企业500强。宗申拥有国家级“企业技术中心”、“机械工程专业博士后科研工作站”，宗申检测中心通过CNAS国家实验室认可。

宗申与意大利比亚乔（PIGGIO）、塞尔瓦（SELVA）、巴贝锐（BARBIERI）、白俄罗斯戈梅利等国际知名企业在中国设有合资工厂；宗申还与合作伙伴在上海、深圳、香港、加拿大、美国等地设有包括水上飞机运营、地产业务开发在内的合资运营公司。
