职位描述：

岗位描述：
1. 负责移动端买家端的体验设计。
2. 根据产品需求，分析用户核心诉求，平衡业务商业目标，提炼设计目标和设计原则，指导设计。
3. 定期进行设计复盘，总结并分享设计经验，保证经验累积可复用，可传递。
4. 熟悉设计验证方法。掌握相关数据分析和可用性分析知识，衡量设计成果。
5. 发起产品的用户体验优化。

岗位要求：
1. 有3年以上互联网工作经验（有移动电商经验者优先）
2. 对电商活动运营感兴趣，了解运营游戏设计法则
3. 能独立负责业务线的整个设计过程，并有案例
4. 善于沟通，逻辑清晰，分析能力强，心态好
5. 关注行业最新设计发展动向和技术

企业介绍：

洋码头：中国知名的海外网购平台。
我们成立于2009年，洋码头网站上线，满足了中国消费者不出国门就能轻松便捷购买到全球优质商品的需求。移动端产品海外扫货神器的推出 ，再现了海外最真实的血拼现场，消费者足不出户即可与海外买手一起疯狂血拼海外折扣季。
目前，洋码头平台聚集了数万名海外商家和买手，他们分布于美国、欧洲、澳洲、日韩等全球20多个国家和地区。
我们在海外已建成10大物流仓储中心（纽约、旧金山、洛杉矶、休斯顿、墨尔本、法兰克福、东京、伦敦、悉尼、巴黎），全球化布局业已完成。每周近20多个全球班次航线入境，大大缩短了国内用户收到国际包裹的时间，用户体验达到最佳。
我们刚刚于2014年年底完成1亿美金的B轮融资，并计划于2015年上半年启动全员持股计划，以及更有吸引力的激励体系，以便搭建最具跨境电商行业特点和最有战斗力及创造力的团队。
我们崇尚目标驱动、结果导向，我们鼓励简单、开放、信任的工作和沟通氛围，反对一切不那么互联网的陈规旧俗。
我们希望邀请到像您这样和我们兴趣相投、气味相投的一帮人加入，一起打造一个有理想、有结果、有文化、有温度的码头团队。
并期待在不远的将来，和您一起骄傲地分享我们的成功！
