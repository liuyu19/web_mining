职位描述：

职责描述：
1、负责笔记本产品的开发工作，对缝产品供应商，协调各相关部门工作，保证产品的品质与项目进度；
2、负责产品项目节点管理，完成产品项目开发的各流程节点，保质保量的按时完成笔记本的项目开发工作；
3、项目管理项目的风险控制，能够预测各开发节点的风险点，进行风险评估和风险预防措施、保证产品的0不良目标达成。

任职要求：
1、公办院校全日制统招，本科及以上学历；
2、计算机、电气自动化、电子信息工程、工业设计等理工类相关专业优先；
3、至少3年以上项目管理工作经验，能够根据项目的发展趋势进行判断，提出解决思路，规避项目风险；
4、了解计算机基本工作原理、部件配置等，熟悉计算机材料特性和产品开发流程。
