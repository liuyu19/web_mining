职位描述：

岗位职责：
1. 负责业务线 SDK 产品以及相关 Sample 的开发维护工作
2. 负责业务线能力展示 Demo 、商业场景应用 App 以及日常工作中效率工具的开发维护工作
3. 负责定制化项目整体解决方案中 Android 部分的开发维护工作
4. 参与项目中技术方案讨论、代码质量把控等日常工作

任职要求：
1. 本科及以上学历；重点院校及计算机相关专业优先
2. 3年以上 Android 端开发经验；有独立 App 开发经验或 SDK 类产品开发经验者优先
3. 精通 Java 或 Kotlin 语言，熟悉常用的设计模式，熟悉软件模块设计；有整体架构设计者优先
4. 精通 Android 开发，熟悉常用组件以及 UI 规范，能独立开发高性能的 Android 应用
5. 熟悉 C++ 及 JNI 开发，有相关开发经验，有 Flutter 开发经验者优先
6. 思维敏捷，逻辑清晰，学习能力强，有良好产品思维者优先
7. 对新技术敏感，有技术热情，愿意迎接技术挑战，有自我驱动力
8. 工作积极认真，有责任心，乐于分享有良好的团队合作精神
