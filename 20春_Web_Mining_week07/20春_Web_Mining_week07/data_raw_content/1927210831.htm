职位描述：

工作内容：
1. 探索品牌及品牌视觉风格，与团队合作完成品牌视觉风格化的定义。
2. 附带创意、情绪的产品高精图库的建立。
3. 可以独立完成，平面广告投放的创意生产与执行。
4. 提升小鹏汽车整体创意、视觉表达的审美级别，推进智能化、国际化品牌的形象。
5. 与业务方配合，带领小组或独立完成相关业务的视觉创意表达支持。

任职要求：
一、美术、设计、广告相关专业，有4A广告公司工作经验8年以上，有汽车行业或服务汽车行业经验者优先；
二、具有深厚的美术功底，快速且稳定严谨的执行视觉任务，有高品质的创作品质，并准确把控视觉创意方向及稳定调性；
三、有丰富的想象力和强烈的创作欲望与能力，了解并掌握国际化前沿视觉表达体系；
四、具备较强的沟通和理解能力，可以有效的将视觉诉求准确的描述给设计团队；
五、极熟练掌握photoshop、illustrator、inDesign，以及了解并能基本操作其他视觉相关软件。
六、有手绘、插画、摄影等能力者优先。

重点能力关注：
1、审美能力：国际化视野，前沿的视觉表达探索、学习能力强。
2、创意能力：在平面广告等需要强创意表达的维度上，有自己深刻的见解，并可以完成相关创意构建，视觉落地执行的支持。

企业介绍：

小鹏汽车2014年中成立于广州，是一家以科技驱动的高科技制造公司。
我们以前瞻科技驱动业务发展，以品质为先实现产品制造，以用户体验推行产品和服务，打造完善和便捷的用车和出行体验。
在这里，汇聚了来自不同国家3000多人的人才，其中60%为研发人员。
核心团队融合了互联网与汽车等行业的专家，保证了小鹏汽车的跨界思维和开放包容。
