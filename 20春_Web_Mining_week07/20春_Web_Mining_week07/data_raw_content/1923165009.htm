职位描述：

职责描述：岗位职责
1.协助视觉设计师进行界面场景概念设定和界面风格尝试
2.根据游戏风格进行场景，部件，道具等3D资源的建模及贴图绘制，增强沉浸感，提升用户体验和视觉设计品质
3.根据游戏要求提出相应的界面内的场景优化方案，进行场景优化
4.具备优秀的3D模型制作能力和优秀的材质贴图表现能力
5.不断学习积累和共享各种风格场景创作经验，对新技术有敏锐的嗅觉，了解当下行业的动态和技术最新进展

任职要求：职位要求
1.热爱游戏，愿意投身游戏行业，能够承受一定的工作压力
2.有较好的美术专业能力，有较高的审美能力，精通于游戏美术各类资源开发的流程和规范。
3.优秀的设计能力，出色的沟通能力和团队协作能力，迅速高效的执行力
4. 熟练操作掌握3dmax/Maya/C4D，Zbrush/Mudbox，Unflod3D, Substance Designer/Painter, Marvelous Designer,PS,AE等其他相关制作软件；
5.对UE4引擎工作有一定游戏制作经验者优先
6.对游戏界面动效设计有一定经验者优先
