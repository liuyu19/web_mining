职位描述：

工作职责：
1，负责产品的交互设计，设计用户行为及产品的信息架构，保证产品可用性。
2，负责产品的规划构思，归纳用户目标和用户任务，并对产品最终的用户体验负责。
3，注重用户反馈，产品数据，行业趋势与发展，持续提升产品的易用性和体验感。
4，根据用户需求及技术驱动开展产品体验的创新研究。
5，熟练掌握交互设计相关软件，如: Axure、Sketch、OmniGraffle、Mindnote、Flinto等。
任职要求：
1，本科以上学历，设计、艺术、计算机、心理学等相关专业。3年以上互联网、软件行业交互设计工作经验，熟悉产品设计相关流程，并有成功案例。
2，输出高质量的产品原型、流程图和线框图，同时清晰的表达设计方案。
3，有交互设计工作经验，熟悉交互设计工作流程和工作方法，有系统的交互方法论
4，良好的职业素养和团队合作精神，以及高效的执行力和协作沟通能力，能够承受较大的工作压力。
5，有电子商务及SaaS平台项目经验优先。有从零到有的独立项目经验优先。
