职位描述：

岗位职责：
1、参与平台模块设计以及产品定制需求设计，能够输出符合业界趋势，并符合平台设计规范的有竞争力的视觉设计方案；
2、组织或协同设计师支持公司产品GUI设计，跨部门沟通推动并提高产品用户体验设计质量；
3、参与团队建设，组织设计经验的分享和传承，提高团队的设计能力。
岗位要求：
1.、3年以上用户体验设计相关工作经验，视觉传达、数字媒体设计、美学、艺术设计等相关视觉设计专业背景，能够带领、组织设计师进行设计工作；
2.、具有深厚的设计理论与娴熟的设计技巧，对业界和行业有深入的了解，善于捕捉流行趋势，并善于分析和总结，有提炼设计语言整合设计规范的能力；
3.、热爱设计，富有激情，拥有宽广的行业（平面设计、互联网、手持应用）视野与时尚的审美标准；
4.、独立的多项目多任务管理能力以及良好的带队设计与跨部门沟通协调能力；
5.、熟练掌握相关设计工具软件，如Photoshop，Illustrator，sketch等；具有较强的动手能力、设计鉴赏能力和优质视觉方案的输出能力。
