职位描述：

职责描述：
1. 负责信息架构及相关软件产品的交互设计工作
2. 参与到产品规划和创意过程中，分析业务需求，完成产品界面信息架构、流程设计和原型设计工作
3. 撰写界面交互设计说明书及交互设计规范，建立和维护交互元件库，提升协同效率
4. 熟悉用户研究、产品分析、数据分析方法，如竞品分析、用户访谈、可用性测试等，配合其他角色挖掘用户需求、分析使用场景、优化产品逻辑、提升产品可用性和易用性
5. 配合视觉设计师完成产品视觉设计，并协同开发团队实现交互效果
6. 分享设计经验，沉淀设计方法，总结设计思想，与团队共成长

任职要求：
1. 对web端应用有深入体验设计经验和理解、见解，对行业内产品和应用保持高度热情
2. 熟悉UED团队工作流程，对产品有整体规划和梳理产品信息架构的能力，对交互设计有较深的理解和实践，可以独立负责和完成整个产品线的交互设计工作
3. 能够积极参与产品用户研究过程，能够将调研结论有效转化为交互设计方案，发现并提出交互设计中的调研需求
4. 对互联网产品的发展趋势，产品总体规划有较深刻的理解，对用户体验有敏锐的把握能力，富有激情、积极主动、关注细节、学习能力强
5. 有3年以上交互设计经验，工业设计、交互设计等相关专业，本科以上学历
6. 全日制本科及以上学历，工业设计、计算机、心理学等相关专业，英语四级及以上，5年以上相关行业的工作经验
