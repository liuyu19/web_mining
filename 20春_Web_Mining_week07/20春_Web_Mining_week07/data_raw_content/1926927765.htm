职位描述：

职责描述：
完成银行产品设计相关基础工作，包括但不限于：
1. 参与和负责银行APP（信贷/支付方向）基础产品功能和流程设计；
2. 负责收集需求、提炼核心金融场景、撰写PRD、输出产品原型及需求文档；
3. 合理安排优先级，协调各部门资源，推动产品的设计、开发、测试、上线、优化迭代的全过程；
4. 根据产品实施效果，配合业务部门持续完善产品功能，优化用户体验；
5. 与各个业务线保持密切沟通，迅速响应合作方提出的需求。
任职要求：
1. 计算机/金融相关专业优先，2年以上金融产品经验；
2. 熟悉消费信贷或支付转账类产品设计经验者优先；
3. 会使用Axure、Visio等产品经理必备原型及流程图软件优先；
4. 有跨文化/国际化工作经验优先；
5. 具有较好的逻辑思维和系统思考能力，乐于分析事物本质，擅长系统解决问题；
6. 良好的自驱力、执行能力和学习能力。
