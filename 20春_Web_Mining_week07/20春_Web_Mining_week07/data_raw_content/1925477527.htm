职位描述：

负责后端的研发
任职要求：
1、 本科及以上学历，3年以上互联网开发经验；
2、 出色的编码能力，精通 NodeJs，熟悉常用的算法，熟悉开源框架，解决问题能力强；
3、 熟悉Git/Github，对后端程序性能优化有丰富经验；
4、 掌握 Redis、MySQL 等存储系统原理及应用；
5、 熟悉Docker，对Linux有深入了解;
6、 对分布式计算，分布式存储的架构设计理念和动机有一定理解；
7、 对 HTTP 协议有深刻的理解，熟悉 TCP/IP 等网络协议；
8、有较强的抗压能力，性格开朗积极乐观，遇事情沉着冷静；
9、有良好的沟通能力和团队合作能力，善于沟通，工作自主驱动，具备良好的问题定位分析能力；
10、 加分项：有 Java、C++ 、Python开发经验；参与或拥有开源项目，为开源社区贡献过代码；有大规模后端服务的设计和开发方面经验。

企业介绍：

  浙江每日互动网络科技股份有限公司（个推）成立于2010年12月，总部位于杭州，在北京、上海、广州等地均有本地团队，核心成员来自于微软、IBM、卓望、百度、新浪、阿里巴巴等知名公司。
      个推专注推送技术7年，至今已服务数十万APP，覆盖数十亿移动终端，典型客户包括新华社、新浪微博、滴滴出行、中国移动咪咕阅读、网易新闻、芒果TV、招商银行掌上生活等众多有影响力的APP。
      以推送技术为基础，依托庞大的数据体系和深度学习技术，个推成立了大数据研究中心，不断突破数据智能研究发展，目前已成为基于大数据的移动互联网综合服务提供商。公司主要利用大数据能力提供面向移动应用开发者的技术服务、面向广告主的移动互联网营销服务，以及面向其他垂直领域客户的数据服务，旗下拥有个推推送、个推大数据、个灯、个信和个旅等众多产品。
