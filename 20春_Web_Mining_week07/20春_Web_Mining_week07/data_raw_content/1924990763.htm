职位描述：

工作职责：
1、负责京东高端客户平台“东家财富APP”UI视觉工作；
2、与产品经理、交互设计师共同迭代产品，打造极致的视觉体验；
3、负责视觉实现的检查，视觉规范制定，监督产品视觉的实现质量。

能力要求：
1、艺术设计、视觉传达设计、平面设计等相关专业，本科学历；
2、５年以上相关工作经验，精通iOS、Android等设计规范；
3、优秀的设计功底，对手机界面/图标/排版/品牌有较高的设计品味，有金融、高端用户产品设计经验优先；
4、追求细节完美，精通Photoshop、AI、sketch等软件，会交互小动画优先；
5、工作积极主动，有创造力和责任感，具备良好的合作态度和团队精神，有压力承受能力。
