职位描述：

【岗位职责】
1、负责大数据可视化项目与产品在PC端、移动端和大屏端的界面视觉图形设计；
2、有一定的交互和用户体验认知，对业务需求能够有准确的分析与判断；
3、参与产品设计体验、流程的制定和规范；
4、负责产品整体表现风格的定位，对产品视觉感受做整体把握。
【岗位要求】
1、统招本科以上学历，美术、设计、计算机等相关专业；
2、3年以上计算机软件系统的UI设计工作经验，对软件系统的UI布局和交互有设计规划能力，能准确把握产品的整体风格和色彩；
3、熟练使用PS、AI、Axure、AutoCAD等设计软件，熟悉三维软件（建模、材质、动画）和AE（动效）；
4、有良好的配色能力、图标制作能力和构图能力，可熟练制作交互原型图，有较好的素材积累；
5、思维敏捷，具备良好的沟通能力和团队合作意识；
6、较强的自主学习能力和创新能力，具备较强的承压能力。
