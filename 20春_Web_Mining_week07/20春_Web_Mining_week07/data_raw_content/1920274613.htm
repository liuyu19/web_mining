职位描述：

职责描述：
1、负责电商C端产品工作，例如交易流程、流量分发、导购营销、用户体系、订单售后等
2、通过挖掘用户各层次的需求、数据分析及调研，优化现有产品路径，提升用户体验
3、对数据敏感，擅长数据分析，对整体付费转化率负责
4、对于业务侧的诉求可给出合理方案辅助业务发展，并协调产品开发资源推动落地与优化
5、多部门合作，保证产品从设计到开发到上线及后期迭代的完整产品周期管理
任职要求：
1、本科及以上学历，3年以上产品经验，有京东、淘宝等电商经验者优先
2、熟悉电商核心业务流程，了解互联网产品特点，有良好的用户体验设计品位
3、具备产品owner意识，善于沟通协作，能够协调各方资源跨团队推进产品落地
4、具备较好的学习能力和自驱意识，思维清晰、逻辑性强，抗压力强，对工作充满激情
5、能够独立策划跟进大型复杂项目优先
