职位描述：

工作职责：
1、根据产品部门需求，完成需求分析并进行对应的功能设计；
2、与UI设计、产品及后端人员及时沟通，保证产品的质量和开发进度；
3、熟悉用VUE/ES5/6/HTML/CSS并结合现代框架来丰富web产品的用户体验；
4、熟悉敏捷开发，编写高质量的、可维护性的代码；构建可复用的代码以及公共库；
5、不断优化现有的产品，努力提高系统速度，稳定性和可扩展性；不断提升用户体验。
职位要求：
1. 两年及以上 Web PC端开发经验，精通vue前端框架；
2. 理解MVVM原理，至少熟悉Vue前端框架、了解nodeJS；
3. 熟悉浏览器运行时细节，精通 HTTP 协议；
4. 注重设计模式、前端工程化、模块化开发及代码架构设计，有前端改进与优化经验优先；
5. 具备前端微服务化、有大数据可视化经验者优先
6. 具备较强的逻辑思维能力，良好的表达能力及团队合作能力。

企业介绍：

  浙江每日互动网络科技股份有限公司（个推）成立于2010年12月，总部位于杭州，在北京、上海、广州等地均有本地团队，核心成员来自于微软、IBM、卓望、百度、新浪、阿里巴巴等知名公司。
      个推专注推送技术7年，至今已服务数十万APP，覆盖数十亿移动终端，典型客户包括新华社、新浪微博、滴滴出行、中国移动咪咕阅读、网易新闻、芒果TV、招商银行掌上生活等众多有影响力的APP。
      以推送技术为基础，依托庞大的数据体系和深度学习技术，个推成立了大数据研究中心，不断突破数据智能研究发展，目前已成为基于大数据的移动互联网综合服务提供商。公司主要利用大数据能力提供面向移动应用开发者的技术服务、面向广告主的移动互联网营销服务，以及面向其他垂直领域客户的数据服务，旗下拥有个推推送、个推大数据、个灯、个信和个旅等众多产品。
