职位描述：

职责描述：
1.根据公司整体战略发展规划，分解、细化和制订本部门的专业职能规划及业务规划，负责编制本部门年度、季度、月度工作计划及预算，并监督计划及预算的落实；
2.负责本部门管理制度、工作标准、业务流程的建立、完善和监督执行；
3.组织开展本部门内的专业业务创新及团队建设工作，提升本部门专业水平；
4.根据总部技发要求，结合公司发展规划，组织制定和调整开发项目的产品标准，完善产品标准体系；
5..不断完善产品配置标准，优化产品设计；
6.检查和监督产品标准执行情况，收集执行中的意见和建议，不断完善产品标准；
7.督导各专业设计人员建立和维护设计类供方信息资源库（设计单位、外部专家、勘察单位）；
8.负责组织概念、方案、本地初步及施工图设计单位的考察，配合合约进行资格审查、招投标、合同谈签订工作；
9.配合组织概念、方案、初步及施工图设计的设计招标计划、招标文件（技术部分）的编制；
10.督导各专业设计人员对设计类供方进行评估，并将评估结果录入供方信息资源库;
11.组织公司相关新项目的概念、方案设计工作；
12.根据产品概念、方案规划草案，组织进行项目概念、方案设计任务书的编制，并组织评审；
13.负责协调和监控设计计划的完成情况；
14.组织对概念、方案设计进行评审 （必要时聘请外部专家），确定概念、方案设计成果;
15.负责组织评审各方面提出的设计变更请求；
16.负责按权限审核设计变更，满足目标成本要求，重大的设计变更按要求上报审批；
17.督导各专业设计人员完成有关设计变更事务与设计单位的沟通，跟踪设计变更的实施情况;
任职要求：
1.3年以上同岗位相关工作经历;
2.具有独立且完整完成项目设计管理的经验;
3.具有多项目管理经验；
4.具有大型房地产项目设计经验；
5.同时具备大型住宅与高端商办项目经验优先考虑；
6.熟练掌握设计工作的业务知识与系统操作，熟悉设计管理流程；
7.较强的建筑设计、产品研发把控能力；
8.较强的成本优化意识和标准化控制意识；
9.优秀的方案表达、沟通协调能力；
10.善于设计团队管理；

企业介绍：

       绿地香港控股有限公司(香港联交所股票编号：337)是世界500强绿地集团旗下控股子公司。
      绿地控股股份有限公司（上市公司简称：绿地控股，上市公司代码：600606）是一家由上海市国有资产监督管理委员会监管的特大型混合所有制企业，创立于1992年7月18日，总部设立于中国上海。
      成立23年来，绿地已在全球范围内形成了"以房地产开发为主业，大基建、大金融、大消费等新兴产业并举发展"的多元格局，坚定实施资本化、公众化、国际化发展战略，旗下企业及项目遍及北京、上海、广州等80多个中国城市以及美国、英国、德国、澳大利亚、加拿大、韩国、泰国、马来西亚等海外九国十二城，实现了国内A股整体上市（600606.SH）及部分资产香港H股上市（00337.HK 01365.HK），构筑起境内外资源整合的资本平台，位列2015年《财富》世界企业500强第258位。
      未来，绿地集团将以整体上市为契机，借助资本市场的力量，加快企业创新转型，不断将绿地打造成一家主业突出，多元发展，全球经营，产融结合，并在房地产、金融、地铁等多个行业具有领先优势的本土跨国公司，努力从"中国的绿地"转变为"世界的绿地"。绿地香港作为绿地集团资本运作的国际平台，及品牌展示的国际窗口，将进一步围绕集团的整体战略，展开资本平台及产业领域的创新转型升级，为其可持续发展提供有力支撑。
      深耕房地产主业的同时，绿地香港大力推进"大金融"战略，成立上海绿地金融信息服务有限公司（简称"绿地金服"）。借力大数据、云平台、移动互联，绿地金服深耕房地产垂直领域，以服务创新、产品创新为导向，为绿地系产业链倾力打造涵盖理财投资、社交金融、社区金融在内的互联网平台，助力绿地集团构建有场景、有温度的房地产互联网金融生态圈。
      依托母公司绿地集团成熟的品牌形象、丰富的优势资源、宏大的规模体系、先进的管理水平和充满激情的企业文化，绿地香港将全面整合现有资源，充分利用香港资本平台优势，积极践行第一个五年规划：力争以年均50%以上的复合增长率，于2018年实现销售收入500亿元以上的跨越式发展，成为香港资本市场内房股标杆企业。
