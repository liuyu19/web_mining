职位描述：

工作职责：
1、基于对界面设计需求的良好理解能力，完成需要的视觉设计提案
2、团队协作设定整体界面视觉风格与创意规划
3、配合团队高效的开展系统化的详细视觉设计
4、本科以上学历； 3-5年以上相关行业的工作经验
5、互联网领域，知名互联网网站工作经验或手持移动端产品设计经验的设计师优先
6、负责分析和跟踪公司产品用户需求，竞品研究，用户反馈和数据分析，负责产品规划与原型设计并负责落地；
7、负责公司产品PC端和移动端的体验设计；具有较强的交互设计理解

任职资格：
1、热衷于前期的设计方向与趋势的视觉探索
2、关注行业设计动态与科技产品讯息趋势
3、良好的设计需求理解能力、需求分析能力与清晰的沟通表达能力
4、扎实的视觉设计表现力，热衷尝试不同的设计风格与视觉表现方式
5、娴熟的相关图形软件的驾驭能力，熟练掌握并运用Photoshop和sketch
6、视觉设计中注重整体感的画面控制力，并对像素级细节严谨把控的专业精神
7、本科以上学历； 3-5年以上相关行业的工作经验
8.、互联网领域，知名互联网网站工作经验或手持移动端产品设计经验的设计师优先

企业介绍：

致力于健康和快乐的人类未来事业，阿里巴巴于2015年9月投资成立阿里体育，旨在构建中国体育经济的基础平台。
置于健康和快乐的黄金交叉点，体育拥有改变世界的力量。而阿里体育亦秉承阿里巴巴的普惠理念，愿14亿中国人，都能成为运动家。
2017年1月，阿里巴巴加入奥林匹克全球合作伙伴赞助计划，成为“云服务”及“电子商务平台服务”的官方合作伙伴，以及奥林匹克频道的创始合作伙伴，共同追逐奥林匹克理想，推动体育的第三次变革——数字革命。因此，阿里体育努力为更多人、尤其是普通人提供优质的体育服务，希望奥林匹克精神与全人类的日常生活真正充分融合，让运动简单。
除了国际奥委会，阿里体育与国际拳联、国际泳联、世界橄榄球联合会、亚奥理事会等体育国际组织建立了合作关系，用数字化思维推广运动项目，同时通过与体育组织/协会的合作，把体育培训标准化，履行运动家培养计划。
阿里体育亦自创赛事IP。2016年，阿里体育启动原创赛事世界电子竞技运动会WESG，首届赛事总计近200个国家与地区、逾6万人报名，实现了电子竞技的全民参与、全民竞技，第二届赛事亦于2017年6月启动，覆盖六大洲
阿里体育在获取头部IP之后，集成运营盘活资源，打通线上线下的运动和体育消费场景，以大数据为手段全面提升体育产业整体能力，让运动更简单，让未来生活更健康快乐。
