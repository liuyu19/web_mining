职位描述：

工作职责：
1. 负责淘宝整体产品的体验设计； 2. 负责产品的整体概念设计和决策； 3. 负责淘宝设计准则的体系化搭建和标准制定； 4. 建立产品的用户体验指标和数据验证；
任职要求：
1. 有成熟的设计经验和完整的设计落地，对设计充满热爱和理想； 2. 对商业目标和用户行为有前瞻性的思考，对设计结果呈现能精准预判； 3. 独立完成全链路和结构化的整体方案，体系化的规则建设并成为团队的设计准则； 4. 能灵活应对不同类型的业务和类型，并具备较强个人的设计特点和观点 5. 良好的沟通协作能力，能与项目团队快速融合、推进方案并与合作伙伴达成共识； 6. 对新事物有好奇心，关注业界发展趋势，有跨领域项目的成果与尝试；
