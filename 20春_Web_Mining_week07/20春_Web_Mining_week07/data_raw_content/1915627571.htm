职责描述：
1.负责公司产品品牌VI设计，平面广告设计，包括企业宣传册、办公区导视、市场运营品牌物料、网站页面设计、H5页面设计等

3．关注行业设计动态与发展趋势，准确把握公司设计风格，具备流行趋势敏锐的洞察力
4. 良好的沟通协调能力以及团队合作精神，能按时、高质量地完成工作任务；
任职要求：
1．平面、广告设计或视觉传达相关专业本科以上学历，5年以上经验
2. 具备优秀的视觉设计表现能力和创意能力，熟练运用各种不同类型的设计表现形式
3.良好手绘能力更加，优秀的视觉设计表现能力；有互联网、摄影、手绘、4A广告从业经验优先；
4．简历投递请附作品（务必）
