职位描述：

职责描述：
1、通过服务体验类相关数据分析、问题挖掘、报表监控，总结用户在服务体验中的问题、痛点、真实需求，归纳用户行为中的规律，能够对用户体验进行客观可量化的评估并提供优化建议，推进业务不断优化和提升用户体验。
2、组织、实施、监督重点服务体验提升相关项目的推动，持续跟进并提升用户满意度指标；
3、不断创新，协助业务端持续加强服务质量管理,提升用户满意度。

任职要求：
1、本科及以上学历, 有客服、售后、物流相关运营或服务质量、用户体验等相关岗位者优先。
2、有较强的结构化分析能力，对数据敏感，逻辑严谨，善于用从数据中发现业务问题并制定改善计划。
3、有较强的学习能力，能快速理解不断迭代和发展的业务。具有创新精神，能够贴合业务开展创新项目,提升客户体验,降低服务成本。
4、善于沟通，有优秀的执行力、团队协同和资源优化能力，责任心强，勇于承担工作压力。
