职位描述：

工作职责：
1、负责服务端架构设计与开发
2、对新功能技术的研究和突破
3、按照项目计划，在保质保量的前提下，按时完成开发任务
4、总结分享开发经验，提升团队技术水平
任职要求：
1、具有2年以上Java 开发经验，计算机或相关专业本科以上学历（或具备同等水平的实践经验）。
2、熟练使用一种或多种编程语言（例如 Java、Python 和 Go），具有良好的编程习惯和技术文档撰写习惯。
3、熟练掌握数据结构，常用设计模式，JVM 底层机制，精通多线程编程，数据库编程和性能优化
4、拥有 Unix/Linux 环境下软件系统开发经验，了解常用的 Linux 命令，拥有 Linux Shell 编程经验。
5、工作能力强，具有良好的沟通（书面和口头）能力。
6、熟悉计算机网络，熟悉 Http 协议，拥有高并发网络应用或分布式系统开发经验优先。
