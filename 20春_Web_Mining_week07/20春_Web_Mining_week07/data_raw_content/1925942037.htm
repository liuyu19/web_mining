职位描述：

学历条件：统招本科及以上毕业,985/211优先
薪水范围：20K+，优秀者面议
招聘人数：30+
工作地点：成都、深圳、北京、西安

岗位职责:
1. 负责Web产品前端UI及交互页面开发及项目定制；
2. 负责前端html、css、js的设计、编码开发及性能调优，能独立完成系统模块的设计与开发；
3. 配合后端工程师完成后端数据的接口设计及联调；
4. 负责前端的维护、优化、重构；
5. 参与项目需求分析和系统设计，输出相关设计文档。

任职要求：
1. WEB前端开发经验，熟练掌握并使用符合W3C标准的HTML5相关技术（JavaScript/CSS3/WebSocket等），对主流浏览器兼容性问题有深入研究；有基于Ajax的应用开发经验。
2. 熟练掌握使用JavaScript进行Web开发，熟悉html5 + css3的新功能, BootStrap框架等；
3. 熟悉至少一种后端开发语言, 熟练掌握API开发和调试经验者优先，具有组件化的思想，精通前端应用构建和组件封装；
4. 精通Vue.js/Reactjs/Angularjs2等前端框架中的一种或多种，并掌握其原理；
5. 熟悉webpack等打包工具，能够独立架构SPA模块；
6. 关注前端前沿技术，各大前端技术社区活跃者、有自己的开源项目者优先；
