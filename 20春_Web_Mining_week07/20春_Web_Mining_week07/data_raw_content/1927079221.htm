职位描述：

工作职责:
1. 负责ARCFOX APP整体运营工作规划及执行，对注册量及日活KPI负责；
2. 负责APP推广策划及执行管理；
3. 负责APP资讯、动态等内容策划、审核、发布管理；
4. 负责APP活动策划及执行管理；
5. 负责APP商城版块商品上下架及订单、发运管理；
6. 负责APP在线客服运营相关话术编写，相关业务对接管理；
7. 负责运营数据统计分析；
8. 负责APP运营相关工作协调沟通。
任职资格:
1. 统招本科及以上学历，熟悉线上平台用户传播及互动特征，能够把控平台运营内容、活动方向和质量；
2. 有一定文字功底，能够独立完成营销文案的编写；
3. 有一定的审美能力，能够根据品牌VI手册指导及把控运营用图设计；
4. 能够站在用户角度思考，知道如何提升用户体验；
5. 有高度的责任心，工作认真细致，学习能力强；
6. 有较强的沟通能力和团队协作精神；
7. 有一定的抗压能力和创新精神。
