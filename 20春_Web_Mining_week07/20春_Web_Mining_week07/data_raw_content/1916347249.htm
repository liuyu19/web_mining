职位描述：

工作职责:
1、负责集团电商产品的需求调研和分析，制定符合公司战略的电商产品规划；
2、负责产品功能设计、原型设计、流程设计及PRD撰写；
3、负责与研发团队紧密配合，提供咨询、指导和培训业务需求，推进产品评审、开发、测试、发布上线等相关工作；
4、负责收集用户反馈数据，持续优化产品，提升用户体验；
5、负责新产品的创新和预研，完成新产品的用户需求定义、产品功能/UI/交互的设计。
任职资格:
1、计算机相关专业本科以上学历，5年以上电商、供应链、交易平台产品的规划、分析、设计工作经验，熟悉OMS/WMS/TMS等系统设计，有大型全渠道项目经验者尤佳；
2、熟悉B2C、O2O网站架构和技术开发,熟悉主流Java框架;
3、熟悉电商业务流程、运作模式分析与设计，熟悉电商供应链管理流程；
4、理解互联网、移动互联网、电商行业的技术发展方向，对新零售、社交电商有一定理解，最好有已落地项目经验；
5、有较强的沟通协调能力，能够与客户沟通需求，有谈判基础能力，与内部有效沟通的能力；
6、有较强的文案和文档编写能力；
7、有网易严选、拼多多、美团、淘宝、京东等电商平台工作经验者优先。

天狮集团有限公司介绍
天狮集团是一家横跨生物技术、健康管理、酒店旅游、教育培训、电子商务、金融投资、房地产等诸多领域的跨国企业集团，拥有全球产品直销、产品供应、泰济生健康管理、奥蓝际德酒店连锁、房地产等五大业态。目前，天狮集团业务辐射全球190多个国家，在110个国家和地区建立了分公司，天狮创新研发的营养保健食品、美容护肤品、保健用品、家居用品等多元化产品，为全球2000多万个家庭的稳定消费群体带来高品质的生活。天狮国际健康产业园占地超过1平方公里，总投资超过70亿人民币，吸纳来自全球超过10000多名管理人员，已形成了“真正的大健康产业集群基地”。
天狮集团荣获了 “全国模范劳动关系和谐企业”、“最佳雇主品牌”、 “中国人力资源管理杰出企业”等荣誉，并蝉联“中国最佳雇主企业”奖项。在此，天狮集团诚挚邀请优秀人才加盟天狮，我们将向您提供国际化、优越的工作环境，有竞争力的薪酬福利，以及完善的培训计划及晋升机制，为各类人才提供国际化的发展平台。
