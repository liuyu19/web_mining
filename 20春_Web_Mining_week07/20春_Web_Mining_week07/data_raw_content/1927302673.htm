职位描述：

职位描述
1.    为业务线提供整体设计视觉规划，根据类目品牌调性设计主视觉KV及延展，保证用户视觉体验。
2.    为类目整体风格的把握，产品定位负责；为活动推广中的各类需求提供视觉创意设计支持；
3.    负责类目线产品页、产品详情描述及服务流程、活动页、专题页、推广图等所有相关设计；
岗位要求
1， 设计相关专业，3年以上互联网平面设计工作经验
2， 在互联网美术、平面设计、艺术设计等相关专业；较强的创意、审美能力、策划能力和美术功底；
3， 思维活跃，深谙品牌设计在传播中的重要性，并善于协作沟通，团队合作精神强；
4， 抗压能力强，保证视觉底线并积极推进项目进度；
手绘能力为加分项目。
