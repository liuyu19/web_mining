职位描述：

1.负责公司品牌市场的视觉活动设计需求，有出色的创意表现能力，能够驾驭大促类型活动的页面设计，准确把控行业设计风格及产出品质；
2.负责活动、专题、营销广告的创意设计；
3.H5、海报宣传册，H5互动创意等设计；
4.能做画面动态实现；
5.具有良好的合作与沟通能力，能与营销策划、运营团队等各部门进行良好沟通，高质、高效保证项目上线
任职要求
1.艺术设计及相关专业本科以上学历，有3年以上互联网电商行业或品牌视觉设计工作经验；
2.手绘功底扎实、色彩感觉良好、拥有丰富的想象力、强烈的视觉表达能力；
3.熟练使用Sketch、Photoshop、Illustrator等绘画软件；
4.对业界流行趋势比较敏感，能够独立设计出高水准的视觉作品；
5.有电商平台经验者优先;
6.有团队合作精神，工作认真细致，吃苦耐劳，责任心强，有较强的自学能力和钻研精神，能承受强大的工作压力。
7.具有强烈的项目归属感以及主动思考能力，极致的设计提升产品用户体验

企业介绍：

　　华米科技成立于2014年1月，是一家以智能可穿戴设备为基础的移动互联网公司。

　　华米科技在2014年1月获得了小米科技及顺为资本的联合投资，成为小米生态链的唯一一家聚焦于智能可穿戴产品的公司。2014年8月16日，华米科技以79元人民币发售了“小米手环”产品，到当年的11月28日短短的3个多月就突破100万枚销量，仅天猫“双11”一天就售出了10.3万枚小米手环，位列当日的天猫销量第一，并远超其他所有可穿戴产品品牌销量之和。同时，华米科技推出了“小米运动”APP及云端数据服务，成为了中国最大的智能手环及互联网服务企业。

　　华米科技在2014年12月获得了高榕资本、红杉资本及晨兴创投一共3500万美元的B轮投资，公司估值超过3亿美元。华米科技目前已经成为全球出货量前三名的智能手环及互联网服务企业。

　　华米科技是一家以标准VIE公司架构设计的美元基金投资的高科技创业公司，公司通过开设中国北京、合肥及美国硅谷三个子公司，邀请了来自全球各地的资深技术工程师、顶级工业设计师及产品专家加入华米创业团队。

　　我们的使命是将人类真实的情感及活动无感地连入互联网，改善人们的生活，我们提供最先进的人体身份识别、活动监测和情绪感知服务，使互联网更好的服务于人类。

　　“小米手环”是由小米科技授权，由华米科技独立运营的品牌。华米科技不是小米科技的子公司或者分公司，而是一家独立运营的创业公司。小米科技是华米科技的战略投资者，不控股，不主导。华米科技是小米生态链企业，是小米生态链中具备唯一性的小米可穿戴产品的排他合作者。
