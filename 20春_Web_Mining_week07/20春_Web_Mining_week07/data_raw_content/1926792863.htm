企业介绍：

     VIPKID是专注于4-12岁在线少儿英语教育的领导品牌，采用纯北美外教1对1授课模式，使用对标美国小学课程标准(CCSS：美国共同核心州立标准)的定制课程，运用第二语言高效的教学方法——浸入式教学法。截止目前，VIPKID共有超过20万的付费用户，外教团队超过3万人且均来自美国和加拿大，学生续费率高达95%，2017年目标营收为50亿。
   2014年10月，VIPKID获得经纬中国领投、创新工场和红杉资本联合投资的500万美元A轮融资。
   2015年10月，VIPKID获得由北极光创投领投，经纬中国、红杉资本和创新工场的近2000万美元B轮融资。
   2016年4月，VIPKID再度获得真格基金的投资。2016年8月，VIPKID获得云锋基金领投，红杉资本跟投的C轮1亿美金融资，创造全球在线少儿英语教育单笔融资。
     2016年8月23日，科比宣布成立风险投资基金“Bryant Stibel”并战略投资VIPKID，这是继阿里巴巴之后，科比投资的第二家中国创新企业。美国教育风投基金Learn Capital促成了此次投资意向的达成，并共同参与了本次投资。
    2016年10月，VIPKID联合全球顶级教育类风投机构Learn Capital、北美顶级学者成立VIPKID北美教育研究院，投入1000万美金用于教学研究，利用前沿科学技术改善孩子的学习水平和效率。2017年4月，VIPKID与云公益基金会达成深度合作，VIPKID也成为首家入驻云公益基金会教育公益平台的专业英语教育机构。
     2017年8月23日VIPKID获得由红杉资本领投，腾讯公司战略投资，云锋基金、经纬创投、真格基金、ZTwo Capital等参与跟投的D轮2亿美金融资，同日正式启动在线少儿中文教育平台“Lingo Bus”。
     我们相信宽松、有趣、融洽的工作环境可以激发团队的创造力，这样才能为我们的用户奉上最优秀的产品。 我们提供丰厚的福利：六险一金、补充医疗保险、交通、餐费补贴、节日关怀、团建。同时，我们还提供了具有竞争性的薪资、弹性工作制以及各种让你意想不到的party及节日活动。空间无限大，成长特别快，就等你到来。
