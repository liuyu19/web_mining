职位描述：

职责描述：
1、产品外观竞争力，结构转化跟踪：负责产品未来三年产品外观，产品形象和人机交互界面的竞争力；对产品的设计方向进行把握，并引进高水平工业设计公司作为设计资源池优化使用；定期梳理现有产品外观，对设计语言VBL进行统一并在渠道内进行区分；
2、外观ID：产品外观竞争力的提升，整合行业的设计风格和未来设计趋势方向，对产品的外观进行设计优化升级；
3、CMF和UI：通过用户研究工业设计企划，将产品的配色，精细化和材料进行优化设计及搭配，提升产品外观装饰及配色统一性和竞争力；通过UI用户交互界面的设计优化，提升产品的档次及功能实用性；
4、负责与内部平台部门进行对接以及项目推进落地；
5、结构转化跟踪优化：针对新设计外观结构转化中存在的风险进行系统评估和中间过程的评审和即时优化，确保产品设计的转化进行和竞争力。
任职要求：
1、工作经验：3年以上相关工业设计经验，具有小家电产品的外观设计经验；
2、学历：全日制统招本科及以上学历；
3、能力要求：
①精通平面及三维软件，优秀的手绘能力，能够独立完成产品方案制作
②熟悉产品开发设计流程、能掌握产品工程结构可行性、了解产品色彩搭配之合理性及产品及表面加工工艺
③设计理念活跃，灵感丰富，对工业产品的审美观符合大众性、行业性、国际性及时代感的要求，能较敏锐的洞察吸收新的流行元素，对造型、CMF处理有充分的感知和把握能力
④创新意识强，具有良好的沟通能力和协作精神，性格开朗，做事积极主动，勤奋踏实
