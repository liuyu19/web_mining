应聘职位

职位描述：

职位详情：
1、负责信息流或社区产品的视觉设计工作，主导项目的视觉定位，制定设计输出规范；
2、参与产品设计的过程评审，负责视觉实现的走查，并监督开发提升产品视觉的实现质量；
3、关注用户反馈，收集产品需求，为产品新功能提出可落地的设计创意想法，持续优化用户体验；
4、不断总结设计经验，参与团队视觉专业建设。

工作要求：
1、3年以上视觉设计从业经验，良好的视觉设计基础，有较丰富信息流、社区相关设计经验，有成功案例优先考虑；
2、较好的总结、归纳能力和项目管理能力，较强理解力与出色的沟通能力；
3、具备较强的插画、3D、动效实现能力优先；
注：此职位的申请需要提交作品集。
