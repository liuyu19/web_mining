职位描述：

职责描述：
1.品牌微信公众号创意概念和排版制作。
2.品牌digital层面的创意设计。
2.微博、ins、小红书等社交平台素材制作。
3.官网设计和执行监督。
4.线下活动类设计。（包括但不限于pop up store、活动海报、赠礼、创意商品等）
5.lookbook拍摄方案协助和版面设计。
6.主题服装大片方案创意和执行方案策划协助。
7.参与服装产品花型图案研发和设计。


任职要求：
1.全日制本科学历，艺术设计相关专业；
2.有一定插画手绘能力，熟练使用PS、AI等平面设计软件，了解且会使用AE软件（如会其他软件制作动图也可），具备优秀的动图制作能力，简单视频剪辑能力。
3.有较高的视觉审美能力，双微设计工作经验丰富，从事过Fashion、Beauty、Luxury等相关行业设计工作。
4.创意能力好，思维开阔及灵活，主动性高，遇到问题能寻求多种方式解决。
5.团队意识强，乐于沟通和交流。
6.热爱艺术设计工作，对于自己的设计有执着。

VGRASS维格娜丝创立于1997年，是中国国内专业的高端时尚女装自有品牌企业。

公司专注发展直营模式及高端渠道，在目前近360家店铺中，直营店铺的比例高达90%，并与恒隆、万象城、王府井等集团建立良好的合作伙伴关系。多年来始终坚持自主研发设计，在上海、首尔、南京设立三大设计研发中心，以国际设计团队为依托，以高品质面料、修身廓形、精良工艺为产品根本，以“时尚、修身”为品牌核心价值理念，为女性创造美丽，传播时尚。
