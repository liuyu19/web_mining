职位描述：

工作职责：
阿里云智能事业群-云安全团队是阿里云飞天平台（阿里云自主研发大规模云计算操作系统）的核心，在全球超过19 个地域及52个可用区，构建高等级的安全基础设施，同时提供业内最丰富的安全产品及解决方案，为云上企业资产保驾护航。 云安全-IDaaS 身份认证中台团队负责阿里云安全应用身份安全服务化输出。产品能满足公有云， 私有云和混合云的身份中台服务能力， 包括应用管控、用户目录、强认证MFA和SSO、STS、RBAC、TBAC、 ABAC权限模型、粗粒度和细粒度审计等。 同时，IDaaS打造零信任网络安全的认证授权基石， 通过机器学习等AI技术， 使用主动动态授权替代传统的被动静态授权。最终，具备支持云安全其它业务、产品的后端开发工作能力。 岗位职责： 1. 参与阿里云安全IDaaS平台高可用、高可靠、高性能工程构架的研发过程，制定测试计划和测试策略； 2. 参与团队DevOp流程发布版本，GA， UAT，性能， 安全性测设， 确保兼容性和回滚；； 3. 开发基于云原生的自动化测试工具， 辅助手动用例，确保人机交互友好； 4. 掌握Docker, Jenkis， Python 等前后端自动打包测试工具； 5. 技术难点攻关，技术驱动业务。
任职要求：
1. 计算机相关专业，5年以上软件开发和测试工作经验，良好的计算机专业基础。 2. 具备丰富的大型复杂系统软件开发和测试经验，有较强的分析设计能力，测试框架研发和方案整合能力。 3. 精通开源工具框架及相关扩展应用，掌握Jmeter,UI自动化，有扎实JS功底，熟悉网络协议（TCP/IP）和操作系统（windows和linux），精通性能测试、自动化测试、安全测试及白盒测试领域之一 4. 了解OAuth , JWT 等前后端交互安全标注，撰写技术文档 5. 同时能用JAVA， Python／VUE 等前后端构建简单DEMO 6. 准确描述DevOps结合自动化测试过程，掌握Git, Ant等工具 7. 有产品商业化经验， 有对外宣讲产品经历 8. 工作积极主动，善于沟通和团队合作，理解和适应变化

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
