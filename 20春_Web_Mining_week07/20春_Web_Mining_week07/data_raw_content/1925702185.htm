职位描述：

工作职责：
1、阿里巴巴大数据设计中心北京团队，负责友盟+开发者云上数据中台业务的体验架构设计工作，为中小企业开发者、垂直行业用户提供数据设计解决方案。 2、可视化分析：负责移动APP、小程序、Web网站等智能统计分析服务的深度商业设计工作，关联线上&线下媒体营销推广场景的用户研究、价值洞见、体验链路构建，为客户市场和用户洞察、经营决策提供清晰的分析动线和体验设计。 3、可视化互动和展示：负责大数据行业峰会、数据节、数据营销等数据可视化设计、空间互动设计，利用最新的互动硬件设备和互动技术、3D可视化、动效设计、品牌设计，为中小企业开发者、媒体公众提供非凡的数据视听和互动盛宴。 4、创意营销设计：负责阿里巴巴动物园成员 “友小盟”吉祥物的品牌营销设计工作，打造数据形象IP的产品化、商业化，推动友盟+品牌影响力建设；针对商业化产品运营进行创意、玩法营销等运营设计。 5、交互、视觉、创意营销设计方向均有职位。
任职要求：
1、三年以上人机交互相关工作经验，HCI、工业设计、心理学、计算机等相关专业本科及以上学历，拥有良好的英文阅读能力。 2、有大型项目设计经验，精通色彩、图形、信息、UI和可视化设计原则和方法，能够敏锐把握设计流行趋势和潮流方向，善于沟通，能够以业务的视角、用户的语言，完整表达方案和设计价值。熟练掌握体验地图、用户画像、流程设计等各种交互设计方法，对商业和用户有系统前瞻性的思考，能够独立承担大型项目的体验策略制定、交互架构和设计。 3、对大数据、云计算、人工智能、企业级产品等有浓厚兴趣，对新技术、新硬件、新交互方式等有强烈的探索欲望；如在以上方面有相关设计案例优先。 4、具备多样的设计手段，有3D模型设计、动效设计、信息和数据可视化设计、互动设计、空间设计经验优先。 5、请随附作品集

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
