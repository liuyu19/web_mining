15612-高级游戏UI设计师（北京） 腾讯

应聘职位

职位描述：

工作职责：
1. 负责游戏中的整体UI风格的设计,以及根据UI风格制作游戏中的UI界面；
2. 制定UI设计规范，字体字色规范；
3. 与策划沟通UI交互问题，制定资源相关制作规范；
4. UI图标设计工作，图标外包质量把控；
5. 与团队其他组员保持良好的沟通，保证项目的品质与进度。
任职要求：
1. 美术相关教育背景，5年以上游戏行业UI设计从业经验，有次世代项目经验优先考虑；
2. 具备良好的视觉审美能力，一定得造型能力，对游戏UI表现，UI动效有丰富经验；
3. 紧跟时下流行设计元素，并能运用到设计当中，在游戏GUI设计领域有系统性的认知和创造性思维，平面领域也需具备基本知识，有运营平面设计经验者优先；
4. 在设计定位、实施、总结模块都有丰富的经验;
5. 熟练操作相关设计软件（PS,AI）等设计软件；
5. 熟悉UI编辑流程，有UE4下UI编辑经验者优先考虑；
6. 善于与人沟通，有团队精神，及良好的执行能力 ，对自己的作品有很高的要求，有责任心 ；
7. 热爱游戏，关注游戏体验，结合本岗位工作配合产品改进游戏体验；
应聘简历务必附作品。
