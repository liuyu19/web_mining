职位描述：

工作职责：
"1. 负责产业与生态产品的移动端页面设计以及线上线下活动广告页的策划，设计，排版等；
2. 能够跟踪产品效果及用户体验，提出具体设计优化方案并执行交付，带动数据上的变化；
3. 以用户需求为导向，对产品的整体美术风格、交互设计、界面结构、操作流程等做出设计；
4. 与内外部团队进行合作，高效率完成工作，对用户的最终体验负责。"
任职要求：
"1、广告设计，美术，计算机等相关专业
2、3年以上移动端UI/H5设计岗位工作经验，在设计中担任主要角色
3、具有扎实的美术基础，良好的手绘能力,优秀的创新意识与娴熟的设计技巧
4、熟练使用Ps,Ai,AE,FW等工作所需软件，了解移动端设计规范，具备用户交互能力；
5、学历：本科及以上"
