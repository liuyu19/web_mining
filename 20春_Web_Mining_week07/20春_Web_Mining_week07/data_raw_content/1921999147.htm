职位描述：

职位描述：
1. 负责PP体育产品的视觉设计，并对内对外传达产品设计理念；
2. 基于产品特性，进行高质量的设计输出，参与视觉设计规范的制定；
3. 在用户角度挖据需求，为产品功能和特性提供合理的系统化设计解决方案；

岗位要求：
1. 具备产品思维，参与调研分析，做出设计趋势的分析并能提出创新建设性方案
2. 设计相关专业毕业，至少3年以上互联网设计经验，熟悉视觉设计流程，能独立owner业务的整个设计过程，并有成功案例
3. 优秀的审美能力和视觉表现力，精通色彩，图形，信息和GUI设计原则及方法，不错的手绘能力，可驾驭各种设计风格
4. 较强的逻辑思维能力，熟练掌握产品需求分析技巧，能对原型快速有效解读并完整准确还原信息层级和关系，能运用设计手法平衡商业和用户需求
5. 对互联网产品和流行趋势有敏锐的设计嗅觉，富有设计创造力和激情，有探索研究精神
6. 较好的团队意识及沟通能力，设计表达和展示能力强者优先，具备大型互联网工作经验优先，对体育和足球有一定了解，热衷足球运动优先
