职位描述：

工作职责：
1.负责前端页面的设计、开发、维护工作，配合后台开发人员完成项目，提升产品UI效果、用户体验
2.前端模块化、工程化的实现及优化，持续改进产品性能、稳定性和可扩展性

任职要求：
1.计算机及相关专业本科以上学历，2年以上Web前端开发经验
2.熟练掌握 HTML、JS、CSS 等前端技术，熟悉页面架构和布局，对表现与数据分离、Web语义化等有深刻理解
3.熟练掌握 Vue2.x / React / AngularJS 中的至少一种前端框架，并了解其原理
4.熟悉前端工程化、组件化、模块化的开发模式并实践
5.对前端性能优化、多浏览器兼容性问题等有过研究和实践
6.善于学习和运用新知识，善于沟通和逻辑表达，有强烈的团队意识和执行力
7.对后端技术（如 JAVA, MongoDB/MySQL 等）和开发模式有一定了解的优先
8.对用户体验、交互操作流程及用户需求有较深理解的优先
