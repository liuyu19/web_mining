职位描述：

职责描述：
1）负责华为智能汽车安全解决方案设计，持续提升产品安全竞争力；
2）负责华为智能汽车版本安全架构设计与安全特性设计，将安全解决方案在产品版本落地；
3）支撑华为智能汽车安全生态建设，牵引开源社区、行业组织安全技术、安全能力向有利于华为的方向发展；
任职要求：
对汽车信息安全标准体系了解，并能参与标准组织讨论，熟悉汽车信息安全标准的推标流程，在标准组织有一定的影响力；
2、 有全面的车载软硬协同信息安全架构设计经验，有8年以上信息安全架构设计经验，能独立完成自动驾驶信息安全架构设计，具备黑客思维，掌握信息安全架构设计的ASTRIDE/STRIDE威胁分析能力；
3、 有信息安全测评实验室建设和管理经验，对车载信息安全威胁有全面认识，并能指导进行软硬件渗透测试，能构建全面的信息安全测评实验室测评能力；
4、 有汽车或其他类似产品信息安全生命周期管理经验，熟悉生命周期开发流程工具，能够结合华为网络安全红线和汽车信息安全V字型生命周期开发流程，识别和部署信息安全关键节点，保证华为自动驾驶信息安全流程能支持设计和测评的完整性和正确性。
专业知识要求：
1，负责华为IT产品线智能汽车业务部的安全加固、安全评估、安全检测、响应取证溯源，发现IT产品线智能汽车产品面临的安全漏洞、安全风险、安全防护短板，促进产品整体安全防护水平及产品线产品安全能力的提升；
2，有CISP、CISSP等相关资质者优先。
