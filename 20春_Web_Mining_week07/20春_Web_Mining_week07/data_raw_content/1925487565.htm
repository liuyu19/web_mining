工作职责：
岗位职责
1. 负责企平基础架构产品：IAM（身份与访问管理）、SSO（单点登录）、ORG（人员组织主数据）、UPM/UAC（统一权限服务）等相关服务端技术架构规划设计与实现；
2. 参与产品的性能优化、故障排查和解决、安全加固等工作；
3. 培养团队成员，组织技术分享，促进团队成员进步。


岗位基本需求
1. 全日制统招本科及以上学历，5年以上互联网研发工作经验；
2. 具有扎实的计算机科学功底，扎实的编程基础和数据结构算法基础，极强的编程能力和问题解决能力；
3. 深入理解面向服务的架构、熟悉常见的JavaWeb应用框架和工具链；
4. 善于交流，有良好的团队合作精神和协调沟通能力。
具备以下者优先
1. 热爱互联网和新技术，具有极强的快速学习能力，研究过优秀开源软件的源码并有心得者优先；
2. 了解权限模型、单点登录，有相关领域开发经验者优先。
