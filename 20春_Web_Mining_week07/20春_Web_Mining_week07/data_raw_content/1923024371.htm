职位描述：

职责描述：
1、独立完成不同项目方案中媒体规划设计，包括媒体外观造型、类型来匹配建筑主体、外形、外墙、景观的媒体等总体设计，并完成效果表现制作；
2、研究并分析项目整体环境和设施，能够配合项目组读图识图（CAD），给出设计意见；
3、参与制作项目设计方案，能够完成工程图纸材料建议；
4、定期更新专业设计软件，深入了解行业制作表现发展趋势；
5、定期和媒体供应商沟通最新技术和展示形式，完成对内媒体动态信息培训。
任职要求：
1、建筑学或美术专业，有一定的美术基础及艺术修养，具备良好理解和沟通能力，有产品设计经验及水晶石效果图高级表现技法培训经历者优先；
2、能较深刻理解建筑表现特点，在色彩、构图、素描关系方面有较强的把握能力，了解设计师手绘效果意图，精准进行效果表现；
3、3—5年以上工作经验，熟练操作3DMAX2014、vary、corona渲染器、AutoCAD、Photoshop及office办公软件，精通Sketchup者优先；
4、具有建筑及内装图纸的识图能力，可以通过施工图纸独立进行效果图绘制；
5、掌握多种异形建筑模型制作技术，有户外大型体育场馆、机场、高铁站效果图建模、渲染后期经验并独立完成案例者优先；
6、熟知新媒体，新媒体技术应用，行业动态，并能高效高度匹配到不同项目场景中。

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
