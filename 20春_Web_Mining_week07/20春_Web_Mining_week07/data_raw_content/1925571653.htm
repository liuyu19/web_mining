职位描述：

职责描述：
1. 负责支付产品服务器端软件的研发工作，包括服务器端需求沟通，架构设计，核心技术研发，线上服务维护等；
2. 学习和研究新技术以满足产品的需求；
3. 负责软件开发过程中的问题分析和总结，提供建议帮助改善研发流程；
4. 和产品/测试/运营进行密切沟通，能够根据需求提出合理技术方案；





任职要求：
1.本科或以上，计算机软件或相关专业毕业
2. 扎实的编程能力，熟悉算法和数据结构，熟悉计算机的基础理论；
3. 熟练使用Java, 熟悉spring，rose等常见的web开发框架。
4. 熟悉mysql的使用，熟悉Linux开发环境,；
5. 有强烈上进心，自我驱动，学习适应能力强，乐观自信，能挑战自我不断追求卓越；
6. 有支付项目经验，分布式系统、大规模线上系统开发或者大规模数据处理经验的优先
7. 工作积极主动，善于思考
