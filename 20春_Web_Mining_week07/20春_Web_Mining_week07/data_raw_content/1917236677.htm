职位描述：

【岗位简介】
团队成员来源于阿里巴巴、搜狗等知名互联网公司，在个性化推荐领域有一线成功经验，极课在K12教育领域拥有海量学生的考试及作业做答数据，专注于教育精准推荐。亟待高级产品经理加入！
【岗位职责】
1. 负责K12个性化推荐策略设计，效果数据分析及策略优化工作
2. 通过策略、机制设计及优化，实现用户体验提升，学习效果提升，产品收入提升等目标
3. 协同算法团队、引擎团队以及运营团队，根据业务状态制定良性的市场机制以及产品策略，达成平台目标
4. 负责教育类策略产品的项目管理，负责试题的智能推荐，智能组卷，个性化课程推荐，帮助K12学校进行精准教学。推动设计、技术、测试等相关团队及合作部门按时完成产品研发、高质量上线，完成产品目标
5.统计和分析教育产品的各项核心数据指标，对智能推荐的改进、商业变现提供数据支持，并针对性的提出策略优化上的建议
【任职要求】
1、至少2年以上个性化推荐产品设计经验，有教育行业策略相关经验者优先，有预测建模、数据挖掘、机器学习或用户画像，知识图谱项目经验者优先；
2、能独立完成需求调研，数据分析，策略产品设计工作；
3、逻辑思维缜密、工作细致有耐心、沟通良好、勇于挑战、善于分析和解决问题、良好的自我管理能力；
4、有较强的沟通和理解能力，具备较强的数据敏感度和分析能力，有独立分析和解决问题的能力；
5、 数据敏感，较强的数据分析能力，严密的逻辑，能够根据业务数据提出投放策略的优化
6、期待不断思维迭代，享受创业公司的工作氛围的你的加入！

企业介绍：

极课大数据是国内领先的K12阶段大数据精准教学服务平台，全国智慧教育示范企业。公司自主研发的EI教育智能系统，在不改变传统大班教学模式的基础上，真正实现因材施教、促进教育公平。

公司采用图像识别、自然语言处理、计算机深度学习等人工智能核心技术，基于作业和考试进行学习过程动态化数据采集和大数据智能分析，使智能评价实时伴随教学行为，为老师教学提供数据决策支持，制定学生自适应学习资料及学习路径，有效实现大数据个性化教学管理及数据智能驱动的精准教学。

公司于2014年在江苏无锡注册成立，现团队主要位于苏州、无锡、北京，拥有全职员工300余人，其中研发人员占公司总人数50%以上。

公司产品目前已在2000多所优质学校常态化使用，覆盖230多万老师、学生及家长。拥有校本题库总量150多万，考试作业数据4000多万条，教师激活率接近90%。

公司于2017年10月获得由丹华资本领投的B轮融资。
