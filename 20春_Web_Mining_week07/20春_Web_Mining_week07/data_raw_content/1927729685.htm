职位描述：

岗位职责：
1. 独立负责浏览器相关产品的大的功能模块，能根据产品需求做技术分解，选取最优方案，按时保质保量的完成产品需求
2. 编写高质量的代码，保证代码的技术选型、架构、稳定性、性能等方面
3. 参与产品功能讨论，根据产品需求完成技术调研、架构和模块设计、编码、测试工作
4. 学习和研究新技术以满足产品的需求，并对产品提出改进建议
5. 分析和研究竞品，学习竞品的技术方案
6. 承担关键任务的同时，能够定期进行技术分享与培训，促进团队成员共同进步
任职要求：
1．5年以上软件开发工作经验，3年以上Android开发经验
2．精通Java语言和Android开发，在多线程、网络编程、UI、性能优化等方面有扎实的技术功底
3．精通常用软件架构模式，熟悉数据结构、算法、软件设计并在工作中熟练应用
4. 具有扎实的编程功底，良好的设计能力和编程习惯
5．责任心强，能主动承担，学习能力强，具备良好的分析解决问题能力，具备技术钻研能力，对软件开发充满热情
6. 具备良好的沟通能力和优秀的团队协作能力
7. 有强烈的求知欲、好奇心和进取心 ，能及时关注和学习业界最新的移动开发技术。
8．有WebKit/WebView相关或Chromium内核相关经验者优先
