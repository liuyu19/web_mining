职责描述：
1、负责公司工业互联网产品的架构设计和功能开发，输出相关文档。
2、负责组织技术研究和问题公关，使产品研发技术处于行业领先水平。
3、配合软件供应商，保证项目按期高质量上线。
4、负责配合销售进行项目推广，为销售提供产品方案技术支持，确保方案落地
5、熟悉工业互联网、模具行业发展动态，对产品不断优化迭代
任职要求：
1、全日制大学本科以上学历，计算机、软件工程等相关专业
2、扎实的JAVA功底，3年以上相关开发经验，有服务化、分布式架构设计实践经验
3、精通互联网和web应用的相关技术，熟练使用Spring、SpringMVC、SpringBoot、MyBatis等框架。
4、掌握多线程及高性能设计、编码、性能调优，有高并发、大容量、分布式、MySQL、NoSQL等框架的设计和开发经验；
5、熟悉大数据开源平台Hadoop、Spark等，有实际大数据处理经验优先
6、具有责任感、工作积极，具有团队合作意识，自住学习能力强，能承受工作压力
7、有模具行业、物联网工作经验者优先
