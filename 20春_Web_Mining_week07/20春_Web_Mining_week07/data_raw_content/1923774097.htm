职位描述：

【服务市场】
职位描述：
负责服务市场、服务平台产品设计：
1.需要有服务市场（服务平台）或是开放平台项目经验；
2.制定服务标准、服务规范、建立服务管理；
3.建立一体化服务市场、服务商门户、服务商城、服务履约；
4.设计服务产品在线交易平台；
5.设计数字合同自动交易、数字化合同跟踪服务采购、服务计费体系；
6.设计服务评价体系。

【开发平台】
职位描述：
1. 负责开放平台官网、开发者接入流程优化、开发者后台服务及运营等产品的设计和规划，参与开放平台API、SDK设计；
2. 协助第三方服务商更方便接入开放平台，持续挖掘企业客户的痛点及需求，助推服务商业务成长，构建开放平台生态圈；
3. 配合项目经理组成项目团队，完成需求分析、产品设计和项目实施，保证项目顺利发布；
4. 配合支持团队为企业客户提供咨询、解决方案、技术支持服务
岗位要求：
1. 4年及以上产品经验，本科及以上学历，计算机相关专业优先；
2. 熟悉PaaS、SaaS，具有技术背景，能与架构师、研发、企业客户顺畅交流，具有平台化运营思维；
3. 逻辑思维能力强，有良好的团队协作意识，具备良好的沟通能力；
4. 有SDK、开放平台类产品经验者优先，有监控行业经验者优先；
5. 有收费类产品设计经验优先。

企业介绍：

汇通达网络股份有限公司（简称“汇通达”），总部设在南京，聚焦中国农村市场，集B2B2C和O2O模式和经验于一体的，以信息化和大数据驱动，融合互联网精神，创新独特的“农村生态电商”模式，是中国农村生态电商平台创领者。
汇通达以全方位平台服务为核心能力，快速汇聚整合乡镇零售网点，并转化为汇通达乡镇会员店，形成服务于农村消费者的网络。
2016年统计，公司年经营规模已超165亿元，带动30万人创业和就业，与5000万户农民家庭建立了联系和互动，平台价值及服务口碑为业界称道。
截至目前，服务网络延伸至苏、皖、浙、鲁、豫、鄂、湘、闽、赣等15个省的14600多个镇，培养了71000多个优质乡镇会员店，成为国内覆盖规模最大的农村生态电商平台。
汇通达整合多方资源，围绕会员店需求，提供商品、金融、信息工具和营销支持，帮助会员店构建线上平台和顾客粉丝群，增加顾客粘度，解决互动难题，获取和运用消费者数据，培育和提升广大乡镇会员店的线上经营能力、线下服务能力和后台管理能力，让会员店快乐经营、轻松赚钱。
汇通达准确把握农村流通消费特点，创新高效的互联网供应链模式，打破城乡商品流通障碍，解决农村最后一公里的物流和售后难题，让优质商品以最快的速度、最低的成本和最好的服务到达农村消费者手中，促进亿万农民生活质量的提高，让农村消费者同步享受与城市一样的商品和服务。
汇聚四海，通达天下！汇通达不仅是引领创新的平台，更是实现梦想的舞台。预计未来几年内，汇通达将实现覆盖20个省、拥有20万家乡镇会员店、年规模突破1000亿的战略目标。
公司坚守“共创价值，共享成长”的核心价值观，汇集四海客商，凝聚优秀人才，员工、客户、厂商和社会机构齐心协力，携手迎接挑战与变化，一起创业、创新和创造，成就非凡梦想，共赢无限未来！
