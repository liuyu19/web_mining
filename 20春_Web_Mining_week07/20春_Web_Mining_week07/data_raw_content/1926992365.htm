职位描述：

职责描述：
1、负责电商处产品线（网站、APP）的整体UI风格、交互体验、产品呈现、页面排版、广告宣传等设计；其他平面设计工作。
2、负责手机软件界面、用户界面设计与参与交互界面的设计等工作；
3、不断优化界面的易用性，改善用户体验；结合用户体验及业务需求，对已有产品提出用户界面和用户体验方面的合理建议；
4、理解各项产品的设计方案，把控产品应用的各类UI设计规范、UI界面视觉风格；
5、完成领导指派的临时工作
6、重点工作按照进度完成
任职要求：
1、专科及以上，视觉设计、美术等相关专业，三年以上相关工作经验；
2、熟练使用Photoshop、Dreamweaver、Flash、Coreldraw、Illustrator等设计软件，熟悉HTML、DIV/CSS、JavaScript、具有深厚的美术功底和良好的创意构思能力；
3、熟悉用户体验设计，对色彩有深刻的把握力、独特的设计风格、独到的创意视点与创新意识。

企业介绍：

天津荣程祥泰投资控股集团有限公司，是一家以钢铁为主业，涉足互联网金融、第三方支付、文化健康、智慧城市建设运营、农林开发、现代物流等多领域的大型现代化企业集团。集团总部座落在天津滨海新区，拥有员工8000余人，资产总值138亿元。集团旗下天津荣程联合钢铁集团有限公司，具有年产铁、钢、材各500万吨的生产能力，在2012年天津民营企业“销售收入100强”、“依法纳税100强”、“科技创新100强”、“社会责任（捐赠）100强”、“和谐劳动关系100强”、“促进就业100强”、“先进党组织100家”七个榜单中全部上榜，成为七项全能！在销售收入100强榜单中，再次蝉联桂冠。 2015年实现销售收入550亿元，位列全国企业500强第241位，中国民营企业500强第57位，连续十二年位居天津市百强私营企业前列。

荣程集团积极响应党的十八大报告中提出的“加快传统产业转型升级，推动服务业特别是现代服务业发展壮大”的号召，自2010年起，相继成立了以两化融合服务、智慧城市业务为主的荣智达高新技术股份有限公司（滨海新区）；以智慧地产、园区经济、健康养生产业为主的天津荣程联合数字城市投资发展有限公司（津南区）；以第三方支付“融宝Reapal”为支付品牌的天津荣程网络科技公司（滨海新区）；以大宗商品电商平台运营的融通物贸（天津）电子商务有限公司（津南区）；未来以互联网金融为发展的荣联国际融资租赁有限公司、荣联国际商业保理有限公司等非钢产业主体公司。此外，还有以俄罗斯林业为发展的额尔古纳鑫源实业有限公司；同时，荣程集团向健康养生产业延伸，在江西武宁、西双版纳建立了健康养生基地。为荣程的干部员工提供健康幸福的生活保障，并向社会推广。同时在健康酒业、生态农业等方面谋划发展。

“十二五”期间，荣程集团科学制定了非钢产业战略发展规划，进一步明确非钢产业按照“钢铁上游产业（资源产业）、钢铁下游产业、非钢相关行业三大板块，实行板块管理”的发展思路，确立了非钢产业重点培育融通物贸大宗电子商务平台、科技信息、以第三方支付、商业保理、融资租赁为核心的互联网金融作为非钢板块支柱产业的发展方向。
