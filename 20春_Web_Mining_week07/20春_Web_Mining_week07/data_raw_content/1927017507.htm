职位描述：

工作职责：
1.负责网易云音乐服务部门的服务策略制定，包括各业务的内容、标准以及流程体系的搭建和落地，降低业务风险；
2.不断发现流程标准中的不足，制定优化方案，并且能够反哺和推动业务方提升业务策略&流程标准，确保平台整体用户的体验。
3.规划服务渠道，设计各渠道的服务引导和内容策略，通过产品化、系统化、自助化的方式推动流程、标准以及内容的落地，提升整体的业务及服务能力。
4.与各业务方紧密合作，确保业务内容及时准确的落地响应，并处理各类业务风险，为平台业务的发展保驾护航。
任职要求：
1. 3年以上的中大型客服中心sop管理经验，具备数字音乐、数字娱乐类行业的服务流程管理能力；
2. 具备独立思考和敏锐的问题洞察能力，善于发现、定位和分析问题，并通过系统化、产品化的思维方式解决问题；
3. 强烈的客户第一服务意识，用户体验的敏感度高，具备服务渠道和流程标准的创新能力；
4. 有独立运作项目经验，并带领协同方完成复杂、多角色的渠道和流程设计优化工作；
5. 极强的学习力、逻辑思维和系统性思考力，良好的沟通协调及整合能力，有非常强的团队合作精神和抗压能力。

企业介绍：

     网易公司由丁磊先生于 1997 年6月创立,凭借敏锐的市场洞察力和努力，网易公司从10多人的初创团队发展至今，在全世界已拥有上万多名员工，并于2000年6月成功在美国纳斯达克股票市场公开上市（NASDAQ：NTES）。随着公司规模的不断扩大，网易已经成为中国领先的互联网技术公司。网易拥有一批活跃的年轻团队，每位成员都能在充满活力的氛围中孜孜不倦地为理想共同奋斗。我们的团队不仅年轻，而且优秀。网易不断领跑中国互联网发展离不开优秀的团队。无论是领先的技术团队、敏锐的新闻团队、专业的销售团队还是高效的管理团队，都为网易的辉煌，中国互联网事业的发展倾注自己最大的力量。
