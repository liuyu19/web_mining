职位描述：

工作职责：
负责移动浏览器的产品交互设计，包括Android和iOS端； 参与产品需求分析和目标制定，提出用户体验优秀的整体解决方案； 参与用户研究，结合可用性测试方法，关注用户反馈； 结合产品上线后数据的分析和评估，提出改进建议，持续优化用户体验。
任职要求：
工业设计、设计艺术、心理学、计算机、视觉传达相关背景，本科及以上学历；5年以上移动互联网产品的工作经验； 熟悉互联网产品设计相关流程，对交互设计有深刻理解，具备完整的理论和技术体系； 熟练掌握相关设计工具，有较强的动手能力，可快速构建可交互原型； 具备较好的产品意识和用户同理心，善于沟通表达、协调合作，工作有热情和责任感，对设计品质有执着追求。
