职位描述：

工作职责：
1.负责阿里巴巴旗下 书旗小说APP的交互设计工作，对大型活动如双11、世界读书日等重点营销项目，支持交互层面的工作。 2.能独立承担日常业务及项目的设计需求，深刻理解业务后通过创新的设计，帮助业务实现产品体验和数据漏斗的持续提升； 3.始终站在客户和用户视角，坚持产品共创和调研，为产品功能和业务目标提供合理系统的设计解决方案； 4.积极参与设计中心基线体验设计、工作流程优化以及视觉规范的设计制定。
任职要求：
1.逻辑思维能力强，从业务需求分析到产品需求分解到设计体验。交互逻辑清晰，低保真视觉表现到位； 2. 较好的设计价值挖掘能力，能够对前期的探索后续的闭环，起到推动作用。 3.对各种互联网产品和视觉流行趋势有敏锐的触觉，富有设计创造力和激情，有探索研究精神；

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
