职位描述：

岗位职责：
1、负责平台Web产品、小程序的整体前端技术架构；
2、以技术驱动优化Web前端应用，改善用户体验；
3、制定Web前端开发规范及开发流程的优化，保障Web前端产品在各浏览器的兼容性；
4、协调后端相关技术资源制定技术方案，并有效地保障方案的执行落地。

任职要求：
1、统招本科及以上学历，5年以上前端开发工作经验；
2、熟悉 Web 前端技术，对 Javascript 框架应用（如 react 、vue 等）有丰富的经验；
3、对 CSS/Javascript 性能优化、解决多浏览器兼容性问题有一定经验；
4、热爱前端技术，有较强的学习能力，有强烈的求知欲、好奇心和进取心 ，能及时关注和学习业界最新的前端技术；
5、3年以上独立技术团队管理经验优先。

企业介绍：

芒果TV，由湖南快乐阳光互动娱乐传媒有限公司（简称“快乐阳光”）负责具体运营，是以视听互动为核心，融网络特色与电视特色于一体，实现“多屏合一”独播、跨屏、自制的新媒体视听综合传播服务平台，同时也是湖南广电旗下唯一互联网视频平台。
随着湖南广电“一体两翼”和“芒果独播”战略的提出，在举全台之力的支持下，芒果TV不仅与湖南卫视共享优质内容资源，也传承了湖南卫视“快乐中国”的核心理念，与湖南卫视齐头并进，实现两大平台无边界的快乐传播。
芒果TV为全球用户提供横跨电视、电脑、移动、互联网电视等全屏的优质内容服务，不仅提供湖南卫视所有自有版权节目高清视频点播服务，并同步推送热门电视剧、电影、综艺、动漫、音乐视频内容，以及部分电视台网络同步直播。目前，芒果TV（快乐阳光）已连续三年获评中国互联网企业百强，蝉联湖南省排名第一，连续两年获评“世界媒体五百强”，成为湖南广电旗下深化推动媒体融合的践行者与排头兵。
