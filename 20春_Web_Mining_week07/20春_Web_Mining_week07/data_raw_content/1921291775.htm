职位描述：

职责描述：
1.根据业务需求和整体软件架构，设计汽车信息娱乐系统以及仪表系统中的软件模块，如多媒体，音频，视觉，语音，导航等。
2.协助产品团队进行业务梳理，指导开发团队和测试团队进行开发测试。
任职要求：
1.有至少5年软件开发编码经验
2.熟练掌握C++或者Java等面向对象编程语言，熟练掌握软件设计模式。有模块接口设计，模块性能设计，安全设计经验。
3.熟悉常见的操作系统如：Linux, Android等。对系统底层或者中间层如bsp，graphics，audio，camera等子系统有开发经验者优先
4.熟悉信息娱乐系统的某些领域如导航，语音等业务者优先
