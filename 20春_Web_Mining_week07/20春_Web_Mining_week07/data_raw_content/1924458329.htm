职位描述：

职责描述：
1. 负责华为手机浏览器，云空间，天际通等app的用户体验优化（交互设计，视觉设计方向）。
2. 核心指标是通过对用户需求洞察，完成交互流程，产品交互设计，视觉设计的优化提升以上产品的用户满意度，用户活跃率指标。
3. 对业界前沿的交互设计理念与技术做相关研究洞察，与部门的用户体验工作相结合，推进新技术新理念的应用落地。

任职要求：
1. 掌握移动互联网产品交互设计，用户体验设计的方法及技能，提升用户体验，并对C端用户心理及行为研究有较深入理解和思考。
2. 对产品交互体验设计拥有较强的逻辑思维和设计能力，以及体验测试跟踪方法；能够基于用户使用行为数据，挖掘交互体验痛点和提升点，并跟踪优化前后的数据对比。
3. 5年以上用户体验，产品交互设计，人机交互，视觉设计类工作经验。
4. 全日制本科以上学历且毕业院校符合985/211；艺术学，设计心理学，交互设计类相关专业优先。
5. 希望你对新鲜事物有好奇心，喜欢钻研问题本质，有用户同理心，对用户体验有自己的极致追求，沟通表达及推动能力强。
6. 后续发展：在用户体验设计/交互设计岗位工作一定时间后，有机会根据个人能力转型产品经理或产品运营。
