工作职责：
负责微信后台的分布式基础组件开发，支持微信事业群的公共需求（搜索、视频号等），包括最近邻（ANN）搜索服务、高性能特征存储服务等
任职要求：
1. 具备强烈的技术热情和良好的学习能力、分析问题解决问题能力，善于合作、乐于分享、责任心强；
2. 扎实的编程能力，有大型系统的代码编写经历；
3. 具有高性能服务系统设计经验及能力，能独立承担任务和有系统进度把控能力，以下任一点精通优先：
1. 精通算法与数据结构;
2. 对Linux内核代码有一定的了解，熟悉内存/存储管理相关部分;
3. 熟悉ANN搜索算法/库，比如 faiss，hnsw；
4. 熟悉并行数据处理，如 MapReduce，Spark 等；
