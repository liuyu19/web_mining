应聘职位

职位描述：

工作职责：
1、负责腾讯企点产品的 Android 端研发
2、和产品经理协作，根据产品需求进行 Android 产品的开发，高质量实现业务逻辑；
3、参与应用架构设计、开发、性能优化等工作；
4、持续不断的改进用户体验，负责相关新技术新架构的实现。
任职要求：
1、本科及以上学历，计算机相关专业，3 年以上Android 平台开发经验；
2、熟练掌握 Java 语言，至少3年以上使用经验，对多线程、常用数据结构等有一定了解；
3、有IM相关产品开发经验者优先；
4、接触过JNI开发，对C/C++有一定了解，有实际开发经验者加分；
5、熟悉Android UI 开发，掌握主流开源框架的使用，并了解其基本原理，对应用性能优化有一定理解和实践；
6、关注安卓技术趋势，对Android Jetpack，以及Flutter/React Native/Weex等跨平台框架有使用经验者加分；
7、具有良好的代码风格，有较强的独立工作能力，掌握Android碎片版本的适配与兼容方案；
8、有良好的学习能力、沟通能力、逻辑思维能力，热爱技术，对技术成长有自己的规划并始终能够付诸实践；
【注：此岗位为腾讯集团旗下全资子公司编制岗位】
