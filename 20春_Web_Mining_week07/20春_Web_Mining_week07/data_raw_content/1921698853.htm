职位描述：

工作职责：
岗位描述：
1.负责美团点评OA、HR、财务等企业信息化产品的交互设计；
2.深度了解业务目标，能合理拆分落地产品需求并提出可执行方案；
3.完成产品框架、流程和界面的交互设计，提高产品易用性和效率； 
4.参与设计规范的制定和迭代，与产品经理、前端工程师紧密配合，做好跨部门和异地沟通。 

岗位基本要求：
1.5年以上工作经验，具有B端产品经验优先； 
2.掌握并运用用研方法，如问卷、可用性测试、用户体验地图等；
3.主导过完整的互联网/软件产品交互设计工作，能较好的驾驭跨平台多端设计；  
4.具备独立思考、设计创新、合作共赢的意识； 
5.产品把控能力强、有良好的逻辑性，善于产品业务逻辑分析及竞品分析；
6.善于沟通，学习能力强，能高效与团队合作，促使产品的迭代与优化；
7.工业设计、心理学、计算机、设计艺术等相关专业。
具备以下者优先：
1.全日制统招本科及以上学历；
2.具有企业管理或协同办公应用等类型产品的设计经验；
3.请附上文字简历及作品集链接，pdf形式为佳。

岗位亮点：
1.能够参与到员工在企业整个生命周期中所涉及的多种业务和系统，全面了解企业管理类、办公应用类产品设计；
2.有机会参与或主导用分析产品和用户现状，提出设计需求，并推动设计方案的最终落地；
3.能与具有丰富B端经验的同学一起协作，输出高质量的设计方案和规范，与团队一起成长；
4.公司提供用研、交互、视觉、产品等多种类型的培训课程，开放的知识学习、分享和传播氛围。

美团的使命是“帮大家吃得更好，生活更好”。作为中国领先的生活服务电子商务平台，公司拥有美团、大众点评、美团外卖、摩拜单车等消费者熟知的App，服务涵盖餐饮、外卖、打车、共享单车、酒店旅游、电影、休闲娱乐等200多个品类，业务覆盖全国2800个县区市。2018年全年，美团的总交易金额达5156.4亿元，同比增长44.3%，年度交易用户总数达4.0亿，平台活跃商家总数达580万。

2018年9月20日，美团点评（股票代码：3690.HK）正式在港交所挂牌上市。
