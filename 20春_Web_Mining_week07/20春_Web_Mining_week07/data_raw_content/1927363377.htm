职位描述：

工作职责：
1、挖掘海外卖家需求，协同环节部门梳理供应链服务解决方案，持续提升海外卖家供应链履约的服务确定性； 2、对海外卖家需求和行为深入分析，依据客户声音，横向拉通流程和产品团队，制定并推动可行性优化方案落地，提升服务效能； 3、针对商家生命周期不同节点的服务需求和痛点，结合海外市场的本土化特性，提升海外卖家线上供应链履约的活跃度和粘性； 4、通过数据分析和需求洞察，设计和落地海外卖家的分层运营策略，提供适配性的供应链服务解决方案，提升海外卖家线上化服务能力和履约交易能力； 5、管理和优化生态伙伴对海外卖家的服务过程，基于现有的服务质量管理体系，建立全球生态伙伴的服务监控指标，通过定制化的生态赋能和培训，提升渠道服务能力和质量； 6、针对不同国际市场的履约服务模式，打通国际站海外卖家的服务承接链路，梳理和设计客户和生态伙伴的日常咨询和异常闭环流程； 7、梳理海外卖家出口服务流程中的关键操作节点，建立服务质量的监控指标，通过数据及线上工具应用，持续改善客户体验，全面保障客户满意度。
任职要求：
1、本科及以上学历，五年以上客户服务经验，具有海外工作或海外客户服务经验优先 2、商业敏感度强，了解海外市场的发展趋势和跨境供应链痛点，有良好的市场敏感度和较强的竞品分析能力； 3、熟悉跨境贸易运作模式和供应链管理，能理解业务规则和业务知识，转化为服务语言形成服务解决方案，解决用户问题； 4、有较强的思维逻辑，善于进行数据分析，并将分析结果和商业洞察结合进行准确的商业判断； 5、善于跨部门沟通，英文沟通流利，有良好的跨文化沟通和协同能力。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
