职位描述：

岗位描述：
1.负责消费者业务官网的产品页和其他品牌产品/营销促销等页面的设计，并持续优化视觉设计上的细节；
2.参与消费者研究及网站交互整体方案设计；
3.负责最新网站内容视觉设计流行趋势分析，参与设定网站的整体视觉风格并建立统一的视觉规范标准；
4.负责网站视觉素材库的搭建和更新，负责部分网站原始图片素材的修图优化等工作，提升品质。

岗位要求：
1.设计，艺术相关专业，从事设计工作3年以上，能独立完成大型网站/电商/APP视觉设计项目，熟悉互联网网页/网站设计方法与流程
2.精通Photoshop，熟练使用sketch、Illustrator、InDesign、Dreamweaver等图形工具，熟悉DIV+CSS、HTML等前端技术；
3.具有丰富的设计美学理论知识，以及对流行趋势和用户偏好的洞察力，具有较强的设计技巧，善于发挥视觉创意和表现力；
4.熟悉视觉设计的系统性知识，能够巧妙运用视觉要素，系统地考虑视觉的统一性。
