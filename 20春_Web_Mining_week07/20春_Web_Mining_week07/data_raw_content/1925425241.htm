职位描述：

工作职责：
1.统筹690智慧园区智能化整体需求规划，对接DTS、安保、行政服务、地产及建筑设计单位；
2.协调智慧安防、智慧交通、智慧会议、信息化基础设施、智慧楼宇等规划及监督实施
3.协调大数据平台及应用、移动应用等相关智慧园区规划及监督实施；
4.对接各产业需求规划建设落地；
5.对接地产进行工程建设管控，对安全、工期及现场文明施工等管理工作；
6.协调新理念、新技术、新工艺、新材料、海尔超前产品等在园区建设项目示范应用等需求规划
7.绿色&智能园区政策申报。
任职要求：
1、工作5年以上，计算机、软件、电子、自动化、通信等相关专业本科及以上学历；2.具备智慧园区规划或实施相关工作经验；3、较强的规划统筹能力，沟通能力及组织协调能力；4、工作认真、负责、敬业，能承受较强的工作压力。
