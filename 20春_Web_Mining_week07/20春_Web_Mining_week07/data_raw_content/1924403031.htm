职责描述：

职位背景：
海运行业是一个专业化和全球化的行业，同时又是一个相对保守和封闭的行业，海运业在严格和细致的分工下安全有序的运行了几百年，但客观的说海运业在模式创新、科技创新等方面已经远远落后于时代的发展，在其他行业已经成熟运行多年的平台电商、网络支付、产业物联网、大数据人工智能等等在海运业还是非常新鲜的概念，但没人可以否认这些新模式和新科技必将给传统的海运业带来巨大的商机和改变。海运在线就是这样一个国际化平台，我们致力于将这些成熟的模式和技术引入海运业，打造全球领先的一站式海运电商服务平台，帮助船东、货主、供应商等海运用户也能分享到模式和科技创新带来的发展红利。为了让我们的高科技团队能够更加深刻的理解海运业的发展现状和各个环节的痛点，从而设计出真正有价值和受欢迎的平台产品，我们特别设计了这个岗位，期望作为海运业专家的您能加入我们一起创新，一起改变，为海运业创造更大的社会价值和经济价值。

职责描述：
1. 编写海运产业链上下游各环节企业整体业务关系，业务模式，总结业务痛点和解决方案
2. 给产品设计、技术开发、运营客服等非海运业人士深度培训和讲解海运业务
3. 代表产品团队和业务团队深入交流，完成各业务板块的产品需求设计书（BRD），包括租船业务、船代、海上供油、船舶供应、船货保险、买卖船、船员劳务以及其他综合海事服务等
4. 参与产品用户体验设计，确保平台功能有价值，方便好用
5. 参与产品测试和验收，结合最初的需求设计全面测试和验收完成的产品，提出问题和改进意见
6. 负责海运产品/服务等专有名词的中英文翻译和检查确保专业性
7. 跟踪行业和竞争对手发展动态，提出系统化的新产品、新服务开发建议
8. 协助业务团队、市场开发团队引荐客户和协助沟通
9. 参与其他支持产品开发和业务开发的实际工作

任职要求：
1. 拥有10-20年海运行业从业经验，至少5年以上一线操作经验，5年以上部门经理及以上管理经验，年龄范围35-45岁；
2. 精通海运产业链上下游各类企业的业务模式，上下游业务关系，主要业务流程和问题痛点；
3. 具有优秀的沟通表达能力，能够用浅显易懂的语言对计算机技术人员进行行业培训和讲解；
4. 对互联网电子商务有较深的理解，能够结合海运业需求和互联网模式提出有价值的结合意见；
5. 具有广泛的行业客户资源，有能力引入体验用户并转换为真正的平台交易用户；
6. 电脑应用熟练，适应互联网公司文化，项目需要时能和技术团队共同加班赶工；

光汇石油集团简介
    光汇石油成立于1992年，是国内知名的民营石油企业集团，被誉为仅次于“三桶油”的国内第4大石油集团。2008年在香港联交所主板上市（证券代码HK0933），2017年营业收入超500亿港元。集团主要从事上游油气田开发开采和生产、远洋油轮运输、石油仓储与码头以及石油国际贸易与海上供油、成品油批发零售等业务。大陆、香港、新加坡三地拥有员工超过2000人。2017年，在全球石油天然气企业中排名第58位、成为中国大陆入榜民营油企排名最高的公司。在2017《财富》中国500强排名中，光汇石油排名第139名，位居民营石油企业首位，排名仅次于“三桶油”。
      光汇石油拥有迪那1 气田、吐孜气田以及渤海湾曹妃甸油田区块三个油气田项目；拥有四艘Aframax远洋油轮、五艘超大型油轮（VLCC），总容量超过200 万载重吨。集团正在建设位于舟山外钓岛的316 万立方米库容的石油仓储设施。该码头将配备13个从千吨级到30 万吨级大小不等的泊位。同时，光汇还在兴建位于大连长兴岛上总库容达719 万立方米的石油仓储设施，该码头将配备13个从千吨至30 万吨级大小不等的泊位。光汇石油集团也是中国最大的海上供油服务供货商之一，服务已扩展全球港口，贸易产品种类包括燃料油、原油、汽油及石油化工和相关石油制品等。集团将继续大力开拓发展上游业务，从事油气田开发开采和生产，锐意成为知名的国际能源资源型企业。
      依托庞大的实体经济体，2016年1月，光汇石油创新性地推出了“石油产业链+互联网”的综合电子商务平台——光汇云油，通过提供大众储油、云储油、储油通、海上供油在线平台等系列产品和服务，进一步整合石油采购、供应、储存、物流服务、金融服务等，打造全产业链的石油能源电商生态圈，引领石油能源行业创新发展。
