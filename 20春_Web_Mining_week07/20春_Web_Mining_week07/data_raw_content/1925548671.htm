职位描述：

薪资，股票面议。

职位描述：
1.支持业务团队进行相关项目的售前/中/后相关业务拓展工作，能够直接面对企业高管和中层业务部门进行沟通，为泛零售相关行业客户的相关项目提供业务咨询、技术咨询、规划、设计等专业服务；
2.协调产品、生态、市场、运营等资源，致力于将已成型的产品快速推向市场，帮助企业解决目前所面临的相关问题；同时，继续寻找新的方向，推动孵化出更多能够帮助不同泛零售企业客户数字化转型的产品和解决方案；
3.聚焦泛零售不同的细分行业的客户业务模式和特点，结合阿里云技术和产品，创造适用于泛零售行业不同细分领域的解决方案。

职位要求：
1. 本科以上学历，计算机、软件工程、管理信息系统或电子工程等相关专业；
2. 熟悉云计算、大数据等技术，具备基于云计算的架构设计经验，能提供业务和技术层面的架构咨询服务，理解客户的业务和功能性需求、非功能型需求、性能及可用性需求，针对客户的业务需求提供基于云技术体系的架构设计方案；
3. 8年以上泛零售行业信息化项目经验，熟悉泛零售行业的业务流程、业务架构、应用场景；
4. 具备具备优秀的文档能力，清晰明了地表达架构意图，能够熟练编写各类售前文档、技术文档与解决方案文档；
5. 具有较强的沟通能力，较强的分析问题和解决问题能力；
6. 具有优秀的团队合作意识，擅长自我驱动，主动学习，做事有责任感，能承受较大的工作压力，能适应出差。
