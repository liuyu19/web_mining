职位描述：

岗位职责：
1. 基于UE4（Unreal Engine 4）引擎，完成游戏客户端的设计和功能模块开发（剧情/情景类）；
2.前后端都需要涉及；
3.能够承担游戏业务对外接口角色。

任职要求:
1. 计算机或相关专业本科及以上学历，3年以上C++游戏项目开发经验；
2. 熟悉UE4（Unreal Engine 4）骨骼动画蓝图以及混合空间的原理及应用;
4.有至少一款成功的游戏项目经验；
3.有一定业内资源，能够尽快组建技术团队；
4. 有FPS\TPS\ UE4项目经验优先。
