职位描述：


岗位职责：
1. 负责依图AI+新行业解决方案中前端、移动端产品的架构设计与实现；
2. 负责依图AI+新行业解决方案中已有产品的前端、移动端优化、修复及插件开发；
3. 负责部分产品中台系统node层开发工作，持续优化前端性能及前后端集成效率；
4. 把控用户体验和前端性能，保证优良的代码可维护性；
5. 把控研发工作的全生命周期，包括产品形态讨论、需求分析、文档撰写、设计编码、测试运维、效果追踪等。
任职资格
1. 统招大学本科及以上学历；
2. 三至五年前端开发经验；
3. 对javascript/html/css有深入了解，熟练掌握ES6、typescript;
4. 深入了解react/redux/saga及相关生态;
5. 掌握nodejs服务端开发语言；
6. 了解移动端混合开发，掌握一种跨平台编程框架react-native/weex/flutter；
7. 拥有良好的逻辑思维能力、沟通能力和文字表达能力。
8. 自我驱动力强，执行力强，学习能力强，适应力强，具备良好的格局观及团队合作精神。
加分项：
1. 对视频直播/富文本编辑/动画性能优化中某一分支有深入理解或经验
2. 具备AI算法/行业背景

企业介绍：

依图智能安防技术主要由人工智能算法、高性能计算、分布式计算和存储、大规模运维等构成。其中视觉智能算法主要涉及人脸识别、车辆车型识别、文字识别、目标跟踪、图像特征搜索等技术。

依图技术团队来自MIT 、Google、阿里巴巴等顶级学术和工业机构，将最前沿的美国学术理念和扎实的工业界实战经验相结合，应用先进的人工智能算法，包括深度学习、统计建模技术，创新性的解决技术和应用结合的难题，能有效处理大规模样本（亿级训练样本）和少量训练样本的机器学习问题，具备丰富的世界领先的技术落地经验，包括超大规模识别（十亿级比对）、超低误报率（亿分之一误报）、多类型终端感知（静态图片、动态视频、低清画质、网纹噪音）、多人种跨年龄段识别等复杂应用场景 。
