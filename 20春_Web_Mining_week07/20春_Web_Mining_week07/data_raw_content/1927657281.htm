工作职责：
负责腾讯动漫版本新功能开发以及相关功能维护；
与产品、测试及其他团队协作，分析产品需求，制定技术方案；
负责产品性能优化、质量指标的优化以及团队项目代码的优化。
任职要求：
1. 本科以上学历；3年以上相关工作经验；2年以上iOS平台开发经验
2. 熟悉掌握Objective-C编程语言，同时熟练掌握iOS应用开发。
3. 了解iOS系统原理,熟悉移动终端特性和解决方案，熟悉API调用、多线程、网络相关知识，熟悉UI布局和控件的使用，有较好的界面还原能力, 同时有一定的自定义控件能力；对iOS性能优化、内存优化有一定的经验；
4. 拥有良好的代码习惯，代码结构清晰、命名规范，提高代码的可阅读性；
5. 积极主动、认真踏实、有良好的沟通能力和团队合作精神，有责任心、抗压。
