职位描述：

工作职责：
1. 参与政务中台系统测试的全流程，包括参与需求分析、设计评审，制定测试计划，设计和执行测试用例，进行缺陷跟踪和软件质量分析等； 2. 带领其他测试开发工程师执行项目测试，包括制定测试计划，分配测试资源，构建测试环境，执行集成测试，回归测试等； 3. 保证被测系统的质量，并通过测试流程和方法创新，努力提升研发的质量和效率。
任职要求：
1. 工科、计算机或其他相关专业本科以上学历，2～6年测试开发工作经验； 2. 熟悉Java/c/c++/python等一门语言，有测试工具平台开发、Android/IOS开发、UI自动化开发经验者者优先； 3. 精通测试流程和测试用例设计方法, 能够独立owner一块领域的测试工作，在原有业务领域拿到具体的成果； 4. 善于沟通协作，能够主导大型跨团队的项目保障工作，以结果和行动为准则，努力追求卓越。

阿里巴巴集团的使命是让天下没有难做的生意。
