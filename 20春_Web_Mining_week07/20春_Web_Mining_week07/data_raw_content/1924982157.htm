职位描述：

主要职责：
1. 负责对语音行业进行趋势调研，输出有价值的调研报告
2. 负责对行业竞品进行动态监控，了解新的体验和技术趋势，对内推动技术迭代
3. 负责语音公众号维护，更新技术进展和研究成果
4. 负责核心粉丝用户的走访，输出体验监控报告
 
任职要求：
1. 有3年以上的语音领域产品运营经验
2. 对竞品和行业趋势进展敏感，能够形成量化的分析结论，推进产品迭代
3. 有较强的沟通交流能力，善于获取行业信息进行分析整合归纳
4. 对小米产品有热情，小米极客粉优先
5. 有AI媒体和用户研究经验者优先
