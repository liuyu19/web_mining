职位描述：

工作职责：
负责波动星球的前端体验和视觉还原工作；
编写交互和视觉层面的JS代码，使用Css3、SVG、canvas等技术实现页面动画效果；
推动公共UI组件化建设，提升前端页面的良好用户体验创造更高项目价值；
关注前端技术发展和应用，了解泛专业相关互联网资讯趋势，参与团队的专业化建设和知识传播。
任职要求：
1. 2年以上Web前端开发经验；
2. 理解Web标准并熟练运用HTML/CSS/JavaScript构建高性能web应用；
3. 对UI组件化开发、响应式、多终端兼容适配、无障碍有一定开发经验；
4. 有node.js/php开发经验者优先；
5. 对Web性能、安全相关有一定的了解；
6. 对语义化结构和SEO有一定的了解；
7. 了解主流前端构建工具，并能根据项目实际情况定制构建工具；
8. 有创新精神并能积极学习业界新技术，顺畅的沟通合作能力。
