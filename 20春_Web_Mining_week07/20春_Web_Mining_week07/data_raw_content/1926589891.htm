职位描述：

工作职责：
岗位职责：
1.根据产品的视觉风格，应用Unity或UE4引擎制作游戏中各类游戏动效，提升游戏表现；
2.配合程序、策划，进行动效资源优化，并及时处理游戏中出现的相关动效资源问题；
3.熟知交互设计流程，关注时下视觉动效流行趋势，为团队提供专业的动效意见和建议。

任职要求：
岗位要求：
1.大专及以上学历，2年以上游戏动效、特效、视频设计工作经验，对游戏、动漫、影视有一定的了解。
2.精通UE4引擎制作游戏中各类界面动效制作，熟练掌握UE4引擎的动画编辑器、粒子编辑器和常用特效制作插件。（有UE4作品的优先）
3.熟练使用3dMAX、U3D、AE或PS等相关软件或各种，具备进行多种风格的三维与二维动效设计制作能力；
4.有较强的美术功底，有丰富动画特效经验，具备画面设计能力、排版能力、后期合成能力；
5.良好的沟通和团队合作，热爱游戏，关注用户体验，有创新精神。
6.能提供特效相关作品；
