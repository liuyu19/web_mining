职位描述：

职责描述：
1、根据公司发展战略与年度经营计划，组织制定并实施供应链战略规划；
2、和集团信息管理部一起设计并改善供应链IT系统，制定并完善切实可行的计划、采购、仓储、物流、生产等管理工作流程；
3、依据销售的年度、季度销售预测，提前做好生产产能规划与供应商资源规划，并完成实际销售订单在各运营中心及分子公司之间的分配；
4、推动技术部门的模块化与标准化设计，有效减少订单交付周期；
5、在集团范围内推进优秀供应商资源共享。建立和健全供应商的开发、维护、跟踪及评估体系,合理控制采购及运输成本；
6、督促各运营中心及分子公司做好生产计划与采购计划，跟踪生产执行状况，确保销售订单准时交付；
7、制定集团供应链体系的绩效考核体系和人员培养计划；
7、指导、监督、检查所属下级的各项工作，做好部门人力资源管理。
任职要求：
1、5年以上供应链相关管理经验，具备供应链管理体系的搭建和流程优化能力；
2、具有较强的逻辑思考能力，具备变革管理技能及抗压能力；
3、具有丰富的设备制造行业机械材料及电气材料采购管理经验；
4、了解光伏电池片生产设备与光伏组件生产设备技术及材料的优先考虑；
5、有供应链计划管理经验的优先考虑；
5、具备良好的沟通协作能力，工作责任心强，心态积极向上。

企业介绍：

  营口金辰机械股份有限公司始创于1996年，坐落在风景美丽、气候怡人的辽宁营口沿海产业基地，现有员工500余人，占地面积9.5万平方米，拥有现代化标准厂房4万平米，研发办公楼1万平米，公司目前在营口、沈阳、苏州、秦皇岛和德国康斯坦茨建有研发中心，在营口、苏州、秦皇岛等地建有生产基地。
  公司属于先进制造领域中的智能制造装备行业，核心技术为自动化控制技术和信息技术，主营业务为自动化生产线成套装备、MES等信息化软件产品、全自动串焊机、高效层压机、EL测试仪、搬运机器人和并联拾取机器人等自动化生产单元及检测设备的开发和应用，为客户提供数字化和信息化的生产线系统集成解决方案，目前在国内太阳能电池组件生产线领域市场占有率达到70%。公司目前所拥有的成套自动化控制、机械、视觉检测、图像分析处理和信息系统软件等关键技术，具有良好的可移植性，可以广泛应用在光伏、食品包装、家具、建材、港口等多个行业。
