工作职责：
1.ToB及ToC品牌话语体系搭建：
了解充电垂直行业品类动态，梳理并制定针对充电供给侧和需求侧品牌话语体系，包括但不限于官网内容搭建、Branding Book、业务及产品手册、品牌传播方案及端内文案、用户案例包装和品牌创意策划、微信微博等新媒体营销工作。
2. 视觉体系管理：
独立对接视觉设计团队，对atl&btl视觉物料负责，结合业务发展动态，实时产出围绕品牌认知、业务宣贯等展开的用户教育内容，包括但不限于POP类广告、创意campaign等内容
3. 渠道模式创新
开拓思路，深度思考商业模型，开展co-mkt和资源置换类异业合作，拓宽充电用户的沟通场景，多维度铺陈品牌教育和业务宣发内容，提高品牌认知度，对传播效果负责
4. 对接PR及Internal 传播工作
独立对接PR和Internal团队，负责把控品牌传播内容方向和其他内部沟通事宜，如管理端内资源位、需求排期、品牌资料库搭建以及市场部日常需求的对接
任职要求：
1.互联网公司或agency经验优先，具备项目独立策划能力；
2.具备优质内容产出能力，具备一定的讲故事能力，有作品和案例优先；
3.具有一定数据分析能力，可参与日报及城市业务数据分析；
4.问题解决型和自驱学习型，能适应变化者优先（英文好加分）

关于我们

滴滴出行是一站式移动出行平台；为5.5亿用户提供出租车、快车、专车、豪华车、公交、代驾、企业级、共享单车、共享电单车、共享汽车、外卖等多元化的出行和运输服务。在滴滴平台，超过3100万车主及司机获得灵活的工作和收入机会。

滴滴出行与监管部门、出租车行业及社群积极协作，致力于以智慧交通创新解决全球交通、环保和就业挑战。滴滴也和汽车产业链企业建立广泛的合作，携手打造面向未来的汽车运营服务平台。

在全球范围内，滴滴与Grab、Lyft、Ola、99、Taxify、Careem构建的移动出行网络触达全球超过80%的人口、覆盖1000多座城市。目前，滴滴通过旗下99平台服务巴西，在墨西哥和澳洲提供滴滴品牌的出行业务；并在日本通过合资公司提供网约出租车服务。滴滴始终致力于提升用户体验，创造社会价值，建设安全、开放、可持续的移动出行新生态。
