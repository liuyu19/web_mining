职位描述：

职责描述：
1.负责闪送产品的交互设计工作，分析用户的操作习惯和偏好，并设计交互流程、内容及页面；
2.根据需求和用户研究结果，完成界面流程设计并改良交互行为和功能，提高产品易用性；
3.对现有产品进行可用测试和评估，提出改进方案，持续优化产品用户体验；
4.维护和更新界面审核标准及对标准和规范的实施；
5.撰写用户调研报告和产品可行性分析报告。

任职要求：
1.本科及以上学历，美术设计相关专业，5年及以上工作经验；
2.熟练使用PS、AI、Flash等软件以及HTML、DHTML、javascript、CSS+DIV布局；
3.色彩感、形式感强，思路开阔，创意丰富，对UI设计有良好的认知和深刻的理解；
4.对用户体验方面的理论有较深的认知和深刻的理解；
5.熟悉互联网产品的研发流程和方法，可以独立完成整个设计过程；
6.精通人机交互理论和原则，可以根据商业需求，设计合理的界面原型及交互流程者优先；
7.具备较强的逻辑思维和总结归纳能力、善于团队协作、具有创新精神以及出色的设计表达能力；
8.具备优秀的沟通与表达能力，并有良好的文档编写能力。
