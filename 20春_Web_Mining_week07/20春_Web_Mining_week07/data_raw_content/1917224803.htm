职位描述：

职责描述：
1.负责活动方案、宣传文案的撰写，根据研究院发展或活动需求，创作各种创意的长、中、短文本；
2.负责研究院微信平台、微博、网站、H5等官方社交平台账号的内容策划、视觉设计与排版工作，配合活动主题提出创意方案，并进行相关的宣传图片、网站页面及素材、H5的设计制作；
3.负责公司视频资料等文字撰写、整理，以及日常宣传物料的设计工作；
4.负责研究院工作内的设计制作，数据图表、信息图等图表设计，协助其他部门同时完成报告、计划书等文档设计制作，负责各类汇报材料、PPT等文件的撰写；
5.上级要求的其他文案工作。

任职要求：
1.本科及以上学历，有自媒体运营经验者优先。
2.文字表达精准，文笔流畅，热爱写作，文学素养较高，思维活跃，有创新创造意识；
3.熟练掌握Photoshop/Illustrator/AE/InDesign等设计软件，能清楚表达设计理念，迭代反应快，熟悉网页、海报、画册、VI形象设计及印刷工艺流程，能独立完成设计，擅于PPT编辑排版；
4.良好的团队合作精神和沟通能力，高度责任感和时间概念，良好的品味和审美能力；
5.优秀的英文写作功底者优先（例如CET-6 500分以上、托福95分以上，或其他同等级参考标准）。
