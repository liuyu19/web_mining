职位描述：

职责描述：
1.面向互联网广告业务的前端架构设计与开发；
2.负责Web前端系统和功能的开发、调试和维护
3.负责与产品、UI设计人员、以及后端开发人员沟通协作；
4.负责前端基础组件的设计与开发。

任职要求：
1.本科或以上学历，计算机软件或相关专业，5年以上相关工作经验；
2.精通web开发相关技术，如：HTML、CSS、JS，熟悉页面架构和布局，对表现与数据分离，Web语义化等有深刻理解，熟悉HTML5/CSS3等常用技术，有H5游戏开发经验者优先；
3.精通JavaScript、Ajax、DOM等前端技术，掌握面向对象编程思想，对js框架应用（如prototype/jQuery/YUI/Ext等）有丰富的经验；
4.熟悉前端MVVM架构，熟悉VUE技术体系者甚佳；
5.在解决浏览器兼容、分辨率适配、组件化设计与开发等方面有丰富实战经验者优先；
6.热爱前端开发工作，具有良好的团队协作能力，并具有较强的学习能力。
