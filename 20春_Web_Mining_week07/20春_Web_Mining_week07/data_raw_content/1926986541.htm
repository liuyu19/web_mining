职位描述：

职责描述：
1.负责公司所有产品相关视觉设计包括H5页面，移动端APP界面，微信公众号界面等，负责运营用户报告，广告等相关视觉设计
2.提出公司各产品视觉优化方案和创意
3.根据对用户体验的理解和分析，提高界面可用性，不断优化产品设计
4.熟知各主流移动平台（ios，android等平台）设计规范

任职要求：
1.本科以上学历，本科五年，硕士三年以上UI设计经验，扎实的美术功底，能够独立完成web/app设计工作，有负责或参与设计的产品项目。
2.熟悉sketch、photoshop、illustartor、ae等设计工具，熟悉axure/xmind/墨刀等产品原型设计工具。
3.逻辑思维清晰，善于概括，擅长创造性的将文字转化为视觉形象
4.有良好的客户思维和业务思维，有良好的沟通协调能力
