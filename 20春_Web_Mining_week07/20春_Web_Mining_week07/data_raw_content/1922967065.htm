职位描述：

岗位职责：
- 作为产品经理，负责建设公司内部应用系统，偏重于业务后场流程（包括但不限于财务，供应链，人事，研发等核心流程）的打通，数据的联通，以及跟前场销售和市场的对接；
- 跟各相关部门紧密合作，梳理和洞察需求，梳理流程，建立数据知识图谱，确定优先级和产品路线图
- 确定系统架构，以及自研还是购买商用软件定制模式
- 形成产品管理的闭环：用户洞察-产品计划-产品开发-测试及用户反馈-持续迭代；
- 对于自研部分系统，跟设计团队紧密合作推进UX/UI的设计工作，跟产品研发团队合作进行开发
- 对于商业软件部分，寻找合适的软件及实施伙伴，推进项目管理
- 建立产品运营机制及组织，保证产品的持续运营和持续改进

任职资格：
- 熟悉企业运作的核心流程，特别是后场的流程经验
- 有过管理ERP或者其他核心系统的实施和运营的项目经验
- 对产品管理有深刻的理解以及实操经验
- 善于沟通，学习能力强，抗压能力强

企业介绍：

依图智能安防技术主要由人工智能算法、高性能计算、分布式计算和存储、大规模运维等构成。其中视觉智能算法主要涉及人脸识别、车辆车型识别、文字识别、目标跟踪、图像特征搜索等技术。

依图技术团队来自MIT 、Google、阿里巴巴等顶级学术和工业机构，将最前沿的美国学术理念和扎实的工业界实战经验相结合，应用先进的人工智能算法，包括深度学习、统计建模技术，创新性的解决技术和应用结合的难题，能有效处理大规模样本（亿级训练样本）和少量训练样本的机器学习问题，具备丰富的世界领先的技术落地经验，包括超大规模识别（十亿级比对）、超低误报率（亿分之一误报）、多类型终端感知（静态图片、动态视频、低清画质、网纹噪音）、多人种跨年龄段识别等复杂应用场景 。
