职位描述：

1. 负责月度、季度与外包供应商的定期沟通培训。
2. 制定学习计划和培训课程，进一步提升外包团队制定整体业务方案能力，实施交付的能力。
3. 针对业务变化点能够快速准确传达，不断改进培训沟通课程，并监督外包实施落地结果，确保用户体验；
4. 实时检查外包商工作开展情况，内容应涉及服务质量、服务态度、安全工作开展等，在检查中发现不足应及时要求外包改进，并对改进效果进行追踪；
岗位要求：
1、业务沟通能力突出，责任心强，敏感度高，具有优秀的客户处理技巧和经验；
2、有良好的数据分析能力，学习能力，谈判力和执行力，敢于担当，乐于奉献；
3、有50人以上外包团队管理培训经验者优先。

企业介绍：

帆软软件有限公司（以下简称“帆软”）数年前从南京大学鼓楼校区起步，成立于2006年，沉淀数据分析领域十余年，致力于为全球企业提供一站式数据分析解决方案；同时也是行业优质交流平台的组织者。经过10余年的发展，帆软在数据分析行业的品牌领导力和品牌影响力上独占鳌头。
   帆软总部位于南京，设有无锡、拉萨两个运营中心，并下设京津区、上海区、浙闽区、华南区、西南区、西北区、华中区、华北区、东北区等九个分公司，35 个线下城市服务网点，省份覆盖率100%，力求为每个客户提供更加细腻和专业的本地化服务。
服务能力：目前公司员工超过1000人，90%以上来自南京大学、东南大学等211/985重点院校，拥有一流的团队。
合作客户：8000+合作客户，覆盖223个行业细分领域；中国500强中285家使用帆软产品；中国244家一级系统集成商，137家牵手了帆软。

我们提供：
1、自由、简单、务实的工作状态，宽松而融洽的成长环境。
2、积极向上的学习氛围，独当一面的机会和平台。
3、有竞争力的薪酬体系：高底薪+激励优秀的KPI绩效+回馈员工的年底分红。
4、一群追求优秀的人、技术大牛、激情又勤奋的小伙伴。
