职位描述：

岗位职责
1. 从用户体验的角度出发，以客服侧相关的用户数据为切入点，发现用户在产品使用过程中遇到的体验问题，同时对接和协调相关部门做好产品和服务的优化
2. 熟练运用多种用户研究方法帮助业务部门做好用户行为分析，挖掘用户痛点和使用需求，提供有价值、可落地的产品优化建议，推动产品优化，提升产品的用户体验
3. 围绕用户体验提升开展相关项目，管理并评估项目有效性
4. 完成上级分配的其他相关工作

任职资格
1. 全日制本科及以上学历 
2. 5年及以上工作经验，3年以上用户体验研究/用户服务管理/产品优化等相关工作经验 
3. 具备较强的洞察力，分析、归纳和总结能力
4. 具备优秀的逻辑思维能力，有较强的业务敏感度和良好的数据分析能力 
5. 熟练使用Word/Excel/PPT/Tableau/SPSS/SQL等软件分析海量用户数据
6. 有突出的跨部门协调沟通能力、语言表达能力和一定的创新能力 
7. 具有大型互联网产品用户体验经验者优先

企业介绍：

【我们是谁】
      我们是喜马拉雅，国家高度重视的新经济龙头企业、互联网行业独角兽。打造国内领先的音频分享平台喜马拉雅，我们拥有海量6亿手机用户，和超5000万海外用户，是中国文化出海的中坚阵地。
【我们的使命】
      用声音分享人类智慧，用声音服务美好生活。我们首创内容生态PUGC，引导行业潮流创新，吸引大咖主播，玩转声音多姿彩，投身音频内容创业。频道覆盖多又广，包含财经、音乐、新闻、商业、小说、汽车等328类过亿有声内容。
【我们的生态】
      秉承万物有声的理念，将有声内容渗透到生活中各个智能终端和场景中。2017年6月，推出国内仅有的全内容智能AI音箱——小雅，布局人工智能领域，打造物联网时代信息获取最便捷的入口。让用户把需求大声说出来，开启了一个语音交互的新传播时代，让声音和知识像水和电一样无处不在，随取随用。
