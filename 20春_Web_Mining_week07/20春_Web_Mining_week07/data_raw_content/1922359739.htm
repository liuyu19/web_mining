职位描述：

岗位职责描述

1. 负责用户画像、用户需求和痛点、用户体验地图，洞察用户使用习惯和态度研究

2. 负责长期追踪NPS和满意度等用户体验指标，提炼体验指标与业务指标关联关系，洞察体验优化点

3. 持续优化服务质量标准，通过神秘顾客调查等研究方法追踪服务质量达标情况

4. 负责用户运营过程中的专题研究，提升用户在运营关键环节中的转化留存

5. 挖掘业务需求，独立完成用户研究项目设计，负责方法设计、执行、数据分析、报告输出和落地跟进



岗位任职要求

1. 本科及以上学历，心理学、社会学、统计学或体验研究相关专业

2. 5年以上互联网用户研究经验，有用户体验研究及服务设计成功落地案例

3. 逻辑思维能力强，对于业务和数据高度敏感，擅长将用户研究与业务策略结合

4. 优秀的沟通能力和项目管理能力，能有效协调跨部门资源，具备强烈的责任心和团队合作精神

5. 熟悉用户深访、专家评估、焦点小组座谈会、问卷调查等用户研究工具，具备SPSS技能者优先
