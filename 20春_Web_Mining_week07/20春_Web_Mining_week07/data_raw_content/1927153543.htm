职位描述：

岗位职责：
1.负责二次元游戏logo icon KV 微信传播图 网页 卖场图等设计；
2.负责展会展板、易拉宝、手册、海报。等纸媒宣传设计；
3.分析产品需求和用户需求，配合营销需求规划各游戏产品在符合每个传播节点的营销视觉方案，定义视觉概念和提取设计元素以完成宣传素材设计；
4.参与项目创意概念构思，并能提出独特见解，将营销类需求转化成高品质的广告作品。
岗位要求：
1.本科及以上学历，美术学、视觉传达、广告设计等专业。（优秀人才，专业和学历可适当放宽）；
2.2年以上大型专业设计、广告、游戏类公司创意/设计工作经验。1年以上移动H5类设计相关工作经验；
3.热爱游戏，有丰富的游戏经验，对游戏美术、动漫等领域有系统的了解；
4.专注用户体验、内容创意和营销创新，对事件营销、互动广告有深刻的理解和创作热情，在游戏视觉表现力与广告创意与设计方面有较深认识，具备扎实的设计功底；
5.精通PS和AI/ID等设计软件，并对AE、 C4D、HTML5 和网络浏览器各项功能的有一定了解。有一项非常规商业的美术设计表达能力者优先（例如动画、摄影、手绘、3D、字体），有一定的手绘能力优先；
6.具备良好的沟通能力和团队协作精神；对设计、创新工作富有激情，对工作有较强的责任感；
7.Behance、Dribble、Zcool等资深用户优先。

企业介绍：

全球领先的互动娱乐品牌
基于全球化的视野，立足于移动互联网时代，英雄互娱致力于为海内外用户提供优质的移动游戏。
以泛娱乐为战略目标，推广“移动电竞”概念，打造全方位的移动电竞娱乐方式，开创全民互动娱乐新时代。英雄互娱的创始团队由游戏领域经验丰富的行业精英组成。 团队成员都有着丰富的移动游戏自研和发行经验。
一流的移动游戏发行团队，庞大而稳固的告联盟网络渠道、终端预装渠道、第三方应用商店、开放平台渠道和各大移动运营商渠道。
除发行自主研发的游戏产品外，还积极寻求优质精品网游产品进行独代、首发和联运合作，与国内外优秀的手机游戏开发商共赢手机游戏市场。
