职位描述：

1. 负责Camera整体软件优化工作，从APP/Framework/Native/HAL/Kernel/Driver等各个层面进行优化和增强，提升用户体验；
2. 负责Camera相关功能的开发和维护, 包括项目新Feature定制、算法集成等；
3. 负责Camera子系统的稳定性，分析和解决各类有挑战性的疑难问题；
4. 负责Camera的新技术预研和模块架构设计，研究行业新技术，参与和跟进Android新特性。


1. 计算机或相关专业本科及以上学历；
2. 熟悉C/C++/Java语言，熟悉Android Camera架构和Linux V4L2驱动架构；
3. 熟悉Android Framework/Native/HAL等开发工作, 有稳定性优化、性能优化和功耗优化经验者优先；
4. 了解Android多媒体/Camera子系统/Linux内核进程调度/内存管理/信号处理等工作原理者优先；
5. 优秀的分析和解决问题的能力，有主人翁意识和使命感，对解决各类疑难问题充满激情；
6. 具备较强的学习能力，积极主动，认真负责，有良好的团队协作意识。
