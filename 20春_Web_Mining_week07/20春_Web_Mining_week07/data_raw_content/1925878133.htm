职位描述：

职责描述：
1. 负责好大夫客户端产品的UI设计工作，具备很好的设计全局观及视觉表现能力；
2. 可完成整条产品端视觉设计工作，有很强的设计风格把控能力。能够根据产品目标设计并提炼最优解决方案；
3. 在研发过程中，能够以专业能力对设计上下游进行推动，并能保证最大限度的方案还原；
4. 明确用户预期，且制定相应设计预期，能够及时发现并解决产品体验问题；
5. 具备交互设计思维，并有能力产出简单交互设计方案；
6. 具备制定设计语言及优化视觉设计规范的能力。

任职要求
1.3年以上互联网和移动互联网设计经验，熟悉iOS，android平台设计原则；
2. 具备带领项目落地的经验；
3. 有创新力，有自己的思考，能总结的设计方法并应用于实践；
4. 很强的自驱力与抗压力，不断追求设计价值的最大化；
5. 对前沿设计探索感兴趣，有很强的设计提案能力；
6. 注重细节，追求极致。
