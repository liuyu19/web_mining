职位描述：

工作职责：
1. 负责开发和落地算法，包括但不限于图像识别和分类、目标跟踪、图像搜索、数据分析挖掘、多传感器融合等相关算法的调研，调试及工程优化。 2. 负责实现服务器端或者移动端的算法优化和部署，具备算法从数据收集、训练到形成稳定服务产出的全链路设计及开发能力。 3. 可以根据实际场景和业务需求不断迭代优化算法性能和服务模式。
任职要求：
1. 对机器学习、深度学习、计算机视觉等领域至少有一项有较为深入的研究。 2. 熟悉以下至少一种或多种深度学习开发框架：Tensorflow、Pytorch、Caffe、MXNet。 3. 有实际开发经验和算法落地经验，编程基础扎实，算法实现能力强，熟悉Python、C/C++，有Unix/Linux下开发经验。 4. 有移动端或者服务器端目标识别检测、ReID、细粒度商品识别，行为识别等算法开发落地和优化经验者优先。
