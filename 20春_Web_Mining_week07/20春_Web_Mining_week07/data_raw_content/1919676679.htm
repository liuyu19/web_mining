职位描述：

工作职责:
1.负责房多多重点App项目的设计工作，这是一个不区分视觉和交互的岗位；
2.分析日常设计上线后的数据，给出最佳设计方案，跟进产品迭代；
3.负责制定设计规范并推动落地；
4.持续关注用户反馈和产品效果, 不断跟踪改进视觉元素，进行产品的易用性和功能分析，优化提高用户体验。

任职资格:
1.本科以上学历，工业设计、视觉传达或其它设计相关专业，有2年以上工作经验，有大型互联网公司经验者优先；
2.有丰富的平台或产品的交互或者UI设计经验，能够独立承担交互、视觉工作；
3.具备一定的产品思维，能独自提出有效的体验改进方案，推动产品数据提升；
4. 关心设计和互联网领域的发展，自我激励和学习能力优秀；
5.良好的沟通能力和团队协作能力，工作态度积极，学习能力强。
