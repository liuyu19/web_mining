职位描述：

工作职责：
1、负责海外钱包的产品体验设计工作；
2、快速理解业务目标，具备梳理复杂需求的能力，高效输出合理的解决方案，专注产品和交互逻辑，并持续跟进方案落地的效果；支持业务相关的其他综合设计工作。
任职要求：
1、思路清晰，做有理有据的设计；
2、善于沟通表达、协调合作，工作有热情和责任感，对设计品质有执着追求；
3、关注数据，有一定分析数据的能力，并通过分析结论，驱动改善线上产品的体验；
4、有全局的体验意识，系统清晰的结合目标产出设计内容。对用户的需求和商业诉求有深入的洞察，敢于创新和突破。
