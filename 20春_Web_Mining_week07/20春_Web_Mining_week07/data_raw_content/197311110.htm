职位描述：

工作职责：
1.通用职责
• 根据公司整体战略发展规划，分解、细化和制订本部门的专业职能规划及业务规划，负责编制本部门年度、季度、月度工作计划及预算，并监督计划及预算的落实；
• 负责本部门管理制度、工作标准、业务流程的建立、完善和监督执行；
• 组织开展本部门内的专业业务创新及团队建设工作，提升本部门专业水平；
• 组织并落实本部门内的培训、绩效管理工作；
• 负责下属员工的工作安排，并对下属进行工作指导、监督，确保本部门工作的顺利开展；
• 组织本部门内的业务知识管理工作；
• 负责代表本部门进行跨部门间的协调与沟通；
• 负责组织完成上级领导交办的其他工作；
2.商业策划管理
• 组织对国家宏观政策和商业地产的研究；
• 负责组织公司商业项目的信息收集和筛选；
• 负责商业地产及住宅地产项目商业部分的定位；
• 组织商业策划方案评审；
• 负责组织商业项目的可研工作，提出项目可研报告；
• 组织提出基本的商业建筑要求(例如层高、柱距、荷载、用电量、燃气、上下水等)。
3.招商管理
• 组织提交商业策划方案和价格方案,设计招商渠道, 提出招商策略，并结合市场价格水平提出明确的租金价格和建议销售价格；
• 组织收集各种招商信息，编制、设计招商宣传数据和招商宣传广告；
• 组织建立完整的营运商档案，统一管理商业资源；
• 负责配合制定公司商业类招标文件、合同的标准格式范本；
• 负责组织项目主力店及项目散铺的招商谈判、签约工作；
4.商业运营管理
• 根据总部下达的年度经营目标，制定各项目商业部分的年度经营目标；
• 审核各项目商业部分的年度经营计划和日常重点工作计划，并监控计划实施情况；
• 负责组织办理承租手续，根据签订的《租赁合同》收取租金等经营收益；
• 组织定期分析商户的经营状况，提出经营改善建议；
• 负责处理经营过程中发生的商户的纠纷和消费者对商户的投诉；

工作经验：
• 8年以上商业管理经验；
• 有大型商业中心成功招商及酒店项目后期营运管理经验优先考虑；
• 有大型房地产工作经验优先考虑；

专业知识与技能：
• 掌握商业策划、招商、运营管理基础知识，熟悉市场营销、招商推广、媒体传播等相关知识；
• 熟悉高端写字楼、商业、城市综合体的招商管理流程；
• 熟悉大型商业项目前期策划定位、招商、营运的全过程管理；
• 熟练掌握office办公软件；

核心素质要求：
• 大型商业项目前期策划定位、招商、营运的全过程管理能力；
• 优秀的计划、统筹和管控招商工作能力；
• 优秀的招商谈判和公关协调能力；
• 较强的团队管理能力；
• 较强的资源整合能力；
• 富有激情和创新思维。

企业介绍：

       绿地香港控股有限公司(香港联交所股票编号：337)是世界500强绿地集团旗下控股子公司。
      绿地控股股份有限公司（上市公司简称：绿地控股，上市公司代码：600606）是一家由上海市国有资产监督管理委员会监管的特大型混合所有制企业，创立于1992年7月18日，总部设立于中国上海。
      成立23年来，绿地已在全球范围内形成了"以房地产开发为主业，大基建、大金融、大消费等新兴产业并举发展"的多元格局，坚定实施资本化、公众化、国际化发展战略，旗下企业及项目遍及北京、上海、广州等80多个中国城市以及美国、英国、德国、澳大利亚、加拿大、韩国、泰国、马来西亚等海外九国十二城，实现了国内A股整体上市（600606.SH）及部分资产香港H股上市（00337.HK 01365.HK），构筑起境内外资源整合的资本平台，位列2015年《财富》世界企业500强第258位。
      未来，绿地集团将以整体上市为契机，借助资本市场的力量，加快企业创新转型，不断将绿地打造成一家主业突出，多元发展，全球经营，产融结合，并在房地产、金融、地铁等多个行业具有领先优势的本土跨国公司，努力从"中国的绿地"转变为"世界的绿地"。绿地香港作为绿地集团资本运作的国际平台，及品牌展示的国际窗口，将进一步围绕集团的整体战略，展开资本平台及产业领域的创新转型升级，为其可持续发展提供有力支撑。
      深耕房地产主业的同时，绿地香港大力推进"大金融"战略，成立上海绿地金融信息服务有限公司（简称"绿地金服"）。借力大数据、云平台、移动互联，绿地金服深耕房地产垂直领域，以服务创新、产品创新为导向，为绿地系产业链倾力打造涵盖理财投资、社交金融、社区金融在内的互联网平台，助力绿地集团构建有场景、有温度的房地产互联网金融生态圈。
      依托母公司绿地集团成熟的品牌形象、丰富的优势资源、宏大的规模体系、先进的管理水平和充满激情的企业文化，绿地香港将全面整合现有资源，充分利用香港资本平台优势，积极践行第一个五年规划：力争以年均50%以上的复合增长率，于2018年实现销售收入500亿元以上的跨越式发展，成为香港资本市场内房股标杆企业。
