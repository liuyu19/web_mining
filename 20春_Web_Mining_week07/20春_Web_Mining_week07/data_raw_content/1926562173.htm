职位描述：

岗位职责
1、紧密结合发展规划战略，分析梳理市场前沿动态、高效管理产品生命周期、收集汇总用户反馈信息、更新迭代企业征信产品。
2、熟悉金融机构对公业务，特别是中小型企业信贷的商业模式和业务需求，设计整体解决方案、研发企业征信产品，包括但不限于信息产品服务和专业咨询服务。 
3、熟悉从业务调研、需求分析到产品发布的整个流程；针对细分领域的业务痛点、难点制定产品设计阶段性目标，主导竞品分析、保持竞争优势。

任职条件
1、双一流高校的金融相关专业的本科或以上学历，8年以上风控相关从业经验；
2、拥有国内知名企业征信或企业评级公司产品设计或咨询服务成功案例者优先； 
3、对于中小企业商业信贷、供应链金融的业务模式有深入了解和见解，主导过商业银行信贷风控、互联网小贷公司、大数据风控咨询公司等行业的全流程产品开发与市场推广，对产品规划和市场定位具有丰富的应用经验；
4、能够独立完成产品需求文档及原型设计文档编写，跟踪产品研发进度、运营情况，分析问题优化产品； 
5、良好的沟通表达能力、需求挖掘及问题解决能力、多方项目协作及团队管理能力、自我激励的工作态度和主动思考的负责精神，思路敏捷逻辑性强； 
6、工作地点：上海或杭州

企业介绍：

同盾科技总部位于浙江杭州，是国内专业的智能分析决策服务提供商，将人工智能与业务场景深度结合，为金融、互联网、物流、政务等行业提供智能信贷风控、智能反欺诈、智能运营等智能决策产品&服务。截至目前，已累计有超过一万家企业客户选择了同盾的产品及服务。

受客户信赖的同时，同盾也备受国内外资本、媒体、行业的认可，已获多轮数亿美金融资，并荣获2015年红鲱鱼全球科技创新100强、福布斯中国互联网金融50强、2017亚洲银行家“最佳云平台应用”、毕马威2017Fintech50、《亚洲货币》中国Fintech领导者大奖、拉姆·查兰管理实践奖、中国社会百人论坛“责任新秀奖”等奖项，入选了CB insights全球Fintech“独角兽”榜单、2018达沃斯论坛上发布的 “中国AI50强”、入选分析咨询机构Gartner《2018年世界人工智能产业发展蓝皮书》，“银行家”杂志“十佳智能风控创新奖”榜首等。

发展至今，同盾科技业务从中国辐射至全球，已在杭州、北京、上海、深圳、广州、成都、西安、重庆、新加坡、印尼等多地设立分支机构。同盾的团队中，超过80%的成员是人工智能、云计算、风险管理、反欺诈、商业智能决策、等领域的资深专家。同盾希望凭借人工智能、云计算等核心技术，赋能金融、互联网、物流、医疗、零售等行业，合纵连横建立智能商业决策平台。
