职位描述：

工作职责:
1. 主要负责内容创新，及在内部的协作下，能够主力完成行业白皮书、深度研究、解决方案优化等推广内容的撰写和规划；
2. 重要市场营销深度稿件策划、撰写及日常推广稿件的审核；
3. 公关策略及市场合作伙伴关系维护、管理；
4. 产品推广方案、活动、行业论坛及公关市场项目策划及管理；
5. 基于商务leads目标的商业活动、营销方案策划和管理；
6. 市场及公关对外形象相关的设计、活动UI等设计沟通及管理；

任职资格:
1， 具有互联网、数据、广告等相关经验，较好的思考、沟通和项目管理能力，能够独立负责大型市场项目、活动的策划、管理及运营；
2， 能够适应快速行业发展，快节奏的工作，并能够结合公司产品及业务方向采用适合的市场和公关策略；
3， 具有较好的文字功底，能够快速实现深度稿件策划、撰写，并且能够把控日常公关内容的大方向；
4， 具有一定的领导能力及突发问题解决能力，较好的对内及对外各种合作关系管理能力；
5， 愿意接受挑战，实现自我突破；
6， 本科及以上学历，较好的英文听说读写能力；
