职位描述：

工作职责：
负责游戏内UI界面及场景和天气特效制作；
负责游戏内角色特效制作；
负责在程序要求的范围内制作出高品质的特效表现。
能负责特效的整合和管理；
有美术外包管理经验能审核修改及外包团队的培养；
确保项目所有特效资源按规格制作并风格统一，最终入引擎以最优化表现顺畅运行。
任职要求：
5年及以上手机游戏或主机游戏特效相关工作经验；
熟练运用3DMAX、UE4、Photoshop、AE，等相关软件；
使用UE4、Photoshop能熟练完成从贴图绘制到特效输出表现；
要求有较高审美与设计能力，对于UE4中粒子特效的应用掌握熟练；
要求对于slg类游戏视角下的特效表现有一定的理解；
对于天气特效，自然类特效有一定的理解；
具有良好的沟通能力,积极主动解决问题，愿意倾听与改变，对于美术效果有自己的追求，并能承受较强的工作压力, 有责任心，有较强的团队合作意识；
有影视特效经验者优先。
