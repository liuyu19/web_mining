职位描述：

工作职责：
岗位职责：

1. 负责美团公交产品、线上+线下闭环体验产品的视觉设计；

2. 参与线上+线下场景体验设计流程制定，持续优化产品的用户体验；

4. 深度了解业务目标，参与产品策划，结合数据、竞品、用户研究等各纬度进行综合分析和拆分，制定行之有效的解决方案；

5. 关注行业内新技术的趋势与应用，提出创新思路和设计方案并推进方案落地和迭代；

6. 积累并分享设计经验，协助提升团队整体设计水平。

岗位要求：

1. 3-5年以上工作经验，具备出行产品设计经验加分，有线上+线下场景产品设计经验优先；

2. 对设计趋势及系统设计有敏锐认知，完善的设计体系及方法论的实践经验；

3. 参与过完整的互联网/软件产品的视觉设计工作，可驾驭跨平台多端设计；

4. 优秀的审美能力和设计表现力，良好的业务理解能力，线上线下设计渗透能力；

5. 有全局观念、独立思考、积极主动、善于沟通、具备良好团队沟通协作能力；

6. 请附上文字简历及作品集链接，pdf形式为佳。

岗位亮点：

1. 有机会全身心地参与到出行产品设计中，和团队一起基于目标，为用户提供优秀的、创新的出行体验；

2. 有机会通过团队方法和资源，主动发现问题，定义目标，提出设计需求，并推动自驱方案的落地和效果验证；

3. 有机会参与到体系化设计规范的制定和迭代中；

4. 能够与具有丰富设计经验的同学一起协作，与团队一起成长。

企业介绍：

美团的使命是“帮大家吃得更好，生活更好”。作为中国领先的生活服务电子商务平台，公司拥有美团、大众点评、美团外卖、摩拜单车等消费者熟知的App，服务涵盖餐饮、外卖、打车、共享单车、酒店旅游、电影、休闲娱乐等200多个品类，业务覆盖全国2800个县区市。2018年全年，美团的总交易金额达5156.4亿元，同比增长44.3%，年度交易用户总数达4.0亿，平台活跃商家总数达580万。

2018年9月20日，美团点评（股票代码：3690.HK）正式在港交所挂牌上市。

当前，美团战略聚焦 Food +Platform，正以“吃”为核心，建设生活服务业从需求侧到供给侧的多层次科技服务平台。与此同时，美团正着力将自己建设成为一家社会企业，希望通过和党政部门、高校及研究院所、主流媒体、公益组织、生态伙伴等的深入合作，构建智慧城市，共创美好生活。
