职位描述：

岗位描述:
1、负责 Android 平台上业务应用的功能设计及开发
2、负责优化客户端软件的模块结构和流程逻辑
3、负责优化客户端软件相关产品适配,升级

岗位要求:
1、本科以上学历,计算机相关专业,有扎实的 Java 语言基础
2、1 年以上 Android 项目开发经验;
3、熟悉 Android 框架及各种特性,熟悉面向对象编程
4、熟悉网络编程、安卓 UI 框架及相关开发工具
5、熟悉 Android 的线程机制,能够熟练使用各种安悉 Android 系统的线程类,及 service、广播等 
6、熟悉 SVN 或 Git 等相关的代码管理工具
7、熟练掌握 UDP/TCP/IP、http 等网络协议
8、熟悉客户端程序设计方法以及安全的应用层协议设计
