职位描述：

职责描述：
1. 负责华云私有云用户手册、技术白皮书、运维类手册、API手册及各种场景教程等的验证和更新，与开发、测试、产品团队密切配合，深入了解产品功能在初稿的基础上进行测试验证并修改，输出终稿；
2. 协助提升产品用户体验，包括但不限于软件界面帮助文档规范化、优化交互体验；
3. 积累收集故障处理和运维案例等，进行运维类手册的持续质量提升。定期整理所有文档资料，保证文档版本统一性，维护文档资料的索引表，定期备份文档资料；
4. 抗压能力强，善于思考，能独立分析和解决问题，责任心强。
任职要求：
1. 计算机、通信、电子相关专业，能够很好的理解云产品的实现原理；
2. 从事过1到2年to B文档工程师的工作或两年及以上软件开发测试、文档编写、技术培训等相关工作经验；熟练使用一种或多种文档开发工具，如dita、idp-lite、markdown等；熟悉资料写作规范和原理，能独立编写项目技术文档、制作各种表格、软件流程图、拓扑图；
3. 具备云计算行业背景、熟悉Linux操作、熟练掌握XML、GIT、Photoshop、Illustrator等工具尤佳；
4. 具体良好的团队协作能力和沟通协调能力，能够主动承担工作，识别风险，积极推动解决风险。且抗压能力强；
5. 具有云产品文档写作经验、两年及以上软件开发测试、技术培训等相关工作经验、英文技术文档项目经验优先考虑。

公司介绍
华云数据集团专注于为客户提供 “自主、安全、可控”的云计算服务，主要面向企业级用户提供定制化私有云解决方案，同时还可以提供混合云、大数据服务、超融合产品、公有云、IDC转云等“全云”服务。
自2010年成立以来，华云数据坚持自主研发，获得了200多项专利，是业界在私有云、混合云和公有云领域均通过可信云认证的




云服务提供商，是国家课题承接单位、中国十大云计算解决方案提供商。
2016年起，华云数据集团连续三年被评为中国独角兽企业，2018年6月8日，华云数据宣布完成10亿RMB  Pre-IPO轮融资（8年内完成6轮融资，总额超过35亿RMB），并发布了企业超融合产品，朝着推动中国企业全面上云，做中国云计算的扛旗者的路上继续砥砺前行。
