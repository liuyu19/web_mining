职位描述：
1、负责花椒直播APP日常开发和维护，参与技术预研；
2、负责公司移动产品的性能体验优化，编写高质量的代码；
3、能独立完成Andorid客户端业务需求开发，和产品经理配合，深度参与产品需求讨论，功能定义等；
4、设计良好的代码结构，不断迭代重构；
任职要求：
1、本科及以上学历，计算机相关专业，5年以上Android客户端开发经验；
2、有丰富的移动端APP应用项目经验和架构经验；
3、熟悉Android系统体系结构以及底层库及常用布局、UI设计，丰富的手机UI设计经验；
5、良好的编程习惯，精通数据结构和面向对象程序设计，编写高效，整洁的代码；
6、熟练使用Git，Android Studio及性能调优工具；
7、参与过React Native，Weex或Flutter等跨端项目；
8、了解主流的插件化框架；
9、具有较好的团队合作精神、主动性与责任心；
