职位描述：

技术必备：
1. 3年以上iOS研发经验。
2. 语言基础扎实，精通iOS框架及各种特性，熟练使用iOS的各种UI组件并了解其内部原理；
3. 具有移动端应用程序性能、内存、稳定性分析优化项目经验。
4. 熟练使用OC和swift 两种语言开发项目。
5. 了解或有组件化开发经验。
6. 了解html5、js、css等相关技术，有Hybrid开发模式经验，熟悉react-native、weex等技术者优先；
7. 有客户端自动化测试或单元测试相关经验者优先。
加分项：
1. 有小程序、web开发、android端、服务端开发经验者优先。
2. 有完整的线上问题收集分析修复等技术经验者优先。
