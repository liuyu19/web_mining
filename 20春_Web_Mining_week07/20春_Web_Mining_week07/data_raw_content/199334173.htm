职位描述：

岗位职责：
1） 根据产品的定位与需求制定产品调研管理制度与方案；
2） 负责产品调研计划的制定、实施和监控；
3） 定期或不定期对工厂、市场、竞品进行前期市场调研、分析；
4） 提交行业动态、产品动态、竞争对手产品和策略分析报告；
5） 调研形成的报告需对产品营销的规划指引与品牌的市场调研起到衔接作用；
6） 调研信息需对产品款式设计起到指导作用；
7） 建立调研的数据库信息管理，并定期针对数据库进行整理；
8） 完成领导交办的其他工作。
任职要求
1. 具有三年以上的珠宝款式从事工作经验及2年以上的调研工作经验；
2.对珠宝款式具有敏锐的洞察力；
3. 拥有较强的数据分析能力；
4. 做事认真细致，拥有高度的工作热情及富有责任心；
5. 具备跨部门协作精神及较强的抗压能力。

企业介绍：

周大生珠宝专注于“周大生”品牌珠宝首饰的设计生产、品牌运营和连锁经营，是国内具有影响力的珠宝品牌运营商之一，也是全国中高端主流市场钻石珠宝领先品牌。2017年4月27日周大生（002867）在深圳证券交易所A股上市。

在近二十载发展征程中，周大生一直坚持爱心、诚信、责任、奋进的核心价值观，以领先战略布局，采取轻资产、整合运营型经营策略，专注于品牌运营、渠道管理、产品研发和供应链整合，不断提升品牌形象和影响力，实现公司快速发展。

截至2019年6月30日，周大生共有门店3599家，形成强大连锁渠道网络，带来非凡的集群效应和资源整合优势。
