职位描述：

职位描述
1、小米手机的包装概念设计，包装平面设计，插画设计；
2、配件及相关衍生品等包装及相关视觉设计；
3、物料印前准备，排版、监督印刷等相关工作；
4、协助完成公司其他部门的平面设计需求。

职位要求：
1、男女不限，年龄不限，包装设计、艺术设计或纯艺术专业，本科以上学历；
2、3年以上相关工作经验；
3、有深厚的美术功底及良好的创意构思能力，色彩感强，有较好的设计理念；能独立按时完成各项设计任务；
4、熟悉Photoshop/Illustrator/Indesign等相关设计软件；​
5、加分项：熟悉印前及印刷工艺；
6、有较好的沟通能力，具备团队合作精神；
7、插画及手绘能力较强者优先；
8、应聘时请携带本人近期作品。
