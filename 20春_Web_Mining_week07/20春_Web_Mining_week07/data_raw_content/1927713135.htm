职位描述：

职位描述
岗位职责：
1. 负责游戏直播/泛娱乐直播的运营活动，参与创意思考，具备运营思维，能有效输出符合要求的设计方案；
2. 负责游戏电竞赛事，配合游戏产品设计线上包装，线下外场设计及舞台设计；
3. 定期总结设计经验，乐于分享，推动提高团队的设计能力；
4. 持续保持对直播行业、游戏电商行业的分析观察，可以定期输出竞品/新品分析报告；

职位要求
1. 本科或以上学历，美术设计或相关专业；
2. 3年以上工作经验，并有成功案例；
3. 有深厚手绘功底，精通并熟练使用PS、AI、C4D等流行设计软件；
4. 学习能力强，工作积极主动，有持续的热情和责任心，追求设计品质；
5. 善于沟通，团队协作精神和责任感，能承受高强度的工作压力；
6. 提交简历时还需同时提供个人作品集（如文件较大，建议以云盘链接形式发送）
7. 加分项：热爱游戏、对游戏行业/游戏用户群深入了解者优先，有直播、语音、电商类产品设计经验者优先；
