职位描述：

职责描述：
1.负责携程商旅网站以及APP相关产品设计工作
2.参与制定商旅Ui及规范工作，以及交互流程相关设计工作。
3.能参与项目前期用户调研工作，对用户需求进行分析和研究，能跟进并追踪国际上优先设计理念，思维活跃，创造力丰富。

任职要求：
1.本科及以上学历；
2.服务设计，工业设计，艺术设计，平面设计，媒体设计等美术类相关专业毕业；
3.精通XMIND，AXURE，SKETCH，PS，AI等设计软件，精通AE等后期以及3Dmax，C4D设计软件优先，具有一定营销设计经验，相关思考及总结；
4. 二年以上互联网或大型广告公司，用户咨询公司经验为佳。
5. 英语能力较好，有国际化视野。
