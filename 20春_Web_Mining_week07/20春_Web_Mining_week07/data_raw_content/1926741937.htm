职位描述：

工作职责：
前端技术中心：
前端技术中心隶属于基础技术部，致力于提升美团的研发效率并降低协同成本。我们的使命是：“CODE A BETTER LIFE”。部门主要工作聚焦在：
1. 研发支持全体员工的项目管理及协同平台，研发工具和强大的DevOps平台，以支持公司级标准化研发与运维流程；
2. 研发支持全体产研员工的Iaas/Paas产品，包含计算、存储、网络、监控等领域，以提供统一，高可用的基础服务；
3.研发前端领域基础中间件服务，包含监控、通信、UI组件等领域，以提升美团前端团队开发效率。
我们是一群充满激情且对技术有追求的人，我们希望你有扎实的理论基础及优秀架构设计能力，和我们一起用技术的力量达成我们的使命。

岗位职责：
1.负责公司私有云移动客户端的功能开发（Native、React Native、Flutter）；
2.根据业务需求，编码实现各模块功能，并且能够确保开发质量与进度；
3.学习和研究新技术以满足底层基础组件性能迭代需求，根据开发过程中的体验对基础组件提出改进建议。

基本任职资格：
1.本科及以上学历，计算机相关专业；
2.经验上：有3年以上大型互联网公司或大型IT企业Android/iOS开发经验；
3.专业能力上:
（1）扎实的计算机基础，对操作系统、数据结构、算法有一定了解；
（2）熟悉iOS或Android原生应用开发；
（3）熟悉ReactNative的基本实现原理，熟悉组件间的通信和数据的管理；
（4）熟悉基本前端技术，包括且不限于HTML, CSS, JavaScript, 熟悉 ES6 和面向对象编程;
（5）对设计模式有很好的理解和实践经验，有良好的编码习惯；
4.学习能力强，热爱通过持续学习优化App性能的过程；
5.个性乐观开朗，逻辑性强，有良好的沟通能力。

优先任职资格：
1.有React Native开发经验优先；
2.有Android/iOS/Web开发经验优先；
3.计算机相关专业硕士优先。
