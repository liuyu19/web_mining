职位描述：

职责描述：
1. 依据创意策略进行主画面及衍生物的创意平面设计；根据功能规划和交互设计，完成视觉风格定义，界面设计及运营设计
2. 负责用户体验的改善、设计和实现，维护及优化页面性能；
3. 能够独立承担项目作业参与内部项目会议，进行创意阐述并能提供相关创意支持；
4. 参与产品规划与功能策略制定，从专业角度提供可行的视觉解决方案

任职要求：
1. 3-6年相关工作经验，2年以上PC端和移动端经验，对安卓和ios的设计规范有了解；平面设计、电商背景优先
2. 深厚的美术功底，具备优秀的活动平面、互动方面设计能力；交互设计能力，同时具备前端能力；
3. 丰富的视觉表现力，色彩控制力和图形表现力佳；
4. 有良好的团队合作精神和协调沟通能力，有与产品、运营等多方密切配合的经验和意识；
5. 提交简历时请同时附上作品集。
