职位描述：

岗位职责：
1、负责京东到家大型促销活动设计+UI改版设计；
2、能独立承担沟通项目且带领团队完成高品质设计；解决内部排期问题及外部需求风险。
3、可沉淀出工作项目经验，宣讲并分享设计经验、推动提高团队的设计能力；
4、配合运营+UI并积极输出页面视觉，保证设计质量并优质上线；
5、理解设计驱动业务，以数据来验证设计从而达到更好的效果 。

任职要求：
1、5年+工作经验，能够主导大型运营活动的设计方向并完成主要设计工作，有较强的创意思维能推动项目。
2、有至少管理4-6人团队经验 ；
3、作品集精炼，有设计亮点及推导过程和验证方法；
4、熟练掌握SK/PS/AI/C4D等软件；
5、拥有较强的视觉表现力，较强的创新思维；
6、学习能力强，具有良好的沟通和设计表达能力，良好的团队协作态度，主动积极、踏实细心、协调能力强；
7、跨部门沟通能力强，具备品牌和营销思维
8、关注行业热点并研究流行趋势，善于总结和优化工作内容，能主动推进工作；
