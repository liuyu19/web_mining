职位描述：

职位职责：
1.负责品牌推广和营销活动推广页面设计，内容包括：活动主视觉、活动页面、插画、H5页面、banner、icon等视觉制作；
2.品牌周边设计，线下各类印刷；
3.参与创意的研究与制定；
4.参与产品前期的规划与构思，与PM紧密配合，分析行业述求以及用户需求，主导设定产品的整体视觉风格并跟进开发落地产品； 跟踪产品效果，提出优化方案；
5.保证产品具有良好的用户体验；
6.跟进开发，测试，保证设计方案最终实现；
7.参与用户调研，深度挖掘用户需求，持续优化产品设计；
8.分析用户行为数据，改进问题，优化产品设计。

任职资格：
1.要求本科及以上学历，美术、设计相关专业；
2.有一定的设计理论基础，关注设计发展趋势；
3.平面设计、网页设计、图标设计、 品牌设计有一定的经验；
4.擅长手绘 插画  熟练操作C4D等设计软件能胜任H5、活动专题等运营视觉类工作；
5.手绘能力强的优先；
6.具备良好合作态度及团队精神，并富有工作激情、创新欲望和责任感，能承受高强度的工作压力。
