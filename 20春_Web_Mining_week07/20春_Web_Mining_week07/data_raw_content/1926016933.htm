职位描述：

职责描述：
1.按照华为云IaaS层服务架构和业务要求，制定测试计划和测试方法；
2.参与华为云IaaS层服务需求和架构设计评审，保证服务的可测试性；
3.通过测试相关流程、策略、方法和工具等创新，提升测试的质量和效率；
4.解决测试过程中的复杂技术问题；

任职要求：
1.三年以上自动化测试或白盒测试工作经验；
2.有C++/Python/Java/Shell其中之一语言开发能力；
3.熟悉测试经验，熟悉测试工具和测试流程；
4.熟悉云计算，有IaaS测试经验优先
