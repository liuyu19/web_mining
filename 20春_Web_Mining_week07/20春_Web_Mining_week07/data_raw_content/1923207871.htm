职位描述：

此岗位为宗申集团子公司重庆物必达招聘。物必达是由宗申产业集团和加班狗科技联合投资创立的产业互联网综合服务平台，“以帮助千千万万个个体工商户更好做生意”为使命，借助移动互联网、大数据、云计算、人工智能等新型技术，致力于为中国7千多万的商户提供完备的一站式新零售解决方案，链接产业上下游，打造开放的产业智能服务平台，为产业数字化转型升级赋能。
我们的愿景：致力于成为一家“以零售服务为中心”的长青企业。
物必达，期待您的加入！
工作地点：巴南区花溪宗申工业园

物必达推出的汽摩零售通，以汽摩渠道商为核心服务对象，为其提供各类场景数字化服务，帮助其更好的实现经营管理，同时构建汽摩行业的数字化综合服务平台，为产业链的制造商和终端用户带来便利。岗位职责：
1.与设计师、产品经理紧密工作，实现产品UI和交互方面的开发需求，确保产品具有一流的用户体验；
2.负责前端架构在各业务线的落地实施，并结合各产品线实际情况，参与前端架构开发工作；
3.负责网站产品性能和代码优化的实施；
4.致力于前端技术领域的研究，并结合公司网站提出改进建议。
任职要求：
1、3年以上前端开发经验；
2、掌握至少一种JS（如ReactJS/Vue等）框架，掌握其原理，能独立开发常用组件；
3、熟悉React Native、Flexbox，并且有用其开发过1个及以上的产品；
4、熟悉移动前端开发，熟悉基于微信端 Web App ，熟悉小程序开发优先；
5、具有原生IOS 或 Android APP端开发经验者优先；
6、熟悉前端构建打包工具者优先；
7、热爱前端技术，乐观，善于沟通，热衷分享，有较强的学习能力和团队协作精神。

企业介绍：

宗申集团始创于1982，现已成长为一家总资产和销售收入均已突破200亿元人民币的大型集团企业。宗申制造工厂（或子公司）遍布在中国重庆、广东、江苏、河南、河北、天津和越南、泰国、巴西等地，正逐步成为“世界一流中小型动力系统集成服务商”。

宗申集团连续16年入选重庆工业50强，连续入选重庆市“纳税50强”，是中国民营企业500强、中国机械500强、中国制造500强，并入选中国企业500强。宗申拥有国家级“企业技术中心”、“机械工程专业博士后科研工作站”，宗申检测中心通过CNAS国家实验室认可。

宗申与意大利比亚乔（PIGGIO）、塞尔瓦（SELVA）、巴贝锐（BARBIERI）、白俄罗斯戈梅利等国际知名企业在中国设有合资工厂；宗申还与合作伙伴在上海、深圳、香港、加拿大、美国等地设有包括水上飞机运营、地产业务开发在内的合资运营公司。
