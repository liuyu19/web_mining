职位描述：

职责描述：
1. 负责汽车票事业部的车票/船票/创新业务交互设计工作；
2. 需要和上下游紧密沟通，完成界面的信息架构、流程设计、原型设计和设计文档撰写，并跟踪设计的落地和验证；
3. 进行用户研究、竞品研究、数据分析和需求梳理，进行产品可用性和易用性测试、评估和优化；
4. 参与建立和维护设计标准和规范，协助提升设计团队合作流程和效率；
5. 根据数据变化/转化率/点击率，做出对应的判断。

任职要求：
1. 3-5年工作经验，本科及以上学历，工业设计、心理学、计算机相关优先；
2. 坚持以用户为中心原则，掌握UED设计流程和常用方法，对设计理论和方法论有理解和应用，有成功的设计案例，具备不同终端设计能力；
3. 熟练使用原型工具和软件，如Axure、Sketch 等；
4. 快速学习了解业务知识，有严密的逻辑思维，能将业务逻辑和数据表现转化成对用户友好的界面设计；
5. 独立高效完成项目，把握设计策略，制定体验目标；
6. 具备良好的团队合作能力和沟通能力。
