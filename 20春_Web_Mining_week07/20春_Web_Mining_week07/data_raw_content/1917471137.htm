职位描述：

职责描述：
1、管理设计部门日常相关创意工作的开展；
2、带领团队对各业务线的需求进行创意设计思路的实现， 在规定时间内提供符合业务要求的创意设计，把握项目进度, 对创意设计质量和过稿率严格把关；
3、负责项目中美术设计及美术效果的指导，充分调动最大创意能力及最佳设计效率，将创意从概念通过视觉表现发展成为具体的设计；
4、带领团队负责大型项目创意提案，与产品团队评审Ui，提出可行性建议。

任职要求：
1、多链路设计，创意总监或艺术指导经验，本科以上学历背景，美术、平面设计、广告设计等相关专业，有互动，4A，营销创意设计工作，Ui经验优先；
2、5年以上从业经验，完整的品牌推广全案创意设计的工作经验;或线上设计经验；
3、具备整合广告策略和创意能力，胜任平面及视频的创意和执行指导工作，有丰富的设计指导经验和较强洞察力。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
