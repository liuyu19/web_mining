职位描述：

岗位职责：
1、 独立完成整套UI的风格设计、并有很好的产品交互意识；
2、 负责项目中各种交互界面、图标、LOGO、按钮等相关元素的设计与制作；
3、 能积极与开发沟通，推进界面及交互设计的最终实现；
4、 根据游戏特色，提出构思新颖、有高度吸引力的创意设计。
任职资格：
1、三年以上社交产品或游戏UI设计工作经验；
2、认真细致，善于创新，对视觉设计、色彩有敏锐的观察力及分析能力；
3、对于用户体验有较深的理解，能够从用户的角度来设计界面；
4、有优秀的图标界面设计功底和设计创意；
5、有团队协作精神，能够快速融入团队。
