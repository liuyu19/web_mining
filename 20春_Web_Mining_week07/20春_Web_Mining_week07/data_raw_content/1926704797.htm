职位描述：

职责描述：
* 根据产品的视觉风格，应用Unity或UE4引擎制作游戏中各类游戏动效，提升游戏表现；
* 运用游戏开发引擎制作引擎动画视频，以满足广告投放和日常宣传视频需求。
* 根据创意需求文档，与组内后期、外部供应商相互协作产出素材，对素材整体质感、节奏等方面进行专业层面的把控；

任职要求：
* 1年以上游戏视频制作工作或相关视频制作工作经验，有在游戏广告宣传片制作公司或引擎动画制作公司工作经验者优先；
* 熟练掌握UE4等现主流的特效制作软件（有Houdini配合UE4制作特效经验的优先）；
* 熟悉至少一款主流游戏开发引擎（UnrealEngine、Unity3D等），会使用引擎动画编辑器完成游戏内镜头制作；
* 精通 Premiere，具备优秀的视频剪辑能力，熟悉 Photoshop、Maya 、3Dmax 等辅助软件，精通AE后期包装者优先，熟悉三维操作者优先；
* 良好的沟通协调能力，善于团队合作，理解力、执行力和抗压能力强。
