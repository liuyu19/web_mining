职位描述：

职责描述：
1. 负责根据集团相关设计标准规范进行软件交互逻辑及UI界面设计；
2. 根据用户及小微团队需求，负责移动端产品的UI设计工作，推进界面的视觉效果实现；
3. 实现界面的视觉和美术设计方面的设计，如主题、背景、配色、图标，制订产品用户界面的风格及标准规范；
4. 参与产品规划构思，深刻理解产品目标和用户需求，协助进行产品设计、提升产品交互能力，优化用户体验。
任职要求：

1.本科及以上学历，人机交互、艺术设计、计算机、心理学等相关专业毕业，具备良好的美术功底、色彩感觉和创意能力；
2. 有独立的UI设计项目经验（3年及以上），完整的界面视觉风格设计经历；
3. 熟悉产品设计流程，擅长UI设计，能理解并善于表现公司要传达的商业理念；
4. 精通UI设计常用的PS、AI、AE、PPT等设计软件及图像处理工具，掌握C4D软件加分；
5. 具备一定的设计前瞻性，重视细节，有设计追求；
