职位描述：

工作职责：
岗位职责：
1. 负责地图相关的UI界面、地图以及开放平台的视觉设计；
2. 参与地图的设计规范制定，持续优化产品的用户体验；
3. 深度了解业务目标，参与产品策划，结合数据、竞品、用户研究等各纬度进行综合分析和拆分，制定行之有效的解决方案；
4. 关注行业内新技术的趋势与应用，提出创新思路和设计方案并推进方案落地和迭代；
5. 积累并分享设计经验，协助提升团队整体设计水平。

岗位要求：
1. 5年以上工作经验，具有LBS、地图、出行相关领域的设计经验优先；
2. 对设计趋势及系统设计有敏锐认知，完善的设计体系及方法论的实践经验；
3. 参与过完整的互联网/软件产品的视觉设计工作，可驾驭跨平台多端设计；
4. 优秀的审美能力和设计表现力，良好的业务理解能力，线上线下设计渗透能力；
5. 有全局观念、独立思考、积极主动、善于沟通、具备良好团队协作能力；
6. 请附上文字简历及作品集链接，pdf形式为佳。

岗位亮点：
1. 有机会全身心地参与到全面的地图产品设计中，和团队一起基于目标，为用户提供优秀的、创新的地图体验，同时全面了解地图类的产品设计；
2. 有机会通过团队方法和资源，主动发现问题，定义目标，提出设计需求，并推动自驱方案的落地和效果验证；
3. 有机会参与到完整的地图设计规范的制定和迭代中；
4. 能够与具有丰富设计经验的同学一起协作，与团队一起成长。
