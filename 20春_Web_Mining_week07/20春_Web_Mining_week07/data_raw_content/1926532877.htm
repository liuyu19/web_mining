职位描述：

工作职责：
l 协助美术团队制定针对各类视觉表现效果需求的技术方案，并通过开发工具、制作实例、提供建议等方式在游戏中实现

l 结合效果和硬件平台限制，与程序、美术团队配合制定渲染效率优化方案，制定资源规范

l 对美术团队执行美术生产规范的情况进行监督和反馈

l 通过编写脚本工具，探索优化工作流程等方式，协助美术团队提升生产效率和质量

l 生产一部分技术性较强的美术资源

l 关注并思考业界新的工具、趋势在工作室内项目应用的可行性

l 对美术等相关岗位进行新技术相关的培训
任职要求：
l 五年以上游戏或影视行业工作经验。

l 熟悉自己常用的工具内脚本语言的编写，插件和自定义UI的开发。例如：python，mel，maxscript，vex，等。

l 优秀的口头和书面表达能力，阅读英语资料无障碍。

l 良好的团队协作能力，有开发大中型项目经验者优先。

l 熟悉商业游戏引擎（Unreal/Unity/CE等），或自研引擎者优先。
