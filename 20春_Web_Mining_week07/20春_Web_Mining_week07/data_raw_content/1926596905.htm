职位描述：

职责描述：
1.作为UI与程序之间的的沟通桥梁，提供技术解决方案，在游戏引擎中将UI设计完美落地
2.配合UI与程序团队对游戏界面交互的操作手感，动画节奏，特效效果等进行调优
3.配合UI团队快速搭建设计原型demo（展示视频/引擎脚本动画等）
4.制定UI资源制作规范和流程，平衡产品落地结果的视觉质量和性能
5.统筹美术制作流程，协调各职能良好配合
6.组建UI技术支持团队

任职要求：
1.本科及以上学历，计算机、美术相关或者交叉专业
2.三年以上TA/主美工作经验，具备较强审美能力和美术敏感性，对交互设计和可用性有一定的认识
3.熟悉计算机图形学，熟悉脚本语言，了解前端工程师实现逻辑，与项目前端工程师落地期望效果，并能组织或配合编辑器前端工程师为设计工作开发和优化各种工具环境
4.至少精通1款3D游戏引擎（Unity3d、UDK、UE4、CE3），并了解其长短处，以及效果极限
5.熟悉完整的美术制作流程，以及各项相关工具的使用，具备一个人做完全流程的能力
6.具备美术资源制作环节的统筹能力,具备较强沟通能力
7.对新技术和工具保持学习的热情，对技术发展趋势有一定敏感性
8.熟练使用美术工具的能力：PS、3Dsmax、ZB、Xnorma、Speedtree等

企业介绍：

     网易公司由丁磊先生于 1997 年6月创立,凭借敏锐的市场洞察力和努力，网易公司从10多人的初创团队发展至今，在全世界已拥有上万多名员工，并于2000年6月成功在美国纳斯达克股票市场公开上市（NASDAQ：NTES）。随着公司规模的不断扩大，网易已经成为中国领先的互联网技术公司。网易拥有一批活跃的年轻团队，每位成员都能在充满活力的氛围中孜孜不倦地为理想共同奋斗。我们的团队不仅年轻，而且优秀。网易不断领跑中国互联网发展离不开优秀的团队。无论是领先的技术团队、敏锐的新闻团队、专业的销售团队还是高效的管理团队，都为网易的辉煌，中国互联网事业的发展倾注自己最大的力量。
