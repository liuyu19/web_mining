职位描述：

岗位职责：
1. 深入理解业务需求，设计和搭建推荐系统；
2. 负责推荐相关算法研发，如特征工程、构建预估、排序、推荐模型等；
3. 不断学习学术界和业界前沿技术并加以应用，持续提升推荐效果和用户体验；
任职要求：
1. 硕士及以上学历，计算机相关专业；
2. 具备扎实的数据结构、算法和开发能力，精通至少一种编程语言，如C/C++、Java、Python等；
3. 熟悉常用的深度学习/机器学习框架，如TensorFlow、MXNet、Pytorch、PaddlePaddle、Spark等；
4. 了解常用的推荐算法和机器学习算法，如深度学习、逻辑回归、GBDT、协同过滤等，有实际应用和调优经验优先；
5. 有大规模推荐系统、计算广告、搜索引擎项目实习经验者优先；
6. 具有较强算法实现能力，有学术比赛经验、发表过顶会论文者优先;
7. 良好的业务理解能力和解决问题的能力。

企业介绍：

Aibee由原百度研究院院长林元庆博士创立于2017年11月，是业界领先的人工智能（AI）整体解决方案企业，专注于提供人工智能（AI）整体解决方案，致力于赋能升级垂直行业，打磨极致AI技术。

自诞生之日起，Aibee就在业内率先提出从行业用户的关键痛点和核心价值出发，搭建数据、算法、技术、产品和服务的闭环，融合计算机视觉、语音识别、自然语言理解、大数据分析、机器人等多模态AI技术、提供完整的AI解决方案。通过降低人力成本、改善用户体验、提高运营效率三个方面，切实帮助垂直行业提升效率与产能，创造巨大商业价值，实现AI赋能升级。成立逾一年，Aibee融资速度与规模均处于AI创企前列，累计融资额达1亿美金。

在商业落地方面，Aibee已迅速取得实质性进展——与线下零售、线下旅游等垂直行业领头企业达成合作：

在线下零售领域，Aibee明确提出“精准零售”战略路线，为线下零售实体实现全面的数字化和AI化，达成“车·人·货·场”的全面理解，扭转线下流量被动变现的困局，提升顾客智能化体验，提高商城运营效率，降低人力成本，创造巨大商业价值。

在线下旅游领域，Aibee首推全域智慧旅游AI升级方案，实现线下旅游场景“人·娱·购”的精准定义，提升游客智慧旅游体验与景区整体运营效率，推动景区二次消费升级，带动周边业态发展。首批项目“刷脸入园”已在国内近百家知名景区投入使用。

在商业落地之外，Aibee始终重视高端人才的储备与AI技术的研发。2018年11月，前阿里达摩院决策智能实验室负责人朱胜火正式加盟，出任联合创始人，将全面负责Aibee的机器学习方向。Aibee在北京、美国硅谷均设有办公室，中美两地同步作业，充分发挥两地技术和市场优势。近期还计划在美国西雅图、波士顿开设新研发中心，持续吸引高端人才，加速业务快速落地。
