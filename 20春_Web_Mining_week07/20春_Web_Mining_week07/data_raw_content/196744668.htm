职位描述：

岗位职责：
1．客户服务相关信息系统建设的规划、设计、健全、完善与推广；重点负责CRM系统建设的跟进。
2．根据客户服务业务发展的需要，完善、优化客服相关系统的功能需求、组织需求讨论与评审，跟进需求实施进度以及测试验收工作，包括客服系统、工单系统、呼叫中心系统、短信平台、慧生活平台、网上营业厅等。
3．推动客服系统建设，包括CRM系统、移动APP等，组织进行项目规划制定、需求调研与评审等工作的推进。
4．编制客服信息相关工作流程与管理要求，组织进行客户信息资料收集。
5．拓展信息化渠道，提升客服信息化水平，包括满意度评价器、摄像系统、客服、触屏、自助缴费机等需求。
6．进行客户数据管理与分析、客户数据支持及数据挖掘。
岗位要求：
一．基本要求：
1. 年龄：30-45岁；
2. 学历：本科及以上；
3. 专业：客服管理、金融、财务、工商管理或城市燃气等专业；
4. 其它：同等岗位5年以上工作经验。具备良好的语言表达能力，良好的人际关系和沟通能力，丰富的信息化系统建设及运维经验。
二．工作经验/能力/个性特质描述：
1．熟悉CRM系统，具有CRM系统建设及后期运营管理经验。
2．有敏锐的数据分析能力，丰富的客服系统管理经验，具备较强的逻辑思维及财务分析能力。
3．熟练操作计算机及Office(含PPT)办公软件，文字功底强，能编制各类专业报告。
4．性格沉稳，善于沟通表达，有较好的团队协作能力及合作精神，能在压力下开展工作。

企业介绍：

中国燃气控股有限公司（简称：中国燃气）是中国最大的跨区域能源供应服务企业之一，在香港联交所主板上市（股票代码0384），通过多年的发展，中国燃气成功构建了以城市管道天然气业务为主导，液化石油气、液化天然气、车船燃气、分布式能源、天然气发电、供热、合同能源管理服务以及燃气设备，厨房用具制造，网络电商并举的全业态发展结构。

中国燃气的主要股东为北京控股集团有限公司，北京控股集团有限公司是北京市政府全资的北京市最大国有企业。

经过15年的高速发展，中国燃气已成为中国燃气行业的龙头企业；2013年，中国燃气被纳入恒生中国内地100指数和富时中国指数成分股。2013、2014、2017年，中国燃气凭借优异的业绩表现和发展获评《福布斯亚洲》“亚太***上市公司50强”。2016-2017年，中国燃气在由美国权威杂志《机构投资者》主办的全亚洲管理团队排名榜评选中脱颖而出，连续两年获评为全亚洲“最受尊敬企业”，并先后获得 “十大品牌企业”、“杰出成就企业”、“中国杰出天然气供应商”、“最具潜力中国企业”、“***爱心贡献奖”等1000多项殊荣。

中国燃气目前在全国29个省、市、自治区进行了广泛的项目布局。截至2017年3月底，中国燃气员工总数近5万人，旗下公司已超过800家，拥有400多个城市燃气项目，629座天然气汽车及船舶加气站项目，110多个液化石油气分销项目、16个天然气长输管线项目，1个煤层气开发项目，8个液化石油气码头，4个大型石油化工基地，燃气管网总长10万多公里，各类管道燃气用户2300多万户，瓶装液化石油气用户600多万户，燃气供应覆盖城市人口超过1.5亿人。集团还与国内外多家金融机构保持良好的合作关系，目前已拥有银行授信额度、发债额度及专项清洁能源基金规模超过1000亿元。

中国燃气全面响应国家治理雾霾、绿色发展的号召，2017年承担华北150万户农村气代煤改造任务，用行动实现保卫蓝天的宏伟目标。目前，中国燃气动员全集团的力量，已经组织了庞大的市场开发、工程设计、施工、客户服务等方面的队伍深入广大农村和城镇，通过构建“4G”（管道天然气PNG、压缩天然气CNG、液化天然气LNG、液化石油气LPG）能源网络，打造整体能源供应链。在气代煤工作中，一方面中燃加快锅炉改造，采用合同能源管理方式，并结合分布式能源、热电改造联产等多种方法提高能源综合利用率。另一方面集中进行农村散煤治理，通过科学规划、周密设计、精心施工、安全运营、优质服务来加快气化乡村的工作。

秉承“气聚人和、造福社会”的使命，中国燃气努力恪守企业公民的社会责任，勇于担当，敢于创新。未来，五万名中燃人将继续致力于推动社会和谐发展、绿色发展、可持续发展，矢志打造长青基业，持续为社会大众创造福祉。
