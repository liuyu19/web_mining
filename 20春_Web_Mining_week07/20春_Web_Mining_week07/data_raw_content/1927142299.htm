职位描述：

工作职责：
1、与服务端工程师、设计师紧密协作，产出高质量的产品前端层；
2、关注前端前沿技术，完成web前端功能开发和重构，保持高效的前端性能，优秀的代码可维护性，良好的浏览器兼容性；
3、负责项目前端版本迭代规划，开发任务分配，过程把控，推进项目落地；
3、能够在理解业务的基础上，通过技术驱动业务的发展，进一步提升用户体验；
4、能够主动的开发出提高团队工作效率的自动化工具，关注团队工作流程的优化。

任职要求：
1、大学本科及以上学历，工作三年以上，计算机相关专业，良好的计算机基础和编程能力，能产出高质量的设计和代码；
2、掌握web前端开发技术: JavaScript(含ES6)、HTML、CSS、DOM、协议等；
3、熟悉HTTP协议，了解TCP/IP的基本工作原理，熟悉各种Web标准规范, 了解服务端开发；
4、至少熟悉一个前端框架（react, vue等），熟悉小程序开发；
5、对前端工程化与模块化开发有一定了解，熟悉常见的前端优化技巧并有实践经验；；
6、较好的产品意识，注重用户体验，将产品效果作为重要的工作驱动因素；
7、对前端技术和团队工作能有持续的热情，逻辑性强，善于进行跨团队的交流和合作；
8、有良好的团队协作精神，勇于接受挑战并能够承受压力，能够主动和团队分享，主动提高自身在团队的影响力。
