职位描述：

职位描述：
1、负责应用平台解决方案服务的特性设计和开发；
2、与团队一起负责应用平台解决方案服务产品的技术竞争力构建，不断提升应用平台解决方案的业界竞争力。

技术栈：
1、计算机或者相关专业本科及以上学历，2年以上工作经验，能力优秀可放宽；
2、有大型分布式系统设计和开发经验，对分布式相关理论有深入的理解；
3、熟悉Java语言，具备良好的算法和编程能力，有云服务化开发经验，DevOps实践经验者优先；
4、熟悉Spring、Spring Boot 、Spring Cloud等各种开源框架，特别在Spring/SpringCloud领域有丰富的项目实践经验者优先
5、熟悉Mysql、Oracle等数据库，有相关数据开发经验
6、精通云计算相关技术，熟悉Devops，熟悉Docker，熟悉Kubernetes应用优先；
7、熟悉Service Mesh、微服务治理、灰度发布等相关技术优先；
8、有较强的学习能力、良好的团队协作能力、良好的沟通表达能力;
