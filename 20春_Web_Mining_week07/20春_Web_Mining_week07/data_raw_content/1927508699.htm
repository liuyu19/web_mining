职位描述：

职责描述：
1、建设和持续优化基础技术平台、共享服务组件，提升系统的自动化、智能化程度；
2、负责基础技术平台及共享服务组件的架构设计、技术方案审核、核心代码编写；
3、进行技术难题攻关、新技术预研，保持系统合理的先进性；
4、制定和完善技术流程、规范，提升研发质量；
5、指导技术团队，不断提升团队研发效能。

任职要求：
1.统招一本及其以上学历，计算机相关专业；
2.逻辑思维敏捷，具有创新精神和优秀的学习能力；
3.具有良好的自我驱动能力，执行力强；
4.计算机技术基础扎实，技术视野宽广；
5.精通JAVA语言和JVM原理；
6.掌握Nginx/Redis/ES/Kafka/Canal/Flink等开源中间件的内部原理，具备源码级改造能力；
7.精通高并发、大数据量分布式系统的架构设计方法和相关技术；
8.熟知软件工程的质量控制方法和常用工具；
9.具有大型互联网系统、高并发中间件、基础技术平台架构设计经验者优先。

企业介绍：

泰康在线财产保险股份有限公司”，成立于2015 年，是由国内大型保险企业发起成立的专业互联网财产保险公司，世界五百强企业泰康保险集团成员公司。
公司以“保险+ 科技”、“保险+ 服务”为主要商业模式，充分融合泰康保险集团现代金融服务元素，专注于互联网保险业务的创新开拓。秉持“让保险更安心、更便捷、更实惠”的理念，泰康在线致力于开发完全贴合互联网生态的保险产品，产品涵盖互联网财产险、车险、健康险、意外险、货运险、责任险、信用保证保险等，同时可开展与互联网保险相关的技术服务与咨询服务。
泰康在线通过完善的线上服务平台，同时基于泰康保险集团强大的线下资源，打造行业内富有竞争力的保险服务生态，实现公司的快速发展。
