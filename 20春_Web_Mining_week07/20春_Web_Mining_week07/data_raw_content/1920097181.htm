职位描述：

职责描述：
1.参与移动互联网/智能电视、智能硬件平台的产品规划构思和创意过程；
2.分析业务需求，归纳出产品人机交互界面需求；
3.搭建界面构架，设计用户操作流程等；
4.研究基于ios/android/winPhone系统的交互设计技术和界面设计趋势，总结设计技能和方法，输出交互设计创新方案并进行推广；
5.参与公司前瞻性产品或战略性产品的用户研究和创意设计
任职要求：
1. 本科及以上学历，工业设计、计算机、心理学、平面设计、广告设计等相关专业；
2. 熟悉User-Centered Design（UCD）方法和工作流程，对交互设计有深入的理解，具备扎实的专业知识，可以独立完成整个设计过程；
3. 2年以上交互工作经验，熟练掌握用户模型、故事板、流程图、竞品分析、界面/产品原型等方法，能对设计创意进行有效的交流；
4. 思维活跃，逻辑思维及分析能力强，对新鲜事物敏感度高；
5. 良好的沟通能力和团队协作精神，能承受较大工作压力。
