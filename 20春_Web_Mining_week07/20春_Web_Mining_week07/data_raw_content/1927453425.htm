职位描述：

工作职责：
1. 算法自动化能力建设：模型自动生成平台（涉及auto modelling、noisy data processing，distributed serving等）；
2. 面向具体业务的平台和服务建设：视频特征引擎，优质视频内容挖掘系统，视频特征聚类系统；
3. 与算法研发和上下游业务方同事沟通，推动算法能力在可自动运维性、可用性和性能上持续改善和优化；
4. 团队建设和人员管理；前沿技术跟进；
任职要求：
1. 具备相关领域工程团队或项目管理经验；
2. 具备机器学习、深度学习相关算法服务设计和开发经验，基于rpc的服务通信开发经验；具备常用数据库mysql、mongodb等开发和使用经验；熟悉常用深度学习框架，如pytorch tensorflow等，了解常用深度神经网络模型运行原理；熟悉主流实时流计算和处理框架如flink、kafka等；
3. 良好的沟通、组织协调能力和强烈的责任心；
4. 对新技术感兴趣， 有技术热情
