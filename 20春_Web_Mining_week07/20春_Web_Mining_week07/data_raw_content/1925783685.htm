职位描述：

岗位职责：
1、负责微信营销内容生产，通过产品包装策划流量推广，实现微信用户（公众号、群、小程序等）粉丝持续增长目标；
2、挖掘新的微信用户场景，结合数据分析和用户调研，通过不断的创新迭代，长期的用户增长策略；
3、善于结合热点事件、关联场景，充分调动用户积极性，实现线上活动到微信关注效果最优；
4、对用户行为及生命周期进行管理，对不同用户分层，并对关键指标持续进行策略优化。

任职要求：
1、3年及以上互联网运营经验，对互联网线上传播有一定的兴趣和见解，关注用户体验；
2、擅长线上用户增长/传播活动文案策划；
3、结果导向，对用户增长目标进行动作拆解，达成用户增长目标；
4、有微信导流投放经验者优先。

企业介绍：

    作业帮是一家专注于K12方向的教育科技公司，致力于为全国中小学生提供全学段的学习辅导服务。作业帮以教育为信仰，以科技为驱动，凭借作业帮强大的技术优势，打造了包括拍照搜题、作业帮一课、作业帮问答、作业帮一练、中英作文和作业批改在内的六款明星产品。截至目前，拥有1.8亿+题库数量、1.2亿+月活跃用户 ，是中小学在线教育领军品牌。
   企业秉承“让优质教育触手可及”的使命，致力于借助科技力量提供学生个性化互动教学服务，提升学生学习效率、助力学生全面发展。作业帮团队怀着对教育事业深深的敬畏之心，怀着对创业初心殷殷的坚守态度，始终坚持务实、进取、用户第一的价值观不断优化产品、完善服务，真正做到让广大中小学生和家长方便快捷的享受优质教育。
   作业帮项目由百度孵化，2015年9月正式独立运营，A轮融资由红杉和君联资本领投。于2016年中完成B轮融资，由纪源资本和襄禾资本领投，A轮投资人跟投。2017年8月完成1.5亿美金C轮融资，由H Capital领投，老虎基金、红杉、君联、GGV、襄禾跟投，是K12在线教育领域迄今为止最大规模的C轮单笔融资。2018年7月18日，作业帮正式宣布完成D轮3.5亿美元融资。此次融资由全球知名投资管理机构Coatue领投，高盛、春华资本、红杉中国、GGV纪源资本、襄禾资本、天图投资、NEA、泰合资本等新老股东跟投，泰合资本担任独家财务顾问。
