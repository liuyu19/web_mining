职位描述：

工作描述
1. 了解和学习触摸屏以及指纹的基本工作原理
2. 根据用户常用的触控操作和指纹使用场景，制定主观常规测试用例和客观常规测试用例，确保正常使用功能完整，性能正常
3. 根据自己对工作原理的理解，制定主观极端场景测试用例和客观极端场景测试用例，合理评价在极端场景下的性能
4. 与研发工程师一起，制定测试标准
5. 竞品机型的触控性能和指纹性能测试分析，并输出测试报告，包含竞品优点和缺点
6. 发起和完善用户体验调查，收集和整理用户反馈。并维护调查反馈系统

岗位要求
1. 细心，有耐心。逻辑思维能力强，有较强分析能力。
2. 善于观察，善于学习。有亲和力
3. 大学本科学历，理工科背景优先
