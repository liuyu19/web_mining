职位描述：

岗位职责:
1. 参与前端架构设计和方案制定，承担公共前端框架、组件的设计、开发工作
2. 负责公司应用系统内嵌网页界面&功能的开发；
3. 负责应用程序的设计、开发工作，撰写相关技术文档；能够将理想的用户体验和交互在用户端的产品落地实现;
4. 通过各种前端技术手段，持续优化用户体验和产品性能，持续跟进应用质量，确保高品质产品的上线并及时响应和解决用户问题；
任职资格:
1. 本科及以上学历，计算机相关专业，5年及以上前端开发工作经验；
2. 精通Web前端技术（包括HTML/H5，CSS／CSS3，Javascript/ES6等），掌握页面架构和布局，熟悉浏览器渲染原理；
3. 精通一种以上主流前端MVVM框架（React/Vue/Angular）,且使用主流框架编写过中大型项目或特色项目；
4. 擅长模块、组件化开发，阅读过源码，能脱离框架编写常用组件及兼容代码；
5. 对 Node.JS 等技术有较深入研究
6. 熟练使用Webpack、Gulp等前端构建&打包工具；
7. 熟悉Git代码托管，会用Git客户端管理工具（如Sourcetree/ TortoiseGit/Fork）；
8. 熟练使用VSCode以及Wiki、Jira等工具，
9. 优秀的代码编写风格和编程习惯，较强的逻辑思维和沟通能力，良好的团队协作能力，遇到问题能够快速定位并提供解决方案；

【加分项】
1. 关注前端技术发展动向，热爱开源技术，参与过开源项目
2. 有 Github 或技术博客
