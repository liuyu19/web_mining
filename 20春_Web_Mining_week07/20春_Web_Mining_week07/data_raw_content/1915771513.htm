职位描述：

职责描述：
1.负责游戏产品营销宣传创意内容项（主视觉、Social活动、创意广告、电竞项目及展会等基础和创新项）的视觉包装、规范制作、创意执行和品质把控；

2. 负责游戏周边产品、活动页面设计，完成产品模型建立，并全面把控设计组产品设计质量；

3.统筹、管理各类供应商的创意项目协作，创意需求、规范制定输出，项目进度把控，出品品质把控等。

任职要求：
1.全日制大学本科及以上学历，广告设计或美术专业优先；

2.3年以上广告或游戏公司设计工作经验，有成功广告或周边设计案例者优先；

3.熟练使用PS、AI等软件，有印刷设计经验，熟悉印刷流程；

4.具备一定的设计理念，出色的视觉表达能力；

5.具备较强的执行力、良好的沟通和思维能力；

6.暴雪游戏资深玩家，熟悉暴雪的游戏文化

企业介绍：

     网易公司由丁磊先生于 1997 年6月创立,凭借敏锐的市场洞察力和努力，网易公司从10多人的初创团队发展至今，在全世界已拥有上万多名员工，并于2000年6月成功在美国纳斯达克股票市场公开上市（NASDAQ：NTES）。随着公司规模的不断扩大，网易已经成为中国领先的互联网技术公司。网易拥有一批活跃的年轻团队，每位成员都能在充满活力的氛围中孜孜不倦地为理想共同奋斗。我们的团队不仅年轻，而且优秀。网易不断领跑中国互联网发展离不开优秀的团队。无论是领先的技术团队、敏锐的新闻团队、专业的销售团队还是高效的管理团队，都为网易的辉煌，中国互联网事业的发展倾注自己最大的力量。
