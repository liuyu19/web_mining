职位描述：

岗位职责：
1、负责京东金融APP上高端理财业务线的产品规划、分析调研和功能设计；
2、制定产品业务规范，完善金融产品线上交易流程，撰写产品需求说明书；
2、通过数据分析跟踪产品设计效果及业务发展状况，持续改善；
3、协调UI、交互、研发、运营、公司各个业务部门等人员，推进产品相关的各项工作。

岗位要求：
1、本科及以上学历，5年以上互联网产品设计经验、2年以上互联网金融行业经验；
2、熟练编写产品需求文档，精通Visio, Axure等产品设计软件；
3、有较强的学习、沟通能力，良好的表达能力、组织协调能力和团队精神；
4、互联网深度用户，对新事物具备独到、深入的分析能力。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
