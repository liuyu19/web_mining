职位描述：

工作职责：
阿里集团-阿里云智能事业群-云智能产品研发板块-基础设施事业部-服务器研发事业部-服务器产品：主要是负责服务器设计和开发，端到端交付成熟稳定的服务器软硬件系统给下游业务，支撑AIS基础设施的底盘。 服务器研发事业部-服务器产品-存储部件团队：负责从服务器存储部件领域的端到端交付，从分析需求、产品规划、供应商合作、定制开发落地、引入交付、系统级测试交付、推动产品上线和解决线上问题。领域包含 HDD、SSD、内存、AEP、存储控制器（HBA卡/RAID/PCH）、Expander等关键核心器件，是服务器系统的基石，需要技术深入和能力突出的人加入来提升团队交付能力。 1、支撑整机架构输出存储模块架构设计和规格制定，负责存储系统的需求分析和转换，输出存储控制器(HBA卡/RAID/Expander) 需求设计方案，结合软硬件适配性能调优，落地到服务器整机系统方案中，支撑高性能、高质量、高可靠性和低成本的目标。 2、基于存储控制器模块的方案需求，定制化开发存储控制器的FW和Expander的FW，定制化开发存储控制器驱动，发布相应的FW和驱动包。 3、基于业务需求，分析行业的控制器，设计对应的方案，并实施引入存储控制器部件，交付上线。 4、负责系统化的解决高密存储的稳定性问题，系统化设计报错方案，优化监控和诊断，提升诊断的准确率 5、持续跟踪存储控制器相关领域的技术发展趋势，结合对阿里业务应用场景的深入分析，并推动相关技术的产品化实现 6、负责数据中心存储控制器疑难问题解决，输出对应的解决方案并落地交付给业务
任职要求：
1、通信、电子、计算机及相关专业，有扎实的计算机底层硬件软件基础知识；深入理解x86/ARM体系结构；具有深厚的相关存储系统架构、设计、存储底层软件架构设计、开发、测试等工作八年以上经验 2、熟悉当前主流控制器及EXP型号及工作原理，有存储控制器芯片（RAID/HBA/Expander）和FW研发经验，熟悉芯片的研发过程和交付过程，能够拉通各个领域识别过程风险点并推动落地 3、熟悉SCSI、SAS、SATA、NVMe等协议，具备大规模嵌入式实时操作系统的编程经验(C语言) 4、熟练掌握Linux Block及块设备驱动工作原理、文件系统和IO调度，具备Linux Driver编程及问题定位能力 5、具备较强的逻辑思考能力、沟通能力、学习能力、合作精神，积极主动，有责任心，抗压性强
