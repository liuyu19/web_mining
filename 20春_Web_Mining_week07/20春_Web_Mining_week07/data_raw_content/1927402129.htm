职位描述：

工作职责：
1、基于业务需求，通过方案设计形成MRD，推动PRD及产品落地，并拿到运营结果； 2、能基于不同服务供应链形态，设计平台所需的服务监控产品，实现对核心节点、消费者痛点的监控； 3、有较强的体验敏感度，能发现不同行业的核心痛点，形成解决方案，并通过平台产品化监控与解决； 4、较强的数据敏感度，能通过数据分析定位问题，能基于业务核心诉求，设计数据指标，形成考核管理体系，赋能业务运营能力提升； 5、推动平台管理流程优化、充分利用平台赋能产品，结合数据指标和机制，推动服务商供应链提升，提升客户体验； 6、能敏锐地识别竞争变化，并快速制定方案有效应对。
任职要求：
1、有5年以上服务供应链服务质量管理工作经验； 2、有标准服务流程管控、服务标准的SOP搭建经验； 3、对到家服务核心行业形态及服务交付管理有一定了解； 4、有很强的执行力和刨根问底的素质，能快速、有效地拿到业务结果； 6、聪明，富于想法、有创新精神、有业务前瞻性； 7、对数据敏感，有极强的数据解决能力，善于将数据作为工具为业务决策提供依据。

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
