职位描述：

职责描述：
1. 公司产品Web前端表现层及与后端交互的设计和开发
2. 配合产品经理和后台开发人员实现产品界面和功能
3. 通过各种前端技术手段，提高用户体验并满足性能要求




任职要求：
1. 实际前端领域开发经验3年以上，能独立完成前端开发工作。且每家公司任职时间不可小于一年
2. 精通并掌握JavaScript语言核心技术DOM、BOM、AJAX、JSON、JQuery等，对面向过程、面向对象等开发思想与深入的了解，对继承、闭包和ES6以及this有深入理解
3. 参与过大型项目、有复杂用户界面与交互应用开发经验者优先，注重页面细节
4. 熟练使用调试开发工具：抓包工具、Chrome调试工具与插件、移动端调试工具、IDE/编辑器插件
5. 熟悉软件项目开发流程、具备较强的团队精神和合作精神，抗压能力强
6. 有使用过表格插件“jqgrid、dataTables”、可视化图表组件“Echarts”、前端UI框架“Layui”的经验
7. 精通VUE/React相关技术栈（webpack、less/sass、elementUI/antUI）， 熟悉常用数据结构和算法

企业介绍：

比特大陆科技控股公司（“公司”）是全球先进的算力芯片公司，拥有全球先进的集成电路工艺设计能力。公司的产品包括算力芯片、算力服务器、算力云，主要应用于区块链和人工智能领域。
公司成立于2013年，总部位于北京市海淀区，在香港、新加坡、美国等地设有研发中心。根据Frost＆Sullivan的统计，公司是全球前十大、中国无晶圆厂集成电路设计企业。
公司秉持让人类数字世界更美好的愿景，聚焦算力芯片。公司在各种芯片制程标准下均具备领先的设计能力，并已掌握基于全球先进的5纳米制程工艺的芯片设计能力。在区块链市场，公司累计出货芯片达数十亿颗，约占全球75%市场份额。公司是世界上少数几家有能力开发云端人工智能芯片的公司之一，并已成功推出四代人工智能芯片。公司的人工智能芯片适用于深度学习领域的专用张量加速计算，可广泛应用于人脸识别、自动驾驶、城市大脑、智能政务、智能安防、智能推荐、智能医疗等诸多人工智能场景。
