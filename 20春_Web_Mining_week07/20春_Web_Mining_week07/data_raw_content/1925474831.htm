职位描述：

工作职责：
1、负责华为IT云在IaaS/PaaS/SaaS信息安全技术架构的战略规划和设计、安全策略和发展路标，前瞻性地规划适配业务发展和技术演进的信息安全战略和路标；
2、负责利用云技术建立与华为业务流程匹配的整体云安全体系架构和信息安全解决方案，以及技术引进和预研，保证技术发展方向正确、先进和可持续发展；
3、负责利用云技术建立公司层面的信息共享机制及端到端解决方案，并推动方案的执行落地，保证共享满意度达到公司要求；
4、针对传统业务向数字化转型的趋势与痛点，主导重大复杂的云安全技术变革，提出安全架构级的解决方案及产品，系统性地解决本领域重大安全技术或业务安全难题，避免发生核心资产泄密、影响业务连续性等影响华为商誉的安全事件。
任职要求：
业务技能要求：
1、熟悉网络及系统技术，能够根据不同网络环境，对信息安全项目进行规划和设计；
2、熟悉业界主流公有云安全架构，熟练掌握云计算知识。
专业知识要求：
如下具备至少3点：
如下具备至少3点：
1、熟悉DevOps流程，熟悉相关工具（Puppet, Ansible, Chef），有一定的编程能力、有云计算平台开发设计经验；
2、精通网络及系统技术，能够根据不同网络环境，对信息安全项目进行规划和设计；
3、精通业界主流公有云安全架构，熟练掌握云计算知识，具备OpenStack，KVM，Vmware相关经验；
4、主导或参加过云计算和云安全领域的技术标准，并具有一定的影响力，或精通所有主流云计算安全相关标准如FedRamp、CSA、SOC1、ISO 27017、ENISA等；
5、参与实现或设计主流云技术关键组件和特性，如网络云化、存储云化、IAM、容器、微服务、服务编排、密钥管理、日志监控等；
6、具备较强的风险评估、架构方案设计能力，能对复杂业务场景或和IT系统进行风险评估、架构方案设计。
