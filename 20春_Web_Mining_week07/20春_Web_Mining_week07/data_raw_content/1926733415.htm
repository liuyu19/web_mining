职位描述：

职责描述：
1、独立负责单条产品线，相关模块、策略的研究和产品设计，关注并研究行业动态；
2、梳理、挖掘整个业务链条上不同用户角色的诉求和痛点，深入理解用户场景；
3、对产品线用户体验负责，关注用户反馈，关注产品使用数据，不断优化用户体验；
4、协调多方资源，推进需求落地和产品迭代。

任职要求：
1、本科及以上学历，2年或以上后台系统产品经验；
2、良好的数据意识，熟悉互联网产品整体开发过程，包括从需求分析到产品发布；
3、逻辑思考能力和结构化思维能力突出，有设计复杂业务系统的能力和经验；
4、有物流或其他2B行业经验优先。
