职位描述：

工作职责：
与产品研发团队以及内部团队，对新项目方向筛选，具体玩法内容设计，研发难度，研发周期等进行评估分析。
与研发团队以及内部团队保持紧密沟通，确保产品相关策划同步/反馈及时。
对于研发团队提出的需求进行及时的支持，积极地支持产品的研发。同时，能根据不同的研发阶段需求，有计划地进行相关策划向支持。
与产品研发团队以及内部团队协作，持续在各阶段对在研产品的品质，玩法，游戏性等进行评估分析且提出建议。
协同内部团队进行未来产品方向进行分析研究。
对热点游戏品类以及目标产品等进行玩法/内容/技术/商业模式等要素的研究评估。
任职要求：
5年以上相关实际研发策划工作经验。
完整AAA/PC/Console研发经验。
多策划岗位相关经验者优先。（LD/GD/CD）
对于游戏玩法，游戏系统，用户体验设计等有专业的认知与实操经验。
深度理解游戏研发/游戏策划的各流程以及需求。
较强的分析提炼能力，能通过分析案例，总结各类经验。
对于各类游戏都有广泛的涉猎，对发展趋势有自我独特的洞察能力。
有较强的自我管理能力，以及多项目同步规划处理能力。具有管理经验尤佳。
有极佳的多团队协作合作经验，深刻理解团队合作的必要性。
具有跨国跨文化沟通能力，熟练掌握英语听说读写能力。
熟练掌握各类工具，Office， Visio， Photoshop，Illustrator，P4等。
