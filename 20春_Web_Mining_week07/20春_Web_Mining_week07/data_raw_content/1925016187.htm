职位描述：

职责描述：
1、负责电商设计、产品信息美化、促销活动等的平面设计、网页设计等美术设计工作。
2、负责产品图片拍摄、处理，新产品详情页设计。
3、负责电商的整体形象设计、整体版面布局、装饰工作。
4、制作各期专题推广海报，及促销活动等平面创意设计，提高点击和转化。
5、制作产品画册、产品折页、海报设计、易拉宝等线下物料设计 。

任职要求：
1、本科或以上学历；视觉设计、平面设计类专业。
2、一年以上设计工作经验，有3C数码或存储行业相关设计工作经验者优先。
3、熟练运用平面设计常用软件。
4、具有良好的审美能力与网页整体布局能力，准确把握线上整体风格设计。
5、具备良好的美术基础，良好的创意构思能力，有独立设计、制作能力。
6、做事认真、细致；有良好的沟通、理解能力及团队协作力，有较强的学习能力。
7、有较强的学习能力及抗压能力，有手绘能力优先。

企业介绍：

   深圳市朗科科技股份有限公司成立于1999年5月，总部设在深圳市南山区科技园。公司历经十逾年的发展，于2010年1月在A股创业板成功上市，股票代码300042。
   朗科公司拥有的闪存盘的核心发明专利“用于数据处理系统的快闪电子式外存储方法及其装置”，已获国家知识产权局和美国专利商标局授权，填补了20年来中国企业在全球计算机领域的发明专利空白。朗科目前拥有专利及专利申请总量逾500项，多项发明专利覆盖中国、欧洲、美国、日本、韩国等国家与地区。
   朗科公司坚持以“知识产权战略、人才战略、国际化战略”为公司发展战略，秉持“品牌先导、持续创新”的经营理念。
