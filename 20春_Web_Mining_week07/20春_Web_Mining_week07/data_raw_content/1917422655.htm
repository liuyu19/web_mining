职位描述：

岗位职责：
1. 为小米海外商城、新零售、集团信息化系统等提供前端 / 客户端支持；
2. 为团队积累和分享最佳开发实践和工具。
任职要求：
1. 具备3年以上工作经验，背景优秀者年限可略放宽；
2. 熟练掌握前端 / 客户端开发技术：前端老三样 + React / RN / Electron / node.js / ES6 / Vue 等皆可；
3. 熟练使用前端构建工具、自动化测试；
4. 熟练使用 Mac/Linux 环境进行开发，掌握必备的 git、shell 等使用；
5. 具有高效团队协作的能力，关注前端领域发展；
6. 具有良好的英语听说读写能力；
7. 熟悉至少一门后端语言。
加分项：
• 认同小米公司文化、价值观；
• 如果你是米粉小米欢迎你；
• 有大型电商系统或 APP 开发经验；
• 关注 UE、UI 设计；
• 代码洁癖偏执狂。
