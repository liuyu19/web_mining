职位描述：

职责描述：
1、负责云化/服务化的分布式数据库系统的特性开发、问题定位和性能调优；
2、主攻分布式云化数据库弹性、一致性及大规模并发读写高性能数据库等技术方向。
3、参与产品云化/微服务化（docker、K8s）架构演进、微服务拆解，构建稳定、竞争力强的服务化架构并落地；面向未来承担关键软件技术的设计与开发。
4、负责看护产品核心技术，承担模块关键技术、疑难问题攻关，提升产品优势。
5、面向下一代网络架构，负责服务框架、网络切片和分布式磁盘引擎等前沿技术预研。

任职要求：
1、熟悉C/C++、Java、Python、Go、Js等语言的一种或多种，具备丰富开发能力和经验，能够熟练运用完成开发工作；
2、具有良好的计算机和操作系统基础，熟悉常见的数据结构和算法，熟悉设计模式，具有一定的架构设计能力和重构经验；
3、熟悉linux操作系统，对进程间通信/socket编程/多线程并发等有相关经验，有PaaS平台开发经验者优先；
4、熟悉MangoDB/RocksDB等数据库者优先，具备数据库或高性能服务器软件设计开发经验者优先；
5、熟悉Paxos/Raft等算法者优先；
6、熟悉Tcp/IP/Sctp或者http，soap，diameter等协议者优先.
