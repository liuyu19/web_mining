职位描述：

职责描述：
1.设计&开发华为公司内部AI平台
2.提升模型训练的效率、提升推理服务的性能、提升资源利用率
3.关注业界AI最新进展，引入AI相关技术并实践应用

任职要求：
1、3年以上架构工作，解耦、特性设计经验， 对大型分布式系统有架构经验，熟悉高并发场景尤佳；
2、有机器学习平台（模型训练、分布式模型训练、部署推理服务）设计开发背景优先
3、熟悉python开发、调优 或掌握机器学习、深度学习框架（tensorflow、pytorch等）、或大数据生态包含hadoop、spark、flink等优先。
4、掌握微服务架构，熟悉spring cloud相关技术栈，spring cloud gateway、etcd、eureka等。
5、熟悉容器、k8s等相关技术栈，熟悉nginx、k8s ingress等。
