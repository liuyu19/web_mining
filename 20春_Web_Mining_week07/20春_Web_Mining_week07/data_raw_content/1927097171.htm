职位描述：

1、参与APP设计开发的全过程，包括需求分析、产品策划、功能设计、前后台模块设计、配合开发、UI/UED工作等；
2、撰写产品相关文档、协调各部门工作，推动产品上线并参与运营工作；
3、分析相关数据，发现问题并给出相关方案；
4、持续关注行业趋势，市场现状及发展方向。
要求：
1、3年以上互联网产品经理经验，有O2O类产品经验；
2、有电商导购工具类APP经验优先；
3、具有较强的数据分析能力；
4、熟悉APP前端和管理后台设计，能够理解前后端，以及B端和C端的设计重点不同；
5、善于与人交流，学习能力强，注重协作，并能承受工作压力；
6、计算机相关专业优先，熟悉电商行业优先。
