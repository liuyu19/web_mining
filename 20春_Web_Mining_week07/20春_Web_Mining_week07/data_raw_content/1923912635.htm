职位描述：

职责描述：

1. 公关策略及市场合作伙伴关系维护、管理；
2. 产品推广方案、活动、行业论坛及公关市场项目策划及管理；
3. 深度稿件策划、撰写及日常推广稿件的审核；
4. 公司对外各平台应用的定位及策略规划、管理，包括：微信、官网、百度百科等；
5. 对内internal marketing，打造员工正确、全面的对公司地位、行业形象、产品定位等认知；
6. 对外的corporate brand and employee marketing建立；
7. 品牌对外形象相关的设计、活动UI等设计沟通及管理；



任职要求：
1. 具有互联网、数据、广告等相关经验，较好的思考、沟通和项目管理能力，能够独立负责大型市场项目、活动的策划、管理及运营；
2. 能够适应快速行业发展，快节奏的工作，并能够结合公司产品及业务方向采用适合的市场和公关策略；
3. 具有较好的文字功底，能够快速实现深度稿件策划、撰写，并且能够把控日常公关内容的大方向；
4. 具有一定的领导能力及突发问题解决能力，较好的对内及对外各种合作关系管理能力；
5. 愿意接受挑战，实现自我突破；
6. 本科及以上学历，较好的英文听说读写能力；
