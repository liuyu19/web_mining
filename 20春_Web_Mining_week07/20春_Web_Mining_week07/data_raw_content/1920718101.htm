职位描述：

职责描述：
1. 信息流广告图片（包含视频创意）设计与制作，根据对客户特点、行业趋势、当下热点以及整体的投放策略设计广告创意，并输出行业广告创意方法论；
2. 对所设计的广告素材的效果负责，协助广告优化师分析广告表现数据并不断优化广告效果；
3. 具备分析用户喜好和行业痛点的能力，根据不断变化的用户喜好和行业痛点需求，设计制作不同种类风格的素材；
4. 视觉把控力强，细节控，手绘功底扎实，有成熟的手绘及视觉设计感强的作品。
