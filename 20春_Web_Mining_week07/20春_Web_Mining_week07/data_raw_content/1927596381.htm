职位描述：

工作职责：
1、负责艺人相关的创意视觉设计； 2、负责结合项目和艺人相关热点设计、制作物料。
任职要求：
1. 平面设计或工业设计等艺术类相关专业本科以上学历，3-5年以上的视觉设计工作经验，有时尚、娱乐、互动等行业视觉设计经验优先； 2. 具有优秀的视觉营销洞察力、对市场信息的敏锐感受能力以及丰富的想象力； 3. 有良好的美术功底、设计表现能力、审美能力； 4. 有较好的沟通能力，具备良好的团队合作精神； 5. 抗压性好，具备良好的沟通能力和逻辑能力； 6. 熟练应用PS、AI、Sketch等设计工具，有手绘能力、动画能力、视频制作剪辑能力优先； 7. 面试需要携带作品和成功设计案例。
