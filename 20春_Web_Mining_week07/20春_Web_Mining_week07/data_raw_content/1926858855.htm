职位描述：

岗位职责：
1.负责Android客户端产品的编码设计和需求开发，维护；
2.独立完成需求设计和参与技术方案讨论，编写技术方案和技术维护文档归档落地；
3.有客户端跨端设计及开发经验；
4.Android新技术的调研，引入，学习，技术攻关和框架设计，组件开发；
5.技术分享和技术培训，与团队共同提升，共同技术成长；
职位要求：
1.大学本科以上学历，计算机相关专业，3年以上Android实际开发经验，具有上线应用；
2.熟悉JAVA编程语言，良好的数据结构与算法，常用的设计模式；
3.熟悉AndroidGUI开发，以及自定义控件开发，熟悉RecyclerView等
4.对手机性能优化有经验，例如：内存优化，UI渲染和布局优化，网络性能优化，APK瘦身优化等；
5.对主流开源开发框架有二次封装和了解行业内的技术解决方案
