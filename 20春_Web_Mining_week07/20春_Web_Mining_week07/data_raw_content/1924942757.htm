职位描述：

职责描述：
1. 主要负责公司产品网站页面，APP软件页面的UI设计
2. 对外推广活动宣传页面设计
3. 结合产品推广，对宣传的文案，PPT进行美化设计
4. 从产品的设计角度，优化公司产品的操作流程，整理设计规范，带领设计团队

任职要求：
1. 本科及以上学历，热爱设计，有自己的设计创意思想，具体有一定的文案能力
2. 3年以上UI设计经验，熟练使用常规设计软件如Photoshop、AE，AI，Axure等设计软件，有会使用Blender，3D Max等工具进行3D建模者优先，有优秀案例或会视频制作者优先
3. 具有一定的美术功底，对色彩敏感、细节精到，能把握不同风格的设计工作,具有创新精神及创意激情
4. 有良好的沟通和协作能力，性格好，有团队协作精神，做事有效率。
