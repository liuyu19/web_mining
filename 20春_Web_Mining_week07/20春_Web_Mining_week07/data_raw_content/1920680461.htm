职位描述：

职责描述：
1、负责各业务线PC后台管理系统&可视化大屏的视觉设计工作，参与产品团队设计讨论与产品规划，归纳用户目标，用户任务。设计产品目标与需求；
2、独立负责产品模块的日常迭代设计以及整体体验优化，与产品经理完善产品细节，持续优化产品，追求极致用户体验；
3、与产品、业务配合，分析业务需求，分解归纳设计需求，进行概念设计并输出高质量高保真demo；
4、参与公司前瞻性、创新性产品的规划构思、产品逻辑和创意过程；
5、分享设计经验，帮助团队成员成长，提升团队整体设计水平。

任职要求：
1、熟练掌握sketch、axure或其他一种以上视觉/原型设计软件；
2、有数据可视化相关设计经验，对大屏数据可视化设计有深入的理解和实践；
3、有较强的逻辑思维能力，熟练掌握业务需求分析、产品需求分解的技巧；
4、有大局观，主动性高，具优秀的理解、沟通与协调能力，较强的文字表达和图形呈现能力，可以在复杂的约束条件下找到平衡或创新的方法；
5、本科设计专业，要求4年以上设计工作经验，3年以上pc端后台系统设计经验。

企业介绍：

平安医疗健康管理股份有限公司（中文简称"平安医保科技"，英文简称"Ping An HealthKonnect"）正式成立于2016年9月，是中国平安 "大医疗健康"战略的重要组成部分与集团智慧城市建设的核心成员。截至目前，公司市场覆盖全国近30个省、200余个城市，为8亿社会公众提供服务。2018年初，平安医保科技完成A轮融资，融资规模11.5亿美元，融资后估值 88亿美元，发展态势持续快速、稳健。
依托平安集团强大的综合金融实力与品牌影响力，公司凭借行业领先的管理经验及核心科技，为医保、商保及医院、医生、医药提供系统、服务和数据赋能。公司以智慧医保系统为核心，咨询、服务、保险为辅助，打造智慧医保一体化平台，全面赋能医保局。同时，公司打造商保产品方案设计、风控、健康服务等TPA服务，并积极延伸至为医院、医生、药店、药企赋能。此外，公司还可为合作伙伴提供多元化专业医疗租赁服务，并为广大人民群众提供便捷、精准、安全的创新型健康检测服务。
展望未来，平安医保科技将以革新中国医疗市场，助力三医联动，实现"一降两提"（降低医疗成本、提升服务体验、提升保障水平）为目标，持续深耕与开拓创新，矢志成为"中国领先的全方位赋能医疗生态圈的智慧科技公司"。
