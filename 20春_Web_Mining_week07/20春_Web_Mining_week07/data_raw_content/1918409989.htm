职位描述：

职责描述：
1.能准确理解产品需求和交互原型，根据产品定位输出优质的界面。
2.参与产品上线前的测试、验收，为所设计的产品的最终呈现效果和质量负责。
3.参与公司其他设计需求。

任职要求：
1.有PC/移动端网站等设计经验，UI相关设计经验；
2.精通Sketch、Photoshop、illustrator、会Principle/AE加分;
3.具有良好的网页及平面设计能力、丰富的网站制作经验，了解色彩搭配，思维活跃、有创意、有较强视觉效果表现能力;
4.具备良好的组织和沟通能力;
5.本科以上平面设计、视觉传达设计等相关专业毕业，能够接受较强的工作压力和挑战，工作态度端正，耐心，细致。
