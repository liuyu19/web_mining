职位描述：

工作职责:
1、根据产品需求，提供前端技术方案，完成高质量的前端开发和维护；
2、对具体的产品进行性能优化，实现优秀的页面加载、执行和渲染结果；
3、在理解前端开发流程的基础上，结合前端实际建立或优化提升工作效率的工具；
4、关注行业动态，在理解业务的基础上，结合前端技术给予创造性的解决方案；
5、对项目与自身进行定期总结，并乐于分享，帮助团队同学共同成长。

任职资格:
1、计算机相关专业本科及以上学历，具备5年以上Web前端开发项目经验；
2、深刻理解Web标准，熟练解决各浏览器兼容性问题；
3、熟悉各类Web前端开发技术，包括HTML、CSS、JavaScript、Ajax等，具有HTML5/CSS3使用经验，熟悉响应式设计理念；
4、有良好的审美能力，对自己的作品有严格的质量要求，熟悉用户体验相关知识，参与推动UI规范制定和落地；
5、有良好的沟通能力和团队协作能力，有较强的自主学习精神；
6、对数据结构、服务端开发（PHP，Java，Python等）或Node.js有一定了解者优先，对移动产品有较强的敏感度。

企业介绍：

     魅族科技由黄章于2003年创立，公司以设计研发优雅、简单易用的智能设备和系统为依托，致力于打造开放共赢的移动互联网智能生态。
      创立初期，通过三年时间魅族成为中国最知名的音乐播放器品牌。2006年全面转型进入智能手机领域，陆续发布M8、M9、MX、MX2、MX3、MX4、MX5、魅蓝note2、魅蓝2、PRO 5、魅蓝metal、魅蓝note3、PRO 6及魅蓝3、魅蓝3S，魅蓝E等多款智能手机；推出基于Android，专为国人使用习惯，而自主深度优化的操作系统——Flyme；在中国智能手机行业占据一席之地。
      2015年，魅族完成智能手机品牌布局，细分为PRO（高端）、MX（中端）、魅蓝（超高性价比）三个品牌，覆盖更为广泛的市场，满足不同层次消费者的需求。
      魅族一直保持对品质的追求，不遗余力地打造用户喜爱的产品。到今天，魅族已拥有4000多名员工，并在俄罗斯、以色列、乌克兰及印度等国家和地区销售。
