职位描述：

岗位职责：
1. 实现设计师的设计方案，编写web前端；
2. 负责公司各类项目的前端开发工作,与后台工程师配合，实现数据交换和系统整合；
3. 配合产品经理和UI设计师，通过技术提升用户体验和可用性
岗位要求：
硬性要求
1、必须熟练掌握React框架及周边工具
2、对实时聊天系统有开发经验（websocket）
职责要求
1、熟练使用 HTML5、CSS3 技术编写前端页面，可对前端程序进行性能优化、代码调试；。
2、熟悉JS 语言原理。熟练使用ES6，基于React(React、Redux、Redux-Saga、React-Route、Ant-Design、dva)开发SPA应用，
3、熟练使用 NPM、WebPack、Jquery 等常用库
4、熟悉 HTTP 协议以及nginx开发环境，能够使用抓包工具分析归纳和解决问题。
5、熟练使用git命令或git相关工具对代码进行版本管理。
