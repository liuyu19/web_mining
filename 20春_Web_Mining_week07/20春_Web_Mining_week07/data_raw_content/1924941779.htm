职位描述：

职责描述：
1、负责公司Web产品的UI设计；
2、参于产品规划构思和创意过程，完成交互原型及实现；
3、对UI设计有自己独到的见解及风格，能根据产品需求设计出相应产品匹配的、有较好契合度的UI接口及图标、需要充分考虑用户体验；
4、对页面进行优化，使使用者操作更趋于人性化，持续优化产品的用户体验。

任职要求：
1、大专以上学历，美术、设计相关专业，2年以上UI工作经验，有已上线的项目；
2、对使用者体验设计与互联网产品有独立的见解与思考，逻辑性佳，具有产品思维，能够提出系统性的UI设计解决方案，具有较强的执行力；
3、有较强的手绘能力、设计能力、创意能力并熟练掌握相关设计软件；
4、有想法、有创意，学习能力强，关注设计流行趋势并有自己见解；

特质要求：善于沟通，关注用户体验细节，有较强理解力与出色的沟通能力，具备一定的抗压能力
