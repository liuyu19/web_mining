职位描述：

职责描述：
n 岗位职责
（1）牵头研究智慧城市相关国家政策、行业标准、前沿技术和发展理念等；
（2）根据行业需求和痛点分析，结合新政策、新技术发展趋势等，独立制作对外交流方案；

（3）负责向公司及时反馈行业、市场、竞争对手等方面的市场分析数据以及产品动向，了解同类产品的情况和特点，为公司的市场方向、产品研发等提供建议；

（4）作为智慧城市行业研究专家，进行团队内部知识传递，提升团队战斗力。

（5）负责开展智慧城市顶层设计和规划、行业信息化总体规划和设计等；

（6）负责面向客户、行业或项目的解决方案咨询，包括需求调研、客户引导、规划设计、撰写相关技术文件等；
（7）协助面向客户的技术交流、方案宣讲、产品演示，进行售前项目的分析、设计和策划；

n 任职要求
（1）本科及以上学历，通信、计算机、经济管理、城市规划等相关专业；

（2）熟悉智慧城市理论、建设方案、规划设计、招投标等知识，对智慧城市、大数据、云计算、物联网、新一代互联网和互联网+等技术有深刻理解；
（3）精通智慧城市业务应用方向中优先，包括：城市运营管理中心（IOC）、智慧交通、智慧园区/小镇、智慧生态环保、智慧建筑、智慧政务、智慧教育、智慧医疗、智慧安监、智慧文旅等；

（4）具备良好的演讲能力、沟通能力与观察洞悉能力，能准确把握客户要求并牵引客户；
（5）具备较强的责任心，组织能力、合作能力和学习能力。
（6）能够适应工作需要的出差。

企业介绍：

京东于2004年正式涉足电商领域。2016年，京东集团市场交易额达到9392亿元*。京东是中国收入规模最大的互联网企业。2017年7月，京东再次入榜《财富》全球500强，位列第261位，成为排名最高的中国互联网企业，在全球仅次于亚马逊和Alphabet，位列互联网企业第三。
2014年5月，京东集团在美国纳斯达克证券交易所正式挂牌上市，是中国第一个成功赴美上市的大型综合型电商平台。2015年7月，京东凭借高成长性入选纳斯达克100指数和纳斯达克100平均加权指数
