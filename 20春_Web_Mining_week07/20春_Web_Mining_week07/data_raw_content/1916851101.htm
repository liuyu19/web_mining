职位描述：

职责描述：
1. 负责华为云产品的前期视觉推导并完成设计方案，新功能用户界面设计
2. 负责web产品与移动端产品的视觉设计，包括线上、线下活动提供美术支持
3. 参与前瞻性产品的创意和动态DEMO的主创视觉设计，设计规范输出并落地

任职要求：
1. 热爱设计，具有大型互联网视觉界面实战经验，曾主导设计3套以上UI风格设计，拥有宽广的行业视野（UI设计、互联网等）
2. 始终掌握UI视觉设计的最前沿动态和工具（sketch，ivision，xd等）利用，能准确，完整的表达自己的设计，并推动设计落地
3. 具备较强的创新能力，创造出耳目一新的界面呈现效果。
4. 具备优秀的沟通能力、学习能力，善于把控细节，有较强的创新欲望与执行力。
5. 全日制本科及以上学历，美术、设计、艺术学类相关专业，英语四级及以上，5年以上相关行业的工作经验。
6. 在数据可视化方面有自我的探索和实践是加分项。
