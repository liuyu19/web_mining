职位描述：

工作职责:
1、负责国际化产品用户体验分析和专题研究，含：不同市场用户需求和特性分析、竞品分析、产品全流程体验等。
2、负责海外平台全流程用户体验项目推动。确定项目方向，引领项目进程，确保项目沟通、控制和验收；
3、相关流程、产品的设计，与产品、技术及其他部门沟通协调；
4、负责相关服务直接提升。

任职资格:
1、有海外工作/学习经验，了解当地用户需求和特性，语言流利；（日语韩语及英语方向）
2、3年及以上国际化互联网公司产品经验，PO/前端/用户增长PM优先；
3、有海外OTA行业工作经验优先
4、以结果为导向，抗压能力强，善于沟通，推动目标达成。
5、能深入分析问题，逻辑思维/数据分析能力优异，用户和产品Sense好。
6、热爱产品工作，持续学习、了解市场/用户/行业，自驱动成长。
