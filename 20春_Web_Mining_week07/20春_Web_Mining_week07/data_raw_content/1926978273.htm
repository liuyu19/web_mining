职位描述：

   在这里，你将从事IT应用层软件、分布式云化软件、互联网APP产品等的设计开发，可以采用敏捷、开源等先进的软件设计开发模式，接触最前沿的产品和软件技术，成为一个软件大牛；你将有机会参与瑞幸的智能化供应链，财务中台，基础服务，用户增长，前沿技术研究等平台的研发工作，包括但不限于：
-负责咖啡产品相关的App开发以及产品维护；
-负责 Android手机端性能相关技术优化工作；
-面向终端平台，进行前沿技术探索和软件架构设计工作。
职位要求：
1.本科及以上学历，计算机软件相关专业；
2. 熟练掌握Java，熟悉常用的数据结构和算法
3. 掌握Android开发平台及框架原理，熟悉Dalvik/JVM工作原理
4. 有多个完整的Android项目经验，并至少在一个复杂项目当中承担核心设计与关键开发工作，有一定的架构设计并实现的能力
5. 良好编程习惯和文档编写习惯，高质量编码能力
6. 对移动端App的UI开发熟悉，并有自己的UI架构心得和经验，具备开发高性能UI的能力
7. 熟悉 React 框架，有React Native 项目线上经验优先；
8. 基本功扎实，可以应付各种APP开发中的复杂问题，并迅速给出解决方案，
9. 积极主动，较强的学习能力，能够思考和优化解决方案，能够独立完成项目的设计和交付。
