职位描述：

工作职责：
负责互联网金融平台产品的视觉设计工作。与产品团队、运营团队以及交互设计师一起工作，做产品运营相关的所有设计，包括策略的设计、落地的设计执行（较强视觉能力），以及平台相关品牌设计、UI设计等工作。
以设计提升产品运转效率，分析数据（可学习）；
以设计推动团队协作，发挥设计的杠杆作用。
任职要求：
5年以上的移动互联网视觉设计工作经验，艺术/设计/动画等相关专业；
有良好的审美能力，扎实的美术功底，较强插画能力，精通PS/AI/SKETCH/C4D等软件；
具有良好的个人品质，积极乐观，认真负责任，能够承担较大的工作压力
在动效或3D表现有特长 / 有大型互联网项目成功案例/有4A公司工作经验者优先
