企业介绍：

是一家专注于人工智能和计算机视觉系统和芯片的科技型公司。英文名NextVPU，蕴意是Next Vision Processing Unit, 即未来的视觉处理器。“肇观”是“开启视觉”的含义。我们的愿景是将视觉赋予给所有的机器人，无人机，无人车以及其他智能设备，开启人工智能的新时代。公司由业界老兵成立，对外低调，专注黑科技的研发和应用。
截止目前，已完成A轮融资，估值数亿美元，公司业务覆盖中国大陆、中国香港、美国、德国、英国、法国、中国台湾、印度、荷兰、日本等全球各大国家及地区,申请国内外专利数十项。作为技术研发机构，我们已获得2017 CES Asia 创新奖、创业奖，2017年度国际创新创业大赛国赛第三，2016首届海外高层次人才创新创业大赛二等奖、2016 Slush Shanghai Top 5等奖项。
核心技术
NextVPU有着世界领先的深度异构网络技术，硅视觉技术，高级视觉定位技术，高级视觉识别技术，虚拟盲杖技术，虚拟盲道技术 ，用于深度感知、视觉定位、动作追踪和视觉识别，为机器装上“眼睛”，赋予其智能。
研发团队介绍
肇观电子拥有一支高学历、高水准、高素质的三高运营团队，核心管理成员均为中国籍公民，毕业于复旦、交大、中科大、浙大、人大、南安普顿等知名大学，研发团队全部拥有博士/硕士学位，来自于Intel、Qualcomm、AMD、Broadcom、Marvell、中芯国际、猎豹移动等国际知名企业。
天使眼智能导盲眼镜
天使眼智能导盲眼镜是世界首款采用计算机视觉及人工智能技术辅助盲人感知世界和出行的智能眼镜，由上海肇观电子科技有限公司研发制造。前方双目立体摄像头模拟人眼采集前方物体位置信息并转换成听觉信号，再通过骨传导耳机传达，即时提示避障，自动定位、智能识别物体。根据听觉信息的提示，天使眼智能眼镜可以作为盲人朋友的感官延伸，辅助其感知世界和安全出行。天使眼是一款功能强大的环境感知设备，集先进技术与时尚外形于一身，可靠、轻便，需链接一个外接设备，辅助视障人士感知世界和出行。
开普勒双目立体摄像头
“开普勒”双目立体摄像头，是一款先进的、功能强大、可靠而外形小巧的环境感知设备，拥有高精度和低功耗的特性，用于深度感知、光磁定位和动作追踪，可应用于室内和室外，出厂前经过精确标定；开发者可以利用开普勒摄像头搭建先进的可以感知周围环境的智能系统。应用于机器人，无人机，无人车的室内和室外避障导航等领域。从镜头、图像传感器到姿态传感器，开普勒摄像头将深度感知、光磁定位和动作追踪提高到了一个新的高度。
