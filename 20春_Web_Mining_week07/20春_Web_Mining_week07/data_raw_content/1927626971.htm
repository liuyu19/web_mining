职位描述：

职责描述：
1.与策划、美术沟通，负责场景，战斗，UI系统和系统玩法等的制作；
2.客户端框架比如UI框架，技能框架，AI框架等的问题解决和优化。
3.客户端引擎的开发和维护，渲染相关的工作。
4.客户端动作相关的工作。
5.客户端各类编辑器的协助工作

任职要求：
1.本科及以上学历，2年以上游戏客户端开发经历；
2.熟练掌握C/C++/Lua/Python，Unity3D/Unreal使用经历者优先，有丰富3D渲染开发和优化经验优先；
3.熟练掌握常用数据结构与算法；
4.熟悉iOS和Android等多平台开发环境优先；
5.有线上游戏维护经验者优先；
6.善于沟通，有团队合作精神，对待工作有责任心；
7.良好的的代码风格和编程习惯，热爱游戏，能承受压力。
