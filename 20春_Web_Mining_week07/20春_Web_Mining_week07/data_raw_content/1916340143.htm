职位描述：

工作职责：
1.负责一起学App的日常活动运营设计及相关推广设计；
2.负责公司线上线下大型运营活动活动的视觉表现
3.有独立承担并执行项目的能力，进行高质量的运营产品类视觉设计；
4.熟悉设计团队的整个设计流程，能够很好对接研发部并保证产品开发落完美视觉呈现

职位要求：
1. 美术、平面相关设计专业毕业。有广告公司、互联网公司工作经历。
2. 有3年以上大型运营专题设计经验，并有上线成功案例。
3. 有较好的沟通能力与理解能力，有极强的责任心与执行力，能承受一定工作压力
4. 插图绘制及创意能力突出者优先。

企业介绍：

一起教育科技是全球知名的K12智能教育平台。怀着“让学习成为美好体验”的使命，一起教育科技致力于用先进的教育科技、优质的教育内容和持续的教育热情，为K12阶段的学校、家庭、社会教育场景，提供更为高效、美好的产品和体验，开启了智能教育新时代。
     一起小学、一起中学是以大数据为驱动的学校教育平台。同步学校教学进度，覆盖练习、评测等教育场景，提供匹配K12全学段、全学科的教育内容。通过大数据和人工智能等教育科技手段，帮助老师减负增效、科学教学，成为老师的智能教学好帮手。通过寓教于乐的产品设计、个性化的学习解决方案和科学的护眼呵护，帮助学生形成良好的学习兴趣和学习习惯，成为学生的学习成长好伙伴。通过学习报告、智能诊断和家长鼓励等功能，让家长掌握学情并为孩子定制科学、个性化的成长路径，从而成为家长陪伴学习的得力助手。
     一起学是以个性化学习为目标的家庭教育平台。依托一起教育科技Socrates智能学习系统，为校外学习场景提供了智能学与练的解决方案，帮助学生和家长们了解自己，有针对性的高效学习。
     一起学网校是一起教育科技旗下中小学在线学科辅导平台。采用真人在线直播互动教学模式，让全国任何地区的中小学生，都可以在家享受一线优秀教师的直播辅导服务。一起学网校，基于一起教育科技多年沉淀的中小学生海量作业数据，以“个性化学习”为本，同步课内基础知识，外延核心素养，帮助孩子更科学地获取学科知识，提升学习兴趣，养成自主学习能力。
      一起公益是连接公益力量的社会教育平台。秉承“知责任、行公益”的理念，借助一起中小学平台连接青少年、公益组织和爱心企业，帮助教育资源匮乏地区，实现优质资源共享，助力教育均衡。
      科技中有温度，数据里有梦想。我们始终以教研为基、以科技为梁，为老师、学生和家长提供优质的产品体验和服务。未来，一起教育科技将努力让知识和能力一起，构建学生核心素养；让乡镇和城市一起，共享优质教育资源；让科技和教育一起，实现学习美好体验。
最新公司数据：
截止2019年3月，一起教育科技产品覆盖全国31个省市、365个城市、14万所学校，用户数超7900万，其中学生用户超5170万，家长用户超2500万，老师用户超250万。
