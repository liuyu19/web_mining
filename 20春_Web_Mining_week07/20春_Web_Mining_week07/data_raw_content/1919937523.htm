职位描述：

岗位介绍
1. 纷享销客的CRM产品以PaaS+SaaS的形态快速成长，以连接的理念重新定义CRM：连接企业内部组织、连接合作伙伴、连接客户。
2. 纷享销客的CRM成长过程需要更多有经验有兴趣有战斗力的产品经理加入。
3. 在这个岗位上，你需要有优秀的商业直觉、业务认知以及问题解决能力。

核心岗位职责
1. 负责PaaS平台产品路线图制定及落地，包括但不限于对象、权限、流程、UI PaaS、开发者平台等。
2. 负责竞品分析、市场调研，针对性给出市场分析与差异化策略。
3. 负责需求详细设计及评审，组织UE/UI团队完成产品原型设计，对最终的PRD及产品原型负责。
4. 负责与RD团队紧密配合，宣讲需求及原型，跟进需求开发全过程。
5. 对负责方向的产品数据负责，运营产品：包括但不限于产品培训、线上运营、支持售前及销售。

任职要求：
1. 熟悉CRM，熟悉Salesforce、Zoho、Pega和销售易等CRM产品PaaS平台者优先。
2. 计算机及相关专业或管理类专业（信息/企业管理类专业优先）,大学本科及以上学历。
3. 3年以上PaaS平台产品经验者优先。
4. 逻辑严密，沟通能力强。负责，闭环。
5. 心态好，抗压能力强。有好奇心，热爱挑战。

我们的特色
1. 连接型CRM的提出者与践行者，我们在改变CRM。
2. 年轻团队，氛围极好，战斗力强。

企业介绍：

纷享销客（北京易动纷享科技有限责任公司）总部位于北京市海淀区中关村，目前已在北京、上海、广州、深圳、杭州等5个城市设立直营分公司；在南京、武汉、成都、郑州、济南等50余个城市建立营销服务中心。

截至目前，员工总数700余人，产品研发团队近300人，是一家具备完善的研发、实施能力的优质SaaS企业。

纷享销客是连接型CRM优质服务商。纷享销客连接型CRM以开放的企业级通讯为基础架构，以连接人、连接业务、连接客户为使命，将CRM、PRM及SCRM融为一体，为企业提供销售管理、营销管理及服务管理为一体的移动化客户全生命周期管理。开放的通讯架构与交互的业务逻辑，帮助企业实现外部业务、内部全员以及微信生态的互联互通，为企业构建起连接企业内外的业务价值链。

为满足企业组织与业务的敏捷变化需求，纷享销客通过“业务定制平台+商业智能平台+开放平台”的平台化战略，为企业个性化需求提供友好的业务自由配置能力，为中国中小SaaS开发厂商提供生态级能力支持。

公司成立以来，先后获得IDG资本、北极光创投、DCM、高瓴资本、中信产业基金和金蝶国际等优秀投资机构投资，累计服务客户60万家，是中国连接型CRM的优质服务商和企业级SaaS生态的构建者。
