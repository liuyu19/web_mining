职位描述：

职责描述：
1、负责提供收入应收各领域的数据需求落地解决方案，以及财经主数据及核心基础数据的管理工作；
2、承担LTC领域各主流作业系统的数据需求分析、数据架构设计、落地遵从管控，及部分核心主数据及基础数据的数据服务设计、实施、上线注册及运维工作；
3、承担数据中台BA等角色，构建LTC领域应用消费所需LTC业务领域信息架构及入湖能力，设计及落地数据中台分析资产，支撑财经数字化建设；
4、规划与设计销售核算的数据与应用架构，识别现有核算与报告管道的不足和演进方向，作为变革方向。

任职要求：
1、工作经验要求：
1）至少7年以上IT领域工作经验；
2）有数字化转型或运营经验者优先；
3）有Oracle、SAP、IBM/埃森哲/PWC/德勤（或同等水准）顾问公司的IT方案实施经验者优先。
2、技能要求：
1）熟悉数据库模型设计方法及规范，能使用数据建模软件进行建模；
2）熟悉软件工程的理论，掌握多种需求获取及分析方法，善于把握需求要点和重点，有较强的用户需求判断、引导及控制能力；
3）熟悉ORACLE、DB2、SQL SEVER、MYSQL至少一种数据库，熟练使用SQL，熟悉分析工具POWER BI、TABLEAU；
4）具备较强的学习能力、沟通能力，具备良好的技术知识储备和业界实践知识。
3、教育背景要求：全日制本科以上学历。
4、语言能力要求：英语四级及以上或同等水准。
