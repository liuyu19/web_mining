职位描述：

职责描述：
-负责百度技术中台用户体验部的产品交互设计工作
-能独立负责一个或多个产品线，熟练运用交互设计方法，进行高质量设计输出
-参与用户洞察、市场分析、产品定位、商业模式分析，有效支撑设计成果的合理性和严谨性
-定义用户体验标准，能用数据验证设计有效性，完成设计闭环
-对交互设计工作能够进行体系化的规则建设，并成为团队设计准则

任职要求：
-本科及以上学历，工业设计、计算机、心理学、艺术设计、广告或相关专业
-有3-5年以上交互设计工作经验，熟悉UED工作流程，有商业B端相关设计经验者优先
-清晰的逻辑思维、扎实的设计理论基础和学习能力，关注业界发展趋势，有跨领域项目成果与尝试
-有较强的沟通表达和执行力，能有效推导设计思路主动发起设计提案，并推动方案落地
-有一定的数据分析能力，设计过程中有使用数据驱动设计的意识和经验
-具有前瞻性、创新性思考
