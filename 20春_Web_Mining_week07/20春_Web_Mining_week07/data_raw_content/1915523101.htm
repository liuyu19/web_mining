职位描述：

职责描述：
1. 负责华为云促销活动页面的平面设计工作；
2. 参与视觉用户研究，负责WEB网站UI设计；
3. 负责WEB网站的整体创意、设计和美化；
4. 配合交互设计师完成页面设计。
5. 负责部门活动推广素材设计制作工作；
6. 负责部门产品icon及吉祥物延展设计工作。




任职要求：
1. 美术、设计等相关专业本科以上学历，从事过网站UI设计五年以上，具良好设计灵感与美术功底，具有一定的手绘能力；
2. 有丰富的设计理论知识和对流行趋势敏锐的洞察力，对网站UI设计趋势有灵敏触觉和领悟能力，推动团队的设计能力；
3. 具有大型网络软件产品界面风格设定以及UI设定（包括界面、图标风格与统一规范等）经验优先；
4. 精通Photoshop、AI、Sketch等设计软件工具，具备较强的艺术哲学理论并能融汇贯通于产品、业务、市场，以及沟通能力和职业作风；
5. 态度积极，工作责任心强，具有较强的独立工作能力和团队合作精神；
6. 拥有自己独立设计风格者优先考虑；
