职位描述：

岗位职责:
1. 负责网站、软件产品移动端与PC端界面风格设计定位；
2. 在充分理解产品交互文档基础上，可以快速理解产品需求及交互原型；
3. 改进软件的交互设计，对软件的用户体验交互性和易用性定期提出改进方案；
4. 及时完成上级领导交代的其他工作。
任职资格:
1. 全日制大专以上学历，3年以上网站、移动互联网等相关行业UI工作经验；
2. 熟练使用photoshop、dreamweaver、illustator等工作相关设计软件；
3. 熟练使用DIV+CSS网页布局及Web前端开发技术；
4. 有扎实的美术功底、良好的创意思维和理解能力，能及时把握需求；
5. 善于沟通，良好的团队合作精神和高度的责任感，能够承受压力，有创新精神，保证工作质量；
6. 请投递简历时附带作品。

企业介绍：

　　湖南科创信息技术股份有限公司是一家上市企业，股票代码300730,是一家专业计算机软件开发及系统集成服务提供商。公司成立于1998年，注册资金6970万元。科创公司致力于为客户提供完善的行业解决方案和优质服务，为客户创造最大的价值。已经成为湖南省最大软件企业之一，国内电子渠道、机器视觉、电子政务等应用的市场领跑者。
　　公司依托211及985高校中南大学的学科性公司，是国家高新技术企业，国家火炬计划软件产业基地骨干企业、湖南省首批认证软件企业，获得了信息产业部计算机信息系统集成貮级资质，涉及国家秘密的计算机信息系统集成资质。2001年ISO9001：2000国际质量管理体系认证，2007年7月通过了CMMI3 V1.2级软件能力成熟度模型评估，2012年通过了ISO20000体系认证同，是湖南省最具成长性的非公有制企业。
　　公司总部长沙，现有员工900余人，另在北京、广州、武汉、云南、郑州贵州设有分公司。公司依托中南大学，具有雄厚的技术实力和人才优势，拥有一大批包括教授、副教授、高工、博士和硕士在内的高素质人才，目前员工总数880多人，85%以上的员工具有本科以上学历。
　　科创信息的核心竞争力是产品研发能力，拥有自主知识产权产品57项，国家专利6项，国家发明三等奖1项，国家八五重大科技成果奖1项。公司潜心十多年研发了系列核心技术平台：科创电子政务应用支撑平台和应用整合平台、科创机器视觉嵌入式软件平台、科创电子渠道技术支撑平台、科创企业应用集成平台、科创教育门户和应用集成平台等。并基于上述平台为政府、电信运营商、企业、教育等行业及机器视觉领域提供了广泛而优质的软件产品和服务。
　 在电子政务领域，公司承建了湖南省委省政府电子政务内网(一、二期)、湖南省电子政务网、湖南省检察院三级网络、湖南省地税金三期网络、湖南省财政厅三级网络等大型网络工程,研发并部署了湖南全省电子公文传输系统及OA群软件,覆盖了全省电子政务内网各级党委和政府及委办局;承建全省行政审批和电子监察系统建设、全省法人库基础信息库建设、财政国库集中支付软件、地税全省大集中业务软件、环保厅全省数字环保综合应用平台,并在省内外的国土系统、公安系统等行业推广多种创新性行业软件.通用电子政务、公安电子政务、国土电子政务、财税电子政务、环保电子政务是公司关注的主要方向.
  在机器视觉应用领域,公司成功研制了一系列具有国际领先水平的视觉检测系统:浮法玻璃缺陷在线检测、玻璃厚度在线检测、玻璃温度在线检测、玻璃表面应力在线检测、玻璃完整性在线检测等系统.其中“玻璃缺陷在线检测系统”荣获国家科学发明三等奖及“八五”攻关重大科技成果奖，列入国家火炬计划，并完成了产品化，在全国各地15个玻璃厂27条生产线上推广使用。公司还为中国铁路研发了列车动态偏移量在线检测系统、铁路桥梁挠度状态多功能检测仪；为中国钢铁企业研发了宽厚板激光厚度在线检测系统。这些视觉检测系统均为国内首创，具有巨大的市场前景。
  在电子渠道领域，公司为湖南移动提供了门户网站、短信营业厅、WAP营业厅、自助营业厅、10086呼叫中心、客服信息管理平台等新型电子渠道产品，以及物资管理等业务系统。为国税提供12366呼叫中心等电子渠道应用。并以电子渠道技术平台为基础，进行跨界创新，为电信、政府、大型企业等需要新型营销与服务平台的组织提供完整的行业电子渠道营销和服务解决方案。
  在企业行业，公司为（联通）中讯设计院、中烟工业公司、长沙银行、浙江巨石集团、中联重科等知名企业开发了包括“企业办公综合信息门户”、“企业质量管理系统”、“企业生产过程集中监控系统”在内的多个应用软件系统；为昆明、柳州铁路局等40多个机务段开发了铁路机务检修综合管理信息系统；为三一集团、株洲时代集团、南车集团等大型企业提供了包括网络、智能建筑的整体信息系统解决方案。
  在教育行业，公司为包括中南大学、湖南大学、湖南师范大学在内的三十余所大学、五十余所中学提供了不同规模的校园网络、数字化校园及应用系统解决方案。并为湖南省教委、教育考试院提供了相关业务系统。是湖南省最大的教育信息化解决方案提供商。
