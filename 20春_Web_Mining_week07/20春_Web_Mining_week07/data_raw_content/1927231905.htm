职位描述：

职责描述：
1. 基于业务线的场景设计，协助对数据进行挖掘和整理
2. 根据公司业务，利用大数据分析手段，协助形成大数据分析报告，并辅助提出可行性的解决方案
3. 根据大数据分析结果，协助团队分析业务模式改进点与创新点
4. 根据业务或应用需求，协助技术团队完成数据建模及模型开发等工作
5. 根据业务要求，参与数据服务方案设计，并协助数据资产团队完成数据资产开发及治理等相关工作

任职要求：
教育：
1. 全日制大学本科及以上学历
2. 计算机或通信类相关专业
知识：
1. 了解通信行业网络与业务知识，了解各类支撑系统如BOSS、CRM、网管系统、ERP及相应数据结构等
2. 熟悉数据库/数据仓库、hadoop生态相关组件工作机理
3. 熟悉软件开发与测试流程等专业知识，熟悉java/python等开发语言
4. 了解项目管理相关知识
技能素质：
1. 了解通信行业网络与业务知识，了解各类支撑系统如BOSS、CRM、网管系统、ERP及相应数据结构等
2. 熟悉数据库/数据仓库、hadoop生态相关组件工作机理
3. 熟悉软件开发与测试流程等专业知识，熟悉java/python等开发语言
4. 了解项目管理相关知识
5. 具备基本的数据库开发技能，熟悉SQL、H-SQL、Mapreduce、Spark、python、R等数据处理方法
6. 了解ETL处理，熟悉数据模型设计和优化方法
7. 熟悉通用的数据分析、挖掘方法和工具，具备数据可视化工具或技术及原型系统设计工具Axure操作技能
8. 具备基本的书面和口头表达沟通技能
经验：具有3年及以上相关工作经验。

企业介绍：

中移信息技术有限公司是中国移动通信集团有限公司下属的全资子公司，定位于智慧运营的推动者、数字服务的使能者、企业流程的固化者。中移信息技术有限公司的成立，是中国移动IT能力提升的重要标志，也是中国移动打造IT核心竞争力的新起点。
公司长期发展愿景是努力成为国内一流、国际知名的数字化技术和解决方案提供商。针对中国移动集团内部，重点推进IT统一规划、企业级大数据平台、集中化IT系统的整合与建设等项工作，为集团各成员单位提供资源共享、能力集中、合理管控的IT运营支撑服务，协助各单位打造极致体验、高效运营、良性增长的数字化服务能力；针对中国移动集团外部，依托集中化IT系统和自主研发经验积累，积极探索IT资源和能力优势的对外输出，面向其他行业企业和海外电信运营商提供大数据等IT解决方案和服务，努力拓展对外服务份额，力争成为业界有影响力的专业化IT服务机构。
为”大连接”战略保驾护航，助力中国移动转型发展。公司将继续推进卓越人才工程战略落地， 引进“高潜人才”、培育“卓越人才”、调动“精英人才”，致力于培养富于远见、充满激情、意志坚定、勤奋务实、诚实守信、认真负责、尊重他人、乐于分享的职业化人才队伍。欢迎有志于投身IT事业的人才积极加入公司大家庭，和我们成为工作伙伴！
