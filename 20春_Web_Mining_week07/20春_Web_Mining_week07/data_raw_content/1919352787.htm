职位描述：

职责描述：
1、负责提供符合客户要求的创意设计思路，与策划共同完成广告方案创意部分
2、在广告创意中给予有效的创意设计策略支持；
3、网络广告方案主视觉设计，同时响应日常工作中的创意设计类需求；
任职要求：
1、 美术、平面设计、动画制作、广告等相关专业，大学本科及以上学历；
2、 2-3年相关工作经验，有成功作品；
3、 具有扎实的美术基础，有一定的手绘能力，色彩控制能力强、想象力丰富；
4、 精通Photoshop、Flash、Illustrator，熟悉flash actionscript；熟悉动效表现，会使用AE软件能独立完成Flash Banner等相关设计工作。
5、 具备良好的沟通能力，有团队合作精神，善于表达；
6、 热爱互联网，热爱广告，关注生活，追求完美，有广告业获奖经历优先；
