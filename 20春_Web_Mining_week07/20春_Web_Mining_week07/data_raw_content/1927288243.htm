职位描述：

岗位职责：
1.负责统筹各电商平台店铺的整体装修设计，不断优化，提高客户体验感；
2.负责天猫平台店铺首页、Banner图片、宝贝页面的制作及美化，维护高质量的品牌视觉形象；
3. 带领设计团队，负责各个品牌推广设计，对项目进度和效果有绝对把控能力；
4. 负责品牌全渠道的视觉呈现，深入理解品牌，独立完成及带领团队指导品牌的创意构思及执行，确保作品质量和完成时间；
5.根据公司不定期促销计划，对公司网络平台店铺进行页面美化设计，配合推广制作推广效果图，并能提供营销策划建议
6.项目设计过程中每一环节的确认工作，把控时间进度，跟踪设计后期
任职要求：
1.3年以上电商平台设计工作经验，有2年以上团队管理经验优先；
2. 美术设计类、广告学及相关专业优先考虑，有5年以上视觉设计经验，
3. 较强的设计管理能力和沟通能力，熟练掌握平面和网页设计的各种技能，良好的审美能力、对潮流趋势的敏锐洞察力；
4.对文案及产品策略有较强理解力，有灵活运用色调风格创意的良好表现能力；
加分项：有较好的文案功底，英语基础；服务过汽车后市场或相关行业、公司的优先考虑。

企业介绍：

◎2011年:以轮胎为切入点进入中国汽车后市场，途虎养车商城正式上线
◎2012年:在上海成立物流中心，逐步实现江浙沪极速达。
◎2013年:全国安装网点超过4,000家。
◎2014年:途虎养车APP上线;途虎汽车保养服务超过30种。
◎2015年:6月，途虎完成近1亿美元C轮融资;10月，完成5,000万美元C+轮融资，品牌理念由途虎养车网变更为途虎养车，同时推四大战略升级。
◎2016年:途虎养车第一家工场店在上海开业;同时，途虎养车在全国布局合作门店超过13,000家，服务能力覆盖全国405个城市;途虎养车2016年销售额超过10亿元人民币，并且在同年获得百度、高盛 领投的近10亿人民币D轮融资。
◎2017年:途虎养车布局工场店超过400家;同时，途虎养车积极开展合作，先后与德国马牌、邓禄普、3M、辉门达成深度合作，基本完成与国内主流零部件供应商的正品直供和合作授权。
◎2018年:途虎养车联合腾讯社交广告发布 《2018中国汽车用户线上养护行为报告》
途虎养车联合人民网成立汽车后市场·诚信服务联盟 途虎养车相继与埃克森美孚、圣戈班达成战略合作，途虎养车&将军轮胎旗舰店揭幕，线下智慧零售迈出全新一步。
