职位描述：

岗位职责：
1、负责美食小程序相关的平面设计，不限于专题活动、banner、活动版块的设计工作；
2、负责线下宣传物料设计工作，包括海报、易拉宝、礼品周边、宣传单的设计；
3、负责与运营、UI、前端合作，完成平面相关的设计工作。

任职条件：
1、有平面工作3年以上工作经验,熟悉Vi  海报、等视觉设计表现,熟悉设计基础，熟练使用PS、Ai等设计软件;
2、有绘画功底，可以提供插画作品；
3、具有较强的美感和色感、能够掌握互联网相关设计趋势；
4、熟练使用Ps、Ai、Ae等工具；
5、有团队精神，沟通能力佳，工作积极不埋怨；

企业介绍：

玖富数科集团成立于2006年，是一家以人工智能技术驱动的数字科技企业，获国家高新技术企业认证，属于科技类企业。2019年8月15日，玖富数科在美国纳斯达克证券交易所挂牌上市，股票代码“JFU”。
玖富数科集团作为集团公司不具体开展业务，负责集团战略规划与投资并购，旗下陆续成立了十多家独立全资或控股子公司，拥有数字科技、数字账户、数字普惠、数字财富、数字国际五大业务板块，各大业务板拥有自己的系列创新产品品牌，包括玖富科技、谛易科技、玖富普惠、玖富钱包、玖富万卡、万卡商城、悟空理财、玖行保险经纪、玖富咨询、玖富证券（香港）、玖富财富（香港）等，并控股或参股了保险经纪、基金销售、银行、融资租赁等多个具有牌照资质的机构。
经过13年的数字技术沉淀和生态链整合，玖富数科集团已成为行业内著名的独立第三方金融科技控股集团，围绕用户开展一站式便捷化的服务，以“连接器”的形式为金融机构、第三方场景合作伙伴提供专业化的数字金融科技解决方案。
在T.E（Tech Enabling）科技赋能战略下，玖富数科集团以“流量、数据、场景、风控、资金、技术”六大要素为核心，全面开放集团技术资源，通过数字科技、人工智能等技术，为大众用户创造数字普惠金融价值，让金融简单而美好！
