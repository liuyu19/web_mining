职位描述：

职责描述：1、负责相关需求分析和技术转换，协助、指导团队成员的开发工作，含技术问题的解决和项目质量把控；
2、并对需求文档和需求变更文档进行复审，分配系统设计任务，包括体系结构设计、模块设计、用户界面设计、数据库设计等，制定产品开发等计划；
3、负责系统设计和具体开发工作，并指导初中级开发人员，核心代码开发，平台或业务核心代码代码的具体实现，解决系统开发中的关键问题和技术难题；
4、按照公司项目技术开发规范、编写相应文档；
5、熟悉linux/win通用部署环境，能够快速分析和排查故障；

任职要求：1、五年及以上java研发经验，相关专业毕业者优先。
2、熟练掌握axure、xmind等工具
3、熟悉各种关系型数据库，如oracle、mysql、mssql、等。
4、熟练使用Spring/Springboot/Mybatis
/Maven/Junit等框架，熟悉产品设计到上线所有环节，包括需求分析、原型设计、需求设计、开发推进和产品迭代等
5、具有出色的抽象设计能力，思路清晰，善于思考，能独立分析和解决问题
6、具有一定的项目规划和管理能力，善于捕捉业务需求、架构设计中存在的问题，并给出有效的解决措施和方法；

企业介绍：

企业概况
人本集团创立于1991年，是中国轴承行业的排头兵,位列世界轴承行业十强之一，获得“中国机械工业100强企业”、“全国民营企业500强企业”、“全国重点行业效益十佳企业”。人本集团具有强大的技术研发实力，不断强化研发投入，人本集团技术中心是“国家级技术中心”、“国家技术创新示范企业”，拥有国家级“博士后科研工作站”等。人本集团致力于轴承制造、商业超市、机电贸易等产业发展，目前拥有员工20000余人。现已形成温州、杭州、上海、芜湖、无锡、黄石、南充等七大轴承生产基地， 2015年公司实现销售收入137.8亿元，完成利税8.02亿元。
轴承制造
人本集团是国家级高新技术企业，集团技术中心拥有一支轴承专业博士和高级工程师领衔的技术研发队伍，年平均开发新产品400余种，在滚动轴承振动噪声、密封、润滑、轴承寿命等方面的研究居于领先水平，并在轴承关键工艺、关键配件和技术研发方面取得了大量的应用成果，掌握和拥有了一批轴承制造核心技术，集团的产品技术水平、质量水平处于行业领先水平。人本集团积极参与国企改革，通过投资、并购、技改等方式，形成了温州、杭州、无锡、上海、南充、芜湖和黄石等七大轴承生产基地，现已拥有60多家轴承成品和配件制造企业，下属成员企业达到150多家。集团能生产内径1.5mm至4000mm范围内的轴承三万余种，年生产能力达8.19亿多套，在国内同行业总收入中名列前茅。
机电物流
人本集团实行代理制和连锁专卖的运营方式，已在全国中心城市成立了22家人本轴承销售总公司和100多家家轴承销售办事处和轴承专卖店，形成了轴承产品全国连锁网络，并在美国和日本设有贸易公司，以一流的产品和优质的服务，满足顾客不断发展的需求，与广大客户携手共赢。
商业超市
人本集团多业并举，稳健发展。1997年进军商业连锁超市行业。浙江人本超市公司开业至今，现已拥有1600多家连锁店，是浙南地区最大的商业超市公司。2004年被中国连锁经营协会评为“中国连锁业百强企业”，人本超市正以“人本精选店”、“十足平价店”和“人本批零中心”三种业态向浙闽两地迅速扩展。此外，人本集团在汽车零部件、精密机械设备、橡胶制品等行业保持良好的发展势头。
