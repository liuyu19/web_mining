职位描述：

职位描述
1. 负责提高浏览器内小游戏相关用户活跃、留存等指标；
2. 推进提升浏览器内小游戏&其他产品的用户体验：主动收集用户需求，归类分析并传达给产品同事；直接参与到产品需求设计、版本迭代、流失分析、制度制定等工作中，积极有效地推动产品改进体验
3. 负责从产品内外获取更多流量&用户，拓展产品的覆盖面与知名度；
4. 负责提升产品的整体收入，拓展更多变现形式，提升产品变现效率；
5. 负责与各个合作方进行日常沟通，完成合作接入，结算等事宜；
6. 数据研究与分析：通过观察日常数据、竞品分析、专项研究等方式发现产品潜力和增长点，跟踪分析各环节漏斗数据，诊断问题并推动优化方案落地；

职位要求：
1. 1-3年互联网从业经验，统招本科及以上学历；
2. 有良好的用户运营方法论，擅长通过数据分析、产业运营等方式提升用户活跃度；
3. 有较强的逻辑分析能力、沟通能力、跨部门合作和团队协作能力，主动学习与分享能力、快速适应能力，用于探索与坚持创新;
4. 英文水平可应付日常工作及交流;
5. 热爱手机游戏，对海外轻游戏行业有相关见解者优先；
