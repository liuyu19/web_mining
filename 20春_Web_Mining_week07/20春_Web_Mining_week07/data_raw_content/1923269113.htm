职位描述：

职责描述：
1、负责境外购物优惠信息功能设计和体验优化；
2、深入理解旅游用户的购物述求，能有清晰的用户购物需求框架，持续提升购物业务产品体验与功能认知；
3、对用户需求和行为有好奇心和同理心，能根据用户调研和数据分析进行产品判断；
4、收集与分析竞争产品、行业产品等相关信息，对境外购物的行业现状和潜在机会点有一定的了解和思考 ；
5、参与产品周期管理，与运营团队紧密合作，跟进和把控产品交互、设计、开发、测试、上线全流程，对用户体验和产品体验负责 。

任职要求：
1、有1～2年以上互联网产品经验，有大公司或购物社区产品工作经历优先；
2、对境外旅游感兴趣，资深境外购物用户优先；
3、注重细节，执行力强，具备严谨的工作态度和prd文档能力、强烈的责任心和团队精神；
4、 熟悉产品和项目的推进过程，包括需求分析、产品规划、功能设计、业务流程梳理、原型设计等；
5、具备良好的产品规划和逻辑思维能力，拥有优秀的沟通协调能力和团队协作能力。

企业介绍：

同程艺龙由同程旅游集团旗下的同程网络板块和艺龙旅行网在2017年底合并而成。
同程旅游集团是一个多元化旅游集团，中国领先的在线旅游服务商和一站式旅游预订平台，是“中国民营企业500强”和“中国旅游集团20强”企业。同程旅游集团下辖同程网络和同程控股两大板块，其中同程网络业务涵盖同程机票、酒店、火车票、汽车票等多项业务。
艺龙旅行网是中国领先的在线旅行服务提供商之一，深耕OTA行业多年，秉承专业团队的管理运作，在酒店等领域享有盛誉。
合并后，同程艺龙定位于领先的在线旅行服务平台，将同程网络和艺龙各自核心业务优势互补，并占有微信钱包中机票火车票的入口和酒店入口。目前同程艺龙已有近两亿的消费会员，2017年服务超过五亿人次，合并交易额超过千亿人民币，并已实现年超过五亿人民币的规模化盈利。
