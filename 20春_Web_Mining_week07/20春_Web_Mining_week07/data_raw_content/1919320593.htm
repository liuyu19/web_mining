职位描述：

职责描述：
1、负责各类系统基础服务的日常运行维护,配置信息更新;
2、负责各类系统数据备份数据监控、应急响应、故障排除、应急处理,保证服务运行稳定,高效;
3、负责各类系统基础服务的设计、配置、维护、监控、调优;
4、负责各类系统基础服务内部管理流程的建立、维护更新和落实执行;
5、在大用户量下高性能服务器系统部署方案的制定及实施;
6、保障服务器与数据库安全,检查并消除系统安全隐患;
7、关注开源社区动态,追踪前沿技术和应用。

任职要求：
1、本科学历及以上,计算机相关专业,LINUX系统管理3年以上相关经验;
2、强烈的责任心,做事认真细致;具备良好的团队合作精神,沟通能力良好,注重知识的分享与传递;
3、精通LINUX/FREEBSD系统安装、配置和管理;
4、精通Linux系统原理,熟悉文件系统调优、内核参数优化、系统安全加固等实施方案;
5、精通Nginx、Apache、PHP、Varnish、Memcache等开源软件的安装配置和维护;
6、精通Shell脚本编程 ( bash,perl,awk,sed,python,etc... );
7、具备批量、自动化管理服务器集群的能力,了解网络相关原理,了解TCP/IP协议;
8、了解Linux网络编程,具有C/PHP/Python编程能力优先。
