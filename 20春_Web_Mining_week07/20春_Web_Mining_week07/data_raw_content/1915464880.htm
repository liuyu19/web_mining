职位描述：

职责描述：

1、负责用户需求分析和竞品分析，持续优化产品的用户体验，推进产品市场化；

2、与研发同事协同，将NLP和AI技术应用于金融行业的产品设计，推进产品的持续功能迭代增强，打造金融行业有影响力的解决方案；

3、负责在产品开发过程中的设计确认、调整等协调工作，确保产品相应文档的编制合乎规 范，如介绍文档、需求说明书、用户操作手册等；


任职要求：
1、本科及以上学历，经济金融及其相关专业毕业，3-5年工作经验；

2、熟悉产品设计工具，熟练掌握 Axure、XMind、photoshop等，能够提供高质量的产品原型、流程图、 线框图等，清晰表达设计方案；

3、了解金融行业，对金融行业相关的业务需求能够做出合理判断；

4、优秀的沟通协作和项目管理能力，能够在复杂环境下灵活调整、强力执行，有很好的抗压力。

企业介绍：

     达观数据是一家专注于文本智能处理技术的国家高新技术企业，先后获得中国人工智能领域 “吴文俊人工智能奖”、中国青年创新创业大赛总冠军、ACM CIKM算法竞赛全球冠军、EMI Hackathon数据竞赛全球冠军、中国人工智能创新企业50强等众多荣誉资质。达观数据利用先进的自然语言处理、光学字符识别（OCR）、知识图谱等技术，为大型企业和政府机构提供机器人流程自动化（RPA）、文档智能审阅、垂直搜索、智能推荐、客户意见洞察等智能产品，让计算机代替人工完成业务流程自动化，大幅度提高企业效率。

     达观数据总部和研发中心位于上海，同时在北京、深圳、成都、西安等地设立产品和解决方案中心。团队由来自腾讯、盛大、百度、阿里等知名企业高管和技术专家组成，申请有六十余项发明专利，出版发表两本人工智能领域著作和数百篇技术论文。达观数据是微软加速器、联想之星、SAP创新营成员，获得宽带资本、软银赛富、真格基金等著名机构的数亿元投资。

     达观数据已与复旦大学、上海交通大学外国语学院等建立了文本挖掘联合研究实验室和语言智能实验室，并已成功服务招商银行、光大银行、浦发银行、中国平安、德勤、华为、中兴、中国移动、中国兵器集团、海尔、长虹、中国科协、国家知识产权局等数百家金融、制造、军工、科技等领域知名企业和政府机构。
