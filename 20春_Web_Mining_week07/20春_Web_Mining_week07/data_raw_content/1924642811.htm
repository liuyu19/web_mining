职位描述：

岗位职责：
1. 与公司客户保持良好业务关系，洞察客户业务需求方向，指导公司相应产品线战略方向。
2. 对负责产品线进行战略规划，构思产品并制定产品规划路线图。
3. 对竞争对手和竞品保持敏感，定期撰写竞品市场报告和功能洞察报告；
4. 了解市场需求，客户需求，从各个渠道收集产品反馈，管理产品的需求池；
5. 根据需求设计产品，管理开发过程中的需求质量；
6. 撰写需求文档与平台原型图，积极与开发团队沟通确保产品需求被准确理解；
7. 负责管理用户体验数据，跟踪产品上线后的产品运营，持续改善产品和用户体验。


岗位要求：
1.计算机相关专业本科以上学历，至少2年或以上互联网公司商业产品策划管理经验，并至少成功策划和运营一款互联网产品；
2. 了解人工智能行业产品发展动态及竞争对手的最新动态，有敏锐的市场洞察力，知识范围广阔；
3. 有深厚的产品设计思维，深刻理解软件生命周期管理，并对其中每一个环节的工作内容有自己的见解；
4. 具备熟练的数据分析，总结能力，能够通过各种运营数据的收集、归纳、分析对产品的设计和运营进行及时的调整及改进；
5. 有良好的沟通能力和反馈习惯，有良好的文档写作能力，口头沟通能力，良好的团队合作精神，良好的抽象思维，理性地做出技术决策，具有风险控制意识。

企业介绍：

图普科技成立于2014年，是一家专注人工智能图像识别技术的高新技术企业。公司以图像识别技术落地应用见长，依托先进的人工智能和计算机视觉技术，迅速落地包括互联网|融媒体图像智能审核、新零售 、泛安防三大领域应用落地实践。2018年4月图普科技凭借技术优势，在MegaFace全球百万人脸挑战测试赛中荣获全球第一的成绩。目前图普科技图像识别日均处理量在10亿次，累计处理超过数千亿次图像，在国内第三方独立识别云平台处于领先位置。
凭借稳定可靠的服务和超越用户期待的产品，赢得了包括迅雷、小米、美图、秒拍、映客、酷狗、爱回收、OPPO、名创优品、美的、金一珠宝、李宁等数百家知名企业和政企机构的稳定合作，是较早将人工智能成功商业化落地的企业，在人工智能领域独树一帜。

产品介绍
互联网AI云服务
图普科技利用自研AI核心技术，通过图片、文本、声音三个维度数据，对互联网平台内容进行审核、过滤，根据用户阅读习惯进行精准推荐，并根据平台运营场景提供定制化部署方案。

新零售BI解决方案
图普科技通过人工智能视觉技术，围绕“人、货、场”建立三位一体的售前数据模型，帮助门店实现运营数字化与消费洞察，辅助商品、运营、陈列、营销升级，从而提升销售业绩。

泛安防解决方案
基于人脸识别、图像识别等技术，广泛应用于校园、商业、工地、水务、公共安防等领域，满足人脸通行、可疑人员身份确认、实名制认证、重点人员布控告警等需求。
