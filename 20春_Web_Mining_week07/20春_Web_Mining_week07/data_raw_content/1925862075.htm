职位描述：

职责描述：

1、负责华为通用计算和异构计算硬件平台服务器或板卡硬件的设计交付，实现产品化，对产品质量负责；
2、主导完成产品开发和验证的全流程，包括方案设计、器件选型、原理图设计、产品的调试与测试，小批量试产指导等；
3、参与服务器产品规划和产品竞争力构建。


任职要求：
1、有产品硬件设计开发成功经验，熟悉ARM、X86架构者优先；
2、具有成熟的电子产品开发调试经验，能解决产品研发过程中遇到的问题；
3、理解产品的可靠性、可生产性、可维护性等要求，并能应用于产品设计中；
4、 具备熟练的英文读/写能力.
