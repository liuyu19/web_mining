职位描述：

岗位职责：
1.  负责国际MIUI媒体变现。能快速掌握流量特性，提供优化策略，为媒体变现效率负责；   
2.  结合客户需求和媒体属性设计广告产品，对接优质的预算，提高广告产品的变现能力；
3.  保证广告质量，平衡商业化和媒体的用户体验；
4.  定期调研、收集及评估用户需求，密切关注竞争对手，推动产品不断创新和完善；
5. 与公司兄弟部门/团队沟通协作，整合资源，推动相关系统、流量渠道的优化，制定广告主解决方案。
 
任职要求：   
1.  本科及以上学历；熟悉产品规划流程，有较强的数据分析能力；   
2.  3年以上商业变现产品工作经验，熟悉移动终端广告SDK产品特性及开发流程；   
3.  逻辑清晰，沟通能力和抗压能力强，乐意打磨细节；
4.  积极主动，有出色的沟通和协调能力，执行力强、团队意识强；
5. 有海外商业产品经验者优先；英语听说读写流利者优先；
