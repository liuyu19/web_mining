职位描述：

工作职责：
1. 结合人力资源业务诉求系统性的完成解决方案设计及对应的业务系统架构规划，并在此基础上负责人力资源信息系统的架构设计，技术演进和关键技术攻关落地工作
2. 通过新产品新技术推动人力资源业务发展
3. 负责制定相关技术管理规范及技术标准"
任职要求：
"1、 计算机相关专业硕士及以上，8年以上相关工作经验；
2、 精通分布式，高并发，高可用，高可扩展性企业信息系统的架构设计和开发；
3、 精通微服务架构及基于微服务架构的企业信息系统的架构设计和开发；有基于容器云构建云原生应用的丰富实践经验
4、 熟悉大型企业信息系统的架构、运作及管理；具备对大型企业信息系统的规划设计、业务需求分析及整合规划能力；
5、 熟悉Sql/NoSQL开发，熟悉常见大数据基础组件如Flume/Hadoop/Spark/Storm/Kafka等
6、 热爱技术工作，喜欢钻研技术，对新技术本身及新技术落地有强烈兴趣和敏感性，并乐于和团队分享，帮助团队技术成长；
7、 善于主动思考和自我驱动、具备较强执行力及抗压能力，敏锐的风险意识和良好的风险识别能力，有很强的独立工作能力和解决问题的能力；
8、 具有优秀的沟通技巧、团队合作经验、敬业精神和学习能力；具备很好的动手能力；
9、 具有大型人力资源IT系统架构设计及开发经验者优先；
10、 有大数据分析、AI相关领域实践经验者优先；"
