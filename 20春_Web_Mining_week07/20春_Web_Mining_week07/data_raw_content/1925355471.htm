职位描述：

资深交互设计师
工作职责
1.        负责京东高端客户“东家财富APP、京东金融APP东家板块”交互设计工作。
2.        参与产品规划、设计，用户研究，挖掘用户真实需求。
3.        为产品用户体验负责，追求极致的用户体验，跟踪实现效果。
 
工作要求
1.        工业设计、设计艺术、心理学、计算机等相关专业，本科及以上学历；
2.        5年以上工作经验，有基金、保险、信托、资管等相关经验优先。
3.        熟悉交互设计基本原理和方法，具备完备的知识体系，熟悉IOS、Android、微信小程序、Web等交互设计规范。善于使用各种交互设计工具，快速构建交互原型。
4.        具有较好的产品意识和用户同理心，有团队合作精神，有较强抗压能力，工作有热情和责任感，追求极致体验。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
