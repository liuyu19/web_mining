职位描述：

岗位职责：
1. 负责部门产品及项目后端的设计与开发，参与业务系统后台模块开发；
2. 负责内部工具、组件、基础服务的设计与开发；
3. 负责协同项目相关方确保跨团队项目落地；
4. 承担系统核心功能代码编写重点项目架构设计研发协助解决项目开发过程中的技术难题；
5. 积极了解业界发展、相关新技术，根据业务需求不断优化，持续改善和提高产品性能、灵活性及稳定性。

任职要求：
1. 5年以上开发经验，具备扎实的Java基础，对JVM原理有扎实的理解；
2. 对Spring、Mybatis等开源框架熟悉，熟悉jvm、web开发、缓存，分布式架构、消息中间件等核心技术；
3. 熟悉spring、springmvc springboot等框架开发,了解springcloud开发；
4. 熟悉常用的前端框架vue,jquery,rectjs等,并能够结合具体的业务场景,给出具体框架的使用建议；
5. 熟悉基于Mysql、mongodb数据库设计和开发、对数据库性能优化有丰富的经验；
6. 具备良好的识别和设计通用框架及模块的能力，熟悉底层中间件、分布式技术（如RPC框架、缓存、消息系统等），熟悉微服务架构；
7. 思路清晰，有良好的需求理解、分析、抽象能力和软件设计能力；对技术有强烈的兴趣，喜欢钻研，具有良好的学习能力，能快速接受和掌握新技术；
8. 具备良好的沟通技能能力，团队合作能力以及推进工作执行，能利用自身技术能力提升团队整体研发效率，提高团队影响力；
9. 有丰富Go或者Python开发经验者优先。
