职位描述：

职位描述：
1、进行产品全流程管理，包括制定产品规划、设计系统功能、收集用户/客户需求、进行市场和竞争分析、升级产品组合等；
2、规划产品发展路径，在项目推进过程中的跨部门协调沟通工作，并推进执行；
3、设计产品具体功能，组织协调其他部门高效完成产品开发；
4、深度研究竞争对手，把握产品和运营现状，判断机会和潜力点，切实有效地采取行动；
5、对业务运营给予支持，通过数据分析定位业务发展的问题，挖掘业务发展潜力；
6、根据机票产品市场及目标消费群体的需求，提出产品研发理念或参与提高已有的产品概念，掌控产品研发计划及产品研发的各种细节；
7、从整个产品研发的各个环节中，负责与开发人员、UI、市场等相关人员沟通协调 。

任职要求：
1、本科及以上学历，对国际机票有强烈的兴趣和爱好且敢于付出；
2、熟悉项目管理流程、文档撰写能力出色；
3、熟悉数据分析、效果评估方法；具备创造性思维、快速学习适应能力，对用户需求有很强的洞察力和判断力；
4、优秀的创新能力、钻研精神，出色的执行能力和沟通协调能力，及独立解决问题的能力；
5、对消费者有足够的理解和分析能力；

企业介绍：

同程艺龙由同程旅游集团旗下的同程网络板块和艺龙旅行网在2017年底合并而成。
同程旅游集团是一个多元化旅游集团，中国领先的在线旅游服务商和一站式旅游预订平台，是“中国民营企业500强”和“中国旅游集团20强”企业。同程旅游集团下辖同程网络和同程控股两大板块，其中同程网络业务涵盖同程机票、酒店、火车票、汽车票等多项业务。
艺龙旅行网是中国领先的在线旅行服务提供商之一，深耕OTA行业多年，秉承专业团队的管理运作，在酒店等领域享有盛誉。
合并后，同程艺龙定位于领先的在线旅行服务平台，将同程网络和艺龙各自核心业务优势互补，并占有微信钱包中机票火车票的入口和酒店入口。目前同程艺龙已有近两亿的消费会员，2017年服务超过五亿人次，合并交易额超过千亿人民币，并已实现年超过五亿人民币的规模化盈利。
