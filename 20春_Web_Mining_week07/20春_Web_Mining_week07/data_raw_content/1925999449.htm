职位描述：

【KLOOK招聘可选用远程视频面试】

【职位描述】：
1. 参与搜索/推荐服务的设计开发以及优化，和算法团队合作支持算法和机器学习模型的快速迭代
2. 改进搜索suggest的在线服务，实现更快更精准的autocomplete以及活动推荐
3. 开发优化并维护实时用户画像系统
4. 参与内容抓取等一些其它基础服务研发工作
6. 解决各种疑难杂症、核心代码编写、参与技术与业务产品平台和组件的沉淀
7. 推动优秀的软件工程方法、流程和规范在团队落地

【职位要求】：
1. 计算机或相关专业本科、硕士学历
2. 3年以上互联网开发架构设计工作经验
3. 对微服务架构、高性能低成本的后台架构有深入了解，有较强的技术规划和实现能力
4. 精通GO或者C++语言开发
5. 熟悉Elastic Search，有线上加载机器模型实现搜索结果排序优化经验优先
6. 熟悉搜索推荐系统以及用户画像系统的系统架构，具备实现及优化的相关经验
7. 了解AB测试框架开发，有框架开发的经验优先
8. 自我驱动力强，有强烈的事业心和责任感，结果导向，具备良好的协调推动能力和抗压能力

企业介绍：

【KLOOK是谁】
KLOOK客路旅行提供世界400个目的地的旅游体验服务，包括景点门票、一日游、独特体验、必吃美食、WiFi及当地交通等。KLOOK与超过10,000位供应商伙伴合作，提供超过100,000种活动及旅行服务。目前，KLOOK总融资额超35亿元人民币，获得创投机构的支持，包括红杉资本中国基金、软银愿景基金、经纬中国、高盛、博裕资本、TCV 、微光创投及华创资本等。


【KLOOK客路团队】
- 于2014年成立于香港，在香港（总部）、深圳（全球技术研发中心）、台北、新加坡、首尔、曼谷、吉隆坡、东京、马尼拉、胡志明市、雅加达、迪拜、孟买、宿务及阿姆斯特丹等地设有29个当地办公室；
- 国际化团队，超过2000名团队成员，来自超过30个国家及地区；
- 平台已汇聚500多万条真实的参与评论，帮助旅行者更轻松地选择的自由行活动；
- 支持41种货币支付及12种语言：英语、简中、繁中（港／台）、韩语、泰语、越南语、印尼语、意大利语等。

【KLOOK荣誉】
-  KLOOK App多次获得App Store和Google Play“年度精选App”荣誉；  
-  KLOOK提倡体验式旅行，上千位KLOOK旅行体验师深入全球，为用户甄选出最精华当地玩乐；
-  KlOOK为用户提供即时、简便的服务，包括会员专享价格，免排队通道以及其他特权。
