职位描述：

职责描述：
1.负责面向k12用户的教育类app的角色/场景/道具的一系列设定。
2.能够独立完成多角色多风格的设计整合，并熟练地把各个风格融入到教育类产品里。
3.高质量还原产品设计需求，并提出有建设性的设计想法和思路。
4.在高效协作的情况下，与其它用户体验设计师/产品经理/需求方进行无障碍的紧密合作。


任职要求：
1.五年以上原画设计工作经验，或教育类相关产品设计经验，有成功案例者优先；
2.熟悉主流卡通风格，对教育产品有一定的了解。
3.良好的沟通能力与团队协作能力 ，能够清楚地阐述自己的设计思路。
4.熟练掌握基本的手绘软件如photoshop等，有过矢量绘画经验者优先（ai，an等）。
5.加分项：熟练掌握后期软件（如ae）以及会3d制作者更优。
5.候选人请附作品。

企业介绍：

新东方教育科技集团（NYSE:EDU）由1993年11月16日成立的北京新东方学校发展壮大而来，集团以培训为核心，拥有满天星幼儿园、泡泡少儿教育、优能中学教育、基础英语培训、大学英语及考研培训、出国考试培训、多语种培训等多个培训体系，同时在基础教育、职业教育、教育研发、出国咨询、 文化产业等方面取得了骄人的成绩。作为中国著名私立教育机构，新东方教育科技集团于2006年9月7日在美国纽约证券交易所成功上市。
截至2017年5月31日，新东方已经在全国65座城市设立了77所学校、20家书店以及850多个学习中心。自成立以来，新东方累计面授学员超过3000万人次。
