职位描述：

职位需求
小米手机系统软件团队正在寻找一名可以在移动平台上设计并实现运动传感器功能的专家。
基本技能
- 至少5年的C / C++语言嵌入式系统编程经验；
- 对面向对象的算法设计深入掌握和理解；
- 在嵌入式系统中开发软件的经验；
- 拥有计算机方面的知识；
- 拥有诊断的技能和利用JTAG进行调试的经验。
 
工作职责
您将从事嵌入式运动传感器相关的软件开发(如加速度计、陀螺仪、罗盘）。
 
附加需求
嵌入式系统编程经验，运动传感器，信号处理、机器学习和衍生算法的相关经验，可以在一个协作环境中快速成长，有很强的沟通能力，对最终交付给用户的使用体验负责。
