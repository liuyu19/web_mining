职位描述：

一、能力要求

1. 负责景观项目初设、施工图阶段的图纸审图工作，把控与指导景观、水电、植物施工图纸质量；

2. 负责景观工程设计初设、施工图、现场配合阶段的技术咨询服务，配合设计师参与图纸会审、施工图交底会议；

3. 负责制定施工图技术标准和质量控制体系；

4. 定期对各项目施工图错误率进行统计和分析，收集、整理新的设计规范、施工技术，指导设计人员提高设计能力。

二、岗位要求

1. 园林景观相关专业本科以上学历；

2. 8年以上景观设计专业工作经历，3年以上审图岗位工作经验，参与过多项大型项目设计工作，有甲级院工作经历；

3. 精通景观类施工图制图技术及规范，熟悉南北植物，具备丰富的现场配合经验，对施工过程中发生的技术问题有高效判断和解决能力；

4. 具有良好的审图能力与专业协调能力，熟悉规范与流程 ；

5. 熟悉城规领域，手绘能力强；

6. 思路清晰，逻辑严谨，擅于分析和解决问题。

企业介绍：

杭州园林设计院股份有限公司始建于1952年，是全国首批甲级园林设计院，是风景园林和建筑设计双甲级、城市规划乙级单位，同时具备文物保护及维修、 工程咨询等资质。公司前身为杭州园林设计院，2002年改制成立了杭州园林设计院有限公司，并于2011年改制为民营股份制企业。
