职位描述：

职责描述：
1. 负责商场固化活动整体视觉的设计；
2. 负责促销、线下、线上宣传物料等相关设计；
3. 满足部门内日常设计需求。

任职要求：
1. 2年以上工作经验，具备综合类促销活动或多品类活动的设计经验；
2. 具备良好的美感判断，创新能力强；
3. 精通Photoshop，熟练使用Illustrator，Dreamweaver等设计软件；
4. 思路清晰、有较强沟通力、抗压力强，具备良好的团队工作能力；
5. 有美术功底，能够灵活控制设计风格，富有创造力和激情。
