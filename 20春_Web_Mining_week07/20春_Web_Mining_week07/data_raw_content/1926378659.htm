职位描述：

岗位职责
1. 实现超大规模多人在线游戏环境的服务器端研发；
2. 参与后台业务模块及框架的设计、研发及调优，并承担后台核心模块研发；
3. 参与游戏AI的研发工作；
4. 跟踪相关领域前沿科技进展，保持产品的领先性。
招聘要求
1. 校招（2020届毕业生）或者社招均可；
2. 扎实的计算机专业基础，熟悉常用的数据结构与算法；
3. 熟悉Linux开发，熟练掌握C/C++编程语言及其思想；
4. 熟悉网络通信，有开发经验者优先考虑；
5. 具备一定的分布式服务器、高性能系统设计等知识技能，有实际经验者优先考虑；
6. 有一定的产品意识，能够注重用户体验。

企业介绍：

寒武纪在全球智能芯片领域有十余年的积累，公司的使命是打造各类智能云服务器、智能终端以及智能机器人的核心处理器芯片，让机器更好地理解和服务人类。

公司创始人、执行官陈天石博士，在处理器架构和人工智能领域深耕十余年，是国内外学术界享有盛誉的杰出青年科学家，曾获国家自然科学基金委员会“优青”、CCF-Intel青年学者奖、中国计算机学会优秀博士论文奖等荣誉。公司骨干成员均毕业于国内外名校，具有丰富的芯片设计开发经验和人工智能研究经验，从事相关领域研发的平均时间达九年以上。

公司在2016年推出的“寒武纪 1A”处理器是世界终端人工智能专用处理器，已应用于数千万智能手机中，入选了第三届世界互联网大会评选的十五项“世界互联网领先科技成果”。公司在2018年推出的MLU100机器学习处理器芯片，运行主流智能算法时性能功耗比全面超越CPU和GPU。目前，寒武纪已与智能产业的各大上下游企业建立了良好的合作关系。
