工作职责:
"岗位职责（工作内容、职责、目标等）：
1. 负责现有接口自动化测试平台的维护，以及新功能开发；
2. 完善UI自动化测试框架

5. 辅助业务测试人员用自动化手段执行测试
6. 帮助团队提高测试技术水平，技术分享;


任职资格:
任职资格（学历、工作经验、专业技能等）：
1. 全日制计算机相关专业本科以上学历，3年以上测试开发经验；
2. 熟悉软件测试理论、测试流程、测试用例设计方法和测试相关技术;
3. 熟练使用Python、Shell脚本语言；
4. 熟悉Django开发框架；
5. 熟悉MySQL、Redis，熟练使用常用的SQL语句进行数据库操作；
6. 熟悉常用测试框架TestNG, Robot Framework, Selenium, Appium等;
7. 对软件测试有浓厚的兴趣和丰富的经验，有很强的分析能力和定位问题的能力；
8. 具备良好的沟通能力、团队合作和质量意识；
