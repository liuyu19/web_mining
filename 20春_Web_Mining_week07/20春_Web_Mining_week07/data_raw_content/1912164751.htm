职位描述：

岗位职责：
1. 负责SCADA产品的详细设计与研发。对SCADA产品的整体系统架构负责，对产品的系统安全性、稳定性、高并发性设计负责；
2. 负责对软件开发团队的技术指导；
任职要求：
1. 软件工程相关专业本科及以上学历,有良好的面向对象设计思想；
2. 6年以上C#开发经验，精通常用工业通信协议：MODBUS，CC-Link，ProfiNET等。精通常用OPC软件的使用，精通TCP编程，有SCADA软件二次开发经验。
3. 极强的文档撰写能力和设计能力，有良好的领域模型设计思想，精通UML设计。
4. 良好的编码习惯，精通.NET体系架构；
5. 了解最新的技术及发展趋向，网络知识经验丰富，懂得怎样衡量各种设计方法的利弊，懂得平衡各种开发局限的制约；
6. 喜欢新鲜事物，极强的自学能力，及时关注并跟踪技术(不限于微软）的发展变化，热衷于研究开源框架。有SuperScoket框架，NoSQL开发经验者优先。

企业介绍：
