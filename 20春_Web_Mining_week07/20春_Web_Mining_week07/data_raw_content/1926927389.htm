职位描述：

岗位职责：

1、负责Android平台上的应用开发工作；

2、负责优化客户端软件的模块结构和流程逻辑；

3、负责优化app的功能，提升app的性能；

4、承担关键疑难问题的解决和攻关。



任职要求：

1、本科及以上学历，计算机相关专业，具有5年以上Android应用开发经验；

2、具备较强的Android应用开发、调试、性能优化能力；

3、精通Android下网络通信机制，对Socket通信TCP/HTTP/HTTPS有深刻的理解，熟悉Sqlite数据库、熟悉Android多线程开发；

4、熟悉Android动画开发，能够按照UI要求开发动画特效；对计算机2D/3D图形库，UI框架，动画引擎有开发经验优先考虑；

5、熟悉Javascript，有React或 React Native开发经验优先考虑；

6、拥有良好的代码习惯，逻辑清晰；熟悉常见设计模式，能独立负责产品的架构规划和架构演进。
