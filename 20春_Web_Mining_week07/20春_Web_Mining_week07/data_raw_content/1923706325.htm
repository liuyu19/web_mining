职位描述：

职责描述：
1、负责分布式场景的权限系统设计与访问控制解决方案的竞争力构建；
2、负责分布式场景下，实现用户、应用程序、服务等的权限系统设计，保证在跨设备场景下的主体客体权限与访问控制一致性；
3、负责分布式权限与主机权限系统的融合，保证主机OS和分布式OS的权限与访问控制体验一致性；

任职要求：
1、具备较为丰富的分布式系统设计经验，尤其是分布式系统的安全架构设计经验；
2、具备较为丰富的服务化、微服务安全架构设计经验，熟悉分布式系统的IAM设计；
3、熟悉场景的权限系统模型如DAC、MAC、RBAC、MLS等，对ABAC访问控制设计经验者优先考虑；
4、具备零信任网络等设计经验。
专业技能：
1、掌握访问控制技术，对DAC、MAC、RBAC、ABAC、MLS等模型技术；
2、掌握服务化和微服务技术，对服务注册、服务访问控制等安全相关特性设计比较熟悉；
3、有大型分布式系统权限设计、IAM方案经验优先。
