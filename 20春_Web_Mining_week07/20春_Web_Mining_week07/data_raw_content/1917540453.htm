职位描述：

职责描述：


1. 负责游戏品牌的设计和规划，提升其品牌调性，并参与各业务品牌整合推广，广告创意等。如平面广告设计，H5，移动端广告设计，banner制作等。
2. 负责主KV、海报、大型活动视觉体系等日常的市场营销素材提供设计支持。
3. 企业CIS设计，延展等的提案及制作，制定视觉规范。
4. 负责网络及户外广告制作的监制指导；与对外执行团队的沟通及美术指导工作。
5. 上级安排的其他工作。


任职要求：

1. 从事设计工作4-5年以上，接受过正规美术/设计的教育和训练；有游戏行业，大型互联网行业或4A公司等相关经验者优先考虑。
2. 熟练使用premiere、3D、AE、PS、AI等软件，有手绘原创能力优先考虑。
3. 具有比较强的沟通与表达能力，善于视觉体系管理，有丰富的线上线下产品设计经验。
具有出色的品牌意识和整合传播能力，有广告创意、文案策划等相关工作经验具备出色的团队协作精神，良好的个人品行和团队合作精神，勤备敬业，能承受较大的工作压力。
