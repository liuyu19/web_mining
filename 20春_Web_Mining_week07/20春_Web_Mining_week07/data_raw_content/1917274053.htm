职位描述：

职责描述：
1. 以公司运营管理角度整合各部门信息化需求，与团队共同规划公司信息化发展目标、实施路线；
2. 负责相关信息化领域方向的研究，跟踪相应领域的业内创新，总结提炼业务管理、项目管理等方面的新管理思想及实践；
3. 深入了解公司内部业务环节和流程，结合行业内的实践经验，通过提供平台、整合系统、优化流程等方式制定解决方案，提高运营和协作效率；
4. 负责企业内部信息化管理系统的产品策划、项目推进、用户反馈及产品运营的全过程管理。


任职要求：
1. 本科以上学历计算机相关专业优先；
2. 具有5年或以上产品的规划及产品设计经验，熟悉企业信息化管理产品，有信息化线上产品设计经验者优先（如CMS、CRM、人力资源管理系统、财务清结算）；
3. 有独立的产品设计以及创新能力；熟练掌握原型设计、协同办公类软件；
4. 有较强的协调沟通能力，逻辑思维和数据分析能力；
5. 有移动版对接经验优先。
