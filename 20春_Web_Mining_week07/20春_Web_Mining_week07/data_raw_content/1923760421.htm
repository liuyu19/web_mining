职位描述：

岗位职责：

（1）参与售前，协助销售了解企业需求，配合撰写方案。

（2）售后实施：云产品的实施部署，与客户制定计划、沟通蓝图、配置系统、完成培训和上线推广

（3）参与产品功能及应用性测试，完善产品功能

（4）基于线上数据，开展多维度统计分析，为客户提供管理改进建议。

（5）配合公司需求制定运营计划，持续改善产品，提升用户体验，促进产品市场化

（6）总结、提炼实施管理经验并持续优化。

任职要求：

（1)本科以上学历，1年以上SaaS系统(MES/ERP/CRM/HR)实施经历，具备3个以上完整项目实施经验；

（2）有相关云产品、工业制造等方面工作经验者优先；

（3）口齿清晰，普通话流利，具备较强的学习能力和优秀的沟通能力；

（4）熟练运用office办公工具，有较强的表达能力和现场演示能力；

（5）工作认真、严谨、敬业，有团队合作精神，有较好的组织能力.

企业介绍：

宗申集团始创于1982，现已成长为一家总资产和销售收入均已突破200亿元人民币的大型集团企业。宗申制造工厂（或子公司）遍布在中国重庆、广东、江苏、河南、河北、天津和越南、泰国、巴西等地，正逐步成为“世界一流中小型动力系统集成服务商”。

宗申集团连续16年入选重庆工业50强，连续入选重庆市“纳税50强”，是中国民营企业500强、中国机械500强、中国制造500强，并入选中国企业500强。宗申拥有国家级“企业技术中心”、“机械工程专业博士后科研工作站”，宗申检测中心通过CNAS国家实验室认可。

宗申与意大利比亚乔（PIGGIO）、塞尔瓦（SELVA）、巴贝锐（BARBIERI）、白俄罗斯戈梅利等国际知名企业在中国设有合资工厂；宗申还与合作伙伴在上海、深圳、香港、加拿大、美国等地设有包括水上飞机运营、地产业务开发在内的合资运营公司。
