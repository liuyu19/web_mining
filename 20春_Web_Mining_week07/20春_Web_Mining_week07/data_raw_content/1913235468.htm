职位描述：

职责描述：
1）负责设备自动化、信息化项目的策划、设计开发、实施和管理；
2）负责自动化设备的外部调研、运行调试、运行维护和管理；
3）负责收集并整合内外部信息，促进生产线两化融合水平的提高。

任职要求：
1）硕士学历，自动化、信息化、机电一体化相关专业毕业；
2）具备自动化开发理论知识和能力，掌握单片机或PLC编程语言，有过自动化开发、可视化软件编程经验者优先；
3) 掌握制图软件，有一定的工程图纸识图基础；
4）喜欢钻研、锐意进取，具备英语读写交流能力。

企业介绍：

     公司是由国务院国资委直属央企中国钢研科技集团发起成立的国家高新技术企业，拥有国家级企业技术中心。公司以先进金属材料为主业，服务于战略性新兴产业，为全球高端客户提供先进金属材料、制品及解决方案。
     公司成立于1998年，2000年在深交所发行上市，股票代码000969。作为国内金属新材料行业领军企业，公司拥有体现核心竞争力的规模产业平台，优秀的协同复合多学科及产业拓展延伸能力，较强的国际国内品牌美誉度和知名度。公司总部位于中关村核心区，在全国拥有六大产业基地，1900亩产业用地，13家控股公司，总资产100亿元，员工总数6000人。
     公司成立以来持续稳步成长，连续入选“中关村科技园区海淀园经济二十强企业”，荣登“中关村Top100实力榜、创新榜”，被授予“中央企业先进集体”、“中央企业先进基层党组织”、“首都精神文明单位”等荣誉称号。
