职位描述：

岗位职责
1、语音识别等相关产品负责人，负责产品规划，根据市场需求，竞争态势和资源状况，收集、分析用户需求，深入把握产品核心价值，加强产品创新，确保产品功能满足用户的需求；
2、定义产品功能，负责产品需求的管理，收集、提炼、整理客户需求，将语音和对话技术在实际场景中获得落地，并且带给用户实用化的价值；
3、与市场、开发、测试、运营等多个部门进行充分沟通协调，保证版本功能的市场曝光与迭代高效
 
任职条件
1、本科以上学历，3年以上互联网产品经理经验，1年以上人工智能产品设计经验；
2、 精通语音识别工作全流程、熟悉机器人工作方式者优先；
3、有国际化产品设计经验者优先；
4、 较强的需求分析能力、沟通能力，团队合作能力，具备较强的抗压能力；
5、 严谨，能够输出高质量的产品产出物，并对UED设计提出指导性意见，可展示作品。

企业介绍：

达闼科技成立于2015年初， 目前有300余人。在北京、深圳、美国硅谷设有研发机构。2016年已完成A轮融资1亿美金。公司由国家“千人计划”首批特聘专家、中国移动研究院院长黄晓庆先生发起成立，他曾研发了世界第一套软交换系统。在中国移动领导了4G新一代移动通信技术、云计算和智能终端等诸多创新领域研发工作，提升了中国通信产业在国际上的话语权和影响力。

我们是谁？
1. 我们是一群完美主义的创新者，我们不仅提供服务，更为工作和生活提供灵感与信心；
2. 凭借丰富的行业理解，依托于可靠领先的云端机器人安全与智能技术，我们致力于服务金融，医疗及其他行业用户；
3. 打造移动信息化与智能互联网的云端机器人平台，与行业共同构筑充满想象的生态环境；
4. 致力于高品质的产品研发，聚焦全方位的客户体验，竭尽所能为客户创造价值；
5. 与全球顶尖的合作伙伴齐心协力构建强大易用的创新性产品，为客户提供企业级整体行业解决方案；

我们的愿景和使命是什么？
1. 愿景：我们不懈努力，在这里专注地创造人工智慧云端机器人；
2. 使命：我们设计与制造领先的云端智能机器人，让人们能够方便享受他们的服务；

我们现在在做什么？
1. 人工智能是未来高科技发展的制高点，我们从“终端、管道、云计算”的智能基础平台设施做起，为今后人工智能的产业链提供一个强大的生态环境；
2. 企业应用的智能终端市场是一个亿万级的空间，其中智能终端的安全性、SaaS应用的多样性有着很高的要求。我们现在从企业应用智能终端的角度切入，来满足巨大的市场需求；
3. 我们通过在“终端、管道、云计算”的智能基础平台上做业务的运营，来满足企业用户的需求，提升用户的体验；
4. 我们的团队来自于谷歌，三星，华为，思科，中国移动，IBM，索尼，小米，联想等这些顶尖的知名企业，走到一起，共创未来；
5. 我们的投资方来自于世界知名投资机构。在强大的资金支持下，加速了我们商业模式的快速实现及市场规模的迅速扩大；
6. 我们的管理模式是互联网化的管理模式，扁平化、人性化（平等、弹性办公等）、创新、高效是基本管理理念，在这里每一位同事都是企业的伙伴，每一位员工在这里都可以充分发挥聪明才智。
7. 我们的薪资待遇、办公环境与世界一流互联网公司看齐，骨干员工将有机会获得丰厚期权。期待有志之士加盟！未来机器人世界等你共同来创造！
