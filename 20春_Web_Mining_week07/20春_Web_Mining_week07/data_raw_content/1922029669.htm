职位描述：

工作职责:
1、参与酒店服务类项目，确定项目方向，引领项目进程，确保项目沟通、控制和验收；
2、制定服务流程及服务指标；
3、监控服务质量，提高客户满意度。
4、针对用户体验提升的需求进行服务策略的制定。
5、从服务产品角度协调业务及运营团队配合部门策略，高效推进酒店服务类项目实施以及服务产品推出，服务于大住宿事业部的总体战略目标的达成。

任职资格:
1、全日制本科及以上学历，1年以上服务运营/用户体验研究/项目运营类岗位经验
2、较强的行动力和项目推动力，良好的服务意识
3、良好的逻辑思维和逆向思维素养，较强的沟通表达能力。
4、熟悉SQL数据库，擅长数据分析，需求分析等。
5、有互联网或者酒店行业经验优先。
