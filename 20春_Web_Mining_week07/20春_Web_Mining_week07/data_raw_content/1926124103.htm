职位描述：

工作职责：
负责腾讯云官网平台以及国际平台产品的视觉设计，参与产品整体视觉风格定位，配合输出腾讯云平台相关视觉输出
参与产品的体验设计，挖掘典型用户场景、分析产品需求，优化产品的信息展示与流程等设计，对产品的可用性测试和评估提出改进方案，对产品的可用性、易用性负责；
关注用户反馈，收集产品需求，为产品新功能提供创意的设计想法；
参与产品设计过程中的评审，负责视觉、交互实现的走查，监督开发并提升产品视觉、交互的实现还原质量；
制定相关的产品视觉设计规范，沉淀产品视觉设计的方法论，共同提升团队的专业水准。
任职要求：
本科及以上学历，美术、视觉设计、工业设计类相关专业优先，或计算机等理工类专业毕业但具备较高的设计水平且有实际工作经验者；
5年左右相关UI设计经验，有能力独立负责较复杂系统的界面视觉设计且具有一定的交互、产品思维者优先；有B端平台设计经验优先；
对UI/UX设计理论、方法、流程等有深刻理解认识；有较高的审美水平，对流行趋势有敏锐洞察力；有一定的技术理解力，能和研发进行顺畅的沟通；
精通sketch、Photoshop、illustrator、等设计软件；熟悉C4D/AE等设计软件；
学习能力强，积极主动、关注细节、善于沟通合作，有较强的执行力和责任心，能承受高强度的工作压力；
懂云计算知识者优先。
