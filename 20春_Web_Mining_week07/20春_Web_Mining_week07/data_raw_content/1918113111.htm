职位描述：

1.基于小米金服的品牌定位负责小米钱包和小米金融的品牌运营设计；
2.负责小米钱包和小米金融常规的活动运营需求设计；
3.把握品牌方向，控制视觉动态的风格，符合品牌策略，跟踪把控项目的品质，达到设定的标准。

任职要求
1. 3-5年以上品牌活动运营设计的相关工作经验；
2. 擅长多种视觉表现手法，对于设计趋势有敏锐的洞察力，有着积极的学习能力；
3. 有金融、4A经历、插画、字体设计、品牌VI、懂文案、会动效者优先考虑。
