职位描述：

职责描述：
1、整体规划北汽集团的信息化系统应用架构、技术架构、数据架构、包括：整体业务应用规划、系统间
架构规划、集团及下属单位统一系统架构规划等;
2、参与基础设施（硬件部署）架构及安全架构设计;
3、制定架构设计的标准体系;
4、参与复杂系统的架构设计;
5、针对集团统建统管项目，进行系统架构设计;
6、进行架构及应用方面的新技术研究，并明确在北汽集团应用的可行性。
任职要求：
1、具有硕士以上学历，软件工程、计算机等相关专业
2、在汽车行业集团化企业IT架构规划经验4年以上；熟悉主流软件架构设计和应用，有大型销售系统、
制造系统架构设计经验。
3、具备以下IT专业知识
①深厚编程功底，并熟悉AOP、MVC、设计模式等应用框架，以及BS/CS、分布式、消息队列等应用设计；有JAVA开发经验者优先考虑
②熟悉应用系统的整体架构设计；熟悉系统群架构设计；包括系统间耦合性设计、实施步骤规划等；
熟悉集团级应用架构设计；包括为应对业务管控，不同业务单元采用的单独流程、数据独立存储等；
③熟悉云应用的架构设计；熟悉软件即服务模式的架构设计，并具有支持海量并发、便捷扩展应用的实际
设计经验；有在salesforce等云服务公司的架构设计经验者优先考虑；
④熟悉集团级数据架构设计；熟悉企业数据总线等；具有多系统间的数据交互设计经验；熟悉各类关系型
数据库，有深厚SQL开发经验者优先考虑。
4、具有国家级软件、计算及相关的职业资格认证
5、良好的逻辑思维能力、具有很强的分析问题及解决问题的能力，对解决具有挑战性的问题充满激情；
6、知识面广，思路开阔，创新能力强，对新技术持有敏感性并愿意致力于新技术的探索和研究。
7、沟通能力及专业能力强
8、具有研发、制造、销售、财务某一方面的深厚业务经验

企业介绍：

     北京汽车集团有限公司（简称“北汽集团”）是中国汽车行业重点骨干企业集团之一，总部位于北京市，经过50多年的不断成长，目前已发展成为以整车制造为核心，融汽车研发、零部件制造、汽车服务贸易、教育、金融和通用航空多业务发展的综合性、现代化、国际化的大型汽车企业集团，目前位列世界财富500强第137位，并在2015年“中国年度最佳雇主”评选中位列前十强，排名第4位，汽车企业第1位。
     北汽集团旗下拥有北京汽车、北京奔驰、北京现代、昌河汽车、北汽福田、北汽有限、北汽银翔等多家整车企业，品牌丰富，是中国汽车产品门类最为齐全的汽车集团之一。北汽集团还是具有新能源汽车三大核心部件开发能力，并完全实现自主生产的新能源汽车企业，已为市场提供新能源汽车产品40余款，产品种类与数量在国内均处于领先地位。同时，北汽集团通过与新西兰太平洋航空航天公司进行战略合作，正式进军通用航空产业。
     北汽集团产业链遍布北京、河北、山东、湖南、广东、江苏、云南等十余个省市，并在巴基斯坦、俄罗斯、印度等十余个国家建设了海外工厂。2015年，北汽集团销售汽车248.9万辆，营业收入3452.2亿元，已形成健康的产业结构布局和良好的发展态势。
     北汽集团秉承“行有道·达天下”的品牌理念，走规模化、高端化、服务化、国际化、低碳化的可持续发展之路，努力建设制造服务型企业和创新型企业，把北汽集团打造成为一个具有国际竞争力的汽车制造商和服务提供商，为追求幸福出行与高效运输的人们提供科技、安全、品质、环保的全方位解决方案，成为高品质美好出行生活的引领者。
