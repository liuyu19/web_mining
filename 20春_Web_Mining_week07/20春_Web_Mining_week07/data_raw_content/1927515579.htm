职位描述：

工作职责：
1. 负责区域市场营销工作，优秀的消费者理解能力，优秀的市场营销经验，结合业务目标制定区域市场营销规划，负责整合营销活动的策略制定、策划;
2. 敏锐关注市场竞品信息及行业发展趋势，熟悉业内创新的营销玩法，能够有效挖掘营销场景下的用户体验及传播价值;
3. 有较强的资源整合能力，数据分析及洞察能力，内部推动能力和解决问题的能力，能够按计划分步骤推动项目上线，并定期跟踪执行情况;
4. 深入挖掘合作伙伴需求，共同制定合作方案，积极应对变化，有较强的客户管理能力;
5.优秀的项目管理经验，能够独当一面独立带领项目策划和推进。
任职要求：
1、全日制本科及以上学历，具有5年以上大型营销活动的组织及策划经验，优秀的数据分析能力；
2、具备经营思维，市场洞见，框架性思维，有体系化运作能力以及创新意识；
3、具备优秀的市场开拓能力，能有效与客户关键人建立关系，有较丰富的行业客户资源者优先；
4、优秀的沟通能力，工作积极主动，能很好的进行跨部门协同，有独立推动大型项目落地的经验；
5、有500强快消行业，一线互联网公司市场背景；
6、能够适应短期出差，思路清晰，具良好理解力、表达及沟通能力，为人诚实、正直，性格开朗、责任心强，具有优秀的人际交往和公关能力。

关于我们

滴滴出行是一站式移动出行平台；为5.5亿用户提供出租车、快车、专车、豪华车、公交、代驾、企业级、共享单车、共享电单车、共享汽车、外卖等多元化的出行和运输服务。在滴滴平台，超过3100万车主及司机获得灵活的工作和收入机会。

滴滴出行与监管部门、出租车行业及社群积极协作，致力于以智慧交通创新解决全球交通、环保和就业挑战。滴滴也和汽车产业链企业建立广泛的合作，携手打造面向未来的汽车运营服务平台。

在全球范围内，滴滴与Grab、Lyft、Ola、99、Taxify、Careem构建的移动出行网络触达全球超过80%的人口、覆盖1000多座城市。目前，滴滴通过旗下99平台服务巴西，在墨西哥和澳洲提供滴滴品牌的出行业务；并在日本通过合资公司提供网约出租车服务。滴滴始终致力于提升用户体验，创造社会价值，建设安全、开放、可持续的移动出行新生态。
