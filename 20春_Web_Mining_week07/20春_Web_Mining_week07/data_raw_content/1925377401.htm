职位描述：

工作职责:
1. 负责媒体行业数字化运营系统整体规划和产品方案设计、产品推广和解决方案封装；
2. 参与和制定产品战略、市场调研及竞对分析
3. 完成产品调研及规划，提炼产品核心能力，并结合客户需求调研，调整系统方案；
4. 撰写详细的产品需求文档及原型设计文档，协调设计部门资源完成UI设计工作；
5. 与客户、销售、项目团队协同工作，对于重点市场，协同各部门联合研发，对产品落地负责；
6. 把控产品推进进度、协调突发问题与质量管理工作；
7. 配合项目经理编制项目实施计划，严格执行项目计划的任务；

任职资格:
1. 大学本科及以上学历，有3年以上ToB产品工作经历，完整负责过至少一个ToB产品的落地
2. 熟悉产品开发的流程、包括用户调研、需求分析、产品分析定位、核心功能点规划，以及系统上线后的数据采集与分析；
3. 能够熟练使用Visio、Axure、墨刀、Sketch、Briefs等软件工具设计系统原型和流程图；
4. 责任心强、跨团队合作能力强，具备良好的沟通协调和自我驱动能力，能承担一定压力
5. 有一定计算机功底，熟悉业界大数据公司和产品，了解基本的机器学习知识者优先
6. 有内容运营、BI相关产品或项目经验者优先；
7. 国内一线互联网公司或企业级服务公司从业者优先；
