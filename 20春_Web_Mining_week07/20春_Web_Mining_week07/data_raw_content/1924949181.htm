职位描述：

职位职责：
1. 负责清河大学在线跨平台产品的界面设计和整体风格把控
2. 参与产品设计讨论，协助产品和研发团队进行易用性和体验改进
3. 根据产品运营需求设计活动所需素材和页面，输出高质量设计作品
4. 基于品牌和产品定位，并结合用户研究，提出视觉风格创意
5. 总结和分享设计经验，推动专业能力建设

任职要求：
1. 本科或以上学历，设计类相关专业
2. 3-5年行业工作经验
3. 有品味，有想法，善于思考，能设计出高水平的产品界面
4. 善于创新，有敏锐的色彩感觉和出色的产品界面分析能力
5. 细心耐心，做事考虑全面，善于把关各种设计细节
6. 态度积极，抗压能力强，具有良好的团队协作能力
