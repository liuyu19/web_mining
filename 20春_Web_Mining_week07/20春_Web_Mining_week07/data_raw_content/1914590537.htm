职位描述：

职责描述：
1. 带领团队管理部门日常相关创意设计工作，确保创意的执行品质；
2. 负责贯彻执行美术创意细节，管理项目的质量和进度；
3. 协助设计总监和其它部门进行良好的协调与沟通，确保高品质、及时地达成预定的任务目标；
4. 按照公司需求完成品牌视觉管理、对内对外的广告宣传、准确表达公司的意图、使之在受众范围内引起共鸣，完成广告宣传任务；
5. 与广告公司协作，开展企业新产品推广、市场开拓、广告创意制作、广告发布、产品促销等市场营销策划活动。配合营销中心各部门开展营销策划、推广工作。
6. 具有线下活动及展会设计和制作经验，能够把控展会制作进度和执行品质。

任职要求：
1. 大专及以上美术设计、艺术设计、视觉传达,传播等相关专业毕业，掌握平面设计相关所有软件；（有线下活动经验优先）
2. 8年以上品牌公司或4A广告公司从业经验；
3. 有较强的创意发想、创意规划和执行能力；
4. 在设计领域内具备丰富的知识和经验，对专业问题有敏锐的判断，能够熟练、有效的解决专业问题；
5. 美术功底扎实，审美意识和艺术修养思维开阔想法独到，手绘功底深厚；有良好的文字解析力
6. 具有团队管理能力，有团队意识，能承受较强的工作压力；为人正直、工作踏实、有进取、敬业精神。

企业介绍：

【我们是谁】
      我们是喜马拉雅，国家高度重视的新经济龙头企业、互联网行业独角兽。打造国内领先的音频分享平台喜马拉雅，我们拥有海量6亿手机用户，和超5000万海外用户，是中国文化出海的中坚阵地。
【我们的使命】
      用声音分享人类智慧，用声音服务美好生活。我们首创内容生态PUGC，引导行业潮流创新，吸引大咖主播，玩转声音多姿彩，投身音频内容创业。频道覆盖多又广，包含财经、音乐、新闻、商业、小说、汽车等328类过亿有声内容。
【我们的生态】
      秉承万物有声的理念，将有声内容渗透到生活中各个智能终端和场景中。2017年6月，推出国内仅有的全内容智能AI音箱——小雅，布局人工智能领域，打造物联网时代信息获取最便捷的入口。让用户把需求大声说出来，开启了一个语音交互的新传播时代，让声音和知识像水和电一样无处不在，随取随用。
【合作大咖】
