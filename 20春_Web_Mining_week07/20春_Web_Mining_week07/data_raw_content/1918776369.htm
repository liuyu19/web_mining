职位描述：

岗位职责
1. 负责品类销售额和销量等重点指标的制定达成；
2. 负责品类活动的规划和基础工作运营；
3. 品类销售运营各类异常情况的协调和处理；
4. 负责品类的行业市场研究、竞品跟踪分析，市场调研，建立符合公司自身产品发展的需求文档；
5. 与横向相关部门协同合作，提升用户体验，并逐渐形成标准。

岗位要求
1. 统招本科及以上学历；
2. 三年以上互联网电商行业工作背景，具有家电、3C类目品类行业工作经验；
3. 能够跟进工作要求和内容，独立梳理业务流程，制定相关工作的KPI；
4. 理解电商品类的划分和运营方法，能独立完成各项任务；
5. 具备数据整理和分析能力，能快速整理数据，输出结论并提解决方案；
6. 有较强的商务谈判能力，能够处理复杂业务关系，处理事务耐心细致、责任心强、沟通及协同能力强。
