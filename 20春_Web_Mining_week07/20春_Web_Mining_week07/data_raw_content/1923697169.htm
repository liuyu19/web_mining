职位描述：

工作职责：
1. 负责携程App端、Online端界面设计；
2. 研究全球用户旅行习惯，提升用户体验；
3. 具备产品思维，对产品提出合理建议，并有能力制定设计规范；

任职要求：
1. 本科及以上学历。艺术设计、平面设计、数字媒体等设计相关教育背景；
2. 1年及以上相关工作经验， 有国外工作经验优先考虑；
3. 具有非常良好的个人品味以及扎实的美术功底，擅长手绘，热爱设计，对时下流行的国际设计潮流有很高的敏感度；
4. 精通并熟练使用Sketch、Photoshop、Illustrator等设计软件，会AE及C4D者优先；
5. 逻辑清晰、沟通力良好，富有团队协作精神，责任感强；
6.欢迎有海外生活或学习经验或外籍设计师投递。
