职位描述：

岗位职责：
1、从用户需求、竞品跟踪、业务模式创新、新技术红利等不同角度，研究探索新产品方向。主导确定产品的方向，并驱动产品整个生命周期完成
2、负责整体产品规划，及关键方案设计，挖掘用户需求，不断改善用户体验；
3、能有效协调组织设计、开发、测试、运营等各部门人员各类资源达成产品目标；
4、持续分析和评估自身产品、整体行业、竞争对手等运营数据，基于数据制定产品规划，寻找优化和突破方向。

岗位要求：
1、7年以上产品工作经验，有过成功创新产品（互动娱乐或流量变现）经验者优先；
2、 战略规划能力，学习能力强，快速熟悉某个领域，洞察行业趋势和本质；
3、对互联网和所在行业发展，有前瞻性、战略性思考，敏锐的市场嗅觉；
4、有社交类和社区类产品经验优先，有一年以上创业公司工作经验者优先
