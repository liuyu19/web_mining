职位描述：

岗位职责：
1.负责PC端的前端页面及业务逻辑开发工作
2.与UI设计师沟通交互、视觉设计的相关问题
3.与后台工程师制定技术实现方案、接口服务等
4.对具体的项目进行性能优化，解决前端遇到的各种技术、性能、兼容问题
5.在理解产品业务的基础上，提升产品的用户体验
岗位要求：
1. 精通js语言和ES6，熟悉node.js
2. 精通vue框架和常用插件（vue-router、vuex、axios），精通webpack配置、vue cli配置
3. 熟悉elementUI和组件扩展
4. 熟悉前端性能优化，有项目优化经验
5. 有vue cli、elementUI使用经验优先
6. 良好的沟通和学习能力
