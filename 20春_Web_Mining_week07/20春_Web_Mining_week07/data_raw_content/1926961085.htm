职责描述：

1. 负责DJI互联网领域Android端产品的设计开发工作；
2. 负责基础服务组件的研发和维护工作；
3. 负责研究Android研发领域的新框架或技术并加以利用；
4. 参与并带领团队进行产品的设计、方案设计和可测性优化和技术改进。



任职要求：
1. 5年以上Android客户端开发经验；
2. 有扎实的JAVA语言基础，熟悉JNI/NDK，具备较强的编程能力素质、质量意识；
3. 熟悉Android开发技术，包括UI、网络等方面；
4. 熟悉Android开发工具和相关开发测试工具的使用；
5. 精通多线程和网络编程，对高性能程序设计、架构有较多的工程经验；
6. 具备音视频类软件开发经验者和团队管理经验者优先考虑。

企业介绍：

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，办公室设于北京、上海、西安、香港、东京、洛杉矶、旧金山、鹿特丹、法兰克福等全球18个国家及地区。

大疆是一家致力于持续推动人类进步的科技公司，坚持做空间智能时代的开拓者，不断为世界创造超越想象的科技之美。2006年创立至今，大疆在全球的员工人数已超过14,000人，客户遍布全球二百余个国家和地区。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十三年间，大疆不断革新技术和产品，开启了全球“天地一体”影像新时代；大疆的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。
