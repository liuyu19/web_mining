职位描述：

工作职责：
负责钉钉统一通讯服务后台的设计和研发工作，交付高质量的音视频，多媒体产品服务： 1. 负责视频会议系统（视频会议，语音电话，直播，办公电话等）系统的研发工作； 2. 负责语音通话（VoIP）服务的设计与研发工作； 3. 负责直播系统系统，以及基于直播、短视频、语音图文的内容系统的研发； 4. 负责办公电话，电话会议系统设计与研发。
任职要求：
1. 计算机或相关专业， 五年以上计算机相关工作经验，java/c++基础扎实； 2. 有丰富的linux下开发经验，具备高并发分布式系统研发经验，有丰富的线上系统经验； 3. 较强的工作责任心和良好的沟通协调能力，能在压力下独立解决问题。有创业之心，用户第一，具备良好的自驱力； 4. 有多媒体，音视频系统研发经验者优 5. 有高活跃用户产品研发经验者优先 6. 熟悉内容平台，算法推荐系统架构的优先
