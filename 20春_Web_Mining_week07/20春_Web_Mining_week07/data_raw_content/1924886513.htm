职位描述：

岗位职责:
1、负责助贷金融产品的产品规划和方案设计；
2、负责助贷产品的创新和预研，完成产品的用户需求定义、产品功能/UI/交互的设计；
3、负责制定项目开发计划并跟踪进度，确保项目如期完成；
4、撰写详细的产品需求文档及原型设计文档，跟踪产品研发进度；
5、负责把控产品或项目的推进进度、协调突发问题与质量管理工作。
任职要求：
1、本科及以上学历，3年以上互联网金融产品经验，曾独立负责过产品版本的迭代；
2、熟悉信贷业务金融产品后台系统的调研、规划、设计、运营、优化等全流程工作环节；
3、拥有清晰的产品设计思路，对信贷产品和流程有全面了解和独到认识，拥有整体把握产品的能力；
4、具备良好的专业背景、逻辑能力好，有较强的创新思维、执行力和沟通能力；
5、能够输出高质量的产品原型和需求文档；
6、热爱互联网，具有互联网思维，熟悉并经常使用相关各类产品。
