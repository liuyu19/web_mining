职位描述：

工作职责：
我们需要行业顶尖水准的体验设计专家，负责智能 APP 及带屏类智能产品（带屏智能音箱、车机、电视/投影仪、一体机等）的系统用户体验设计工作，产出最具行业领先性的设计解决方案，定义新的领域标准。
任职要求：
1. 四年以上的设计工作经验，体验设计能力突出，有成功案例/代表作品； 2. 较好的用户视角、价值导向意识、体验设计思维，对人机交互领域与产业技术有深刻认知； 3. 有激情、创新欲、责任感和自主性，具备良好的跨职能团队合作能力； 4. 对设计趋势及潜在价值有敏锐的洞察力，能在产品初期综合运用相关体验设计方法推动创新方案落地，有成功案例； 5. 具备系统性与框架性思维，有系统级产品设计能力，能通过建立、优化规范和机制，保障落地品质。 ​​ 一些加分项 1. 有 智能语音／ IoT／智能软硬件／智能车机 / 跨端跨平台声屏交互 相关产品设计经验； 2. 软硬件一体化交互体验设计经验； 3. 对设计系统有深入研究。
