职位描述：

工作职责：
负责游戏系统UI交互设计，把控游戏整体UI交互;
负责游戏交互总结及系统设计，跟进系统功能开发和执行;
负责游戏系统交互资源跟进和维护，并对游戏最终交互体验负责。
任职要求：
本科及以上学历，逻辑严谨，责任心强；
两年或以上游戏交互设计/视觉设计工作经验，熟悉主流交互设计工具；
优秀的UI交互设计能力，沟通能力和推进执行力，有系统策划经验者为佳；
经历过实际的游戏项目开发，并主导过游戏重要功能的交互设计及推进开发；
熟悉游戏行业，有休闲类手游UI交互设计经验优先。
