职位描述：

岗位职责
1. 参与设计结合公司产品符合市场需求的课程，包括设计产品的案例、教学方案设计、教学视频内容等；
2. 参与教研工作，包括课件制作、教案编写、视频录制等；
3. 参与国内外课程编写合作项目，并对课程内容与质量进行评估；
4. 参与授课后的监控与教学反馈，不断优化课程内容，提升课程质量。

任职要求
1. 本科及以上学历，软件工程和计算机专业优先；
2. 热爱机器人STEAM教育编程，有教育行业培训、教研等经验优先；
3. 有一定的用户体验意识，接触过STEAM教育产品的课堂使用场景优先；
4. 良好的沟通表达能力，较强的学习能力，能短时间内了解各科课程的专业知识特点；
5. 能根据产品的特点与公司的课程理念设计出完整的1节课程，有课程作品优先。
