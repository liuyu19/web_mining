职位描述：

职责描述：
1、有两年及以上生活类电商新媒体内容运营经验，熟悉互联网传播机制，善于挖掘并运用各类传播手段提升数据；
2、有一定的文字功底，反应迅速且对热点极度敏感，并能与产品结合策划内容和活动；
3、了解养宠用户人群，喜爱宠物并对宠物有一定的了解；
4、沟通能力强，有跨部门合作经验；
5、热爱电商行业，有激情、有责任心、抗压能力强。



任职要求：
1、在用户运营矩阵中负责微信服务号的内容策划、创作及运营，对GMV负责；
2、负责用户运营组微信服务号用户互动的内容设计及执行；
3、对平台内容运营进行数据追踪，并根据反馈结果优化运营策略；
4、对其他内容平台进行研究，定期给出竞品分析报告。
