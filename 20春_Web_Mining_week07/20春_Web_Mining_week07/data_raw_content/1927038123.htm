职位描述：

工作职责：
负责参与产品的前期视觉用户研究、设计流行趋势分析；

任职要求：
5年以上手游UI相关经验，了解多类型游戏UI设计风格，负责游戏美术视觉设计工作，制定游戏UI的制作规范和标准，把控产品UI风格，能够独立完成界面，图标，外宣设计。负责广告，图标的外包质量审核及把控外包进度，保持风格统一性和资源规范性.
本科以上学历，设计先关专业，熟练掌握Photoshop，illustartor等图像设计软件，有深厚的美术功底，色彩感觉敏锐，想象力丰富，有独特的审美能力
交互设计经验丰富，逻辑思维及分析能力强，有较强的判断能力和项目推动力并有强烈的责任感。
