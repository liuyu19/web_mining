职位描述：

职责描述：
1、负责所属展陈项目及空间设计类的方案设计工作；
2、把控空间设计的创意方向、水准及进度；
3、评估并改进设计方案、把握设计质量、具有可操作性；
4、设计成本控制，不断改进设计水平；
5、负责空间项目设计工作进度及效率、质量把控；
6、解决设计难点、参与工艺、材料改进；
7、负责所属项目中设计突发事件的解决；
8、完成领导交办的其他临时性工作。
任职要求：
1、广告、美术专业相关大学专科及以上学历；
2、熟练使用CAD、3DS MAX、PS等绘图软件；
3、5年以上工作经验，多个中标项目经验，展示馆、展厅、博物馆、企业馆等相关项目经验；
4、熟悉创意策略和视觉设计的表现与执行；
5、熟悉展览行业的工作内容和方法；
6、具有专业美术素养、思考领悟能力、沟通协调能力。

企业介绍：

景域驴妈妈集团为中国领先的旅游产业链一站式服务生态圈企业，由驴妈妈集团、景域旅游投资集团、景域旅游科技创新中心三大板块构成，已连续七年进入“中国旅游集团20强”，入选“2018中国服务业企业500强”，被评为国家高新技术企业、贸易型总部企业。
驴妈妈集团由驴妈妈旅游网、驴妈妈目的地运营集团和奇创智慧旅游集团三大子集团构成，全面覆盖了当下旅游业发展所需要的三个核心驱动力：互联网渠道能力、目的地运营能力和智慧服务能力。
驴妈妈旅游网是中国知名的综合性旅游服务平台，具备“线上OTA服务+线下超千家门店服务”的360度旅游全服务能力，在景区门票、周边游、定制游等品类处于行业领先地位，已成为全球门票领军者、中国排名前列的自助游资讯及预订平台、自助游领军品牌和白领喜爱的旅游品牌，入选2018中国互联网100强。
驴妈妈目的地运营集团由多家全资子公司和控股、参股公司组成，专注旅游资源的运营管理。在国家大力发展“全域旅游”的背景下，目的地运营集团将通过特许运营、委托管理、合资公司及投建运一体化等多种形式，重点打造500家优质旅游目的地，成为核心旅游资源的供应商，为全国乃至全世界旅游景区、酒店及目的地创造更大价值。
奇创智慧旅游集团成立于2004年，是中国旅游实效咨询的领军企业与旅游IP创意落地的实践者，是驴妈妈集团核心的智慧中枢与IP孵化器。深耕中国文旅行业十五载，奇创以战略咨询，策划创意、体验设计、产品开发、IP文创、内容营销、智慧科技等能力，开创从系统规划到科学运营的全生命周期服务，累计完成2000多个品牌项目，足迹遍及全国，为众多政府、开发商、景区挖掘并创造可持续的价值增值，以智慧势能为传奇而创造！
景域旅游投资集团是集景区、景点投资，旅游综合体项目打造，实施国家乡村振兴战略，旅游新业态和商业模式创新，旅游产业基金设立和管理，融资租赁金融服务以及旅游IP投资运营等为一体的国内知名旅游投资集团，以股权投资为载体，以投资、建设、运营一体化为核心业务，通过旅游内容丰富和提升，文旅产业融合与升级，金融和资本赋能，IP创新和植入，为游客提供美景、美游、美食、美购、美宿和场景体验式美好生活方式。下设景域旅投、景域文旅、景域金融、景域资本和景域酒店集团五大核心业务板块，拥有帐篷客、歌璞、歌遥、歌笙、狐巴巴冒险园、亲密花园、虹之汤等多个IP品牌，是中国知名的旅游IP投资运营商。
景域旅游科技创新中心旗下拥有旅游科技应用平台，是面向全球市场的旅游科技应用服务平台，分为WEB、PARK、MEDIA、FORUM、ASSOCIATION等5个模块，融交易、内容、社交、企业孵化、社群运营为一体，线上线下互动，依托母集团生态圈优势，搭建科技行业与旅游行业更为快捷、高效的对接桥梁。旗下的景域智能科技公司参与多省市智慧项目建设，为智慧城市、智慧目的地等供给侧改革带来新的探索，为5G时代下文旅智慧化融合开创旅游新业态。
十多年来，驴妈妈创始人、景域集团董事长洪清华率领全体员工恪守“诚信、激情、创新、多赢”的经营理念，奉行“一个家庭 一支军队 一所学校”的管理理念，坚持以客户为中心，不断践行“愉悦你心”的企业使命。
