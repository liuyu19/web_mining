职位描述：

职责描述：
1、配合公司要求完成门店各类项目平面设计工作。
2、能充分理解创意意图并准确体现于创意设计中。
3、配合完成项目的设计与实施。
4、活动现场的执行、沟通、跟进。
任职要求：
1、本科以上学历，广告视觉、平面设计专业。
2、具有平面创意设计公司等有平面设计任职或实习经验者优先考虑。
3、精通使用Photoshop、illustrator、Coreldrew等设计软件。
4、有海报、包装设计，画册及宣传印刷品设计、VI创意设计、网站UI设计的经验者优先。
5、具有良好的视觉品味，设计理念、形象及色彩表现力突出。
6、工作主动性强，有责任心，善于沟通，个人乐观积极。能清晰阐述设计思路和理念，有较强的理解和分析能力。
7、应届毕业生也可，接受实习生，表现优秀者可转正。
8、面试时请携带作品进行演示。
