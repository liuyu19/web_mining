职位描述：

岗位职责:
1、负责公司可穿戴智能产品的研发工作，制定研发发展规划和具体的研发方案、计划，并推动实施落地；
2、负责公司产品设计、测试和标准化等的技术标准、指导文件的制订；
3、关注业界新技术和新产品的动态；负责可穿戴市场的调研、行业竞争对手的分析、主流可穿戴产品的解读，通过与用户的深度交互等方式分析提炼用户需求；负责产品设计的变更、评审及修改工作，及时满足市场需求；
4、协调、处理设计和生产过程中出现的问题，为产品小批量试制、量产和产品售后服务提供完善的技术支持；
5、负责可穿戴产品的检测认证，确保新产品通过国家电子产品质量检测相关标准，及其他相关的检测认证程序；
6、结合互联网营销模式，协助运营团队完成新产品开发的全流程的用户互动与产品运营。
任职资格:
1、学历要求：本科及以上学历，电子工程、计算机软件/硬件等相关专业；
2、工作经验：5年以上可穿戴产品开发经验；拥有行业上下游资源（包括新技术、元器件/模组、产品方案、软件开发、生产代工等）者优先考虑；
3、知识要求：熟悉研发流程、嵌入式软件开发流程和方法、检测流程和方法，熟练使用行业常用软件和工具，有扎实的模拟、数字电路基础，有较强的电路分析及解决问题的能力；
4、具有医学、中医或营养保健方面的知识和经验者优先考虑；
5、具有量子物理学常识着优先考虑；
6、其他要求：具有良好的互联网思维，在智能产品软硬件创意方面有较高的敏锐度；较强的分析能力、沟通能力和解决问题的能力。

企业介绍：

天士力控股集团于1994年5月成立，2002年8月，集团所属核心企业——天士力制药股份有限公司上市。公司成立十五年来，坚持自主创新，走新型工业化的发展道路，全力打造大健康产业第一品牌，全面推进国际化。形成了以“生命安全保障产业”为主线，包括现代中药、化学药、生物药、特色医疗等产业；以“生命健康需求产业”为拓展，涵盖保健品、化妆品、健康食品、安全饮用水等领域的高科技跨国企业集团。
