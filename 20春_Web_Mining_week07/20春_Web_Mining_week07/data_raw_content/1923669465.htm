职位描述：

岗位描述：
1、负责浏览器的产品方向规划，能结合战略目标制定产品阶段性的重点和策略，为整体客户端的用户活跃度负责；
2、对C端用户敏感，通过设计产品整体框架，让不同用户可以更快捷消费合适的内容，并保证用户的产品使用体验；
3、制定版本节奏，协调开发人员等相关角色推动项目整体进展
4、负责互三南京分部产品横向职能，包括参与南京已落地产品的规划、目标对齐、项目推进等，承担业务指标

岗位要求：
1、有3年以上C端产品经验，独立负责过百万级体量产品经验优先，有信息流、浏览器产品经验优先；
2、热爱互联网产品，对互联网行业动态有自己的见解，能结合产品特质挖掘潜在用户需求；
3、善于运用交互设计原则、用户行为分析等方法完成可用性高的产品设计，对细节关注；
4、逻辑清晰，沟通及推动能力强。
5、有团队管理经验优先
