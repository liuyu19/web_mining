职位描述：

职位描述

1. 负责游戏客户端引擎功能研发及美术工具开发。

2. 负责游戏客户端的性能优化及资源优化。

3. 协助美术持续改善制作流程，优化开发效率。

4. 协助策划将相关游戏玩法在引擎中落地。

职位要求

1. 掌握C++，熟悉至少一门脚本语言如Python；

2. 熟悉Unreal4引擎程序开发，掌握通过引擎进行游戏玩法开发，有次世代项目经验积累；

3. 在渲染、动画、材质、UI、AI、美术工具构建等任一方向有具体实践或专精；

4. 有3年以上的客户端开发经验者优先考虑；

5. 乐于沟通合作，有团队合作精神，可以主动的推进业务；

6. 热爱游戏，对效果和体验有追求，有游戏经验者优先；

7. 可承受一定强度的工作压力。
