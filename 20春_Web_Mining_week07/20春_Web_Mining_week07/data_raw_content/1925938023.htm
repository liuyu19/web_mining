职位描述：

职责描述：
1、组织对接项目收集建筑设计资料，拟订建筑设计计划，编写专业设计调研报告，组织专业调研工作；
2、参与各阶段设计成果评审，提出优化意见并负责沟通落实；
3、负责与设计单位的工作联系，督促其出图进度和质量；
4、审核施工图纸的设计深度和质量，保证设计方案构思的实现；
5、负责施工过程中的技术配合、控制设计变更，配合现场施工、质量检验、竣工验收、设计回访及设计技术总结工作。

任职要求：
1、大学统招本科及以上学历，建筑学相关专业；
2、3年以上建筑设计工作经验，其中2年以上大型房地产公司工作经验；
3、具有较强的建筑设计管控能力，熟悉设计管理程序，熟悉设计规范、施工验收规范和验收标准，了解国家建筑相关信息，对设计方案有很好的把控能力。
4、工作责任心强，工作严谨，具备良好的团队合作精神和职业操守。

企业介绍：

中南集团始创于1988年，目前已发展成为拥有“房地产业”、“建筑产业”、“商业产业”、“工业产业”等产业板块的大型集团化企业。下辖江苏中南建设集团股份有限公司（股票代码：000961），中南房地产业有限公司、南通建筑工程总承包有限公司（特级资质）、金丰环球装饰工程（天津）有限公司、北京城建地铁地基市政工程有限公司等70家子、分公司。
   凭借着优秀的管理团队和杰出的经营理念，中南集团自2005年探索第一个2100亩大型城市综合体项目开发以来，始终致力于 “打造时代精品、铸就百年基业”的历史使命，努力实现2015年500亿、2020年1000亿、2027年1500亿产值的宏伟战略目标，追求卓越的步伐从未停止。从单一的建筑施工、住宅开发，到以二、三线城市综合体项目为主的“城市运营商”定位，从传统施工工艺向NPC全预制施工技术的推广，从第一代西班牙风格大众产品向低碳、节能、环保型第五代主打产品的探索，无不折射出中南集团锐意进取、与时俱进的发展意识，在激烈的市场竞争中建立了良好的品牌优势，积累了丰富的大型城市综合体项目开发经验，成为江苏省领先品牌企业。目前，房地产业集团共有30个建筑面积100万平米以上的开发项目，土地储备量40000亩，分别分布于海南省、江苏省、山东省、辽宁省、四川南充。建筑产业集团在海内外共承建100多个施工总承包项目，并且成功实现向EPC模式的全面转型。
   中南集团旗下建筑产业雄居江苏省建筑行业前列，共荣获“鲁班”奖14项，“詹天佑”奖2项，曾荣获“江苏省建筑业最佳企业”，“江苏省建筑业企业综合实力第一名”等称号。房地产业“中南世纪城”品牌是江苏省最具知名度的房地产品牌之一 ，2009年荣获江苏省房地产企业排行榜第八名，2011年荣获中国房地产企业500强第76位。中南集团2010年排名中国500强企业第355位，2011年荣获中国企业慈善排行榜十大慈善企业称号。
