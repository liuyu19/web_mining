职位描述：

岗位职责：
1.网络架构设计及规划：协助网络架构的整体设计，规划，相关设备与软件的选型，并负责实施；
2.信息安全架构设计及规划：协助制定信息安全策略、技术架构和管理制度的规划、设计，并督促实施；
3.日常维护： 负责制定相关制度、规范及其实施细则，并监督执行；
负责网络的日常维护工作；
负责评估和优化现有IT安全策略和制度，提出改进建议并指导执行；
定期对公司网络及基础设施、信息系统进行安全监测与评估；
定期进行安全弱点及漏洞分析，编制信息安全报告、制定改进措施及安全解决方案；
负责调研并导入信息安全技术产品或防范工具；
4.应急响应支持： 负责日常网络、安全事件管理，安全响应、识别、分析、整改，拟定安全事件报告；
负责建立完善的信息安全防范及灾难恢复机制；
5.维护报告编制： 负责记录并定期提交网络、信息安全运行报告；
配合外部信息安全审计，各类安全审计材料、专项申报的编制工作；"
6.重大IT项目支持：参与重点项目网络和信息安全的规划，并提出专业意见；
参与为项目交付团队提供IT应用系统安全设计的专家意见，对重大项目的建设，从IT运行风险控制角度提出评估意见；
7.其他相关任务: 为各项IT基础架构运维管理提供支持，保障公司IT基础设施的稳定性。

任职要求：
1. 具备扎实的计算机专业、信息安全理论知识，精通IT安全方面的知识，对常见安全问题的解决方法有深入的研究；
2. 具备大中型企业（金融优先）网络架构经验，精通主流网络技术；
3. 熟悉思科、华为网络规划、安装、调试及优化；
4. 熟练掌握信息安全技术及安全管理理念、方法；
5. 精通各安全管控系统，包括防火墙、IPS、漏洞扫描工具、终端符合性系统、防病毒系统等信息安全产品；
6. 熟悉各种攻防技术以及安全漏洞原理，有过独立分析漏洞的经验；
7. 具有较强的应变能力，及沟通协调能力；
8. 具有IBM、HP、Cisco等相关认证。

企业介绍：

远东宏信有限公司
远东宏信有限公司是一家以体量庞大、高速发展的中国经济为依托，专注于中国基础产业，并以金融及产业相结合的模式服务于产业中最具活力的企业的中国领先创新金融公司。秉承“金融+产业”的经营理念，远东宏信致力通过不断创新产品与服务为客户提供量身定制的产业综合运营服务，为实现“汇聚全球资源、助力中国产业”的企业愿景努力前行。
成立20多年来，远东宏信已由一家单一金融服务机构逐步发展成为立足中国、放眼全球、致力于推动国民经济及社会可持续发展的产业综合运营服务机构。远东宏信在医疗、教育、建设、电子信息、城市公用事业、包装、交通、工业装备等多个基础领域开展金融、投资、贸易、咨询和工程等一体化产业运营服务，创造性地将产业服务和金融资本融为一体，形成了具有自身特色的以资源组织能力和资源增值能力相互匹配、协调发展为特征的企业运作优势。
远东宏信总部设在香港，在上海和天津设业务运营中心，并于北京、沈阳、济南、郑州、武汉、成都、重庆、长沙、深圳、西安、哈尔滨、厦门、昆明、合肥等多个中心城市设立办事机构，形成了辐射全国的客户服务网络。在海内外，远东宏信设立了金融服务、产业投资、医院投资及运营、设备运营服务、高端教育、贸易经纪、管理咨询、工程服务等多个专业化的经营平台。
2011年3月30日，远东宏信正式在香港联交所主板挂牌上市。

远东国际租赁有限公司
远东国际租赁有限公司为远东宏信的全资子公司，是远东宏信在金融服务领域的核心企业，是中国领先的融资租赁服务商。
远东租赁提供以融资租赁为核心的综合金融解决方案，帮助客户解决发展过程中所遇到的财务及资金问题，并积极为客户提供财务管理、商务运作、资产管理、管理咨询等全方位增值服务，提升客户满意度。
作为中国融资租赁行业的先行者和领导者，远东租赁在医疗、建设、教育、工业装备、电子信息、城市公用、包装、交通、轻工等多个产业形成突出优势，并持续深化和拓展事业疆域，创新服务体系，成为推动中国融资租赁行业创新和发展的中坚力量之一。
