职位描述：

职责描述：
1、 主导平台设计与体验的交互设计方案（含可视原型），设计方案具备可客观度量的竞争力，包含完整的信息架构及任务流程设计。
2、 完成用户体验设计/前端实现工作，产品可用性成为产品竞争要素之一；
3、 进行用户体验研究/测试，输出的报告能够帮助产品准确了解体验现状，挖掘问题和定位原因，识别关键体验竞争力机会点，支撑产品体验竞争力目标达成；
4、 有效开展领域内用户体验规划，持续积累关键设计能力，推动体验设计方案的持续改进

任职要求：
1、视觉设计等相关专业，熟练掌握用户研究方法与人机交互设计原理，10年以上网站与软件开发相关行业的工作经验；
2、能够提供高质量的产品原型、流程图、线框图等清晰的表达设计方案；
3、熟练应用Sketch, Axure等原型工具及Adobe 系列视觉设计与前端开发相关软件
