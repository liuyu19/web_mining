职位描述：

工作职责：
1、负责优化搭建海外重度游戏发行体系，建立和完善海外发行渠道；评估海外游戏项目水准与潜力，共同参与游戏引进评估；与发行团队密切合作，共同拓展海外发行市场，形成自有竞争力。

2、热爱游戏，自身是资深的游戏玩家和行业关注者；拥有丰富的海外游戏行业资源，能够快速搭建在主要海外区域的发行体系框架，以服务实际发行业务；

岗位要求：
1、负责重度游戏产品在海外市场的发行策略, 制定运营战略、布局并组织和监督运营实施；
2、负责一款或多款产品，定制产品生命周期的运营计划；
3、负责游戏产品在海外市场的用户增长，敏锐把握市场热点，进行游戏品牌塑造、市场竞品状况监测等；
4、实时跟踪产品数据，多维度Review，通过数据分析、用户反馈及竞品的运营模式；
5、提出产品优化建议，持续推动提升游戏产品品质，用户体验。
