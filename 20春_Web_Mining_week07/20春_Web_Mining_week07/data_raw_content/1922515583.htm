职位描述：

职责描述：
1. 负责医疗APP和微信端整体视觉创意设计，配合产品或运营进行界面风格设计、页面设计、整体的视觉效果贯彻与实现；
2. 配合医疗APP和网站策划及开发人员进行相应的专题推广页面设计制作、图片处理等；
3. 分析医疗健康APP设计潮流，配合运营思想，制定相应页面的设计风格优化及视觉效果。




任职要求：
1. 美术类/设计类/计算机类本科以上学历，两年以上APP UI设计经验，精通Photoshop、Illustrator、Flash、AI等设计软件 ；
2. 对APP用户界面、交互设计、信息构架和用户体验有较深刻的认识，善于创新，扎实的美术功底，足够的视觉设计和把握能力。；
3. 有iPhone，android产品UI设计经验，对用户界面（UI）和用户体验（UE）的设计理论有全面的了解，能够独自完成高品质的界面和友好的体验设计。 ；
4. 做事认真细致，工作效率高，思维敏捷，良好的沟通能力和团队协作精神，能承受高强度的工作压力；
5. 有大型医疗APP和网站设计制作经历者优先（必须有成功上线的APP作品）；有平安好医生、阿里健康、微医、丁香园等医疗健康平台或类似平台设计经验优先。
6. 面试请携带最近两年内的UED作品。
