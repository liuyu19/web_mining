职位描述：

工作职责：
"职位描述：
1. 参与需求分析、设计评审，制定测试计划，设计和执行测试用例，按照产品架构和业务需求，评估需求合理性和复杂度，提高产品质量;
2. 负责项目的系统测试，构建测试环境，执行集成测试，回归测试等，并进行缺陷跟踪，把控质量和风险；
3. 独立开发自动化测试脚本，能完成稳定性、故障、性能、安全等方面的专项测试，并能根据业务特点，做针对性的测试；
4. 通过技术手段，提升团队内开发、测试的质量和效率。在流程、方法、工具等方面能够主动创新，不断提高研发和测试的效率；
5. 带领团队进行必要的测试类型拓展，能够推动项目测试流程等改进工作开展。




任职要求：
资格要求：
1. 计算机相关专业本科或以上学历；
2. 具有3年以上服务端产品开发或者测试经验，熟悉常用的服务端架构设计；
3. 精通测试方案和方法的设计和落地，熟悉业界主流测试模型、测试平台和工具，能够根据实际业务问题的要求选取、设计和编写测试工具和系统以及指定针对性的测试用例；
4. 熟练使用Linux操作系统，对linux常用命令，有一定的了解和使用经验；
5. 熟练掌握Python、Java、Shell编程，熟悉至少一种接口或性能测试框架，如pytest、rally等，能够完成大规模接口测试、性能测试或测试工具的开发；
6. 有人力资源相关业务系统测试、开发以及分布式系统测试、稳定性测试和持续集成(CI&CD)经验者优先。
7. 很强的逻辑能力，具备成熟的团队合作经验和冲突管理能力；"
