职位描述：

职责描述：
1、洞察电子制造行业变化与挑战、制造系统的技术发展趋势。结合通信类产品特点，组织规划产品精益制造系统的发展战略、场景化解决方案及DFX能力，包括可维护性、可扩展性、可获取性等能力。
2、组织分析内外部客户需求，开展重点产品精益制造系统架构设计，包括价值流设计、精益生产线设计、实物流设计、信息流设计等内容，支撑重点产品制造竞争力领先。
3、借鉴行业精益制造系统设计经验、方法和工具，完善华为产品制造系统架构体系，包括设计需求基线、设计规范、设计流程、度量标准等能力，提升产品制造精益设计平台能力。




任职要求：
1、具备敏锐的制造领域技术洞察力和行业洞察力，在一流的制造行业标杆企业至少8年以上工作经历。
2、对精益理论有较深的理解和熟练的运用能力，具有项目策划、项目管理推进和总结能力，具备良好的团队协作和沟通能力。
3、具备丰富的产品制造系统设计及DFMA设计经验，5个以上亲自负责或深度参与的项目实践。
