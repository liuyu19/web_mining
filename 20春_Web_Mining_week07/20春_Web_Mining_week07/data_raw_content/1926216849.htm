职位描述：

【岗位职责】
1、与产品、技术团队合作，明确业务需求，了解用户使用场景，规划并产出交互设计创意与方案，包括界面结构、用户操作流程、交互细节与动效等；
2、负责软件产品的整体美工创意、设计；
3、控制检查产品交互设计的实施效果，协作开发实现设计概念，全局把控产品视觉效果；
4、进行产品用户体验评估与测试，并能够根据评估结果持续优化产品，提升易用性，达到更好的用户体验；
5、整理并制定产品交互设计规范，推动规范落地。

【任职要求】
1、拥有工业设计、计算机、心理学、平面设计、广告设计等相关专业背景；
2、有优秀的交互原型设计能力，熟悉UCD方法和工作流程；有一定的设计理论知识和优秀的用户界面设计能力，熟练掌握可用性设计方法，基于可用性原则提出可行的设计改善方案；
3、熟练使用Axure、Photoshop等设计软件，有设计作品；
4、有良好的逻辑思维能力和沟通能力，有责任心和团队合作精神；
5、有创意、有激情，关注细节。

企业介绍：

        福建星网锐捷通讯股份有限公司成立于1996年，公司秉承“融合创新科技，构建智慧未来”的经营理念，是国内领先的ICT应用方案提供商。2016年公司品牌价值206.55亿元，并被世界品牌实验室授予中国500最具价值品牌。2010年6月，星网锐捷成功上市（股票代码：002396），成为一家以自主创新打造核心竞争力的公众公司。
        作为国家首批创新型企业，星网锐捷锻造了以企业研究院为核心、开放合作的自主创新体系。星网锐捷现有员工8000余人，其中研发技术人员占公司总人数的40%以上，每年持续将10%左右的销售收入投入研发，2016年研发投入达到7.89亿元。基于强大的创新能力，星网锐捷在网络通讯、交换机、云计算终端（瘦客户机）、支付终端、桌面云、无线接入、宽带接入、融合视讯、视频信息应用等各产品领域均形成了独特的领先优势，同时也致力于在智慧网络、智慧云、智慧社区、智慧通讯、智慧娱乐、智慧物联等应用领域为客户带来前瞻的应用解决方案，并成功进军欧洲、美洲、非洲、大洋洲等近百个国家和地区，向全球客户展示着“中国创造”的独特魅力。
