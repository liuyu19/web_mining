职位描述：

【高级后端研发工程师/后端研发架构师/业务线技术负责人】
职位描述
负责今日头条金融业务的后台业务研发；
负责在线大流量高并发系统设计；
主要实现语言为Golang/Java
职位要求
本科及以上学历，计算机、通信等相关专业；
有扎实的编程能力，有优秀的设计和代码品味；
深刻理解计算机原理，有良好的数据结构和算法基础；
熟悉MySQL、NoSQL、消息队列等常用WEB组件；
岗位本身不限制编程语言，但最好能熟悉Golang/C++/Java/Python等任意一种或多种编程语言；
有高并发服务设计和实现经验优先；有互联网金融相关业务开发经验优先；
积极乐观，责任心强，工作认真细致，有良好的团队沟通和协作能力；
热爱编程，有较强的学习能力，有强烈的求知欲、好奇心和进取心，能及时关注和学习业界最新技术

【支付研发团队- 研发工程师】
岗位职责：
1. 参与字节跳动核心支付业务系统开发和维护。
2. 参与公司级别的春节活动、双十一等活动，应对高并发、高可用场景需求。
3. 理解掌握常用架构设计、性能优化、高可用保障理念、并灵活运用于核心链路系统的优化。
4. 参与设计并实现具备可扩展性和可维护性的系统方案。
5. 充分理解支付业务，抽象领域模型，推进平台型系统建设。
6. 沉淀技术能力和行业经验，跟随平台一起成长和发展。
岗位要求：
1. 本科或本科以上学历，工作2年及以上。
2. 扎实计算机基础知识和强大的编程能力。
3. 熟悉常用的数据结构和算法，熟悉计算机网络基础知识。
4. 熟悉MySQL，kafka，redis等常用的基础组件的使用和实现原理。
5. 有服务治理，稳定性提升，容量规划等分布式系统的实际经验，熟悉分布式框架、消息、缓存等技术。
6. 对支付业务有兴趣，有志于在行业持续发展。
7. 能快速融入团队，善于协作，内心成熟，皮实，工作认真、严谨、敬业。
8. 具备金融行业、银行业、第三方支付等核心系统开发、设计工作者优先。
