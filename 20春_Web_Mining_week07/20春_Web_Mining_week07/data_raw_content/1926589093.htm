职位描述：

工作职责：
1. 负责公司我的世界项目组的开发者相关功能设计，设计产品交互界面结构、用户操作流程等；
2. 了解用户需求和使用情景，归纳产品界面需求；
3. 完成产品的信息架构、流程设计、用户界面设计；
4. 跟踪产品开发流程并推动落实，制定并输出相关设计规范；
5. 进行用户体验研究和分析，优化产品的易用性，持续推动用户体验优化和交互创新。


任职要求：
岗位要求：
1.本科及以上学历，优秀的设计表达能力，热爱做设计，交互设计/工业设计/产品设计相关专业；
2.两年以上产品交互设计经验，熟悉多平台（移动端、电脑端、web）交互特征；
3.学习能力强，观察力敏锐，能不断学习吸收最新流行设计趋势；
4.具有良好的需求把握和逻辑分析能力，善于沟通表达，具有设计大局观，能系统地有关联地设计功能；
5.熟练使用Photoshop，脑图等相关软件，掌握流程图、线框图等交互设计方法；
6.有一定的视觉功底优先，有成功上线项目优先；
7. 热爱游戏，喜欢“我的世界”或沙盒类游戏者优先；

**此岗位有外包编制和正式编制，编制类型根据面试综合评价而定**
