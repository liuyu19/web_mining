职位描述：

工作职责：
岗位职责：
1.负责高性能、分布式服务通信框架研发；
2.负责万亿级以上服务治理系统研发；
3.负责服务性能优化、SET(单元)化架构等核心系统研发。

岗位要求：
1.CS基础扎实，熟练使用常见数据结构和算法；
2.熟练掌握Java语言；
3.掌握Java生态(Spring/Mybatis/Linux)及常用工具体系(Git/Maven/IDEA)；
4.熟悉主流开源通讯框架相关项目(Grpc,Netty,Thrift，Dubbo)，研究过源码优先，能够较快上手；
5.掌握TCP网络通讯底层机制，熟悉网络问题诊断优化方法；
6.对分布式架构组件有实践经验，关注并理解架构的高可用性与高容错设计；
7.具有较高的技术钻研能力、技术难点攻关能力，分析问题解决问题的能力；
8.善于总结、思考、跨团队协作；
9.有良好的自驱力和较强的学习能力；
10.全日制统招本科及以上学历。

具备以下者优先：
1.在通讯框架技术方向有一定学习或经验积累。

岗位亮点：
1.有机会全身心地参与万亿级服务框架及系统的设计、迭代；
2. 能和团队内多位资深技术同学一起工作，共同学习进步。
