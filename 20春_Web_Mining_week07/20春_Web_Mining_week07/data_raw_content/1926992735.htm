职位描述：

职位描述
1、 负责赋能产品线落地，数据产品的调研、规划、设计。
2、 负责新产品线的整体调研、规划、设计，支持线上、线下、国外、海外、自营、三方平台等多渠道，跨场景的产品形态建立。
3、 有较强的市场调研，需求分析和总结能力。
2、对现有系统进行迭代升级以满足不断丰富的各类功能；沉淀核心业务逻辑，输出标准产品方案，不断迭代升级。
3、项目推进，参与产品全流程项目管理，把控项目进度，协调项目中的问题。
4、熟悉分发、内容、推送等互联网产品现状，对行业整体发展趋势具有敏锐的洞察力及创新意识；具备快速发现行业机会的能力。

任职资格
1、本科以上学历。
2、有良好的逻辑和结构性思维，优秀的用户需求挖掘、产品体验把握、产品设计规划，且有极强的语言文字表达能力。
3. 善于跨部门沟通协调，有大型项目实施能力，有较强的责任心和团队合作精神，抗压能力强。
4、有创业/新产品规划和推进经历优先。
