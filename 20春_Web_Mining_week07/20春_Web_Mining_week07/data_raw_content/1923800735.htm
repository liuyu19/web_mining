职位描述：

职位描述】
1. 基于大数据平台产品的Web应用的前端设计与开发；
2. 负责重点项目和产品的前端技术方案和架构的研发和维护工作；
3. 负责产品前端框架或组件的研发和维护工作；
4. Web前沿技术研究和新技术调研；
5. 维护和升级现有软件产品，快速定位并修复现有软件缺陷。
【任职资格】
1. 计算机软件、数学等相关专业211硕士毕业；
2. 5年以上web前端UI开发经验；
3. 精通前端各项技术例如JavaScript、CSS、HTML等，并至少熟悉一种后台语言，例如PHP, JAVA等，熟悉前后端协作迭代开发流程，可熟练使用SVN,GIT等版本管理工具；
4. 精通前端常见开发框架，例如Vue, React, Backbone等，乐于研究和贡献，有开源框架开发经验优先，最好在github上已有比较受欢迎的开源组件；
5. 需要有ToB业务或者大型前端工程开发经验，对前端技术架构安全、性能、稳定等方面有研究者优先；
6. 有良好的代码习惯，结构清晰，逻辑性强；
7. 具备优秀的沟通和协调能力，能承受比较大的工作压力，自我驱动力强；
8. 具备良好的团队合作精神，为人正直、勤奋、踏实，有责任感，并能注重细节；
