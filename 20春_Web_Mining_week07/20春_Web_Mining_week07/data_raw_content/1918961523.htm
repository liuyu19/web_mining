职位描述：

1. 负责宝宝树孕育APP的用户体验设计，包括新功能的设计以及风格迭代、设计规范制定等；
2. 结合竞品分析和用户研究与产品协作，从用户体验角度给出合理建议和解决方案；
3. 配合前后端工程师完成页面搭建，并跟踪产品呈现效果，保证上线质量；
4. 持续学习相关业务，有能力根据用户和数据反馈，不断提升用户体验；
5. 持续关注设计趋势，分享设计经验、与团队共同提升设计能力。


任职资格
1. 5年以上用户体验设计相关工作经验；
2. 对互联网领域的视觉设计、交互设计和用研方面的理论及实践拥有深刻见解与分析能力；
3. 具备出色的设计表现和创新能力，能迅速有效的将想法表现为设计方案；
4. 具备结果导向、快速执行、自我驱动意识；
5. 有独立思考和优秀的沟通能力；
6. 有团队意识，亲和力强热爱生活。

企业介绍：

   宝宝树集团成立于2007年，是专注年轻家庭的在线母婴类社区。集团总部设在北京，全球业务布局包括上海、杭州、武汉、厦门、广州、印度、北美等地。我们致力于连接及服务年轻家庭，并以“让年轻家庭享受美好的生命旅程”为使命。
目前宝宝树集团旗下包括旗舰平台“宝宝树孕育”APP，社交记录媒介和线上早教内容及工具在线平台“小时光”APP和母婴产品类电商平台“美囤妈妈”APP。经过12年的发展历程，宝宝树已在中国母婴在线平台中树立起备受信赖的品牌，并于2018年11月27日，于香港成功上市（股票代码：1761.HK）。
