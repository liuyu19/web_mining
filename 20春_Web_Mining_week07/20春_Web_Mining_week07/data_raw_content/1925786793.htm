职位描述：

职责描述：

1、参与深度学习领域异构编译工具链的设计和开发，包括AI编译器、实时库、Profiler等；
2、参与AI异构加速全栈系统的设计和开发，包括ASIC集成、资源池化、设备管理和分配、性能优化等。


任职要求：

1、计算机科学及相关专业，熟练掌握C/C++/Python等编程技能，有扎实的编程基础、良好的编程风格和工作习惯；
2、熟悉了解芯片架构/微架构（X86、GPU、ARM、FPGA、ASIC等)；
3、对机器学习/深度学习算法有一定基础；
4、有异构计算加速和性能调优相关的开发经验；
5、熟悉编译原理和算法实现，有GCC，LLVM和Open64等开源编译器相关开发经验尤佳；
6、熟悉MLIR/TVM/GLOW等深度学习编译框架，有开发和优化经验尤佳；
7、熟悉Caffe/TensorFlow/PyTorch/MXNet等深度学习系统，有框架内部Internal开发和优化经验尤佳；
8、具有独立解决问题的能力，良好的团队合作意识和沟通能力；
9、有责任心，抗压性强。
