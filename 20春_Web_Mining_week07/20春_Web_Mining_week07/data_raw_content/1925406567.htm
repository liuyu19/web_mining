职位描述：

工作职责:
1. 清晰掌握品牌的发展方向，结合赛事与商品制作相关图文编辑及创意设计。

2. 配合内容编辑或独立创意，设计编排自媒体图像。

3. 即时掌握社会、行业、网路动态，熟悉运动项目与商品。

4. 负责运营YYsports App平台，编辑内容美化等工作。

5. 据公司经营目标与上级交办事项，执行各项视觉设计产出与支援活动内容。

任职资格:
1. 大专以上学历，广告、美术相关专业。

2. 3-5年以上从事品牌、零售或媒体相关平面、广告设计相关工作经验。

3. 精通Photoshop、AI等图片处理软体。

4. 积极、主动的工作态度，并可独立完成创意作业与执行。

企业介绍：

宝胜国际（控股）有限公司是世界顶级体育用品零售批发商，2008年在香港上市（HK,3813)，是恒生指数企业之一。其年销售额超120亿人民币，拥有2.1万员工，6000家实体门店（其中3500家为直营店）。宝胜国际是Nike，Adidas大中华区最大的零售企业，同时经营的领先品牌还有：李宁，kappa，Reebok，Puma，Converse，Asics，Hush Puppies、Nautica、Wolverine及Umbro等。

宝胜国际是Nike，Adidas大中华区最大的零售企业，同时经营的领先品牌还有：李宁，Reebok，Puma，Converse等。除零售业务之外，宝胜亦经营特许经营品牌业务，目前作为O’Neill及 pony的独家品牌特许经营商，这些品牌允许宝胜在订立产品价格、供应链管理、产品设计、市场营销及发展，以及产品推销方面具一定程度之自主性。
