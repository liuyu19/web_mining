职位描述：

工作职责：
1. 参与大型分布式的搜索引擎系统的开发和优化； 2. 参与在线、离线的搜索服务和算法服务的开发、设计； 3. 基于Tensorflow开发大规模的在线算法服务工程。
任职要求：
1. 本科及以上学历，五年以上的Linux平台下的C++、java开发经验，精通TCP/IP等协议； 2. 精通Linux高性能网络服务器编程，熟悉通信、多线程、高并发处理、内存管理等技术； 3. 熟悉Linux内核，有分布式系统的开发和性能优化经验者优先； 4. 精通C/C++、掌握java、Go、python等开发语言，有使用开源项目的经验更佳，如flink、hbase、hadoop、storm等； 5. 熟悉常用的一些算法和tensorflow知识； 6. 工作场景中有基本的英语的听说读写能力。
