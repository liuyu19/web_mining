职位描述：

职责描述：
1、负责携程金融相关视觉设计、运营设计、情感化设计、包括但不限于UI设计、用户体验
2、参与产品讨论，从视觉设计和用户体验的角度提出建议与解决方案
3、把握流行趋势，以及产品视觉的拓展和延续；
4、规范产品在用户体验上的统一性和一致性，制定、维护和更新UI设计视觉风格规范；
5、及时跟进设计结果并能保证设计实现质量及视觉呈现效果，关注用户反馈与数据表现，具备一定的数据分析能力，并根据分析结果持续优化设计方案；
6、参与团队的专业建设，提升整体设计能力。
任职要求：
1、本科以上学历，4年以上UI设计和运营设计经验，有插画绘制及创意能力突出者优先；
2、具备较强审美能力、设计中注重整体感，画面控制力强，良好的把控色彩搭配；
3、随时关注行业热点、视觉设计趋势研究，并乐于分享，具备较强的沟通能力；
4、积极主动、执行能力强、有良好的沟通能力；
5、熟练使用PS、AI、Sketch等设计软件。
