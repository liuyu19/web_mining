职位描述：

职责描述：

1. 负责服务端系统的设计，实现，重构与调优    
2. 负责核心业务模块的架构设计和代码实现；保障服务端系统高性能、高可靠运行，高性能中间件的设计与研发  
3. 对接产品团队需求，高质量高效率的完成
4. 负责服务端团队的建设和管理，包括基础技术建设、团队梯度搭建、组织文化建设等 

任职要求：
1. 计算机相关专业本科以上学历；5年以上大型互联网公司工作经验，2年以上团队管理经验 ；有语音、智能硬件领域经验者优先
2. 深入掌握 PHP 语言，并对常见的 PHP 框架和扩展有深入的了解；深入 Java 或 Go 者优先；
3. 掌握 MySQL / Redis / ES 搜索 / Kafka / Nginx / Docker 等常用服务，并了解其基本原理，了解其性能优化方法；
4. 熟悉微服务、SOA 等一种或多种架构模式，应对不同复杂度的业务有自己的选型能力和思考方法；
5.良好的编码习惯和严格的编码规范, 能够编写开发，设计文档
6.有较强逻辑分析和组织协调能力，能够适应创业公司氛围，抗压能力强
