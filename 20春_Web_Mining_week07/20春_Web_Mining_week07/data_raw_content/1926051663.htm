职位描述：

工作职责：
负责富媒体（视频，图文，UGC类）类安全管理平台的建设和研发工作，确保相关系统的稳定可靠运行； 参与系统的需求分析、评审、运营及业务运维；负责运营的问题反馈响应，跟进和解决；要求积极主动，能抗住压力；
任职要求：
计算机或相关专业本科及以上学历，三年以上相关工作经验；
精通PHP/GO任意一种语言；
熟悉常用的数据结构与算法；
熟悉常见的设计模式，有一定的架构能力和良好代码规范；
熟悉网络编程，TCP/IP协议以及http协议；
熟悉常用的互联网技术，包括但不限于rpc、nosql、mysql等
熟悉linux开发环境，熟练掌握Go语言，基础扎实，有完整go项目经验优先；
具有高并发、高性能服务系统设计经验及能力，能独立承担任务和有系统进度把控能力，熟悉云、容器使用；
具有三年以上互联网业务开发经验或大容量网络服务相关经验者优先；
具有安全审核平台能力建设的，特别是视频，图文等能力相关系统建设者优先；
有较强的逻辑思维能力，思想上开放，主动积极有责任感，能够承担工作压力；
