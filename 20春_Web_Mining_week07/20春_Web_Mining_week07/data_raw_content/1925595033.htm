职位描述：
不管你是 UI 设计师，视觉设计师还是交互设计师，我们的产品设计师需要参与整个产品设计的各个环节，沟通、思考、分析，给出有效的解决方案，为最终的产品负责。

工作职责
1、负责企业级移动化产品的整体设计工作，包括 Web & Mobile App 产品
2、在产品设计前期与前后端工程师沟通，理解产品目标及意义
3、负责产品的 UI 及部分视觉设计，输出高质量的设计
4、负责产品的交互设计。发散思维，分析竞品或从其他产品设计中获取灵感，为产品注入更合适、有效的交互、UI 展示及落地的解决方案
5、负责日常的产品更新及维护，不断提升产品的用户体验

职位要求
1、视觉或其他设计相关专业，热爱互联网产品和设计，宽阔的设计视野
2、2年以上 Web/Mobile App UI 或产品设计经验，对交互有较深的认识和研究
3、优秀的设计思维和表现，合理把控细节和全局，较强的逻辑思维及分析能力
4、良好的沟通和表达能力，较强的团队协作能力和自驱力

加分项：
1、有完整、成熟的产品设计经验
2、熟悉 HTML、CSS 等 Web 专业知识，了解saas行业及产品功能
