职责描述：
1、参与公司WINDOWS平台C++ 方面软件的开发，调试，维护工作；
2、参与软件开发项目实施；
3、解决软件开发过程中的一般技术问题，支持、协助处理其他相关的技术问题；
4、独立完成软件配置及根据客户需求完成模块的编码；
5、参与公司新平台的研发和设计等工作。
任职要求：
1、精通C、C++开发语言，熟练使用VS2005、VS2008等集成开发环境，有实际项目二年以上MFC软件界面开发者工作经验；
2、熟悉MFC界面设计，理解MFC消息机制以及熟练使用MFC编写程序；
3、 精通Windows API编程；有图像处理相关知识或设计经验者优先；
4、有责任心，有良好的团队协作精神，钻研精神，良好的沟通能力和文字表达能力。
5、有手机摄像头行业者优先。
