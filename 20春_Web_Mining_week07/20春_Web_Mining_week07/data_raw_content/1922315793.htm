职位描述：

工作职责：
1、负责腾讯为村产品的前后端、活动、宣传等视觉设计工作；
2、参与产品需求分析，服务于功能与特性设计易用优雅的用户界面；
3、参与⽤户研究，关注用户测试结果和用户反馈，持续优化用户体验；
4、参与设计标准制定，保证设计输出高效规范。
任职要求：
1、有扎实的平面设计功底和设计理论知识，在用户界面设计有较为丰富的经验；
2、熟练掌握Sketch，Principle，Photoshop，Illustrator，Axure等设计工具，有动效设计相关经验者优先；
3、饱满的工作热情，⾼度责任感，积极主动沟通，与其他职能岗同事合力推动项⽬落地，能承受高强度的工作压力；
4、保持好奇⼼，保持学习和思辨的能力，乐于分享；
5、美术、设计相关专业本科以上学历；
6、具备快速手绘能力者优先，⾯试请携带相关作品。
