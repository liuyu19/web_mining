职位描述：

岗位职责：

1.服务全国各地的销售体系，做产品支撑，协调公司内外部的研发资源，做出产品版本计划。需要较强的沟通协调能力、项目管理能力、风险把控能力、具备良好的团队合作意识，较强的抗压能力；
2.深入洞察线上客户需求，到客户身边调研，指出产品方向，给客户最好的产品体验；
3.产品售前辅助，对接一线客户经理销售打单。

任职要求：
1.本科及以上学历，3年及以上电商，收银，支付产品设计相关工作经验，有开发经验优先；
2.具有场景化、数据化思维，有丰富的用户行为和用户体验数据分析经验；
3.具备优秀的逻辑思维能力和业务抽象能力，具备良好的业务建模、产品架构能力；
4. 熟悉收银，支付，电商营销、商品、交易、会员等一个或者多个业务域；
5.能主动收集评估多方需求，联动各方，协调资源，完成从产品设计到上线的全链路工作，推动项目快速开展；
6.有良好的行业敏感度和用户场景感，具备快速判断用户价值的能力，能精准理解用户需求并清晰描述传达。
