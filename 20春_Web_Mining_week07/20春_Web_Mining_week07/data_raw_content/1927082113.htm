职位描述：

岗位职责：
1. 主导公司自有操作系统产品及第三方操作系统产品与不同芯片服务器底层适配；
2. 为大规模项目交付可靠的、稳定的、高性能、低能耗的底层基础设施；
3. 协调内部与外部团队完成硬件产品功能、形态、需求的定义，输出系统设计规范，系统验证规范；
4. 与内部与外部团队共同完成硬件选型、详细设计文档输出、设计检视、调试、硬件单元测试与问题定位。
任职要求：
1. 具有国内云平台公司或渠道商的集成适配工作经验优先；
2. 对服务器系统设计中的，管理、散热、结构、电源、信号完整性有深入了理解，了解linux、BIOS、BMC以及虚拟化技术；
3. 对x86系统、存储、网络的硬件体系与设计有坚实的基础；
4. 有很好的系统级别的复杂问题问题定位，调试，测试经验；
5. 对于硬件设计验证的方法有深入理解，并有动手经验和能力；
6. 有领导ODM供应商完成设计并交付产品的经验更佳。
