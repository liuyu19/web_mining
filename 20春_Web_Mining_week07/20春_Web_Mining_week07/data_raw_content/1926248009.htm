职位描述：

职责描述：
1、负责AR特效产品的迭代，产出满足业务需求的特效功能，如AR、美妆、3D特效、卡通人物等（包括需求分析、市场和竞品分析、产品体验与功能设计等）；
2、协调各部门资源推动产品功能开发，跟进项目的进度，与内外部团队有效沟通，提高项目整体质量。
任职要求：
1，研究生及以上学历，计算机相关专业，5年以上互联网娱乐app相关产品经验，或特效SDK产品经验。
2、逻辑强，产品感优秀，能够深度分析和理解客户需求，不断迭代自己；思维活跃，脑洞大开（有创造力）；
3、负责任、学习能力强，需要很好的适应能力与抗压能力，能够融入快节奏的产品团队；
4、从事过对AR特效，美颜美妆，等相关产品经验者优先；有2B业务产品或项目经验优先。
