职位描述：

职责描述：
1. 流程设计：负责呼叫中心业务全流程的设计、优化、梳理、制定及审核。推进公司内部流程体系信息化建设；
2. 流程审核：负责呼叫中心业务流程实施状况的监督，协助各部门进行业务流程的实施落地，推动业务的标准化建设；
3. 流程优化：定期对流程效能进行测量、诊断及分析，并结合竞品及行业动态，定期组织端对端的流程优化工作以节约成本，提升客户满意度。

任职要求：
1. 大学本科以上学历；
2. 5年以上服务设计、流程管理、项目管理等相关工作经验；
3. 熟悉呼叫中心及服务管理常用工具和方法，必须获得COPC证书；获得6sigma黑带证书优先；
4. 有COPC流程搭建及ISO审核相关经验；
5. 具有极强的用户诉求感知能力和数据驱动思维；具有深入了解产品、业务和竞争环境的能力；具有流程信息化系统化的工作经验；
6. 英语能作为工作语言优先。

企业介绍：

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，办公室设于北京、上海、西安、香港、东京、洛杉矶、旧金山、鹿特丹、法兰克福等全球18个国家及地区。

大疆是一家致力于持续推动人类进步的科技公司，坚持做空间智能时代的开拓者，不断为世界创造超越想象的科技之美。2006年创立至今，大疆在全球的员工人数已超过14,000人，客户遍布全球二百余个国家和地区。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十三年间，大疆不断革新技术和产品，开启了全球“天地一体”影像新时代；大疆的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。
