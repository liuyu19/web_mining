职位描述：

职责描述：
1、负责美图系产品交互设计，并对产品最终的用户体验负责；
2、在有限的时间及项目资源内，给出合理的交互解决方案，并推动方案的落地；
3、进行交互设计规范的输出，并推动规范在整个项目团队执行；
4、能够主导用户研究，根据用户研究的结果对产品体验和设计方案进行优化；
5、进行数据分析和可用性测试，评估产品上线后的效果，并提出用户体验改善计划。

任职要求：
1、本科及以上学历，2-3年互联网产品交互设计经验，并有成功案例；
2、熟悉产品设计流程，能够独立完成交互原型、撰写交互规范文档和交互组件梳理定义；
3、掌握Axure、Sketch等常用产品原型、流程图、线框图工具的使用；
4、熟练运用交互设计各种方法，对交互设计理论有较深的了解和认识,有数据埋点和数据分析相关经验者优先；
5、工作积极主动，有持续的热情和责任心，对互联网行业有热情；
6、提交简历时需同时提供个人作品集（必要项）。

企业介绍：

美图公司成立于2008年10月，以人工智能为驱动，是中国领先的影像处理及社交平台。美图公司的使命是“用美图的产品，让世界更美好”，围绕着“美”创造了一系列软硬件产品如美图秀秀、美颜相机、美拍、美图拍照手机以及美图魔镜，改变了用户创造与分享美的方式。2016年12月15日，美图公司在香港联合交易所（“联交所”）主板挂牌上市，股票代码1357。
2018年08月，美图公司开启了社交化的战略转型。在2018年12月当月，图片及视频内容浏览量已达80亿，并有接近51%的月活跃用户已成为社交用户。截至2019年6月，美图秀秀月活跃用户数增长5.1%至1.23亿，创下历史新高；同时，美图秀秀社交用户的日均使用时长已经超过12分钟，平台上传的内容（即图片及视频）数量，较2018年12月大幅增长超过5.5倍。这些数据表明，美图秀秀用户的粘性一直在增加，用户在平台上的社交心智已经逐渐形成。
截至2019年06月，美图的影像及社区应用矩阵已在全球16.9亿台独立设备上激活（数据自2015年7月起取数）。如今，“美图”不仅是“修图”的代名词，更是一个代表着美丽、时尚与年轻的品牌。

美图产品的核心是一系列基于人工智能的影像技术，并拥有庞大用户数据及丰富用户反馈为支撑。
2010年，美图成立了美图影像实验室（MTlab）致力于研究计算机视觉及深度学习相关的前沿影像技术。如人脸技术（该项技术可精准检测及识别人脸和面部特征）、极智美颜技术（通过分析使用者年龄、性别等人脸属性并据此定制个性化美颜方案）、图像分割技术等。美图已开发一系列专有技术，截至目前，已注册超过100项专利，并持有超过120款软件产品的著作权。
2018-2019年，美图影像实验室（MTlab）先后参与了CVPR、ISIC、ECCV等计算机视觉国际顶级会议，并斩获了ISIC Challenge 2018皮肤癌病灶分割赛道、ECCV 2018图像增强技术比赛、CVPR-NTIRE 2019图像增强比赛的冠军，累计发表学术论文7篇。
未来，美图将在严格遵守用户隐私保护法则的基础上，通过强大的研发实力以及百亿级数据支撑，进一步增强美化算法、创建新功能及开拓使用美图人工智能的新业务模式。
