职位描述：

地址：深圳 / 南京
职责描述：
1、 负责华为云产品交互设计，协助产品经理，对产品功能、信息架构、操作流程进行梳理，输出低保真原型；
2、具备互联网产品经验且有一定的产品概念，能够在前期进行用户调研及产品规划，梳理设计需求；
3、对现有产品的可用性测试和评估提出改进方案，持续优化产品的用户体验；
4、负责参与设计体验、流程和规范的制定，负责分享设计经验、推动提高团队的设计能力；

任职要求：
1. 全日制大学本科及以上学历，其他学历结合业务经验和技能综合评估
2. 5年以上交互设计经验；参与过互联网类产品的交互设计，并经历过完整的产品设计周期；
3. 熟练掌握专业的设计和原型开发工具，如Axure、墨刀等；
4. 对交互设计有较深的理解和实践，可以独立完成整个交互设计过程（对流程图、线框图等到交互设计方法能熟练应用）；
5. 做过至少2年的toB产品设计，了解面向企业业务的体验设计特征和知识；
6. 熟练掌握用户研究流程，能够通过调研、访谈、数据分析等方式了解用户需求；

备注：知名互联网公司和云服务提供商，如BAT、京东、AWS、Azure等优先考虑。
