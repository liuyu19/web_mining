职位描述：

职责描述：
1、负责企业数据指标,企业客户和企业业务服务主题等数据产品设计；
2、针对不同渠道（PC端、APP）进行相关产品功能设计；
3、负责跨部门协调和沟通，推动产品实施项目施落地；
4、对产品数据进行监控、跟踪、分析，评估上线效果，并迭代持续优化；
5、将业务诉求和业务场景转化为业务需求文档，跟进业务需求。
任职要求：
1、本科或以上学历，5年及以上互联网产品经验，3年及以上支付产品工作经验；
2、熟悉第三方支付系统和银行系统信息流及资金流相关流程；
3、有知名互联网支付产品工作经验者，特别是账户、支付结算、对账等相关经验者优先；
4、具备深入理解和行业产品从业经验，能独立完成需求分析、产品设计、开发推进和产品迭代等完整产品环节；
5、自我驱动型，具有优秀的沟通协调能力、逻辑思维能力，较强的责任心及执行力，优秀的团队合作意识。

企业介绍：

    京东数字科技的前身是京东金融。2018年11月，京东金融品牌升级为京东数字科技，旗下包括京东金融、京东城市、京东农牧、京东钼媒、京东少东家五大子品牌。
    京东数字科技的经营宗旨是用数字科技连接金融和实体产业，助力产业提升互联网化、数字化、智能化水平，推动实体经济的发展，创造社会价值。
    京东数字科技以数据技术、AI、IoT三大时代前沿技术为核心，已经完成在数字金融、数字城市、数字农业、数字营销、数字校园等领域的全面布局，在客户群体上实现了个人端，企业端，政府端的三端合一。未来，京东数字科技将在深耕现有业务领域的同时，进入更多实体产业领域。
