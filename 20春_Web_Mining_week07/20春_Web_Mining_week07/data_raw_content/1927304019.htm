职位描述：

工作职责：
我们需要行业顶尖水准或者有极高潜力的体验设计师，负责产品体验设计、软硬件一体人机交互、系统级UI等体验设计工作，产出最具行业领先性的设计解决方案，定义新的领域标准。
任职要求：
1. 多年互联网、智能产品体验设计工作经验，有成功案例/代表作品，曾多次主导设计和制定大型项目的体验方案与推动落地，有系列化产品设计经验，能通过建立、优化规范和机制，保障落地品质； 2. 拥有良好的审美和优秀的视觉手段的表现能力，对设计流行趋势敏锐且具有前瞻性，并擅长基于品牌基因与产品特色产出富有创造性的设计方案； 3. 拥有较好的用户视角，对体验设计的商业价值及用户价值有自己的理解，同时对于体验设计有系统化的抽象和整合能力； 4. 综合能力强，拥有独自驾驭复杂产品UI 的架构力、表现力、控制力； 5. 有激情、创新欲、责任感，能够自我驱动和快速学习，能够推进自己的想法，具备良好的合作态度及团队精神； 6. 良好的思辨总结与沉淀能力，能引领团队的专业建设与发展。 一些加分项 1. 有 智能语音／ IoT／智能软硬件 / 跨端跨平台 相关产品视觉设计经验； 2. 有系统级产品设计能力或对设计系统有深入研究。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
