职位描述：

工作职责:
1. 面向B端、G端客户群体，基于产品战略定位，给出品牌化视觉体系的设计方案；
2. 独立完成数据可视化大屏、峰会PPT及产品单页等的设计；
3. 对接需求部门：了解客户行业管理项目需求，守时交付，为设计质量负责；
4. 对接研发部门：跟进研发充分合作，为最终产品呈现的视觉效果负责；
5. 在UED团队内：参与软件产品体验设计，如：色彩风格、数据图表可视化、icon、插画、动画等创意类视觉设计等；
6. 定期的客户行业知识学习，竞品分析，趋势调研，分享经验，推动提高团队的设计力；

任职资格:
职位要求：
1. 有丰富的大屏展示、3D、动效、多媒体设计经验；
2. C4D或Blender、AE 专家型使用者，Sketch/AI/PS熟练型使用者，能用PPT、World表达设计思想；
3. 有长期的绘画积累，美术专业院校毕业，有逻辑思维能力，设计有故事感；
4. 正能量，思维开阔，可以自发解决问题；
5. 提供相关作品集或链接；

企业介绍：

北京旷视科技有限公司成立于2011 年，是国内人工智能独角兽企业。以自研视觉感知算法引擎为核心，致力于持续打造在各商业领域的AIoT 操作系统，以及深度构建具备连接百亿物联网设备能力的生态系统。旷视持续聚焦算法与工程创新，通过底层统一的AIoT 操作系统（AIoT OS）建设，旷视正在为200 多个国家和地区的数十万开发者，及上千家行业客户提供智能物联服务及解决方案。
目前，旷视在北京、上海、南京、成都、西雅图设有五大独立研究院，员工中研发团队占比超过60%，拥有国内外在申及授权专利1000 余件，并代表行业领先技术提供方参与了19 项人工智能相关的国家及行业标准制定，在多项国际人工智能竞赛与世界技术测评中夺冠（其中包括在ECCV 2018 COCO+Mapillary 联合竞赛中，旷视独揽4 冠）。
经过8 年时间的高速发展，除北京作为全球总部，旷视已在上海、深圳、成都、南京、杭州、武汉、宁波、青岛、徐州、美国西雅图等城市建立分部，现有员工已超过2000 人。由旷视设计研发、交付实施的智能物联服务及解决方案，被广泛应用于城市管理、物流、零售、地产、教育、手机、互联网、金融等核心领域，合作伙伴包含阿里巴巴、蚂蚁金服、菜鸟网络、富士康、联想集团、华为、OPPO、vivo、小米、凯德集团、华润集团、秀水街、鲜生活等行业头部企业。
我们有智力超凡、码力超强的技术团队，经验丰富、专业可靠的商务团队，简单温暖、软妹云集的运营团队；我们提供无限量零食饮料、新鲜水果，休息区、羽毛球、篮球、足球、聚餐、outing等团建活动，弹性工作制，刷脸进出、不打卡。
如果你对人工智能、互联网、高科技充满兴趣，如果你追求极致、简单可靠，那么，欢迎加入我们！
