职位描述：

视觉设计岗工作职责要求：
1、负责产品前期的设计分析，协作设定产品的整体风格；
2. 负责公司软件产品界面设计和视觉输出；
3. 负责UI设计规范的创建，不断提升用户体验；
4. 为日常运营活动及功能维护提供视觉设计支持；

任职资格
1、有APP UI 设计方面的经验和设计案例，有自己成熟的设计方法论；
2、能够从设计语言的推导到完稿，整个设计的流程都比较熟稔，中间有自己的特长；
3、具备自驱力，自我驱动成长，善于总结，乐于分享；
4、手头功夫硬，熟练掌握常用的PS、AI、Sketch等工具；
5、本科以上学历、3年以上工作经验；
6、优秀者以上条件可放宽；
