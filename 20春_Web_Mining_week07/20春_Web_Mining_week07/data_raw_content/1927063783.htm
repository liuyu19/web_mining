职位描述：

职责描述：
1.        负责集团内部IT支持体系的完善，包括但不限于IT流程管理、IT内部服务质量管控、IT桌面运维、办公网络运维、内部办公系统运维、IT资产管理、供应商管理等相关工作；
2.        完善已有的IT运维支持与管理流程，定期识别并建立常用的服务模型与故障模型，确保其在团队内部的有效执行，出具分析报告，并能根据分析内容，实施、跟踪改进措施；
3.        负责内部IT体系相关的项目协调与管理工作；
4.        负责职场基础设施与办公设备的日常管理和维护；
5.        配合政府主管部门与职能机构，协调集团内部满足相关的合规要求；
6.        协助组织公司的各种大型会议与活动；
7.        负责IT支持与行政性支出项目的预决算。

任职要求：
1.        本科以上学历，计算机及相关专业；
2.        五年以上工作经验，不少于三年IT管理工作实践经验；
3.        熟悉ITIL、ISO20000、DEVOPS等服务管理标准，具有IT服务管理体系规划设计经验；
4.        具备较强的部门统筹管理能力、应急处理与协调能力，能推动整个部门工作的有效开展 ；
5.        具有出色的沟通技巧和语言表达能力，文档编写能力；
6.        具备ITIL Manager、ISO20000、DEVOPS、COBIT、PMP证书优先。

企业介绍：

沪江是专业的互联网学习平台，致力为用户提供便捷、优质的全方位网络学习产品和服务。目前，沪江旗下设有四大业务体系，包括学习资讯、学习工具、社群学习平台、优质课程平台，其中涵盖了语言、留学、升学、职场兴趣等丰富的内容。
近年来，沪江大力扶持互联网教育创业团队，积极打造在线教育生态圈，实现产业共赢。同时，与传统学校合作探索，缩小教育差距，推进教育公平，把优质的教育资源通过互联网传递到世界的每个角落。
沪江诞生于2001年，历经5年公益化运营后，自2006年开始公司化运营，现已成为拥有1500名全职员工、2000名兼职员工，拥有1.3亿用户的大型互联网教育企业，处于行业领军地位。随着移动互联网的不断兴起，沪江团队推出多款移动端产品，构建移动学习生态圈。短短两年时间汇聚了超过1亿移动学习者，更自由的学习方式，让知识触手可及。
沪江致力于与国内外的伙伴们一起为全世界亿万学习者提供最优质的学习资源。作为专业的互联网学习平台，沪江同500余家机构展开深度合作，合作老师数量超过2000人，累计公开课超过20000堂。
