职位描述：

职责描述：
作为网易云音乐设计团队成员，参与网易云音乐直播产品相关视觉推广设计工作，具体职责包含：
1.负责云音乐对外推广/市场/招募等设计支持；
2.配合云音乐运营活动进行推广视觉设计支持，参与前期风格讨论与探索、差异化分析设计，对视觉传播方向设计有深入了解或研究，并根据运营、品牌的需求做出合适的设计；
3.较强平面功底和创造力、画面把控能力，了解平面构成、文字排版、文字设计；
4.关注设计前沿趋势，对设计探索感兴趣 ，有系统化的设计思维方式；
5.能够独立负责（包括带领和指导高级设计师）并完成一个中大型项目；
6.较高自驱力、对产品的热情和投入度。

任职要求：
1.本科及以上学历，3年以上平面设计经验；
2.有扎实的平面设计理论知识和对流行趋势敏锐的洞察，对互联网设计趋势有灵敏触觉和领悟能力；
3.有丰富的移动端推广 、互联网品牌相关设计经验；
4.深究细节，重视细小的体验，熟悉互联网产品的设计理念和规范；
5.对视觉用户研究有一定经验和见解，对互联网产品可用性有深入认识； 6.良好的团队意识，富有工作激情，创造力和责任感。
