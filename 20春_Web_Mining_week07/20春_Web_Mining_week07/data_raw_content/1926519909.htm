职位描述：

岗位职责:
１、负责前端开发工作，页面及相关业务逻辑的实现；
２、负责前端用户交互，用户体验及浏览器兼容，性能优化工作；
３、参与前端系统分析、技术选型与架构设计工作；
４、遵守团队编码规范，编写高质量、结构清晰、易读、易维护的代码；
５、参与项目CodeReview，持续改进系统架构及代码质量；
任职资格:
１、计算机相关专业，本科及以上学历；
２、4年及以上前端编码经验，拥有前端架构能力，熟练掌握HTML、CSS、JavaScript、ES6等前端基本技术；
３、熟练使用Git版本管理工具进行多人协作开发；
４、至少熟悉2种以上开发框架，精通Vue，React，Angular其中１种，深入理解MVVM框架原理，熟悉前后台分离的开发模式，与后台开发配合高效完成接口联调工作；
５、熟悉Web前端组件化、模块化开发模式，独立完成项目脚手架搭建，具备独立分析问题、解决问题的能力；
６、具备优秀的团队沟通协作能力，有强烈的责任心和创新意识，思路清晰，业务逻辑理解与分析能力强；
７、对于web前端技术有强烈兴趣，追求技术卓越，具备较强的自我学习能力，能够不断学习新技术并运用到工作中；
８、具备微信或钉钉小程序开发经验者优先；

加分项: NodeJS、TypeScript、前端工程化；
可提供GitHub，技术博客;
