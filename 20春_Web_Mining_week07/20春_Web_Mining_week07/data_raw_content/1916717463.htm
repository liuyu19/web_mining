职位描述：

职责描述：
1. 负责京东大客户活动页面视觉设计；
2. 站内外促销banner、推广所需H5活动页面设计、宣传单页等的各类线上线下宣传物料等相关设计；
3. 品牌活动logo及衍生品设计；


任职要求：
1.本科及以上学历，3年及以上视觉设计经验；
2.具备良好的美感判断，对自我要求严格。关注当前设计流行趋势。有美术功底，能够灵活控制设计风格，富有创造力和激情；
3. 精通Photoshop，Illustrator等设计软件，熟悉活动后台搭建； 
4. 有手绘基础、掌据H5、CSS基本技能者优先；
5. 思路清晰、有较强沟通力、抗压力强，具备良好的团队协作能力；
6. 有电商行业设计背景，或有丰富产品、页面设计经验者优先；

京东于2004年正式涉足电商领域，2017年，京东集团市场交易额接近1.3万亿元*。2018年7月，京东第三次入榜《财富》全球500强，位列第181位，在全球仅次于亚马逊和Alphabet。
