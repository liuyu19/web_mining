职位描述：

岗位职责：
1.负责公司终端产品和移动产品的研发，编写高质量的代码
2.和项目经理配合，参与公司算法SDK的封装

岗位要求：
1.熟悉Cocoa Touch框架，熟练使用OC、Xcode，熟悉iphone开发等技术，包括UI、多线程、Socket网络通信
2.熟悉iOS界面开发规范及Appstore上架流程和规则
3.熟练掌握Autolayout布局技术能独立封装sdk
4.熟练掌握java编程语言，掌握计算机软件工程、数据结构、算法等专业知识，有SDK架构设计
5.熟悉View的绘制过程，事件分发动画绘制以及进程通信

企业介绍：

南京甄视智能科技有限公司，即小视科技（Minivision)，成立于2015年4月，是一家基于
Machine Learning技术，以“人脸”为入口，坚持图像应用和大数据双轨道并行，打造一体化
智能服务生态的高科技公司，目前公司主要涉及金融服务、智能安防、智慧商圈等多个业务领域。
公司拥有多年的技术储备和深入的市场经验，先后获得南京江宁高新管委会、五百强财团、上市
公司、中泽资本、浙商产融航母基金等十余家顶级机构战略投资，累积融资近亿美金。
目前，公司在沈阳、徐州、太原、印尼等多地成立分子公司，团队成员近四百名，拥有中国科学
院院士、俄罗斯工程院院士、中国国家“青年千人计划”学者，及来自微软、华为、中兴、
Marvell、新加坡国立大学研究所等多领域的人才。
