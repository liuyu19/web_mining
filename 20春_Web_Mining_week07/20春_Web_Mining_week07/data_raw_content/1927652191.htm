职位描述：

工作职责：
1、统筹网易有道官方商城平台的运营工作，制定平台运营目标、运营计划和总结并跟进，协调处理电商业务需要公司内部团队支持的工作事项，达成平台销售目标；
2、根据公司情况和特点，结合官方商城运营情况进行相关数据分析，提出产品和流程改善策略，针对系统功能需求进行整理和汇总，进行功能测试和评估，优化商城客户体验；
3、负责项目整体策划及落地执行，包括项目进度策划、工作分配、公众号配置、系统配置、商城平台功能完善、页面设计、直营门店信息汇总、流程制定、门店快递确认、会员制度制定、帐号权限设置、活动策划、推广策划等；
4、负责网站对外合作开发，积极寻求市场推广资源拓展，并对效果进行跟踪评估，提出改善措施；
5、负责会员体系和业务流程的建设和落地；
6、负责和其它协助部门有效的沟通，进行资源优化与整合。

任职要求：
1、本科或同等学历以上；
23年及以上社交电商经历优先；
3、有较强的数据分析能力和数据敏感度；
4、熟悉电子行业／电子产品优先；
5、敏锐的市场意识、应变能力、领导能力和独立开拓市场的能力，学习能力强；逻辑性强和良好的语言表达能力；
6、具有强烈的责任心、进取心，精力充沛，身体健康，乐观豁达，爱岗敬业，富有开拓精神；

企业介绍：

     网易公司由丁磊先生于 1997 年6月创立,凭借敏锐的市场洞察力和努力，网易公司从10多人的初创团队发展至今，在全世界已拥有上万多名员工，并于2000年6月成功在美国纳斯达克股票市场公开上市（NASDAQ：NTES）。随着公司规模的不断扩大，网易已经成为中国领先的互联网技术公司。网易拥有一批活跃的年轻团队，每位成员都能在充满活力的氛围中孜孜不倦地为理想共同奋斗。我们的团队不仅年轻，而且优秀。网易不断领跑中国互联网发展离不开优秀的团队。无论是领先的技术团队、敏锐的新闻团队、专业的销售团队还是高效的管理团队，都为网易的辉煌，中国互联网事业的发展倾注自己最大的力量。
