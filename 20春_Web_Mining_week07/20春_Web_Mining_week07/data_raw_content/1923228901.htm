职位描述：

工作职责：
1、制定行业安全业务规划和业务拓展战略；
2、为重点行业客户提供高端信息安全规划和咨询设计；
3、挖掘行业需求，设计行业安全新业务模式，探索和推动行业新型安全产品设计；
4、推进重点行业信息安全标准的制定和完善；
5、整合公司安全产品和服务，设计行业整体安全解决方案；
6、推进和指导公司安全产品和服务的行业性改造和完善；
任职要求：
1、具备10年以上信息安全咨询和安全解决方案工作经验；
2、具备综合安全解决方案能力，熟悉各类安全产品、国内外安全标准、安全服务；
3、行业安全咨询或安全管理经验不少于5年；
4、近两年内主导和完成过2000万以上行业综合安全项目；
5、具有CISSP/CISP/ISO27001/CISA等相关专业证书;
6、沟通，表达和协调能力强，PPT和文档能力扎实；
7、四大会计师事务所安全咨询，全球大型IT和互联网安全解决方案架构师，甲方安全管理工作背景优先。
