职位描述：

一、团队介绍：
阿里云-视频云团队是一个年业务增长速度超过300%的团队，拥有全球top的视频、图片、文件分发技术，为国内80%以上的视频直播和点播平台提供服务，支持千万级并发及具有上百PB级的存储能力，拥有多位国内知名开源软件作者与业内大咖，众多前沿科技领域的探索：人工智能、边缘计算、大数据等。
阿里云音视频通信团队，构建全球范围的音视频通信系统，安全可靠、端到端低延迟、高清音视频、简单易用， 应用于互动直播、在线教育、互动游戏、远程医疗、企业会议等多个场景，以建立下一代互联网RTC基础设施为目标，AliRTC让跨国沟通250(ms)。
你想引领视频云服务整体市场的发展吗？想致力于为客户提供优质的视频体验，希望以自己的多媒体知识服务上亿客户吗？请加入我们，一起造未来！
二、职责描述：
1、负责构建覆盖全球的稳定、实时、高效的媒体处理服务；
2、构建分布式实时媒体处理平台，构建实时多媒体网关，实现PSTN、VoIP等异构互联；
3、负责设计和实现服务器微服务架构，弹性伸缩和高效维护；
4、负责实时音视频在多业务场景技术落地，包括通话，会议，直播，连麦等。
三、任职要求：
1、至少7年工作经验，且最近5年内有过至少3年的音视频相关开发经验或者图像处理经验；
2、熟练C、C++、GOLANG等服务器开发语言，精通Linux平台音视频方面的特性及多线程开发，掌握大规模分布式媒体服务器的研发和架构经验；
3、至少精通2种媒体封装协议或者传输协议，如WebRTC、FLV、HLS、MP4、TS、PS、RTP、RTMP、RTSP、SIP、H.323、GB28181等；
4、有丰富的视频会议MCU、SFU服务器的研发和工程经验优先；
5、熟悉多媒体开源架构(如SRS、WebRTC、FFmpeg、OWT等)，有基于某开源框架进行开发者优先；
6、学习能力强，善于折腾，喜欢挑战，精益求精，丰富的多媒体行业经验，对视频处理方面具有前瞻性和自己的见解和规划。

企业介绍：

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
我们的愿景
我们旨在构建未来的商务生态系统。我们的愿景是让客户相会、工作和生活在阿里巴巴，并持续发展最少102年。
