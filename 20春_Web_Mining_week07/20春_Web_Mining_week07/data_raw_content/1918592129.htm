职位描述：

【工作职责】
负责边缘云的系统设计、开发和上线，以及边缘计算复杂场景的解决方案设计；
【任职要求】
1、拥有计算机或相关专业本科以上学历，熟悉C/C++/JAVA/Go/Python编程语言之一；
2、2年以上有云服务设计、开发经验，对微服务有较深入的理解，具有丰富的设计、开发和运维经验；
3、熟悉云计算、DevOps，IaaS、PaaS等相关领域，有k8s/mesos/OpenStack等系统开发经验优先；
4、熟悉边缘计算/雾计算/IoT物联网等概念，有工业互联网、智慧城市、智能家居等解决方案经验优先；
5、丰富的系统分析/设计经验，有独立的大型系统或模块设计经验者优先；
6、良好的沟通能力和责任感，能够承担工作压力，独立分析和解决问题。
我们的优势：
1、边缘计算风口：主航道，业务刚刚起步，快速发展期，行业快速成长，用户多多，机会多多
2、技能快速提升：在这里可以接触到最前沿的技术，玩转k8s，docker，微服务，AI边云协同（AI往边缘发展是当前重要趋势）等，技术能力能够得到快速提升，技术研究成果快速落地产品，产生价值，成就感满满，只要愿意接受挑战，一定大有可为。
3、拥抱开源： 积极拥抱社区，回馈社区，崇尚开源文化，从我们团队开源的项目kubeedge（https://github.com/kubeedge/）已经并被CNCF基金会接纳为正式项目，成为开源界边缘计算领域重量级作品，加入我们团队将有机会贡献开源社区，提升个人业界影响力。
4、金牌团队：边缘计算团队荣获2018年华为云金牌团队，团队氛围融洽，积极，和谐，团队大牛众多，每个人都能发挥所长，与优秀的人一起共事。
