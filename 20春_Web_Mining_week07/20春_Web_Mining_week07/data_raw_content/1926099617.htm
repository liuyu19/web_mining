职位描述：

职位描述：
1. 负责小米国际浏览器客户端的产品规划和需求制定，完善客户端功能、优化UI/UX，提升用户体验，推进项目进度，确保按计划高质量完成产品迭代；
2. 收集分析新老用户反馈，调研对应国际市场用户需求，完成针对特定国际市场的浏览器+内容分发类产品的竞品调研和需求分析；
3. 根据功能需求，整理埋点数据，进行AB测试，输出数据分析报告，确保通过数据验证来优化产品；
4. 帮助优化小米国际浏览器的商业化变现，挖掘商业化新场景，优化商业化功能的客户端逻辑，提升商业化效率，确保项目组可以完成营收目标；
5. 具备预估/测算商业化回报的逻辑和能力，对接信息流产品、大数据推荐算法团队、业务组商务及运营，了解内外部合作方需求，并根据需求进行设计产品方案，并进行优先级排期，保证项目及时按预期交付。
职位要求：
1. 985/211院校本科及以上学历，良好的英语听写能力，英语可作为工作语言；
2. 3年以上产品岗工作经验（有浏览器/信息流产品岗经验优先），良好的逻辑思维能力，优秀的数据分析能力，能做到数据驱动；
3. 优秀的产品文档撰写能力，有产品大局意识，能够输出业务打法和中长期业务规划；
4. 沟通能力强，优秀的情绪控制能力，能够对接团结内外部合作团队；
5. 以结果为导向，执行力强，目标和责任感强，强大的自我驱动力，不惧困难做到使命必达。
