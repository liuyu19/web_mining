职位描述：

【岗位职责】
1.主要支持公司iOS、Android、小程序等移动产品及PC端的界面设计视觉；
2.把握产品视觉风格，传递符合产品定位的设计理念；
3.参与制定和维护产品视觉设计规范；
4.参与前瞻性、创意性的产品概念设计。

【职位要求】
1.本科以上学历，3年以上视觉设计经验，人机交互、工业设计、视觉传达等相关设计学专业；
2.了解产品设计流行趋势，熟悉Android与iOS平台设计规范；
3. 能独立完成符合趋势与规范的设计需求，具备较强版式设计、图标设计，界面设计能力；
4. 具备优秀的沟通与表达能力，开放的学习心态，耐心细致、乐观勤奋。

企业介绍：

     我们叫大搜车，是一家有伟大梦想的公司，成立于2012年，正在为车商的日常经营提供数据分析、营销管理、金融及交易服务。我们作为国内先进的汽车新零售平台，从诞生初期就期待用互联网+智能化思维重新定义这个行业，通过链接、赋能产业链上下游，打造汽车全流通生态，推进汽车产业互联网发展。

    创始人姚军红是原神州租车的创始团队成员、前执行副总裁，有过丰富的创业经验和带领团队IPO的经验，公司的英雄也主要来自一线互联网公司，都是一群充满想象力，有激情的牛人，公司成立之初便获得了国际一线VC的青睐，截至2018年9月，大搜车先后获得阿里巴巴集团、蚂蚁金服、晨兴资本、华平投资、春华资本等机构超过12亿美元融资。2017年12月，大搜车进入由硅谷全球数据研究机构PitchBook评选的“2017年全球新晋独角兽”名单。

     大搜车还很年轻，大搜车还可以变得更大。
IPO不是梦，上市不是终点，我们以推动汽车产业数字文明为使命，以建成年交易额超过2万亿的汽车产业互联网平台为愿景。这个行业未来可期，等待你一起深耕，我们有优秀的领导，优秀的团队，充裕的资金，可以给你实现梦想的机会，
