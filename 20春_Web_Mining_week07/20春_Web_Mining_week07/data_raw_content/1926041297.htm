职位描述：

1）负责产品PC端、WEB端、后台等产品的UI设计；
2）准确且快速理解产品需求和交互原型，设计出符合逻辑且易于后期迭代优化的多端界面；
3）能积极与开发沟通，推进界面及交互设计的最终实现；
4）根据阶段性用户反馈及产品反馈，分析结果不断优化升级产品UI，提高产品易用性；
5）主导设定产品的整体视觉风格、负责参与设计体验、流程的制定和规范、与产品和交互设计师密切配合，提升产品整体用户体验
6）引领团队视觉设计思维意识的创新，实现产品视觉呈现在用户体验上的突破；
岗位要求：
1. 三年及以上的UI/UE设计经验，熟悉产品的设计流程及规范, 具备证券行情软件设计经验者优先；
3. 具有优秀的审美和丰富的视觉表现力，追求极致与完美;
4. 精通Sketch、Photoshop、Illustartor等设计工具；
5. 熟悉PC、Web、iOS、Android平台设计规范，具备产品全局观，对PC端产品有较深入理解；
6. 具备较强的逻辑思维能力，可以快速理解和分解设计需求, 能够独立胜任产品设计；
7. 有强烈的Owner意识，积极主动、团队合作，沟通协调能力强，并具有高度责任感；

企业介绍：

银科投资控股有限公司（简称“银科控股”）是中国领先的个人投资与交易服务商，致力于通过金融科技与移动互联网向客户提供优质的金融信息、工具和服务。
目前，银科控股的业务范围主要涵盖黄金TD经纪、证券投顾投教、期货经纪、海外证券期货经纪、海内外资产管理等。
面向未来，银科控股正不断拓展新的业务领域，旨在向客户提供更加多元化的产品和服务。
银科控股成立于2011年，并于2016年4月27日在美国纳斯达克交易所成功上市，股票代码为“YIN”。
银科控股总部位于上海，并在北京、广州、深圳和香港设有办公室。
