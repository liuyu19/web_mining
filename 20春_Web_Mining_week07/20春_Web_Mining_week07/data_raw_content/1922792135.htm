职位描述：

岗位职责：
1、根据产品业务需求，定制测试策略和计划；
2、对测试需求进行分析，指导测试工程师，参与测试用例设计并组织审查；
3、管理和控制整个测试过程，并解决过程中出现的问题和风险；
4、分析总结产品的测试结果，推动问题的解决，提高最终的交付质量；
5、制定测试流程、规范，并推动团队的流程规范的实施，并根据实际情况不断对流程规范进行改进和优化；
6、持续改进提高测试团队的测试覆盖度和测试工作效率；
7、测试团队的培养和建设，帮助团队在业务知识和专业技能方面持续成长。

任职要求：
1、计算机相关专业本科以上学历，5年以上软件测试经验，2年以上测试技术管理经验；
2、具备测试领域系统化的知识结构，熟悉完整测试环节，能独立完成功能测试、接口测试、UI自动化测试集成测试；
3、熟悉主流自动化测试框架，如Selenium/Loadrunner/Jemter/Testng/Jenkins等；
4、熟练linux等操作系统及常用命令，熟悉Mysql/oracle等主流数据库及sql语法；
5、熟悉Python/Java/C++中的一种开发语言；
6、优秀的软性素质、具有较强的学习能力和分析、解决问题能力，具备良好的团队合作精神，结果导向型思维。
