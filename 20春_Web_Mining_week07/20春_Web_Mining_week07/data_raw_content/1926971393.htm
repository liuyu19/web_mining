职位描述：

岗位职责：
1、负责新媒体运营，以微信平台为主；
2、负责新媒体热点、公众号的内容策划；
3、熟悉用户增长路径，完成账号粉丝增量；
4、根据品牌与产品策略，制造有影响力的新媒体传播事件。
任职要求：
1、 至少从事过 3 年以上新媒体工作，热爱新鲜事物，对热点具有敏感度；
2、文案功底扎实，能独立策划选题、并且进行文案创作；
3、极强的用户洞察能力、文字创作能力或美术设计能力，
4、具备一定PS、视频剪辑能力者优先考虑
