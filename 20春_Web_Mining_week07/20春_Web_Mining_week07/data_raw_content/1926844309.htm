职位描述：

工作职责：
1. 负责RDS（Oracle或MySQL或DB2或Mongo DB或高斯DB）服务的构建，负责架构高可用性建设和持续优化；
2. 对公司内部数据库的可靠性负责，包括系统架构优化、性能优化及可用性提升；
3. 负责公司内部数据库服务产品的架构设计及规划。
任职要求：
业务技能要求：
1. 5年及以上Oracle或MySQL或DB2或Mongo DB DBA从业经验；
2. 有大型数据库管理经验优先，如OLTP超过100TB 或者 OLAP超过300TB；
3. 有Python,Shell等开发经验。
专业知识要求：
1. 深入理解Oracle或MySQL或DB2或Mongo DB的体系架构，对数据库优化，架构设计有较深入的研究和实践经验。
2. 熟练掌握数据库调优工具，并且精通分库分表、高并发等场景的管理及故障诊断。
3. 具备高并发压力下的数据库管理经验，参与过重大数据库升级迁移的实施项目。
4. 熟悉Linux操作系统，熟悉进程间通信、内存管理、网络通信机制。
