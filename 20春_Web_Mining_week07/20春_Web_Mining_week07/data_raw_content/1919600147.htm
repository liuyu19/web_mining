职位描述：

职责描述：
1.与相关部门配合，进行调查研究，明确培训目的；
2.制定与实施专项培训计划；
3.对公司员工进行业务.产品等方面培训；
4.组织公司员工培训并执行培训计划和培训管理规范；
5.开发培训课题，编制培训教材，编写培训教案；
6.完成早会计划及早会培训工作；
7.向培训学员.员工.管理层提供有效培训信息及协助；
8.设计培训评估体系并做好培训评估备案；
9.完成直接上级交办的其他工作。

任职要求：
1.大专及以上学历，男女不限；
2.能力要求：熟悉员工培训制度建设及操作流程，按需求制定培训计划，具备开发设计培训课程和编写教材，熟练运用PPT等办公软件，具有良好的沟通协调能力和组织能力，能动性强，重沟通，有较强的责任心；
3.经验要求：三年以上培训工作经历或保险从业3年以上。
