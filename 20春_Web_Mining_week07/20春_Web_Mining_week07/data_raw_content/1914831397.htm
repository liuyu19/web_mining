职位描述：

岗位职责:
1. 负责企业信息化的基础服务体系的搭建，能够支撑所有内部系统的公共依赖需求，降低内部系统的开发成本和提升开发效率。
2. 负责推动各职能部门梳理企业内部流程，解决企业内部的效率问题。
3. 负责搭建团队，完成企业内部的办公系统、效率工具的设计，提升各职能部门的效率，提升员工协同的效率。
4. 带领团队对企业的基础设施建设、办公系统和企业内部效率负责。
任职资格:
1. 有大型互联网企业内部信息化系统建设经验，对企业信息化的基础服务建设和信息化系统的架构设计有成功经验，且有自己的方法论输出。
2. 有很强的发现问题、解决复杂问题的能力，有很强的推动力和执行力。
3. 有大局观、有担当、有很强的规划能力，可以站在集团快速发展的高度来发现问题、并推动各部门协作来解决问题。
4. 有很强的组建团队能力和管理能力，能够在短时间内组建一支有战斗力的产品团队，推动企业信息化建设快速发展。
5. 有很强的沟通协调能力，可以和公司内部后职能部门（IT、行政、人事等部门）很好的协作，支撑这些部门的需求，和他们一起为企业提效、赋能。

企业介绍：

瓜子二手车，成立于2015年9月，致力于用创新重塑二手车产业，为用户创造更大的价值。瓜子二手为用户提供二手车检测定价、居间服务、汽车金融、售后保障、维修保养等一站式服务。截至目前，瓜子二手车业务覆盖全国200多个主流城市。

基于大数据与人工智能能力，在国内率先实现了二手车非标品的标准定价——瓜子价，该定价将逐步成为中国消费者买卖二手车的重要依据。在强大技术基因的助力下，瓜子二手车正在推动二手车车况与车价的透明化，对于行业的高速发展起到了革命性的作用。

2017年3月，瓜子二手车推出新零售业务——瓜子二手车严选直卖店，构建线上线下高度融合的一站式新消费场景。营业面积高达数万平米的超大型瓜子二手车严选直卖店为卖家提供实车寄售，买家一次多看、一站购车的综合服务，在技术能力的助力下持续推动服务效率与用户体验的优化升级。瓜子二手车严选直卖店将成为用户线下二手车消费的城市地标，将实现二手车服务超越国际水平。
