职位描述：

岗位职责：
1、负责线上客服中心（热线服务、在线服务）生产运营策略，服务生产流程、技能设计，客户接入感知保障等，对客服经营成本负责
2、负责线上客服中心（热线服务、在线服务）生产现场管理，包括制定资源拟合、需求预测、排班管理、监控调度、现场安全等流程规范
3、负责客服中心（热线服务、在线服务）生产运营分析，不断改善对客户服务流程的设计、服务流程的合理性，优化服务成本及效率
4、负责客服中心（热线服务、在线服务）团队管理工作，包括团队生产管理、核心指标提升、文化建设等工作
岗位要求
1、本科或以上学历，专业不限
2、有2年及以上汽车行业或客服中心相关工作经验优先，有150人以上团队管理经验优先
3、风险敏感度高、逻辑分析能力强、沟通能力强
4、有较强的数据分析能力

企业介绍：

小鹏汽车2014年中成立于广州，是一家以科技驱动的高科技制造公司。
我们以前瞻科技驱动业务发展，以品质为先实现产品制造，以用户体验推行产品和服务，打造完善和便捷的用车和出行体验。
在这里，汇聚了来自不同国家3000多人的人才，其中60%为研发人员。
核心团队融合了互联网与汽车等行业的专家，保证了小鹏汽车的跨界思维和开放包容。
