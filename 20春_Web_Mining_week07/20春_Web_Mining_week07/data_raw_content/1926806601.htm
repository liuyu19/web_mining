职位描述：

职责描述：
1. 在UE4引擎中还原真实世界场景；
2. 确定材质编写流程；
3. 提供场景光照渲染方面解决方案；
4. 维护游戏美术内容开发流程及规范；
5. 编写美术相关工具提升美术工作效率。



任职要求：
1. 三年以上一线美术工作经验，本科及以上学历；
2. 精通UE4引擎材质，蓝图，动画，特效等模块；
3. 精通材质Shader编写，精通PBR材质制作流程，对Substance系列软件，Quixel系列软件有较深积累；
4. 精通3DMax/ Rhino/Maya/ZB/3Dcoat等相关软件制作写实风格低面模型，对拆UV，烘焙法线有较深刻的认识；
5. 精通场景搭建，地形，植被系统，对Worldmachine，Speedtree有较深积累，精通场景光照，营造真实世界气氛；
6. 精通PS，AE等2D美术相关制作软件，熟练掌握MaxScript等相关语言优化美术工作流程；
7. 在美术性能优化方面有一定积累，在场景仿真的情况下保持帧率稳定，了解三维重建工作流程优先。

企业介绍：

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，办公室设于北京、上海、西安、香港、东京、洛杉矶、旧金山、鹿特丹、法兰克福等全球18个国家及地区。

大疆是一家致力于持续推动人类进步的科技公司，坚持做空间智能时代的开拓者，不断为世界创造超越想象的科技之美。2006年创立至今，大疆在全球的员工人数已超过14,000人，客户遍布全球二百余个国家和地区。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十三年间，大疆不断革新技术和产品，开启了全球“天地一体”影像新时代；大疆的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。
