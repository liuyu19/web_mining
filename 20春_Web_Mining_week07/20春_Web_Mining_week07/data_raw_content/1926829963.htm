职位描述：

职责描述： 1.制定品牌的全年规划和产品线规划，为品牌全年销售额及总体市场预算负责；
2.带队制订、统筹和执行产品的营销策略，包括新品概念、推广策略、促销策略、其他产品的长年运营和销售分析等，为新品销售额及相应市场预算负责；
3.负责线上线下渠道营销活动定制方案，专供新品的开发策略和上市推广策略；
4.负责制定品牌打造策略，包括传播定位，视觉设计及VI规范等；
5.参与制定内容运营年度规划，包括图文、直播、短视频等的主要策略。
6.负责开拓公司为线下引流蓄客的工具与产品，从营销获客、转化成交、服务消费者到口碑传播，为线下门店提供整体完善的解决方案。

任职要求：
1、本科以上学历，有家居家具品牌管理经验优先，有整合营销能力，线下营销经验优先；
2、3年同行业相关工作岗位经验，有2年以上独立负责品牌建设和新媒体管理工作经验；
3、精通品牌管理、市场策划、产品推广、消费者促销、广告和创意管理等知识，有较强的品牌战略理念和创新意识，具备良好的沟通组织能力；
5、性格外向，热爱市场营销，对品牌有浓厚兴趣，有良好的职业道德和置业操守，抗压能力强，有良好的团队合作意识，能独立带领团队工作。
