职位描述：

岗位职责:
1.作为某一业务领域合作伙伴，负责和各业务部门沟通，洞悉用户需求，进行业务分析，充分结合现有信息系统或结合第三方系统，分析问题并提出创新解决方案；
2. 负责某业务领域信息化产品设计、建设和运营，提升信息化水平，以提升业务部门的业务能力与运营效率；
3、与业务 owner 一起推动相关信息化产品的有效运营，确保信息项目的真正落地，并持续优化迭代更新产品，提升产品用户满意度，切实达到提升效率和价值的目的；
4、信息化项目管理，以推进信息化项目建设的成功开发与实施；
5、定期对此业务领域的业务动态、最佳实践、信息化解决方案进行调研学习，保持对前沿技术进行嗅探，并分析其与业务结合点，在恰当的时机推动应用和转化。
任职资格:
1、具有3年或以上互联网产品的规划及产品设计经验、或从事企业信息化产品规划、项目实施；熟悉企业信息化管理产品，有信息化线上产品设计经验者优先（如OA、CRM、财务管理系统、需求管理）；
2、有独立的产品设计以及创新能力；熟练掌握原型Axure、Xmind等工具；
3、熟悉某一领域业务（财务、供应链、人事、产品开发IPD\CMMI等）；
4、项目管理经验，成功实施过信息化项目。
