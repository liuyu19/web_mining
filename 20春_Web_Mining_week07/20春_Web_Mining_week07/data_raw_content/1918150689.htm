职位描述：

岗位职责：

1. 负责公司政府业务相关产品的需求调研分析，产品设计，交互设计，编写产品文档，制定产品实施方案，推动产品上线交付等工作
2. 负责或配合公司业务部门，参与政府相关部门发起的招投标工作
3. 与客户或公司业务部门同事沟通，挖掘客户需求（包括隐形需求），帮助/引导客户将需求描述清晰、完整，输出内/外部文档
4. 配合公司整体规划，协同多产品间的并行推进，业务与技术间的沟通
5. 持续深入一线理解业务，深刻了解用户需求，竞品情况，不断加深对行业和业务的理解


任职要求：
1. 3年以上产品经理工作经验，独立负责过至少1个toG产品，熟悉产品经理常用的工具，包括但不限于：Axure、脑图，了解用户体验，熟悉产品开发流程
2. 具备良好的文案撰写能力，能输出高质量的对内/对外产品文档（word、PPT等）
3. 了解政府招投标流程，参与过政府招投标工作者优先
4. 出色的沟通及表达能力，具有良好的团队意识，工作责任感及使命感
5. 专业不限，本科以上统招学历
