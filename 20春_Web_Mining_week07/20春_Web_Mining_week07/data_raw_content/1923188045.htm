职位描述：

【岗位要求】
1、根据产品需求，对产品业务需求进行分析、转化、决策，完成界面的信息架构优化、流程优化等，提高产品的易用性；
2、分析产品需求、优化产品易用性、改善用户体验并能够跟踪产品实现进展；
3、在产品推进执行中能良好的传达交互设计思想和意图；
4、撰写交互设计文档，不断完善和规范化；
5、除产品外，支持定制化项目需求的原型交互设计。
【任职资格】
1、本科以上学历，5年以上企业级应用产品交互设计工作经验；
2、具备用户研究、需求分析、原型设计、可用性测试等能力；
3、逻辑能力强，善于思考，善于分析产品需求和模拟用户使用场景；
4、具备视觉设计技能优先；
5、熟练使用 Axure 、PS、AE 等软件推进交互设计实施； 熟练运用交互设计各种方法来表达设计思路并传递信息；
6、主动性强，具有优秀的理解、沟通、表达、协调能力。
