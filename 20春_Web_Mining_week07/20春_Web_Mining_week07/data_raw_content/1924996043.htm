职位描述：

职责描述：
1. 制订体验规则，不断优化规范并完善；
2. 通过对于体验相关运营数据的分析，聚焦业务异常体验的问题点，并制订具体改善方案；
3. 从用户体验的角度出发，进行客服服务流程规划，协调各业务部门配合执行，并对执行结果负责；
4. 定期通过异常体验case分析等专业方法对产品&流程的设计进行多维度分析，发现共性问题，制定有效策略和计划，协同业务方持续改进，保障整体服务质量；
5. 定期对用户服务过程中的关键时刻进行分析以提高用户忠诚度，提出可行性改善方案，并推动落地执行；
6. 定期通过调研等方与客服内部员工沟通，挖掘系统，流程，产品方面的建议，并协助推动优化。
任职要求：
1、3年以上工作经验，本科及以上学历，具备一线互联网公司用户运营经验优先；
2、项目推动力强，善于发现用户问题和需求，出色的人性洞察力，对用户体验有深入理解；
3、具有较强的逻辑思维能力，良好的系统思考能力，解决新业务和新项目遇到的各项挑战；
4、优秀的沟通表达能力和团队协作能力，能够有力推动项目进程，跨部门共同达成目标。

企业介绍：

时空电动汽车股份有限公司（时空电动）是国内领先的清洁能源出行集团公司，公司以电动汽车定制、城市高频出行和移动电网服务为核心应用场景，业务涵盖电芯制造、电池制造、整车工业、能量网络、市场运营五大板块，从产业最上游到终端消费者完整布局，贯穿纯电动汽车全产业链。公司掌握纯电动汽车动力电池、电机、电控三大核心技术，拥有近百项专利，其中时空制造的电池已经成为业内公认的纯电动汽车动力，时空设计研发的纯电动汽车“蓝色大道”已经有过万辆投入实际运营。时空电动同时成为了东风汽车在新能源领域最初、最核心的合作伙伴，也是滴滴出行全国两大运力合作伙伴之一。时空电动以纯电动汽车为主要应用场景的清洁能源流转体系，努力成为中国新能源汽车产业生态圈的构建者。目前正在开发的“蓝道出行”APP，公司主打的顺应“互联网+绿色出行”趋势，通过互联网大数据与城市智慧交通出行领域技术，结合丰富的能源出行行业服务经验，致力于为用户提供更清洁、便利、前卫的出行方式。2019年，时空电动获创投协公布的估值达10亿美金以上的杭州独角兽企业。时空电动总部设在杭州，目前拥有1000+名员工，时空电动已经初步有美国上市计划。
