职位描述：

工作职责：
1. 深入理解服务管理领域的模式和流程，发现潜在的服务运营风险点，和产品技术、业务运营高效沟通，设计合理的服务安全运营策略架构 2. 与BI团队紧密配合，通过大数据挖掘，找到行为违规类的特点，快速形成有效的打击策略，持续迭代优化某个业务或场景的服务安全运营效果 3. 针对某个业务或场景建立合理的指标体系，过程中不断完善监控体系，对服务链路中关键控制环节的数据进行分析提炼，与BI配合形成可视化的监控系统快速发现业务风险并形成风险预警指标
任职要求：
1. 本科及以上学历，统计学等应用数学等相关专业尤佳 2. 对业务和数据敏感，能快速发现问题和解决问题 3. 利用经验、统计分析或机器学习等手段，从风险案例或风险数据中理解和挖掘出风险手法和风险特征，并归纳出有效的风险变量 4. 思维开阔，有良好的逆向思维、逻辑思维和结构化思维。有自驱力，能主动思考和学习，勇于接受挑战和承压

阿里巴巴集团的使命是让天下没有难做的生意。
我们旨在赋能企业改变营销、销售和经营的方式。我们为商家、品牌及其他企业提供基本的互联网基础设施以及营销平台，让其可借助互联网的力量与用户和客户互动。我们的业务包括核心电商、云计算、数字媒体和娱乐以及创新项目和其他业务。我们并通过子公司菜鸟网络及所投资的关联公司口碑，参与物流和本地服务行业，同时与蚂蚁金融服务集团有战略合作，该金融服务集团主要通过中国领先的第三方网上支付平台支付宝运营。
