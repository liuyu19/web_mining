职位描述：

职责描述：
1. 根据零售团队的需求独立创意设计方案；
2. 根据设计方案独立制作或指导供应商制作SI；
3. 执行SI培训；
4. 执行设计打样验收、培训、指导；
5. 协助海外国家SI适配落地；
6. 搜集竞品优劣势分析，输出报告；
7. 零售管理部其他协助事项。
8. 地区有中东非、拉美、亚太、东北欧等。



任职要求：
1. 大学本科以上，英语四级以上，视觉表达、美术学、平面设计等相关专业；
2. 有服务于国内外知名品牌的从业经验；在SI设计、陈列规范与设计领域有5年以上工作经验，能够独立出设计；
3. 有过大型项目的设计、创意等实战经历或成功案例，具备从策略制定到具体执行的能力
4. 熟练掌握材质、工艺、基础预算；
5. 熟练掌握 3D MAX、Auto CAD、Phtoshop、Powerpoint；
