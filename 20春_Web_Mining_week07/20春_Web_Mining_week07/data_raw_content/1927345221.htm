工作职责：
1.负责参与SaaS&APaaS服务端系统架构设计与开发。 2.持续优化系统架构，提高系统在高并发、大流量下的容灾容错能力，保证系统的高可用性。 3.对现存或未来的系统能进行宏观的思考，规划形成统一的框架、平台、体系或组件。
任职要求：
1.扎实的JAVA基础，熟悉常用的数据结构、类装载机制、多线程并发、IO、网络等基础知识。 2.熟悉常用面向对象设计原则，熟练掌握设计模式及应用场景。 3.对分布式场景常用算法及中间件原理了然于胸。 4.掌握 Linux 操作系统和大型数据库，有线上运维经验。 5.具备良好的算法、设计模式、操作系统、数据库功底。 6.喜欢学习最新的技术，追求编写优雅的代码，从技术趋势和思路上能影响技术团队 备注： 如果你在github上有很棒的开源项目或是stackoverflow、quora上有精彩的回答、喜欢分享知识、有自己的技术博客都请直接联系我们，希望这里能成为你梦想起航的起点。
