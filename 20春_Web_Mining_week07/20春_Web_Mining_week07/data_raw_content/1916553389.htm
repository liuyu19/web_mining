职位描述：

岗位职责：
1、负责京东到家公司级别大促活动。
2、负责banner、活动页面、促销页面、设计。
3、负责协助制定相关设计规范、模板或分享经验推动产品视觉力量。

任职要求：
1、3-5年工作经验，能够精准把握产品调性、品牌调性，深刻了解电商视觉营销防弹的特性，具有良好的产品感；
2、能够了解当下运营设计的潮流趋势，熟悉受众喜好；
3、精通PS/AI/AE/C4D，有动效制作等相关经验；
4、有完整的相关设计作品，美术，设计相关专业本科以上学历（优秀人才，学历可以适当放宽）3年以上运营、平面设计工作经验。
5、具备良好的合作态度和团队经验，较强的沟通能力，富有工作激情。有责任感，承受一定强度的工作压力。
