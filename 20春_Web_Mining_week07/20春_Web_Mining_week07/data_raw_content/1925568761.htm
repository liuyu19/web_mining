职位描述：

岗位职责：
1.立体视觉测量方案设计与优化；
2.立体视觉工程问题的分析与定位，包括量产相关问题的技术支持；
3.视觉测量系统标定与数据处理算法的设计与实现，算法工具的开发与维护；
4.视觉测量相关最新技术跟踪与算法预研。
职位要求：
1.较强的C++代码能力，熟练掌握面向对象与模块化的开发方式，丰富的Windows/Linux开发经验(必备)；
2.计算机视觉、摄影测量、光学测量等相关专业，良好的线性代数、概率论、统计学基础，硕士及以上学历；
3.熟悉立体视觉相关优化理论与常用开发库；
4.熟悉视觉测量系统，有较强的工程能力与问题分析能力；
5.有TOF项目经验者优先。

企业介绍：

奥比中光成立于2013年1月，是我国人工智能3D传感领域独角兽。公司总部位于深圳，在美国西雅图、上海、广州、西安设有分部。奥比中光是继苹果、微软、英特尔之后，全球第四家，也是全亚洲第一家能量产消费级3D传感器的厂商。目前，奥比中光已推出多款3D传感摄像头，产品已应用于智能手机、智能机器人、智慧客厅、新零售、VR/AR、工业测量等多领域。2018年5月，奥比中光完成由蚂蚁金服领投的超两亿美金D轮融资，估值已超10亿美元。
