职位描述：

岗位描述：
1、负责集团企业信息化业务服务端的整体架构规划，并推动落地；
2、负责对现存或未来系统进行宏观的思考，规划形成统一的框架、平台或组件；
3、负责对业务逻辑进行合理抽象，高效地满足架构、业务需求；
4、负责设计、开发支持全球化多IDC的基础架构服务；
5、参与技术方案和系统设计评审，把握复杂系统的设计，掌握核心代码逻辑，确保系统的架构质量；
6、负责在后端大规模分布式系统等领域开展前沿技术研究和落地；
岗位要求：
1、计算机、应用数学等相关专业硕士以上学历；
2、10年以上服务端开发经验，5年以上互联网服务端架构设计经验；
3、精通JAVA技术，至少熟悉GoLang、NodeJS、C/C++、Python其中一种语言；
4、具备海量数据高并发和大规模分布式系统的设计和开发经验；
5、具备良好的基本功，熟练使用各种数据结构和算法，在互联网后端技术领域具有较完整的知识体系，并能结合到实际工作中；
6、具备大规模系统的故障诊断与性能优化能力；
7、具备优秀的逻辑分析能力，能够对业务逻辑进行合理的抽象和拆分；
8、具有很强的沟通协调能力，善于推动跨部门复杂项目的实施；
9、在产品、技术、工程等方面有深入思考和中长期的产品和技术视野，对技术架构演进有清晰、成熟的思路；
10、负责过多条业务线或整个产品线的业务架构工作，组织过中等以上规模项目者优先；
11、主导或参与过大型的开源项目者优先；
12、有技术文章输出，有社区影响力者优先；
13、有企业应用大数据行业经验优先；
