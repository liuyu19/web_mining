职位描述：

业务技能要求：
1、深入理解并精通C/C++/Java/Python/Go等一种或多种语言的开发应用，具备大型软件产品设计和开发经验
2、负责软件架构优化以及整改，微服务流程逻辑的高层设计和开发、服务级测试，提升产品的竞争力；
3、参与虚拟化、云化、运营商领域软件系统核心模块的技术竞争力构建，架构设计，跟踪分析业界发展趋势，完成技术竞争力识别，标杆分析和指标设定。
4、5年及以上项目开发经验，独立承担过模块开发、架构设计工作。
5、对技术有激情，喜欢钻研，能快速接受和掌握新技术，有较强的独立、主动的学习能力。
责职要求：
1、承担核心模块的高层设计、代码设计、代码编写、测试架构设计、静态检查等工作；
2、参与软件系统核心模块的技术竞争力构建，架构设计，跟踪分析业界发展趋势，完成技术竞争力识别，标杆分析和指标设定。
