职位描述：

工作职责：
1. 为政务钉钉的应用场景提供极具创新的设计方案； 2. 参与建立和维护平台产品的设计标准和规范； 3. 坚持以用户为中心的设计原则，参与建立平台产品的用户体验指标和数据分析，并藉此不断跟进持续优化产品服务； 4. 利用自身设计经验和优势，帮助和推动团队提升整体设计能力和协作能力；
任职要求：
1. 五年以上设计工作经验，对用户体验设计的方法和流程有深入的理解，形成较完整的设计思路；有独立设计交付能力且有过政务设计的经历； 2. 对企业级服务和应用领域感兴趣，能够通过深刻理解业务，形成完整的设计方法论和创新的解决方案； 3. 具备不同终端的熟练设计能力；有视觉设计的经验； 4. 能够独立承担项目的整体设计策略，深刻理解产品和用户痛点，制定体验指标，高效地获得设计成果； 5. 具备严谨的思辨能力，能够针对复杂业务逻辑下的交互界面提出优秀的设计方案； 6. 有热情，具备积极沟通和主动服务的精神
