职位描述：

职责描述：
1. 协助直营体系门店设计平面类延展物料，包括不限于KV延展，POSM，围档画面，开业筹备物料设计等；
2. 辅助开发新产品别创新物料2D设计，包括不限于创新道具，铭牌，通用类平面物料等；
3. 协助SI完成门店分层分级分场景陈列指引输出。

任职要求：
1. 熟悉使用adobe illustrator、photosho等平面设计相关软件；
2. 有高效执行力和理解力，接到项目能快速反映并输出最优解决方案；
3. 有较强主观能动性，主动发掘业务痛点并解决，不断优化及提升部门专业度；
4. 有4A广告公司经验优先。
